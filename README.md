日本商业航天公司小型火箭发射第三度失败
日本商业航天公司“太空一号”再度尝试发射小型火箭“凯洛斯”三号，不过火箭在上升过程中飞行中断。这是继2024年两发两败后，这一系列火箭第三次发射失败。
继星期三（3月4日）发射临时中止后，“太空一号”公司（Space One）于当地时间星期四（5日）11时10分（新加坡时间10时10分）从位于和歌山县（Wakayama Prefecture）的纪伊太空发射场发射“凯洛斯”三号。直播画面显示，火箭升空后很快出现飞行姿态不稳，飞行轨迹异常。“太空一号”公司判断火箭难以完成送卫星入轨的任务，遂决定采取飞行中断措施。
新华社报道，“凯洛斯”三号搭载五颗超小型卫星。其发射最初定于今年2月进行，但因为天气等原因多次推迟。星期三上午，火箭发射前安全系统被触发，导致发射紧急停止。
“凯洛斯”（Kairos）火箭采用四级串联构型，全长约18米，直径约1.35米，重约23吨。其近地轨道运载能力为250千克，太阳同步轨道运载能力为150千克。“太空一号”公司原先希望把这款火箭打造成全球范围内从签约到发射所需时间最短的航天运载工具。
“凯洛斯”火箭曾于2024年3月和12月两次尝试运载卫星上天，均以失败告终。其中，“凯洛斯”一号升空数秒后在空中爆炸解体，二号在发射后不久便飞行中断。

美伊冲突后中国航司首次重启中东航线
中东局势因2月底爆发美伊冲突升级以来，中国首次有航司恢复飞往中东地区的航线。
综合第一财经、微信公号“民航观察”消息，中国国际航空星期四（3月5日）重启飞往沙特阿拉伯的航线。
据报道，国航CA789次航班已于星期四下午1时30分许从北京首都机场起飞，预计当地时间傍晚7时40分许到达沙特首都利雅得机场。
这是中东局势出现突变，中国至中东航线停摆后，中国航司飞往中东的首个航班。
从飞行航路来看，这条航线预计经云南出中国空域后，经过孟加拉国、印度、阿曼等国空域，不会飞越主要冲突交战空域。
美国和以色列2月28日空袭伊朗、伊朗随后展开报复性袭击，战火蔓延至波斯湾国家和更广泛的中东地区。目前包括伊朗、以色列和重要国际中转枢纽卡塔尔在内的数个中东国家，空域仍全面关闭。

日官房长官：中国正迅速增强军力
中国今年军费预算增长7%，日本官房长官木原稔说，中国正迅速增强军力。
据共同社报道，木原稔在星期四（3月5日）的记者会上，回答中国政府工作报告相关问题。
对于中国军费预算增长7%，他说“中国在缺乏充分透明度的情况下，正广泛且迅速地增强军力”。
对于中国今年GDP增速目标为4.5%至5%，木原稔说，将继续关注中国的经济动向，这可能对日本乃至全球经济产生重大影响。
中国全国人大会议星期四（3月5日）在北京召开，中国总理李强做政府工作报告时说，今年发展主要预期目标是经济增长4.5%至5%。
中国今年军费预算为1万9095.61亿元人民币（3528.94亿新元），同比增长7%。过去三年，军费增幅一直维持在7.2%。自2022年以来，中国军费预算增幅连续五年超过7%。
