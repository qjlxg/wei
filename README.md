中国新能源发电在建项目规模继续领跑全球
尽管电力面临浪费的风险与日俱增，中国公用事业规模的风电和太阳能发电厂在建项目规模仍继续领跑全球。
据彭博社报道，追踪能源基础设施的机构“全球能源监测”（Global Energy Monitor，简称GEM）星期二（7月21日）发布的一份报告显示，截至今年6月，中国约有262吉瓦（GW）的太阳能和251吉瓦的风电项目正在建设中。
GEM高级东亚策略师于爱群表示，这两个数字均高于2025年底的水平。这表明，尽管官方公布的新增装机数据近期出现大幅放缓，中国可再生能源设施的建设仍保持高速节奏。
这波建设热潮主要源于2025年6月起实施的新电价政策。该政策促使企业加快建设进度，以在最后期限前完工并网，导致装机量的大量前置。于爱群指出，中国整体的建设活动保持相对稳定。
“如果你看看在建项目，这股势头从未真正减弱。”她分析称，“我们绝不应低估中国以如此规模建设风能和太阳能产能的意愿。”
庞大的在建项目预示着今年下半年的新增装机量可能会出现反弹。尽管截至5月底，中国仅新增了60吉瓦的太阳能面板装机，但彭博新能源财经预测，到今年年底，中国的新增装机总量最终将达到255吉瓦。
当前，可再生能源面临的更大风险是“限电”（即弃风弃光），也就是电网为避免过剩电力压垮网络，而迫使风电和太阳能发电厂停机。
GEM指出，全中国范围内的限电率都在上升，而在建项目高度集中的西部地区尤为严重。
为了解决这一问题，中国一方面建设长距离输电线路将电力向东输送（西电东送）；另一方面则将高耗能产业向西部转移，以就近利用当地充足的电力供应。
于爱群认为，这种策略虽然合理，但如果不辅以优先保障可再生能源消纳的机制，反而可能进一步巩固煤炭的使用。数据显示，中国现有的长距离输电线路所输送的煤电量，大约已是可再生能源电力的两倍。
与此同时，旨在消化过剩清洁电力的绿氢设施建设项目，也面临着沦为能源转型“形象工程”（window-dressing）的风险。
她警告，这可能会让企业借此轻易满足监管指标，却未能对煤化工等重污染行业产生实质性的减排作用。

预计明年供过于求 中国锂价跌至五月低点
路透社引述海通期货分析师称，尽管短期需求依然强劲，但由于市场预期制造电动汽车电池所需的原料碳酸锂将在2027年出现供过于求，中国锂期货价格在星期二（7月21日）跌至五个月低点。
据报道，广州期货交易所交投最活跃的碳酸锂合约星期二曾一度下挫至每吨13万6800元人民币（2万6072新元），创2月10日以来最低水平。在早盘结束回弹至每吨14万1280元人民币，跌幅收窄至4.45%。
此外，中国财政部等三部门上星期五（17日）发布公告称，自今年9月1日起，将对用于电动车和储能系统的锂离子电池征收2%消费税，自2027年9月1日起提高至4%。
高盛认为，此举可能在消费税实施前拉动碳酸锂的需求。

美国出台原铝关税优惠 弥补产能不足
美国总统特朗普星期一签署公告，依据《1962年贸易扩展法》第232条款等为在美国投资原铝生产项目的企业提供原铝进口关税优惠。
综合法新社和新华社报道，此举旨在弥补美国原铝（Primary Aluminum）产能不足。
美国对进口铝征收的关税通常为50%，但根据这项新计划，相关企业可享有减半的关税税率。
美国白宫星期一（7月20日）发布公告说，美国商务部将建立机制，鼓励在美投资新建、扩建或翻新铝冶炼厂。符合条件的项目需要承诺在2029年1月20日前开工。
美国商务部将允许项目所在企业每年进口与新项目预期年产量相同数量的原铝，并按照《1962年贸易扩展法》第232条款适用的关税减半缴纳进口关税。
2025年6月，美国将进口自除英国外所有贸易伙伴的钢铝产品关税从25%上调至50%。今年4月和6月，美国先后出台政策，弱化钢铝关税力度。
彭博社报道，美国政府原本希望通过50%的钢铝关税提高美国国内产能，然而这一关税措施成效有限。美国原铝产能不足，严重依赖从加拿大和中东地区进口来满足国内需求。
