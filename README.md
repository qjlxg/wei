根据《菲律宾星报》报道，约有至少200万菲律宾公民在中东地区居住或工作。
菲律宾外交部发言人埃斯卡洛纳说，在伊朗战争爆发背景下，联合国是小马可斯倡导和平、保障全体平民安全的“最合适平台”。
埃斯卡洛纳说：“总统将代表本地区及发展中国家发声，呼吁各方保持克制，尊重包括《联合国宪章》与人道主义法在内的国际法，并敦促所有相关方重返谈判桌，实现和平。”
她也说，小马可斯预计将与联合国秘书长古特雷斯会面，讨论中东局势，并重申马尼拉对多边主义的承诺。

联合国难民署：中东危机构成“重大人道主义紧急事件”
联合国难民署称，中东地区不断升级的危机已构成一场“重大人道主义紧急事件”，需要在整个地区立即采取应对措施。
联合国难民署分管紧急情况和项目支援的负责人伊藤礼树星期五（3月6日）在瑞士日内瓦的一场例行记者会上说，近期中东地区局势升级，袭击持续发生，引发了伊朗、伊朗周边国家以及更广泛地区的人口流动。受影响地区难民已经达到2500万人。美国与以色列对伊朗发动打击后的最初两天，伊朗官方数字显示已有约10万人离开家园，但这一数字很可能是低估数字。
新华社引述伊藤说，联合国难民署工作人员正努力向受影响国家提供人道主义援助。他敦促有关方面为需要转移或跨境的民众提供安全通道。

联合国人权高专：世界迫切需要遏制中东战火蔓延
联合国人权事务高级专员沃尔克·图尔克说，世界迫切需要遏制当前中东战火蔓延的态势，制止杀戮进一步升级。他呼吁各国领导人明确承诺捍卫国际秩序。
图尔克（Volker Türk）星期五（3月6日）在日内瓦举行的记者会上说，一周前美国和以色列袭击伊朗以及伊朗的后续反击，给伊朗、以色列以及至少十几个其他国家造成了重大损失，并有可能对全球经济和环境造成重大影响。他敦促当事方立即采取措施缓和局势，“给和平一个机会”，并希望其他国家明确立场，要求当事方及时收手。

下午察：专家学者为何频频翻车？
美国和以色列上周联手对伊朗发动空袭，并在首日行动中“斩首”伊朗最高领袖哈梅内伊。这一突如其来的军事行动不仅震惊国际社会，也“打脸”了此前断言美国不敢对伊朗动武的部分专家和学者。

法国航母“戴高乐”号抵达地中海 西班牙护卫舰已入编队
据沙特阿拉伯阿拉比亚电视台报道，法国“戴高乐”号（Charles De Gaulle）航空母舰抵达地中海。
新华社也引述RTL电台星期五（3月6日）的报道说，“戴高乐”号航空母舰于星期五下午越过直布罗陀海峡（Strait of Gibraltar）进入地中海（Mediterranean），但尚未抵达最终目的地。
法国总统马克龙星期二（3日）证实，“戴高乐”号航空母舰已派往地中海，以应对该地区日益恶化的局势。此舰此前一直在瑞典进行军事演习。
此前，西班牙国防部星期四（5日）发布公告宣布，将派遣一艘海军护卫舰前往塞浦路斯，提供防御性军事支持。这艘名为“克里斯托瓦尔·哥伦布”号（Cristobal Colon）的护卫舰星期二加入法国“戴高乐”号航母编队，执行护航和训练等任务。这个编队将前往地中海向塞浦路斯提供防御性军事支持。

军舰遭袭 伊朗外长与印度和斯里兰卡外长通话
伊朗外长阿拉格齐与印度和斯里兰卡外长通电话，强烈谴责美国在没有任何警告的情况下在国际水域袭击伊朗护卫舰“德纳”号。
新华社引述伊朗外交部星期五（3月6日）发表的声明说，阿拉格齐分别与印度外长苏杰生、斯里兰卡外长维吉塔·赫拉特通电话。
阿拉格齐在通话中讲述了过去几天以色列、美国的“罪行”，强调各国政府及联合国有责任对美以的行为予以明确谴责。他强烈谴责美国在没有任何警告的情况下，在距离伊朗海岸2000英里（约3200公里）的国际水域袭击“德纳”号（IRIS Dena）护卫舰的行为，并强调伊朗将通过国际机构追责。
阿拉格齐同时感谢斯里兰卡为救援“德纳”号护卫舰船员所作的努力。
星期三（4日），伊朗海军“德纳”号护卫舰在斯里兰卡以南海域被美军潜艇用鱼雷击沉，斯海军在附近水域打捞出87具遗体。有报道说，“德纳”号刚完成印度组织的国际海军演习，遭袭时正在返航途中。
伊朗称，美国在印度洋击沉的伊朗护卫舰“无武器”且“无弹药”，刚刚参加完由印度主办的国际海军演习。
伊朗副外长赛义德·哈提卜扎德（Saeed Khatibzadeh）星期五在新德里说：“那艘船是应我们印度朋友的邀请去参加国际演习的。它是礼仪性质的。它没有武器。它没有弹药……许多参加演习的伊朗年轻水兵丧生。下手之人不能放过。”
