马尼拉讯 — 菲律宾食品药品管理局（FDA）周二表示，愈来愈多在网上销售的假冒和未注册产品，尤其是药品，令监管机关深感忧虑。
FDA总干事保罗·泰斯顿（Paolo Teston）在菲律宾欧洲商会主办的论坛上表示，正考虑要求电商平台在卖家账号上公布市场准入（market authorization）证明，以证明所售商品，尤其是部分药品、食品及护肤品，已在FDA登记。
他称此举须谨慎推进，不希望被贴上“反贸易”标签，目前仍在评估具体可行性与执行方式，并指出实施难点在于平台是否会向热销商品的卖家索取该证明。
泰斯顿还表示，FDA持续向多家平台提出下架要求——包括Meta、TikTok、Shopee、Lazada、Shein及Carousel.ph等，并实行周度下架，但下架后常常又有新刊登出现。
他警告，未注册产品多为劣质，可能危害健康，近期网上出现假狂犬病疫苗的案例有所增加。
泰斯顿承认FDA人手不足，正在制定一项为期五年的发展计划，以改善流程并争取相应预算与增补人力。他并承诺要减少产品注册与经营许可的积压，目前累计约1.7万件。
他指出，在解决药品监管中心（CDRR）约1.6万至1.7万件积压的同时，机构能够更有效处理占比60%至70%的非处方药及家庭用药等主流、低风险申请。今年希望启动非处方药的在线注册。

马尼拉讯——供水公司Maynilad加紧行动，透过侦测并修补漏损，力求减少非营收用水（NRW）。NRW指已处理但未向用户计费的用水，成因包括实体漏损、商业损失，以及消防、公用接管与系统维护等获准未计费用途。
公司向媒体展示了听漏棒、漏损相关器与移动漏损探测器等设备，表示此举旨在提高公众认知，推动减损工作。当前Maynilad的NRW为30.9%，目标在2027年底降至25%，并在2030年降至20%。
Maynilad称，2025年通过强化漏损侦测、管线更换、压力管理与网络监控等措施，日回收水量约2.56亿公升，相当于两座净水厂的产能。公司指出，降低NRW可将更多供水输送至低压地区，改善用水可及性。公司今年已在营业计划中为NRW管理拨出77亿披索。

亚马逊云端服务（AWS）菲律宾区负责人 Precious Lim 表示，数据主权不仅关乎数据的物理存放地，还涉及谁控制数据、谁可访问以及适用何种法律。AWS 为客户提供工具和合同承诺，允许客户选择数据存储与处理地点并保持对数据的控制。

KARAWANG，印度尼西亚——靠近雅加达的一处无窗微软数据中心持续嗡鸣，成为席卷东南亚的科技建设热潮一部分。随着人工智能需求攀升，科技巨头正投入数十亿美元在该地区扩张，仓库式数据中心在亚洲尤其快速增长。法新社获准对该印尼中心进行罕见访问；外墙无公司标识，访客须经严密安检，技术人员甚至在宗教假期仍在现场维持运行。
KPMG预计东南亚数据中心容量将自2025年水平到2030年增长三倍。微软基础设施经理Alistair Speirs称，未来数年“每个应用、每项工作流程、每位用户”都将部分使用AI。微软云运营负责人Noelle Walsh表示，更高性能芯片带来更大能耗，公司已调整机房设计并采用“闭环”水冷等散热方式以防过热。
但扩张也加剧对以化石燃料为主电网和当地水资源的压力。智库Ember预计印尼数据中心用电到2030年可能增加四倍；印尼约70%电力来自燃煤。微软在雅加达的项目属于17亿美元投资，若达超大规模将需数百兆瓦电力，公司称通过与公用事业合作并签署十年期协议在印尼提升约200兆瓦可再生能源，推动电网“绿化”。
雅加达都会区因地下水抽取部分下沉，政府拟逐步迁都。新加坡国立大学学者Olivia Jensen警告，数据中心会加剧本已过度开发且管理欠佳的水资源问题。微软预计用水量到2028年前上升，随后随着更多闭环系统投入运作，2029年可稳定在约6.6亿升；Walsh称公司正朝“日常零用水”方向演进。顾问Trung Ghi提醒，若下一代系统更大更耗能耗水，压力将进一步加剧。
