美国第四季度劳动生产率增幅超过市场预期，进一步表明企业正努力提升效率以控制成本。美国劳工统计局周四公布的数据显示，第四季度生产率——非农雇员每小时产出——环比折年率增长2.8%，此前的第三季度增幅被上修至5.2%。经济学家的调查中值预测为增长1.9%。近期效率提升的趋势，有助于抑制工资压力，印证了美联储官员的观点，即劳动力市场已不再是通胀来源。对许多企业而言，劳动力成本是最大支出，因此公司会通过引入新技术和设备来提高员工效率。对人工智能（AI）等技术的投入，使部分企业能够以更精简的人手维持运营，这也在一定程度上导致去年招聘偏弱。单位劳动力成本——即企业为生产一单位产出而向员工支付的成本——在第四季度同样上升2.8%。尽管去年年底经济增长放缓，但这在很大程度上反映了美国史上持续时间最长的政府停摆所带来的影响：联邦政府支出出现自1972年以来最大降幅。不过，企业投资仍以稳健速度增长。劳工统计局的报告还显示，2025年全年生产率增长2.2%，劳动力成本上升1.9%。

法国允许美军支援飞机使用基地 但不得参与对伊朗行动
法国军方宣布，美军支援类军机已获准进入法国南部伊斯特尔空军基地，但前提是相关军机不得参与美国在伊朗境内的军事行动。
法国商业调频电视台星期四（3月5日）引述法国武装部队总参谋部的声明说，获准使用伊斯特尔（Istres）空军基地的是支援类军机（非作战飞机）。此次授权不涉及法国在中东地区的军事基地。
新华社引述法国军方说，上述安排是在北大西洋公约组织框架内的例行程序。鉴于当前局势，法方已明确要求，相关军机不得以任何形式参与美国对伊朗的军事行动，仅能严格用于支持法国在该地区伙伴的防务需求。军方声明说:“法国已获得这方面的充分保证。”
法新社报道，此前，法国总统马克龙星期二（3日）指出，美国与以色列在伊朗的军事行动“违反了国际法”。
