港府拟立法保障外卖员等权益 盼年底提交草案
香港特区政府计划立法保障外卖员和送货员工伤和死亡补偿权益，希望能在今年底将条例草案提交立法会审议。
综合香港《星岛日报》和香港电台等报道，香港劳工及福利局星期五（7月10日）向立法会人力事务委员会提交立法框架，拟为平台工作者设立工伤和死亡补偿机制， 先涵盖外卖员和送货员。
劳福局局长孙玉菡星期六（11日）在一档电视节目上受访时说，根据现行条例，外卖员在工作期间死亡的最低死亡赔偿额，倾向参考《雇员补偿条例》下的相关赔偿金额51万港元（8万4000新元）。雇员若因工受伤休假，雇主必须支付原本收入约八成的补偿金，最多两年。
孙玉菡指出，平台工作者与传统雇员的性质存在差异，因此平台方支付的工伤补偿需采用多层次计算方法。这包括考虑相关人士从事平台工作的时间、厘定按收入计算病假津贴的比例，以及获补偿的最长期限。
在罚则方面，孙玉菡称，工伤补偿机制要求平台公司在指定时间内呈报平台工作者受伤，法定要求与现行《雇员补偿条例》非常类似。因此，政府在制定罚则时会参考现行条例下逾期未呈报工伤的最高罚款金额5万港元。
孙玉菡透露，政府未来一两个月将重点讨论具体细节，希望在暑假内就工伤补偿计算有较为清晰的安排，随即开展法例草拟，争取在今年底将条例草案提交立法会审议。
他补充，若死亡和工伤补偿等核心方案能达成平衡，他倾向一次过推行工伤和死亡补偿机制，但如果若部分条文需要更多时间斟酌，也可分开执行。
孙玉菡最后说，政府下一步将研究如何把网约车等平台工作纳入上述机制。

（纽约综合电）俄罗斯作为仅次于美国的全球第二大柴油出口国，本周禁止柴油出口，冲击全球能源市场，进一步加剧工业燃油供应短缺，并推高燃油价格。 
由于全球石油市场高度相互关联，即使美国和欧洲多国早已停止购买俄罗斯柴油，也难以避免受到波及。美国超低硫柴油期货周三（7月8日）暴涨11%；同一天，欧洲低硫柴油期货上涨至历史新高。

柴油是全球消费占比最大的油品，广泛用于工业、农业、交通和发电等领域。后冠病疫情时期需求强劲，原本已令供应持续紧张；美伊战争爆发后，供应压力进一步加剧。
路透社分析指出，全球少了俄罗斯柴油供应后，巴西和土耳其等国跟欧洲国家和其他柴油进口国竞购美国柴油，这可能对电力和农业领域产生连锁反应。

全球能源咨询公司FGE NexantECA的炼油部门主任谭麒麟（译音）说，霍尔木兹海峡运输受阻后，美国成为欧盟和英国的首选柴油供应国，“但现在，美国每向拉丁美洲转运一桶柴油，就意味着流向欧洲的供应减少一桶。更何况，美国及ARA地区（欧洲三大重要港口阿姆斯特丹、鹿特丹和安特卫普）的柴油库存，已远低于往年同期正常水平”。
他也说，中东局势再度紧张也意味着，中国7月份放宽燃料出口限制的举措，未必能延续至8月。换言之，来自亚洲市场的潜在供应可能受阻，无法缓解全球柴油供应吃紧的局面。

欧盟：减少对华依赖须投入资金支持
欧盟正在制定一项“团结工具”，支持欧洲企业把关键物资供应来源从中国转向其他市场，并在发生贸易冲突、北京采取报复措施时，缓冲企业所受到的冲击。
彭博社星期六（7月11日）报道上述消息。报道引述据知情人士称，这项“团结工具”将带来成本，需要获得资金支持。目前，欧盟成员国正就欧盟下一轮多年期预算讨价还价。
报道称，目前尚不清楚需要投入多少资金，因为这将取决于中国可能采取的反制措施规模等因素。但鉴于所涉及的产业和贸易规模庞大，所需资金可能相当可观。欧盟展开相关讨论之际，多个成员国正推动削减开支。
欧盟也在推动中国延长关键物资出口许可证的有效期、简化进口程序，同时要求北京减少市场准入限制并降低出口规模。
知情人士说，中国则要求欧盟作出对等回应，即如果北京同意作出调整，欧盟也应放宽自身的管制措施。
双方把10月设为取得进展的期限。欧洲委员会主席冯德莱恩6月表示，如果对话无法取得成果，欧盟将果断采取行动。

伊朗：除非美国“放弃立场” 否则不会谈判
据伊朗法尔斯通讯社报道，伊朗表明，除非美国“放弃立场”，否则伊朗不会与美国谈判。
伊朗外长阿拉格齐星期六（7月11日）抵达阿曼，就关键的霍尔木兹海峡问题举行会谈。阿曼曾斡旋推动伊朗与美国谈判。
霍尔木兹海峡一直是伊朗与美国谈判的关键症结所在。阿拉格齐坚称，伊朗一直信守承诺。
据法新社报道，阿拉格齐说，德黑兰“迄今为止一直信守诺言，不像美国财政部长，他违反了谅解备忘录第九条”。
根据备忘录第九条，在双方达成最终协议之前，美国“不会实施任何新的制裁，也不会在该地区部署更多部队”。
美国财政部星期五对一名在迪拜的伊朗商人阿里（Ali Ansari）和其他13名个人和实体发出制裁令，指阿里将公款转移到国外广泛的房地产和商业地产投资组合，为自己、伊朗政府经营和伊斯兰革命卫队图利。
