（巴黎／柏林综合电）法国总统马克龙宣布，法国将扩大核武库，并可能首次在盟友境内部署装载核弹头的战机。
马克龙星期一（3月2日）在法国西北部的核潜艇基地发表讲话时说，八个欧洲国家包括德国、波兰、瑞典、英国、荷兰、比利时、希腊和丹麦已同意加入这个核威慑计划。
按计划，法国“战略空军”将可暂时部署在盟友境内，从而扩大欧洲大陆的防御范围和能力。马克龙说，盟友也可参与法国的核活动，例如英国部队近期参与了法国的军事演习。但他强调，法国将保留使用核武器的最终决定权。法国拥有世界第四大核武库，估计约有290枚核弹头。欧洲另一核武国家是英国。
马克龙也透露，他已下令增加法国武器库中的核弹头，而且从今以后法国不会再披露其核弹头数量。
他说：“当前地缘政治动荡，充满风险。面对多重威胁，我们必须加强核威慑。在充分尊重国家主权的前提下，我们必须考虑将威慑战略渗入欧洲大陆。”
马克龙演讲过后，法国和德国发布联合声明，宣布将加强在核领域的合作。两国将成立一个高级别指导小组协调战略，德国部队将参与法国核演习、联合视察战略要地以及与欧洲伙伴共同发展常规和导弹防御能力。
联合声明称，此安排是为北约本身的核威慑计划提供额外补充，而不是取而代之。
欧洲长期以来依赖美国提供的核保护伞，但在特朗普政府对俄罗斯态度缓和、对美国传统盟友采取更强硬立场、以及威胁以武力夺取格陵兰的情况下，欧洲各国不免担心美国不再可靠。
加入法国核威慑计划的丹麦首相弗雷德里克森说：“加强合作将提升欧洲的威慑能力。很不幸的，这是必要的，因为未来几年来自俄罗斯的军事威胁预料将加剧。”不过丹麦防长波尔森说，与法国的核合作不包括在丹麦境内部署法国核武器。
马克龙的总统任期只剩一年，而他目前的支持率很低。有学者指出，马克龙必须谨慎行事，他的核政策可能在2027年总统大选前引发强烈反弹。法国极右派已对马克龙发出警告，国民联盟领袖勒庞的顾问奥利维尔在社媒写道：“如果马克龙认为他可以把法国的核武器交给欧盟，他将因叛国面对弹劾。”

美国与以色列联合空袭伊朗后，伊朗的报复行动已波及海湾周边。阿联酋与卡塔尔正紧急斡旋，联合盟友游说美国总统特朗普寻找“下台阶”，以缩短美国对伊朗军事行动时间，防止战事扩大并冲击全球能源市场。
彭博社星期二（3月3日）引述知情人士报道，两国正积极筹建广泛联盟，旨在通过外交手段迅速平息战火。此举的紧迫性在于，随着战事升级，海湾国家的基地、基础设施与民用设施正日益暴露在报复风险之下，对地区安全与航运稳定构成巨大压力。
在冲突不断外溢的背景下，越来越多原本不愿卷入战事的国家，正面临被动卷入的压力，逐渐形成一种“被迫联盟”。即便只是有限参与，也难以真正置身事外，潜在风险巨大。

“一方面，美伊冲突会促使它们收紧与美国的安全关系；但另一方面，美国被视为一个并非完全可靠的伙伴，使这些国家心存顾虑。”
海湾沿岸国家面临的直接危险，比任何其他国家都更为严峻。当地的油港、炼油厂、机场和航道，无一不在伊朗导弹和无人机的射程之内。
澳大利亚迪肯大学全球伊斯兰政治学教授格巴顿（Greg Barton）受访时指出，若海湾国家的民用基础设施持续受到影响，或将触发某些国家采取象征性甚至实质性的报复措施，从而对伊朗构成风险。
巴顿说：“这是伊朗政权面临的风险，但考虑到正处于关乎政权存亡的生存之战，它很可能愿意承担这一风险。毕竟，相较于美国和以色列已造成的打击，海湾国家即便采取报复行动，所能增加的冲击也十分有限。”　
星期天（3月1日），海湾合作委员会发声明强烈谴责伊朗针对成员国的攻击，并强调在遭受侵略时，各成员国拥有行使个别和集体自卫的权利，包括采取一切必要措施保卫国家安全和领土完整。
这是海湾国家首次提及“可能军事回应”。
多西认为，单纯观望或发表声明，终究难以持续。“在某个时间点，海湾国家将被迫作出回应。但这并不意味着必然诉诸军事回应；在避免局势升级的前提下，它们更可能选择制裁、外交施压或其他非军事手段。”

IndiGo宣布将开行四班往返沙特吉达的航班，以“逐步恢复”两地间的运行；Air India Express自周二起恢复往返阿曼首都马斯喀特的航班；低成本航空Akasa Air也将开展部分吉达航班。航空公司同时表示，除个别航线外，往返巴林、科威特、卡塔尔、沙特和阿联酋的大部分航班仍处于暂停状态。
印度总理纳伦德拉·莫迪已与包括巴林、沙特和约旦在内的多国领导人通话，感谢他们“照顾印度侨民”；他还与以色列总理内塔尼亚胡通话，表达了对最新局势的关切并强调以保护平民为优先。
冲突已造成数名南亚公民死亡：据报道，阿布扎比至少一名尼泊尔公民遇难，巴林和阿联酋各有两名孟加拉国公民丧生，印度媒体另报导一名印度公民在阿曼海域的一艘船上遇难。

美国国务院周一下令，因地区安全形势恶化，要求约旦、巴林、伊拉克、卡塔尔和科威特的非紧急美方政府人员及其家属强制离境。
国务院表示，约旦、巴林、卡塔尔和科威特面临来自伊朗的持续无人机和导弹袭击威胁，并且商业航班严重中断；同时警告约旦和巴林存在恐怖袭击风险，伊拉克则有高度暴力与绑架风险。
这是自美以对伊发动对伊朗军事行动以来，首次对三个使馆下达非紧急人员强制撤离令，反映伊朗报复导致地区安全风险上升，目标集中在美方军事和外交设施。
美国驻沙特使馆周二夜间遭疑似伊朗无人机袭击并在利雅得附近发生两起袭击；美国驻科威特使馆亦遭袭。多国使馆已发布就地避难指示并敦促美民众同样避险。
下令离境并不等于所有使馆关闭，但科威特使馆已宣布暂停开放直至另行通知。其他使馆将仅保留关键人员。国务院敦促中东地区美国公民“尽快”通过商业航班离开，但多数航班已停飞，美国政府尚未启动撤侨航班。 来源：CNN

总统承诺长期解决燃气储存问题
（3月3日）总统阿努拉·库马拉·迪萨纳亚克在国会表示，因用气量骤增，国内出现燃气短缺危机。他称问题在于储存能力不足，政府正准备必要措施应对。
迪萨纳亚克指出，国家总储气量约为8000公吨，最多只能维持约一周供应。Litro公司日需约1000至1200公吨，若需求突增至1800公吨，供应不到五天便会告罄，可能引发危机。他说，每两天约有8000公吨通过港口入境，目前还有船在马尔代夫滞留，小轮分批运送，现行体系需改变。
总统表示，Litro已订购10万个新气瓶，运输船预计3月12日抵达，气瓶短缺将缓解；总体上未来发生燃气和燃料危机的可能性很低，政府会寻求并实施长期解决方案。
