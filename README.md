马尼拉——众议院议长博希·戴（Bojie Dy）周一宣布，众议院已展开对副总统萨拉·杜特尔特的弹劾审查，强调议会“将履行职责”。他表示，这是宪法程序，不是政治表演，也不是借大众舆论审判的场合；司法委员会将依照既定众议院规则审查投诉的形式与实质，确保公平、正当程序与透明处理。
针对杜特尔特的弹劾案主要指控贪污与渎职，称其在担任副总统及教育部长期间挪用机密经费。杜特尔特否认指控，并已宣布谋求2028年总统大选；官员表示其参选计划与弹劾程序无关。
去年众院亦曾对其提出弹劾，但最高法院因程序性的一年禁限条款驳回，并未对指控本身作实质裁断。担任司法委员会主席的巴坦加斯议员格维尔·鲁伊斯特罗（Gerville Luistro）指出，最高法院的裁定仅限程序性争议，未证明被告无罪，并强调弹劾是宪法程序，不是社交媒体竞赛——“不是標籤決定，也不是迷因判斷”。
依菲律滨宪法，弹劾将移送参议院审判；若定罪，涉案者将被禁止从政，届时其总统参选希望将被彻底摧毁。分析人士并认为，杜特尔特的参选宣示也可能用于稳住支持者并示意其派系有望重返权力核心。（综合法新社报道）
