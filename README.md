另一方面，中国外交部也说，王毅欢迎达尔专程赴华出席世界人工智能大会并代表巴基斯坦政府签署成立世界人工智能合作组织协定，表示巴基斯坦的加入符合巴基斯坦国家和人民长远利益，也为中巴全天候战略合作伙伴关系注入新的时代内涵。
王毅也说，双方要保持高层交往，加强政治互信，推进重点领域合作，筑牢反恐安全保障。

辽源市委书记沈德生突发疾病逝世
中共辽源市委书记沈德生突发疾病逝世，享年56岁。在离世前数日，他的全部公开行程几乎都围绕防汛抗洪工作展开。
根据微信公众号“辽源发布”，吉林省辽源市沈德生同志治丧工作小组星期五（7月17日）发布讣告称，沈德生于2026年7月17日清晨突发疾病，经抢救无效在辽源逝世，享年56岁。
讣告称，遵照家属意愿，丧事从简。
公开资料显示，沈德生出生于1970年1月，历任吉林省地税局发票管理处处长，辽源市地税局局长，吉林省地税局副局长，长春市地税局局长，吉林市委常委、副市长，吉林市委副书记等职。
2020年1月，沈德生当选吉林市人大常委会主任，后任吉林省委编办主任、省委组织部副部长。他自2023年1月起任辽源市委书记，主持全市工作。
据中国媒体梳理，沈德生离世前数日仍持续奋战在防汛一线，频繁参与抗洪调度和应急指挥工作。辽源地处丘陵地带，城市内涝和地质灾害隐患点较多，每年主汛期防灾压力较大。今年7月又受到台风“巴威”影响，当地防汛形势尤为严峻。
据报道，沈德生最后一次公开露面是在7月13日。当天，辽源发布山洪红色预警，他连夜主持全市防汛视频调度会，连线两县两区实时了解险情，逐项部署水库巡查、堤防值守、危险区域群众转移等工作，并要求紧盯各类风险隐患、强化闭环处置。

朝中社批韩国参加美国主导海上军演 沦为“傀儡”
朝鲜官方媒体对韩国参与由美国主导的海上军事演习提出批评，称韩国为“傀儡”。

环太平洋军演号称是全球规模最大的国际海上联合军演，每两年举行一次。据环太平洋军演官网消息，今年约有30个国家参加，包括日本、加拿大、澳大利亚等。
朝中社称，在“军事勾结日益公开”之际，“韩国傀儡部队作为主力参与此次军演”，这里指的是近期韩日军事合作的加强，以及韩国与北约关系的深化。
这是朝鲜近一年零三个月以来再次使用“傀儡”一词来形容首尔。上一次是在2025年4月18日晚，当时韩国空军一架KA-1型轻型攻击机在江原道山区进行夜间飞行训练时，两个机炮吊舱和两个副油箱意外掉落。
朝中社还说，环太平洋军演不只是“针对假想敌的例行演习”，而是美国与盟友针对印太地区国家的“战争演练”。
报道说：“所有这些事实都指向真正动摇世界和平与安全根基的势力，并预示着如果不加以防范，朝鲜半岛和本地区可能会出现令人不愿看到的局面。”
朝中社还进一步批评美韩海军陆战队联合军演，称此次联合空中演习模拟了美国海军“埃塞克斯”号（USS Essex）两栖攻击舰“深入敌后渗透”。
朝鲜正大力推动海军现代化建设，朝中社7月初报道，朝鲜最高领导人金正恩视察了新型5000吨级驱逐舰“姜健”号（Kang Kon）上的战略巡航导弹发射和武器系统测试。

月之暗面发布全球最大规模开源模型Kimi K3
中国人工智能（AI）初创公司月之暗面发布新一代大模型Kimi K3，是全球目前规模最大的开源模型，据称性能已接近美国AI公司Anthropic的前沿模型Fable。
新华社报道，2026世界人工智能大会暨人工智能全球治理高级别会议星期五（7月17日）在上海举行前夕，月之暗面星期四（16日）发布参数规模达2.8万亿的Kimi K3大模型。
公司介绍，Kimi K3原生支持视觉理解，具备100万词元上下文窗口，面向软件工程、知识工作、深度研究、多模态理解等复杂任务场景进行优化，能够在单次提示中处理和保留远超前代模型的信息量。
路透社报道，美国出于国家安全考量，上个月暂停向外国用户开放使用Anthropic的Mythos和Fable模型。Kimi K3的发布，也凸显中国开放模型与美国前沿AI系统之间的技术差距正在缩小。
报道指出，包括月之暗面、智谱和MiniMax在内的中国AI企业，正以更低成本推出越来越强大的模型，且模型发布周期也在不断缩短。
月之暗面称，在GPU内核优化这一专项能力测试中，Kimi K3的表现已接近Fable 5。多个第三方测评结果显示，Kimi K3在整体表现上已跻身全球第一梯队 。
在资本层面，获中国互联网大厂阿里巴巴和腾讯支持的月之暗面持续扩张。彭博社上月报道，该公司正寻求新一轮约20亿美元（26亿新元） 融资，估值约300亿美元，为潜在赴港上市做准备。

巴新关闭台北经济办事处 萧美琴：台外交处境非常辛苦
巴布亚新几内亚宣布关闭驻当地的台北经济办事处后，台湾副总统萧美琴说，台湾的外交处境“非常非常辛苦”，北京持续打压台湾的生存空间及扩展国际空间的机会。她称这是一次挫折，但强调台湾政府仍将持续争取扩大与国际的合作。
综合台湾《太报》和ETtoday新闻云等报道，萧美琴星期五（7月17日）出席“2026亚洲生技大展”前受访时说：“我们的外交处境真的非常非常辛苦，中国到处在打压我们生存以及对外开拓国际空间的一个机会。”
她说，巴新的决定“是一个挫折”，但无论是在服务海外侨胞，还是拓展台湾与国际社会的合作关系方面，政府都会继续努力争取。
巴布亚新几内亚外交部长特卡琴科（Justin Tkatchenko）星期四（7月16日）在社交平台脸书发文说，巴新“重申对中华人民共和国的‘一个中国’政策”，将关闭驻巴新的台北经济办事处。
对此，台湾外交部批评巴新政府未经协商便单方面宣布关闭代表处，并称特卡琴科是在毫无预警的情况下发布相关声明。外交部对此表达严正抗议，并称将持续通过管道与巴新政府交涉。
台湾外交部还强调，台湾不接受巴新政府片面宣布的决定，驻巴新代表处将持续维持正常运作，并依相关规范维护台湾权益及提供台湾民众必要服务。
中国大陆外交部发言人林剑星期四在例行记者会上回应称，北京对巴布亚新几内亚此举表示高度赞赏，并愿同巴新继续在涉及彼此核心利益问题上坚定相互支持，共同推进两国全面战略伙伴关系向前发展。
巴布亚新几内亚曾在1999年短暂与台湾建立外交关系，但双方目前并无正式邦交。
