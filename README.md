中国油价今将调整 大概率上调
中国油价将在星期一（3月9日）午夜12时调整，本轮调价大概率将迎来上调。
红星新闻报道，据卓创资讯数据，截至上星期四（3月5日）收盘，中国第九个工作日参考原油变化率为11.35%，预计汽、柴油上调每吨500元（人民币，下同，92.86新元），远超出每吨50元的上调红线。
按照目前的幅度计算，折合升价后，92#汽油、95#汽油、0#柴油将分别上调0.39元/升、0.41元/升、0.42元/升。
南方网报道称，这已为年内最大涨幅。
中国成品油价格定价不随国际原油“秒更”，而是遵循“十个工作日”原则。只要前十个工作日国际原油均价较上一轮调价时上涨超过调价红线，中国成品油限价就会顺势上调。
按照惯例，本轮油价调整的落地时间为星期一午夜12时，最终调整幅度以中国国家发展改革委官方宣布为准。
由于中东局势没有缓和迹象，油价已飙升至每桶100美元（约128新元）以上水平。
