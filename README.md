芬兰研究：长期夜间噪声 恐致胆固醇升高
芬兰奥卢大学参与的一项研究发现，长期夜间交通噪声不仅影响睡眠，还可能改变人体血液中的代谢指标，增加心血管疾病风险。
研究指出，当噪声水平超过50分贝时，相关生物指标已出现变化，显示环境噪声或构成真实健康威胁。
新华社报道，公报称，此前研究已将交通噪声与心脏病、糖尿病等疾病联系起来，但其如何在生物学层面影响人体仍不清晰。此次研究提供了新的证据，表明长期夜间噪声与一些血液指标的变化相关，而这些变化可能是相关疾病的前兆。
研究人员分析了英国、荷兰和芬兰27万名成年人的数据，并利用国家权威部门发布的“噪声地图”，将交通噪声的情况模型化并与参与者居住地匹配。研究团队在这些参与者血样中检测多种代谢标志物，将参与者暴露于夜间噪声的情况与其代谢变化进行关联分析。
研究人员发现，当夜间交通噪声水平达到50分贝时，暴露于这种噪声中的人体内一些血液指标就会发生变化。随着噪声水平升高，这种变化更加明显。噪声水平若超过55分贝，总胆固醇、低密度脂蛋白（“坏”胆固醇）等指标水平明显升高。
研究人员指出，这项研究所揭示的生物学线索可为解释交通噪声与心血管疾病及相关代谢紊乱之间的关联提供支持。环境噪声不应仅被视为干扰因素，更可能构成真实的健康风险。有关部门可以通过城市规划与交通治理优化、降噪措施以及建筑设计等手段降低夜间交通噪声，为人们营造更安静的睡眠环境，减轻潜在的健康危害。
相关研究成果已发表在美国《环境研究》杂志上。

马尼拉讯——菲律宾外派劳工部（DMW）表示，尽管中东部分地区紧张升温，目前并无对海湾及以色列的海外菲工实施大规模遣返的命令，事态尚未达到需强制遣返的“警戒4级”。
DMW秘书汉斯·卡克达克称，约有80至100名身在迪拜的菲侨、52人在以色列申请返国，但因空域关闭及多国机场停航，暂无法立即运送回国。
目前DMW会同海外劳工福利署（OWWA）与各地劳工办着重提供境内援助，包括餐饮、临时避难、转运与现场协调，并与外交部、驻在国、国防部及私人运输公司协商撤离方案。
3月2日，共57名在香港国际机场滞留、原定前往阿布扎比的菲劳已安全回到马尼拉并获DMW与OWWA接应；另有滞留者获临时住宿与交通协助。
外交部统计，截至2025年约有200万菲劳在中东。OWWA强调其1348热线全天候运作，截至目前已处理92件求助，相关国家专线与使领馆联系方式亦在官方社交平台公布。
当局提醒在阿联酋及其他地区的菲侨遵守驻在国安全指引，如感不安全或欲返国，可透过DMW、OWWA、当地劳工办或使馆联络协助，相关应急预案已就绪。
