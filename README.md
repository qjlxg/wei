特朗普这次援引一项从未实际动用过的法律依据《1930年关税法》（Tariff Act of 1930）第338条，作为征收新关税的法律基础。
白宫周一（7月20日）说，新关税将在30天后，也就是8月19日生效，涵盖葡萄酒、冰球杆和水泥等多种商品，但不适用于能源、钾肥和那些已被征收行业专项关税的商品。
不过，在美墨加贸易协定（USMCA）下获得保障的商品，这次也被纳入征税范围。
美国贸易代表办公室说，这些关税将适用于价值近200亿美元（约258亿新元）的加拿大进口商品。
根据美国普查局的数据，美国在2025年从加拿大进口价值3820亿美元的商品，这约占其中的5.2％。
加拿大总理卡尼周一在回应特朗普最新加征关税举动时指出，加拿大将同美国加紧谈判，并已提出全面的建议来解决双方贸易争端，同时，也在推动美墨加贸易协定现代化。
卡尼说，这是美国一系列单边贸易行动的最新举措，违反了美墨加协定。
白宫周一指加拿大是继中国之后，针对特朗普2025年起实施的大范围关税采取报复行动的另一个国家。
白宫说，过去一年加拿大从美国进口的机动车辆减少了22％，酒类产品进口减少了81％。
特朗普的最新举动可能会进一步加剧美国与第二大贸易伙伴加拿大的紧张关系。几天前，因为加拿大林火的浓烟飘入美国，特朗普便扬言要对加拿大加征关税追责。
美国智库卡托研究所的学者林西科姆（Scott Lincicome）说，这是1930年关税法第338条首次被用于征收关税，显示特朗普愿意动用甚至滥用任何现有法律条文来达到目的。
前美国贸易官员马杰鲁斯（Ryan Majerus）说，特朗普这么做，显然是为了在与加拿大谈判时增加筹码。
他指出，美墨加协定的谈判仍在进行，美国与墨西哥的谈判进展比与加拿大的谈判更快。而关税延后30天生效，显示美方仍为加拿大留有谈判空间。“这可能是为了推动谈判进程。”

下午察：广东揭阳虐狗事件舆情传到海外
中国广东省揭阳市四名未成年人在街头虐狗的事件让许多网民愤怒，也让民间反对虐待动物立法的讨论再起。
不过，随着官方通报为事件定论，中国媒体减少报道、公众人物发贴后被删，一些网民和动物保护人士开始转向境外和线下，在香港、台北、纽约等城市的人流集中区域投放声援海报，呼吁人们持续关注事件，推动动物保护立法。
6月28日，四名未成年人当街以木棍抽打、淋汽油烧死母狗“旺旺”及其四只出生半月的幼犬。据报道，旺旺是流浪狗，被坪埔村村民们投喂长大。

中国未来五年将改造8万公里交通基础设施
（北京综合讯）中国计划在未来五年内更新与改造约8万公里的交通基础设施，同时改建升级农村公路，目标在2030年基本建成现代化综合交通运输体系。
综合央视新闻与中新网报道，中国交通运输部副部长徐成光星期二（7月21日）在发布会上说，“十五五（2026年至2030年）规划”时期，中国交通运输发展将进入新的发展阶段。
徐成光说，在优化交通运输结构方面，重点将推进交通基础设施的更新和数智化改造，到2030年实施8万公里左右的设施改造，并启动新一轮农村公路提升行动，新改建农村公路50万公里。
关于完善基础设施的布局，将重点推进跨区域跨流域大通道建设，推动国家综合立体交通网主骨架建成率提升至95%，完善铁路、公路、水运、民航、邮政快递等基础设施网络，加快建设物流网。
此外，未来五年中国将完善现代化都市圈综合交通运输体系，实施都市圈城际通勤效率提升工程，加快建设20个国际性、80个全国性综合交通枢纽城市。
徐成光说，到2030年，中国基本建成交通强国将取得决定性进展，现代化综合交通运输体系将更加完善。
根据中国政府目标，到2035年将基本建成“交通强国”，同时正加大农村交通基础设施的投入力度。中国交通运输部此前通报，今年1至6月，中国全国农村公路完成固定资产投资1443亿元（人民币，274亿新元），新改建农村公路3.4万公里。

美贸易代表：10%关税将到期 很快会有新行动
美国贸易特使格里尔星期二（7月21日）说，随着总统特朗普的临时关税将于本周到期，美国将“很快”对数十个国家加征新关税。
法新社报道，格里尔告诉消费者新闻与商业频道（CNBC）：“美国法律禁止销售涉及强迫劳动的商品。多数其他国家没有相关法律，即使有，也并未真正执行。我们应该很快就会有一些行动。”
格里尔发表此番言论之际，正值特朗普政府近期提议对一些国家加征10%至12.5%的新关税，理由是这些国家未能采取行动打击强迫劳动。
