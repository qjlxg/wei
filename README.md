澳大利亚总理宣布将制定全球首套人工智能国家标准
澳大利亚总理安东尼·阿尔巴尼斯宣布，将制定全球首套统一的人工智能国家标准，该标准将明确大型数据中心的运营规则，保护版权，并规范人工智能开发资源的使用。
阿尔巴尼斯在悉尼大学发表讲话时表示：“今天，我宣布，为了把握人工智能带来的机遇并引导其发展，我国政府将制定澳大利亚人工智能标准。”
据他介绍，新标准将把大型数据中心的各项要求整合到一个统一的强制性监管体系中。政府计划在与企业界和贸易伙伴磋商后，于明年初向议会提交相应法案。
澳总理还宣布，将在总理和内阁部内设立人工智能办公室，负责协调政府在该领域的政策。
阿尔巴尼斯表示，版权保护将受到特别重视。他强调，未经版权所有者许可，任何公司都不得使用艺术作品、书籍、音乐或媒体内容来训练神经网络。
“澳大利亚的作家、音乐家、艺术家和记者必须保留对其作品的所有权和控制权。任何公司都不得在未经作者监督的情况下使用澳大利亚的书籍、音乐、艺术作品或新闻来开发或训练人工智能……否则就是盗窃。”澳总理说道。
他还指出，澳大利亚将成为首个将数据中心选址、电力供应和用水量要求整合到国家统一标准体系中的国家。

日本出台新版AI战略计划 加强自主研发和安全
由于美国和中国的人工智能（AI）技术快速发展，日本政府星期二（7月14日）出台新版人工智能战略计划，以加强应对AI技术的风险和拓展自主研发。
《日经新闻》报道，尽管日本一直希望带头制定人工智能国际规则，但它在研发能力和研究基础设施方面，一直落后于美国和中国。
在AI领域方面，日本私企每年投资额为11亿1000万美元（约14亿3000万新元），排名世界第14位。排名首位的是美国（2859亿美元），中国第二（124亿1000万美元）。
星期二出台的AI战略计划是高市政府去年上台后的第二个版本。
根据计划，当局要扩大投资框架，推进“人工智能转型”（AX）。拟在2040年将“垂直人工智能” （医疗保健和工业等领域）上的投资额拉高到23.1万亿日元。“物理人工智能”（自动驾驶以及机器人领域）为10.5万亿日元。
日本将AI和量子技术列为优先投资的17个战略领域之一，并且拟通过扩大科研补助金等措施支持日本学府以及企业的AI前沿研究。
在培养人才方面，日政府将建立一套支持大学系统性地促进研究人员和研究生海外派遣的机制。按计划，到了2030年，每年培养至少2万名相关领域的博士毕业生，并向海外派遣3万名研究人员。 
自民党制定AI安全法则的议员平昌明说，AI已成为国家安全和经济发展极其重要的问题，在使用上有必要加强政府的参与，并且避免过度依赖特定国家或公司。为此，日本必须加强AI自主研发。
据悉，美国AI技术公司Anthropic最近发布具备发现网络安全漏洞和高性能攻击能力的旗舰模型Claude和Mythos，促使当局着重于AI的安全问题。 
据报道，日本当局优先考虑金融领域的应对措施，以及石油天然气和电力公司等关键行业的AI使用，不但要加强这些基础设施的检查，还将建立风险评估体系，增强应对网络攻击能力。

德总理：不反对中国接管德国汽车工厂
德国总理默茨星期三（7月15日）说，他不反对中国汽车制造商接管陷入困境的德国汽车工厂，但他同时警告说，这并非解决行业问题的长远之计。
法新社报道，当被问及中国企业可能收购德国汽车工厂的问题时，默茨回应道：“这取决于各公司是否愿意。”

德国汽车行业正面临诸多挑战，包括欧洲需求疲软、美国的关税以及来自中国的激烈竞争，汽车行业的就业人数持续下降。
由于德国许多汽车工厂的产能利用率不足，一些人建议中国制造商可以利用部分生产线，甚至完全接管这些工厂。

默茨还指责中国低估人民币的汇率，从而使中国的出口产品在海外市场更具价格优势。
“从欧洲的角度来看，我们无法长期接受必须与一个货币被低估了25%至30%的合作伙伴展开竞争。”
“各企业可以有权利做任何事，但如果这个问题得不到纠正，我们将始终处于不利地位，尤其是在高额进口和补贴产品方面。”
近年来，德国对华贸易逆差急剧扩大，出口大幅下滑，而进口却稳步增长，机械制造、化工和汽车等行业都受到冲击。

据法新社报道，德国汽车业近年来受到欧洲需求疲弱、美国关税以及中国企业激烈竞争等多重因素冲击，导致行业持续承压，就业规模不断缩减。
德国汽车制造商大众汽车首席执行官布鲁姆（Oliver Blume）星期一（7月13日）对员工说，在此前已达成约五万个裁员名额的基础上，公司仍可能进一步裁减多达五万个岗位。
由于德国许多汽车工厂产能利用率不足，外界开始讨论让快速扩张的中国车企使用部分生产线，甚至直接接管部分工厂的可能性。比亚迪等中国电动车制造商目前正积极在欧洲寻找生产基地，以配合公司欧洲业务扩张。
对于中企收购德国汽车工厂的可能性，默茨星期三（7月15日）在柏林举行的记者会上说：“是否愿意这样做，应由企业自行决定。”
但他强调：“我认为这是一种应急解决方案，而不是解决我们自身结构性问题的办法。”
德国车企长期抱怨能源和劳动力成本高企，以及行政监管繁琐；而批评者则认为，德国汽车企业本身也需要推进改革，改善经营管理模式。
布鲁姆今年4月曾说，不排除让大众中国合作伙伴使用其德国工厂的可能性。不过，大众集团此后已淡化市场对短期内达成相关交易的预期。
除德国外，欧洲其他汽车制造商也在加强与中企合作。欧洲汽车巨头斯泰兰蒂斯今年5月宣布，与东风汽车成立合资企业，共同开展欧洲市场的制造、销售和工程业务。

欧盟乌克兰达成无人机协议 将建立合资企业
欧盟委员会主席冯德莱恩星期三（7月15日）说，乌克兰和欧盟达成一项“无人机协议”，旨在将基辅的专业技术与欧盟的工业能力相结合，建立联合项目并扩大生产规模。
路透社报道，冯德莱恩在基辅举行的乌克兰国庆日仪式上发表讲话时说：“我们需要结合彼此的优势。这项协议将把乌克兰的创新精神和欧洲的工业规模相结合。”
冯德莱恩说，欧盟可以为乌克兰提供“巨大的技术和工业产能”以及“安全可靠的生产基地”等优势。
这项协议是首个旨在覆盖欧盟所有成员国和企业的无人机协议。欧盟执行机构委员会说，双方的伙伴关系将在2028年前扩展至反弹道导弹领域。
欧盟委员会在一份声明中说，将与参与这份协议的19个创始合作伙伴开展合作。这项新倡议将打破官僚主义壁垒，制定共同标准，使乌克兰和欧盟企业更容易开展合资项目。
泽连斯基在仪式致辞中表示：“我们将竭尽所能，整合欧洲所有反弹道导弹能力，构建欧洲反弹道导弹系统。”
这反映出，欧洲政界人士和企业日益渴望利用乌克兰在俄乌战争中积累的专业知识，以增强自身抵御莫斯科的能力。
冯德莱恩对泽连斯基说：“你们在无人机和反无人机系统操作方面积累的知识独一无二。”
“我们深知欧洲在这一领域面临的威胁，因此我们必须共同利用好这些知识。”

美军称击中一艘试图驶向伊朗港口的油轮
在恢复对伊朗港口的海上封锁后，美国军机周三向一艘试图驶向伊朗港口的空载油轮发射导弹，使其失去航行能力。
美国中央司令部星期三（7月15日）在社媒X平台发布消息说，遭袭的是悬挂库拉索旗帜的贝尔马号油轮，油轮当时正经由公海驶向伊朗石油出口枢纽哈尔克岛，违反美国的封锁令并无视多次警告。一架美军飞机向该船发射“地狱火”导弹，使其丧失航行能力。
美军从美东时间星期二下午4时（新加坡时间星期三凌晨4时）起，恢复对进出伊朗港口和沿海地区船只的海上封锁。美军说，在执行封锁任务后的24小时内，已迫使两艘商船改变航向，并使一艘“不配合”的船只丧失航行能力。
周三早些时候，美军对伊朗发动了新一轮打击。美国中央司令部说，此次打击旨在削弱伊朗威胁霍尔木兹海峡船只航行的军事能力。
伊朗首席谈判代表、议长卡利巴夫说，如果伊朗无法从与美国达成的谅解备忘录中获益，那“我们就没有理由遵守这一协议”。
卡利巴夫周三在Telegram发文说，伊朗的国家安全取决于在霍尔木兹海峡维持“伊朗的安排”。
他指出，伊朗在应对与美国的对抗及相关停战谈判时，应立足于国家利益、国家安全及长远考量。他并补充道，德黑兰别无选择，只能依靠自身力量。

皮尤调查：中国全球好感度排名超过美国
皮尤研究中心的一项调查显示，全球对中国持正面看法的人士比例高于美国，为至少2023年以来首见。即便在美国盟友中，舆论也出现了转变。
据彭博社报道，调查显示，随着美国形象恶化，今年调查的36个国家中，中国在大多数国家获得的好感度高于美国，这些国家包括加拿大、澳大利亚、法国和德国等美国盟友。
中国的优势在亚太和中东国家中尤为明显。仅印度、日本、菲律宾、韩国、以色列和波兰六国对美国的评价高于中国。
在20个拥有可追溯至2023年数据的国家中，近一半受访者对中国持正面看法，而对美国持正面看法的占比仅36%。
三年前，这一比例分别为中国32%、美国58%。这也标志着中国国际形象在冠病疫情期间跌至历史低点后，出现了改善。
报告指出：“去年特朗普开启第二任期后，全球对美国的看法进一步恶化，不过多数人当时对美国的评价仍高于中国。今年这种情况已不复存在。”
皮尤另一份聚焦中国的调查显示，包括部分欧洲国家在内，不少国家对中国的好感度已升至或接近纪录最高水平。
研究还显示，至少自拜登总统任期后半段以来，全球对美国领导人的信心一直在持续下降。
外交政策方面，17个中等收入国家的受访者对美国的担忧多于中国。在这些国家，更多人认为中国是“可靠伙伴”，也有更多人认为相比美国，中国对全球和平与稳定“贡献很大”或“有相当的贡献”。
美国在“尊重个人自由”这一指标上依然领先于中国，更多受访者认为美国更尊重个人自由，但两者之间的差距也在缩小。
皮尤研究中心于2月8日至5月13日对超过4.2万人进行了调查，调查期间美国对伊朗发动战争。
