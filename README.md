法国和欧洲多国极右势力因一名年轻活动者被杀而迅速联动。
23岁的极右翼活动者昆廷·德兰克（Quentin Deranque）在里昂于2月14日与极左人士冲突中被殴致死，此事被部分极右人物比作“法国版的Charlie Kirk时刻”。事发后一周，极右组织跨国到场，在里昂举行游行，约3,000人参加，现场出现纳粹敬礼并高喊种族主义与恐同言论；另有至少二十多个欧洲城市举办悼念活动，包括罗马、德累斯顿和萨格勒布。
参加者包括德国“认同”运动（被德国内政情报机构列为极端组织）、意大利的CasaPound与Lealta Azione等被指具有新法西斯渊源的团体。法国政府消息称确有意、瑞、德极右组织到场。德国估计2024年国内极右翼人士约50,520人，较2020年增长50%。
尽管2023年法国议会报告曾认为国际间主要是理念交流、并无“实质行动联结”，报告作者埃里克·普利亚认为事态已在演进。分析人士称，极右在社交媒体与跨国联络上越来越成功，且相关主张如“回迁”等已从边缘语境渗入某些主流政党话语。英、德、法等国当局对极右动员表示警惕：里昂在入城要道部署警力，法国情报机构正监控可能的报复性袭击，外长巴罗警告极右势力可能削弱民主。一些国家亦曾拒绝极右人物入境。
此次活动的组织与参与显示出更紧密的跨国网络：德国媒体集体Filmkunstkollektiv、意大利Lealta Azione成员参与游行；组织方之一为曾以“为里昂之爱”名单参加过市选的反堕胎活动家阿列特·埃斯皮厄（Aliette Espieux）。奥地利极右人物马丁·塞尔纳在维也纳守夜时呼吁“唤醒泛欧运动”，参与者中的网红“阿尔米纽斯”称极右只有协同合作才有力量。

中国许多人工智能（AI）公司近期纷纷上市，但券商跟不上新股挂牌的快速步伐，分析师对相关公司的覆盖出现缺口。
彭博星期二（3月3日）的汇编数据显示，过去一年在香港上市的27家AI相关公司中，近80%获得分析师覆盖的人数不到三位。这些较少受到关注的公司，包括上海壁仞科技和臥安机器人，其股价自上市以来已飙升超过60%。
在全球市场因AI颠覆性变革而动荡下，这批新兴的晶片设计公司和大型语言模型公司，帮助中国市场能够脱颖而出。
随着这些新秀即将迎来首个财报季，分析师提供的指引却寥寥无几。这使得评估进一步投资的价值变得困难，尤其是在大多数公司尚未获利的情况下。
美国银行亚太区股票策略主管吴旖（Winnie Wu）表示，从投资角度来看，这确实让事情变得更加困难。目前市场需要更深入的分析和理解，包括对行业、公司创始人及管理团队、公司治理、财务诚信的判断；这对远在3000英里（4828公里）之外的海外投资者来说，获取相关信息可能很困难。
中国大陆的上市尤其属于这一类。自12月初在上海上市以来，晶片设计公司摩尔线程的股价已飙升超过400%，但目前没有分析师对其追踪研究。
沐曦集成电路在12月IPO时，涨幅高达693%，打破摩尔线程的纪录，目前获得两位分析师覆盖。
相较之下，美国大型公司通常在上市25天后开始获得分析师的覆盖。卫星制造商约克太空系统公司（York Space Systems）1月筹集了6.29亿美元（8.02亿新元），金额约为摩尔线程发行规模的一半，目前却被彭博追踪的10位分析师覆盖。
建筑科技公司EquipmentShare.com在1月进行了8.59亿美元的IPO，与壁仞同月的IPO规模大致相当，获得10位分析师关注。
有迹象表明，至少对于规模最大的公司而言，相关的研究正在增加。摩根大通和高盛最近开始覆盖Minimax Group Inc.和北京智谱华章科技公司，两者股价上个月均上涨超过60%。
美国银行的科技分析师近期开始覆盖几只股票，部分原因是因为中国AI公司在全球供应链中的重要性日益提升。晨星公司也计划选择性地扩大覆盖范围。
摩根大通对Minimax和智谱均给予买入评级，预计两家公司将在2030年实现盈利，市盈率可能达到30倍，理由是“该行业的高增长以及相对于中国互联网同行的溢价”。
高盛给予智谱中性评级，其依据是对该公司到2035年息税前利润率的预估。

法国和欧洲多国极右势力因一名年轻活动者被杀而迅速联动。
23岁的极右翼活动者昆廷·德兰克（Quentin Deranque）在里昂于2月14日与极左人士冲突中被殴致死，此事被部分极右人物比作“法国版的Charlie Kirk时刻”。事发后一周，极右组织跨国到场，在里昂举行游行，约3,000人参加，现场出现纳粹敬礼并高喊种族主义与恐同言论；另有至少二十多个欧洲城市举办悼念活动，包括罗马、德累斯顿和萨格勒布。
参加者包括德国“认同”运动（被德国内政情报机构列为极端组织）、意大利的CasaPound与Lealta Azione等被指具有新法西斯渊源的团体。法国政府消息称确有意、瑞、德极右组织到场。德国估计2024年国内极右翼人士约50,520人，较2020年增长50%。
尽管2023年法国议会报告曾认为国际间主要是理念交流、并无“实质行动联结”，报告作者埃里克·普利亚认为事态已在演进。分析人士称，极右在社交媒体与跨国联络上越来越成功，且相关主张如“回迁”等已从边缘语境渗入某些主流政党话语。英、德、法等国当局对极右动员表示警惕：里昂在入城要道部署警力，法国情报机构正监控可能的报复性袭击，外长巴罗警告极右势力可能削弱民主。一些国家亦曾拒绝极右人物入境。
此次活动的组织与参与显示出更紧密的跨国网络：德国媒体集体Filmkunstkollektiv、意大利Lealta Azione成员参与游行；组织方之一为曾以“为里昂之爱”名单参加过市选的反堕胎活动家阿列特·埃斯皮厄（Aliette Espieux）。奥地利极右人物马丁·塞尔纳在维也纳守夜时呼吁“唤醒泛欧运动”，参与者中的网红“阿尔米纽斯”称极右只有协同合作才有力量。

以色列承诺向在持续中东战争中一名遇害菲律宾外劳家属提供生活援助。
以色列驻菲律宾大使达娜·库尔什周二在记者会上表示，政府“会尽可能为这个家庭提供一切必要的帮助与慰藉”。她指出，以色列有法律规定，无论受害者是否为以色列人，若因战争或恐怖袭击受到伤害，均可获得终身生活保障。
库尔什说，这无法挽回玛丽·安·维拉斯克斯·德维拉（Mary Ann Velasquez de Vera）的生命，也无法消除家属的痛苦，但可在日常生活上提供一定缓解。家属将获月度津贴，具体数额尚未公布；丈夫亦可获得额外照护和必要的心理支持。
对于遗体遣返时间，库尔什表示因以色列领空关闭、进出航班受限，目前尚无法确定何时遣返，“我们正在尽最大努力”。
报道指出，这场战争始于周六的一次袭击，导致伊朗最高领袖阿亚图拉·阿里·哈梅内伊死亡。作为回应，伊朗向包括沙特、卡塔尔和迪拜在内的多地发射导弹与无人机。
攻击已扰乱地区能源流通，关键的霍尔木兹海峡——约占全球石油运输的五分之一——实际上被阻断，外界担忧此举可能引发新的能源危机并推高通胀。

迪拜消息——菲律宾国防部（DND）警告称，网上出现越来越多由人工智能生成的视频和其他内容，歪曲中东局势，传播虚假叙事，可能对公众安全与秩序造成不良影响。
国防部提醒民众切勿转发未经核实的材料，并指出近期曾发现一段在网上流传的AI合成视频，虚假描述中东冲突的实情，易引起不必要的恐慌与混淆。
国防部在声明中表示：“目前冲突局限于中东，没有对我国领土以及在菲的菲律宾人和外籍人士构成可信的直接威胁。我们呼吁对假消息保持警惕，谴责在此时制造恐慌的人。”
同时，国防部重申若局势变化，将随时准备采取必要行动以保障菲律宾人民的安全，并呼吁公众以批判性思维对待网络信息，避免助长错误信息的传播。

耶路撒冷消息 — 以色列军方3月3日对黎巴嫩数十处地点发布新一轮撤离警告，要求民众立即离开并避开疑似目标建筑。
军方阿拉伯语发言人Avichay Adraee在Telegram上公布了约50个地点，多数分布在南黎巴嫩，称“为安全起见，你们必须立即撤离住所”。
他并单独警告贝鲁特南部的Ghobeyri和Haret Hreik社区居民，避开两栋“靠近真主党设施”的建筑，“立即撤离这些及相邻建筑，并与其保持至少300米距离”。随后又提醒哈达斯（Hadath）等南郊居民避开第三栋建筑。
此前在3月2日，以色列誓言在真主党对以方发射火箭弹与无人机后加大打击力度，并令其付出“沉重代价”。同日黎巴嫩政府罕见宣布禁止真主党的军事与安全活动，遭该伊朗支持组织强烈反弹。真主党称于2日凌晨发射火箭与无人机“为报复美以袭击中杀害伊朗最高领袖阿里·哈梅内伊”。
以色列随即对贝鲁特南郊及南黎巴嫩数十个村庄实施轰炸，黎巴嫩政府称此次攻击已造成至少52人死亡、154人受伤。 来源：法新社（AFP）

泰国总理下令紧急调整撤侨计划，优先撤离在伊朗约270–300名泰国公民与官员，尽快安排航班回国。政府已与驻中东各国大使馆及外交部区域办事处会商，各使馆将全力协助有意回国者办理出境手续。
总理表示，目前最关切的是国民安全，伊朗被认定为最危险区域，驻地服务系统部分受阻，需动用一切可行方式协助撤离。若情况恶化，将视需要调整措施。
针对油品供应，总理称国内石油与炼厂产能可满足需求，尚未到危机程度。泰国日炼油量约1.7亿升、日用约1.3亿升，另每日对老挝输油约700万升。政府可依法限制出口，若形势恶化将重新分配出口量，并要求能源部與國營PTT對外说明以安定民心。
政府将于明日召集能源部與商務部，研议尽可能稳住民生相关价格，并成立新闻发布中心每日通报最新情势。治安與相关部门已接令紧急备援。政府强调将尽一切努力，将冲击降至最低，并持续负责国家运作与国民安全。

马尼拉讯——菲律宾武装部队（AFP）周二表示，尽管中东局势紧张，但美方及其他志同道合国家对菲律宾在西菲律宾海的支持并未明显缩减。
AFP西菲律宾海发言人、海军少将罗伊·文森特·特立尼达在阿古纳多营的记者会上说，尤其是多边海上合作活动（MMCA）和即将举行的联合演习仍在进行，年度“巴里卡坦”演习与美军的合作也未见变动，目前未接获任何调整的公告。
当被问及公众能否“高枕无忧”时，特立尼达表示可以，“中东冲突是地区性事件，我们并非冲突方。AFP的重点是保障在中东的菲侨福利”。
AFP发言人帕迪利亚上校强调，中东局势与菲律宾的防务立场不可相提并论；菲美为长期盟友，但菲律宾的防务举措以国家利益为本，属防御性，并非完全依赖他国。
AFP正与相关部门做遣返应急规划，但尚无具体命令或方式决定。空军表示已就绪，初步拟由空运（可能以C-130为主）执行，但中东领空封闭等因素需审慎评估。海军则表示，一旦接获指示，具备提供海上运力的能力并随时准备支援。
