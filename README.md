《纽约时报》星期三（3月4日）引述中东官员和西方国家官员报道称，这一提议是通过一个未具名国家的情报机构提出的。
白宫和中央情报局尚未对此作出回应。
不过，报道称，华盛顿官员对伊朗或特朗普政府是否在短期内真的准备好“退出战争”，持怀疑态度。
伊朗驻日内瓦联合国大使星期二（3日）排除与美国进行任何谈判的可能性。
美国总统特朗普星期二称，德黑兰方面希望对话，但为时已晚，因为美国仍在继续对伊朗进行军事行动。

上千伊朗人遇袭死亡 以色列料展开袭击长达两周
美国和以色列对伊朗的袭击已造成1045名伊朗人丧生；以色列媒体称，以色列计划持续袭击伊朗约一到两周。
伊朗塔斯尼姆通讯社星期三（3月4日）引述伊朗烈士和退伍军人事务基金会报道说，死亡人数已上升至1045人。
另据路透社报道，国际原子能机构星期三说，伊朗伊斯法罕核设施附近的两栋建筑物遇袭受损，但存放核材料的设施未受损，目前没有放射性物质泄漏的风险。
《以色列时报》星期三报道，以色列军方计划在伊朗至少再进行一到两周的军事行动，目标是打击数千个伊朗政权有关的目标。
