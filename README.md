311地震為一次罕見且發生機制複雜的「連動型地震」。其震央雖然位於宮城縣對開海域，但整體的斷層破壞面卻異常廣闊，並由岩手縣一直延伸至茨城縣，合共長達約500公里，東西闊約200公里。在短短五分鐘內接連引發了至少三處大範圍的斷層錯動，累積釋放的能量極為龐大。
災後十五年，受災地區目前在安全性、基礎設施、觀光及出口等方面持續進行復興工作。然而，部分災民的心理創傷，以及福島第一核電廠為降低潛在輻射風險而持續推進的廢爐作業（如近年展開的取出核燃料碎片樣本、處理水排海及移除部分機組乏燃料），依然是這場災難留下的長遠挑戰。

台湾气象署星期四（3月12日）晚间在官网发布的地震报告显示，花莲县当晚8时14分发生规模5.7地震，地震深度15.1公里，震央位于花莲县政府南方24.7公里处，属花莲县近海。
地震最大强度达4级，在花莲县和南投县地区。台东县、台中市、宜兰县、嘉义县、彰化县和云林县地区为3级，苗栗县、新竹县、嘉义市、新竹市、台南市、高雄市、桃园市、新北市和台北市地区为2级，屏东县和澎湖县地区则为1级。
台湾宜兰近海今年2月24日发生5.6级地震。台湾气象署表示，这次地震是去年12月27日规模7级地震的较大余震，后续预计不会有太多余震。
