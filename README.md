数以亿计的人向聊天机器人寻求建议，科技公司也开始推出专门应对健康问题的AI服务。1月，OpenAI推出ChatGPT Health，声称能分析用户病历、健康应用和可穿戴设备数据以回答健康问题，目前需排队等待；竞争对手Anthropic也为部分Claude用户提供类似功能。
两家公司均强调，这类工具不能替代专业医疗，不应用于确诊，而是可用于解读复杂检测结果、准备就诊或从病历与应用数据中识别重要健康趋势。
专家提醒五点：
1) 相较于谷歌搜索，带有病史上下文的聊天机器人能给出更个性化的建议。加州大学旧金山分校医学教授Robert Wachter认为，负责任使用可获得有用信息，建议尽量提供详细信息以改善回答。
2) 出现呼吸困难、胸痛或剧烈头痛等可疑紧急症状时，应立即就医，勿依赖AI。斯坦福医学院院长Lloyd Minor警告，在重大医疗决策上不应仅依赖大语言模型。
3) 在上传健康数据前要注意隐私：美国联邦的HIPAA隐私法不适用于聊天机器人公司。OpenAI和Anthropic称健康数据受额外保护且不用于训练，且需用户主动同意并可随时断开。
4) 测试显示AI仍会出错：牛津大学一项1300人研究发现，使用聊天机器人并未比在线搜索或个人判断带来更好决策；在与真实用户交互环节AI表现较弱。该研究未使用最新版本如ChatGPT Health。
5) 求取“第二个AI意见”有助判断信息可靠性。Wachter认为，若工具能像医生一样追问细节，将更有用；同时把信息交给多个AI比对可增加信心。

人形机器人和外交成中国两会两大热点话题
“您怎么看待人工智能发展？”“怎么看待人形机器人？”“您担心被机器人取代吗？”
这是中国政协十四届四次会议星期三（3月4日）开幕当天，明星委员们被媒体围堵时，最常被问到的问题之一。
中国多家人形机器人公司上月扎堆亮相央视春晚，它们在舞台上花式耍大刀和空翻的武术表演吸引全球目光，相关产业获得舆论高度关注。 
这股热潮延续至今年中国两会（全国人大、全国政协年会）上。不少人工智能（AI）相关的科技、企业人士在会场外成为媒体蜂拥追问的对象。
中国网络安全公司奇安信董事长齐向东被一众媒体从室内的会场门口一路追问到室外。他对AI的发展表示乐观，但也理性地指出，人形机器人目前依然只是一种理想化的概念，实际工作价值预计需要三到五年的时间才能实现。
科技巨头360集团创始人、董事长周鸿祎也强调，目前看到的机器人多是“运动小脑”的能力，要让机器人真正进入家庭工作，需要强大的“大脑”。
以农牧与食品产业为主的新希望集团董事长刘永好提到了类似观点，他说，公司开始在研究如何运用机器人，但“能不能干活还不一定，关键在于它的大脑和小脑”。 
AI话题并非科企人士专利，甄子丹、张凯丽等多个演艺界政协委员同样被问到相关话题，并作出回应。
对于“人形机器人是否会取代武打演员的工作”，“叶问”甄子丹直言：“这很难说，你看春晚，全世界都觉得机器人非常厉害！”
他打趣道，若与人形机器人同台竞技，自己可能会被“比下去”。
不过，甄子丹有信心自己不会被取代，他认为人形机器人能帮助演员提高工作效率，而演员仍能通过情感的引导使这些高科技工具更表现力。
张凯丽则说，“AI发展成什么样，我们说了不算，只能是随着它的发展，而让自己更强大”。
除了机器人，中美、伊朗、中日等近期热门的外交话题同样是今年两会焦点。中国前驻法大使卢沙野刚从安检口进入大厅就被媒体围住。记者问到关税问题对中国外贸的影响，卢沙野简短回道：“其实我觉得现在中国的处境是非常主动的。”
中国驻日大使吴江浩的出现也引起媒体骚动。面对中日关系的追问，他微笑回应：“开完会再说。”   
中国前常驻联合国代表张军回应伊朗问题时说，当前局势下，最重要的是立即停火，恢复谈判，政治解决问题。
