乌军袭击迫使关键水道关闭 俄面临“霍尔木兹海峡式”危机
俄罗斯对其通往黑海门户的控制力正在减弱。本周，乌克兰发动的一系列袭击迫使莫斯科暂停了这一关键水域的航运，从而限制了克里姆林宫与世界其他地区进行贸易的能力。
美国有线限电视新闻网（CNN）报道指，这一事态发展标志着俄罗斯处境的重大逆转。多年来，亚速海（Azov）一直处于基辅的控制范围之外，不仅为俄罗斯提供了攻击乌克兰的便利集结地，还将俄南部广阔地区与世界大洋连接起来。
然而，乌克兰无人机技术的进步改变了这一局面。乌克兰无人机部队指挥官布罗夫迪（Robert Brovdi）周三告诉CNN，基辅在过去短短九天内就打击了116艘位于亚速海的俄罗斯船只，这一数字急剧增长。
此前，针对俄罗斯船只的打击行动主要局限于“影子船队”和军用舰艇。尽管CNN无法独立核实布罗夫迪的说法，但乌克兰官员分享的近期部分打击行动的视频显示，这些攻击均直接命中了目标。
这种持续不断的攻击迫使俄罗斯暂停了亚速海的航运，并关闭了该海域两端的关键通道。
黑海和亚速海是通往俄罗斯和乌克兰的关键水道。黑海北接乌克兰，东北方向则是俄罗斯，是石油和其他商品的重要贸易航道。亚速海则通过刻赤海峡（Kerch Strait）与黑海衔接。
乌克兰在打击俄罗斯运输受制裁燃料的“影子油轮”船队方面成效日益显著，但封锁亚速海可能会产生更广泛的影响，波及小麦或葵花籽油等未受制裁的出口商品。
总部位于美国的冲突监测机构“战争研究所”周二指出，乌克兰在亚速海发起的打击行动标志着，乌克兰努力切断被占领的克里米亚与俄罗斯后勤网络联系、并干扰俄海上航运路线，特别是针对石油产品和谷物的运输行动进入新的阶段。

全台最大虚拟货币商勾结诈骗团体洗钱23亿 主谋判监22年
全台湾最大虚拟货币商币想科技被控勾结诈骗团体洗钱23亿（新台币，下同，9213万新元），有黑帮天道盟背景的主谋施启仁一审被判处22年监禁。
综合TVBS新闻网和ETtoday新闻云等台媒报道，币想科技与诈骗团体勾结，借由全台45家门店对外贩售泰达币（USDT），造成逾1500人受骗。
士林地方法院在审理后，于星期四（7月16日）作出一审判决，认定币想科技负责人施启仁未完成洗钱防制登记，仍非法提供虚拟资产服务，判处一年四个月；另就加重诈欺、洗钱共485罪分别量刑，定应执行22年有期徒刑，并没收犯罪所得4371万8505元，不足部分追征。
起诉书指出，施启仁有天道盟正义会背景，是境外虚拟资产交易所CoinW（币赢）台湾区负责人，杨纪文为亚太区商务总监。因CoinW未完成台湾金管会防制洗钱通报，施启仁等人2023年10月间以1920万元等值61万3223颗泰达币，买下已通过金管会洗钱防制法令遵循声明的币想科技公司。
施启仁等人利用币想科技外壳，与诈骗团体勾结，对外招募加盟股东，短短数月间，在全台开设45家实体门店，对外贩售泰达币。诈骗团体也透过网络假投资、假交友、假征才等手法诱骗民众投资。
检方统计，自2024年1月至2025年4月间，共有1539名被害人依指示前往门店买币，受骗金额达12亿7500万余元，整体洗钱金额更高达23亿余元。
士林地检署2025年8月侦结此案，依加重诈欺、洗钱、发起及指挥犯罪组织，以及未经洗钱防制登记提供虚拟资产服务等罪嫌，起诉施启仁等14人。
检方认为，施启仁矢口否认犯行，犯后态度不佳，建请法院从重量刑，并定应执行有期徒刑25年。
