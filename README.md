日本众院通过修改皇室法 确保皇室男丁不凋零
日本众议院星期五（7月10日）以超过三分之二的压倒性票数，通过了执政党提交的《皇室典范》修正案。这项旨在缓解皇室人口危机、确保皇位稳定传承的重大改革法案，允许皇室收养前皇室的男性后裔以养子身份纳入皇室，并允许皇室女性成员在婚后保留皇室身份。
这是日本政府自1947年颁布现行《皇室典范》以来，针对皇室延续制度所作出的一次重大改革。
面对日本皇室“阴盛阳衰”的现实，如果按照现行《皇室典范》关于“皇位由父系男性继承”的严格规定，目前的皇位继承人仅剩德仁天皇唯一的弟弟文仁亲王，以及文仁的儿子悠仁。
为了增加男性继承人，首相高市早苗上个月提交这项修正案，主张从1947年被盟军废除皇籍的11个旧皇族分支中收养男嗣。修正案规定，养子本人没有皇位继承资格，但如果他们的后代为男性，则拥有继承资格。
关于女性皇族，修正案删除了第12条，即“与天皇及皇族以外人士结婚时失去皇族身份”。不过，对于婚后仍留在皇室的女性皇族，修正案并未写明如何处理其配偶与子女的身份。
据众议院星期五的表决结果，除了占众院三分之二议席的执政党自民党以及执政联盟日本维新党投赞成票外，反对党中的温和派政党如中道改革联合、保守派政党国民民主党和参政党也支持修改。
不过，高市政府在法案中流露出的“重男轻女”倾向，引发在野阵营的强烈不满。他们指出，法案旨在彻底排除出现“女性天皇”或“母系天皇”的可能性。
针对反对声音，内阁秘书长木原稔在会上辩护称：“维持稳定的皇位继承是关乎国家根基的重大考量。现行《皇室典范》第一条就明确规定皇位应由皇室男性后裔继承，这是自古以来男性后裔继承制从未间断、维持这个历史传统的重要体现。“
为确保法案顺利过关，高市内阁也作出弹性调整，表明关于养子继承皇位的具体细节仍有待进一步讨论。
执政党计划于7月14日和15日在参议院特别委员会对法案进行审议，目标是在17日前让法案正式通过。
《每日新闻》的最新民调显示，日本有六成以上受访者不反对出现女性天皇。历史评论员小原俊夫也说：“男性继承皇位的观念在日本历史上虽然悠久，但将继承权授予长子，无论性别，才是最理性的做法，也是皇室血脉断绝风险最小的做法。”

中国多人因捏造散布汛情谣言被拘
中国应急管理部公布多起捏造、散布涉汛情灾情谣言案件，相关人员被依法行政拘留。
中国应急管理部官方公众号星期六（7月11日）发文提醒称，目前，正值一些地方防汛救援关键时期，编造、传播虚假汛情信息极易引发群众恐慌、扰乱公共秩序，严重干扰防汛救援工作大局。
据通报，第一起典型案例是35岁廖姓男子从网络搜集洪涝汛情视频，通过剪辑拼凑二次加工，搭配“杭州建德暴雨过后大量车辆被淹”的虚假文案，并刻意将账号发布位置修改为建德，随后在社交媒体平台对外发布。浙江建德公安接到群众报警反映，固定证据后，建德警方于7月5日将该男子抓获，公安机关已依法对其作出行政拘留七日的处罚。
另一起案例是广西钦州，一韦姓网民编造“7月6日17时钦州全域停电”的不实消息，该消息在各类社交群组内快速传播，一时间引得当地市民人心惶惶，甚至出现盲目囤货抢购等乱象。公安机关对其作出行政拘留7天的处罚。
第三起案例，是湖北大冶市金牛镇一名程姓网民，道听途说“金桥世家小区还有一名九岁小孩没有找到”，未经核实，7月8日12时许在个人抖音账号发布“好惨，还有一名九岁小孩没有找到”的视频，浏览量达8000余次。
通报称，视频经查与事实不符，行为已构成虚构事实扰乱公共秩序，大冶市公安机关依法对其处以行政拘留三日的处罚。

特朗普曾扬言封锁 美加大桥7月27日通车
加拿大和美国之间一座备受期待的新大桥将于7月27日正式通车。美国总统特朗普曾经因为两国之间的贸易摩擦，扬言全面封锁大桥。
据法新社报道，加拿大基础设施部星期六（7月11日）发声明说：“今天，在美国政府的支持下，加拿大和密歇根州已达成一致，戈迪·豪国际大桥（Gordie Howe International Bridge）将于7月27日通车。”
加拿大基础设施部也说，这座大桥“将成为加拿大和美国之间重要的经济纽带，并在未来几十年内创造数十亿美元的经济效益”。
这座耗资47亿美元（约82亿6000万新元）的大桥连接密歇根州底特律和加拿大安大略省温莎市，工程在2018年展开。
加拿大总理卡尼上个月说，加拿大应美国的要求，已无限期推迟大桥的通车日期。
今年2月，特朗普威胁全面阻止这座大桥的建设，坚称美国在大桥建设过程中受到不公平对待，并要求大桥“至少一半”由美国拥有。
根据温莎—底特律大桥管理局发布的情况说明书，这座大桥完全由加拿大出资建设，并将由加拿大和美国密歇根州共同拥有。
大桥以已故的加拿大籍美国国家冰球联盟传奇球星、底特律红翼队球星戈迪·豪的名字命名，象征加拿大和美国之间的团结。

孟加拉洪灾致44人死亡 逾百万人受灾
连日暴雨引发的洪水和山体滑坡已在孟加拉国东南部造成至少44人死亡，百万人受困灾区。
灾害管理部星期六（7月11日）发布消息说，洪水席卷了吉大港、科克斯巴扎尔、班达尔班、朗格马蒂、卡格拉恰里、毛尔维巴扎尔和哈比甘杰这七个地区，导致26万8000户人家受困灾区。
路透社报道，电力中断、道路损毁和通信中断延误救援工作。房屋被洪水淹没使得好些灾民数日无法做饭，另一些居民则在厚厚泥浆覆盖的厨房和起居室中艰难度过。
吉大港居民努鲁尔说：“家里还有积水，我们没法做饭。之前储备的干粮也吃完了，晚上因为停电，我们只能和孩子一起摸黑睡觉。”
成千上万的家庭依靠无须烹饪的扁米、膨化米或饼干等干粮以及紧急救援物资度日。然而，由于洪水冲毁道路和桥梁，救援人员难以将物资送抵一些灾情严重的社区。
救援队争分夺秒地向受灾社区运送援助物资。陆军和海军人员正用船只向偏远社区运送食品、饮用水、药品和其他必需品。
本周的暴雨还引发了科克斯巴扎尔（Cox’s Bazar）罗兴亚难民营区的山体滑坡，造成16名难民死亡，其中包括妇女和儿童。100多万罗兴亚难民居住在这些搭建在陡峭、植被稀疏的山坡上的临时住所中，在季风季节生活尤其危险。
孟加拉的季节性季风降雨经常引发洪水、河岸侵蚀和山体滑坡。气象部门说，受孟加拉湾低压和活跃季风影响，孟东南部分地区近期出现强降雨。

美国民主党众议员访西岸 遭以色列定居者持枪挟持
美国民主党联邦众议员卡纳本周访问约旦河西岸期间一度遭以色列定居者挟持，军队到场后，他的车子才得以继续通行。
卡纳（Ro Khanna）说，此次访问让他亲眼目睹以色列占领造成的人道主义灾难。他说，这些定居者持有美国制造的步枪。
