台湾执政的民进党立委林楚茵回锅担任党部发言人。兼任党主席的总统赖清德说，期待林楚茵与其他三名发言人，共同襄助秘书长徐国勇的党务推动、政策宣传、社会沟通跟媒体服务等工作。
综合中央广播电台网和《自由时报》报道，民进党星期三（3月11日）召开中常会，赖清德在会中宣布，现任民进党不分区立委林楚茵接任党部发言人。
赖清德说，林楚茵过去曾任新闻主播、政论节目主持人，对新闻舆情以及媒体运作十分熟悉，加上两届立委的历练，也熟悉政策沟通。
他表示，期待林楚茵未来与吴峥、韩莹、李坤城等三名发言人，共同襄助徐国勇的党务推动、政策宣传、社会沟通跟媒体服务等工作。
林楚茵2023年曾担任民进党发言人，协助备战2024年总统选举。
民进党中央党部发言团队因卓冠廷、戴玮姗投入新北市议员连任选战请辞，一度少了两人，去年12月由另一名民进党立委李坤城递补。

北京批日本拟部署远程导弹：新型军国主义成现实威胁
日本披露将在3月底前于靠近中国的西南地区部署远程反击导弹后，中国国防部批评此举表明日本“新型军国主义”已从危险苗头演变为现实威胁，并警告日本若胆敢武力侵犯中国主权安全，必将遭受更加彻底的失败。
据法新社报道，日本内阁官房长官木原稔星期一（3月9日）在记者会上说，陆上自卫队计划把射程约1000公里的12式地对舰导弹部署在九州熊本县。
据“国防部发布”微信公众号消息，中国国防部新闻发言人蒋斌星期三（3月11日）在记者会上就此事说，一段时间以来，日本右翼势力加速推进“再军事化”动作频频，推动修改“和平宪法”，加快修订“安保三文件”，妄图修改“无核三原则”。
他并指，如今日本明确部署射程远超本土范围的远程进攻性武器，彻底撕下“专守防卫”“被动防御”和“自卫”的伪装，充分证明日本“新型军国主义”已不只是危险苗头，而是赤裸裸的现实威胁，是对地区和平安全的严重破坏。
蒋斌说：“我们正告日方，穷兵黩武的老路是自取灭亡的不归路。若日方胆敢武力侵犯中方主权安全，必将遭受迎头痛击和更加彻底的失败。”

人民日报评“龙虾热”：要拥抱新事物也要推动监管创新
全球人工智能（AI）智能体新宠OpenClaw（俗称“龙虾”）在中国走红后，中共中央机关报《人民日报》发文称，面对新生事物，既要敢于拥抱，也要推动监管同步创新。
《人民日报》星期三（3月11日）刊文称，海外智能体“龙虾”近期火遍全网，多地发文宣布下场“养龙虾”，掀起试用、创业跟进的热潮。同时，中国国家互联网应急中心、工信部等及时发声提示风险，呼吁审慎使用。
文章认为，这股热潮看似突然，实则是中国创新生态长期积累的结果。一款海外智能体能在中国迅速落地并商业化，反映出中国是全球最具活力的“创新实验场”。
文章并称，“龙虾”走红的同时，风险提示与安全预警也随之而来。面对快速迭代的新技术新应用，公众在追求新潮的同时也应提高安全意识，保护好自身信息安全。政府则应统筹发展和安全，坚持审慎包容监管，既为创新留出试错空间，也牢牢守住安全底线，确保创新不走偏、不脱轨。
文章指出，未来创新竞争不仅取决于技术突破，也取决于在安全前提下实现可持续创新的能力。“中国既有开放包容的创新胸怀，又有完备扎实的生态支撑，更有守牢底线的治理智慧，完全能在创新浪潮中抢占先机、行稳致远，让新技术真正造福社会、赋能发展，在安全轨道上保持全球引领优势。”
由奥地利工程师施泰因贝格尔（Peter Steinberger）创建的OpenClaw今年1月对外开源，被形容为“真正能干活的AI”，主打不只能对话，还可替用户执行任务，包括收发邮件、管理日程、预订机票、运营社媒等。
