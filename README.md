位於菲律賓以東海域的潛在風暴預料較大機會週末在本港以東登陸，但其與本港的距離須視乎中心整合位置和速度，以及季風槽和副熱帶高壓脊的互動。
潛在風暴過去一兩日發展速度較慢，尚未建立和鞏固中心。人工智能模式預測潛在風暴短期內中心整合速度較快，繼而較早進入南海，預測登陸位置亦略為偏東。部分傳統電腦模式則預測潛在風暴仍須一段時間整合中心，因此起跑時間較遲，預測登陸位置亦略為偏西。
人工智能和傳統電腦模式對風暴初期中心整合的預測有別，在去年數個案例亦有出現，但現時未有足夠數據判斷誰優誰劣。
至於潛在風暴進入南海後的路徑北份量，則須視乎副高往南伸展的幅度。早前傳統電腦模式預測季風槽有殘餘渦度，因而阻礙副高南伸。然而，傳統電腦模式過去一兩日有加大副高南伸幅度，以致潛在風暴預測路徑往東修正的趨勢。此外，新一輪西風波動會在週末通過中國東北部，亦令副高短期內較難進一步往西伸展。因此，現階段預測潛在風暴東登的機會較大。
東登風暴須較接近本港，以及登陸後採取較為偏西的路徑，才會對本港造成顯著的風雨影響。相信潛在風暴中心完成整合後，本港週末風雨預測會更為明朗。

日本氣象廳對菲律賓以東熱帶低氣壓發出烈風警報，預料該系統24小時內增強為熱帶風暴。
若該系統獲日本氣象廳升格為熱帶風暴，預料將命名為「紅霞」。紅霞是朝鮮提供的名字，顧名思義是指紅色的天空。

天文台表示，會在明日稍後考慮發出一號戒備信號。
天文台會視乎熱帶氣旋的強度變化、其強風區與本港的距離及本地風力情況，評估是否需要在星期六稍後改發更高熱帶氣旋警告信號。
天文台指，現時評估風暴較大機會在惠州至汕頭一帶登陸，但與本港的距離仍存在變數。

---

預料本港週六驟雨增多，局部地區或受強雷雨區相關的雷暴和強陣風影響，加上會有湧浪，原定週末有戶外或水上活動的讀者須留意天氣帶來的潛在影響。
菲律賓以東熱帶低氣壓會在週五晚間至週六凌晨進入南海，隨後靠近廣東東部沿岸。由於副熱帶高壓脊未穩定控制珠江口一帶，加上受風暴外圍環流和日間升溫影響，本港附近週六強對流風險有所增加。然而，此類強對流的發展時間、位置和覆蓋範圍均較隨機。
現時預測風暴週日凌晨登陸廣東東部沿岸，但週六至週日的實際風雨程度，仍須視乎風暴登陸位置。只要東登風暴與本港距離稍有差異，本港風雨程度已足以有較大變化。MET WARN 會繼續跟蹤此風暴的動向。

【非官方預測：MET WARN 最新評估】
MET WARN 目前評估，天文台就熱帶氣旋紅霞發出三號強風信號的機會為「中等至高」。
【預測：紅霞相關的強風區靠近珠江口以東】
傳統模式及人工智能預報過去一日將預測路徑稍微往西修正。按照現時預測，紅霞會以颱風級或以上的強度，在本港以東200公里內的廣東東部沿岸登陸。
預計紅霞相關的強風區會在星期六稍後靠近珠江口以東一帶，屆時本港西北風會逐步增強；風暴登陸後，本港會轉吹更為當風的西南風，潛在強風威脅會在週日大部分時間維持。
【變數：未能完全排除較遠距離東登】
紅霞轉向西北移動的位置及角度，會影響風暴實際與本港的距離。基於現階段集成預報，未能完全排除紅霞以更遠距離東登的可能。在此情境下，紅霞相關的強風區未必能持續影響本港。
在本港東面登陸的風暴，初期帶來的西北風較受地形屏蔽；轉吹西南風時的風力增強幅度，則取決於雨帶發展和低空風力。因此，東登風暴路徑的細微變化，會對本港最終風雨程度有顯著影響。
註：「中等至高」代表基於現有資料，MET WARN 評估天文台較可能發出相關信號，但有一定機會出現「唔掛波」的例外情況。我們會視乎最新資料，判斷須否更新風球概率評估。

俄罗斯国家杜马7月22日通过一项由政府提出的法案，自2027年1月1日起，将建立覆盖商品流通全链条的价格监测机制，对纳入俄罗斯联邦国家统计局消费者价格指数核算范围的商品和服务开展价格监测，覆盖从生产商或进口商到零售终端的各个环节，以分析价格上涨的动态及成因。据《俄罗斯报》报道，首批监测对象可能包括肉类、蔬菜、牛奶、谷物等社会重要商品，具体名单将由政府确定。
根据法案，俄罗斯政府将指定有权获取数据的部门，由俄罗斯联邦税务局提供企业销售量、收入、支出和利润等经营数据，信息来源包括纳税申报表、财务报表及税控收银设备记录。相关部门将在法定权限内利用这些数据分析价格走势及形成因素，除《俄罗斯联邦税法》规定的情形外，相关商业数据不得公开。
对于该法案，支持者认为，新机制有助于监管部门利用交易数据更准确分析价格上涨环节及其原因；部分学者和零售业专家则担忧，扩大政府获取商业交易数据的范围可能增加信息泄露风险，而物流、租金、人力等成本并不能完全从税务数据中反映，因此该机制对降低商品价格的实际效果仍有待观察。

公司说，它位于圣彼得堡（Saint Petersburg）的两处物流设施以及位于莫斯科吞并的克里米亚半岛辛菲罗波尔（Simferopol）的一处物流设施遭到袭击。
公司首席执行官塔季扬娜·金（Tatyana Kim）在社交媒体上说：“所有仓库人员都已迅速疏散……根据初步数据，没有人员伤亡。”
周边列宁格勒州州长亚历山大·德罗兹坚科（Alexander Drozdenko）说，此次袭击造成三人受伤，至少有59架无人机射向圣彼得堡。
近几周来，乌克兰加大了对俄罗斯能源、军事和后勤设施的远程无人机袭击力度，此次袭击的目标是圣彼得堡。乌克兰称，这些袭击是对莫斯科夜间以无人机和导弹袭击其城市的正当报复。
俄罗斯国防部说，其部队一夜之间在十几个地区和克里米亚半岛（Crimea，俄罗斯于2014年吞并的黑海半岛）击落了约571架无人机，这是战争期间规模最大的单夜袭击之一。

泰国查假亲子登记案 调查曼谷1483名儿童国籍记录
泰国警方逮捕一对中国籍夫妇和两名泰国男子，调查涉嫌通过虚假亲子登记为外国籍儿童非法取得泰国国籍的案件，并进一步审查曼谷涉及1483名儿童的出生记录。
泰国《民族报》报道，警方星期四（7月23日）突击搜查两处地点，包括曼谷孟斋区一栋豪宅，逮捕四名嫌疑人。
警方说，这对中国夫妇是两名儿童的亲生父母，两名泰国男子则涉嫌冒充孩子的父亲。两名儿童持有两国护照。
四人被控共同向官员作虚假陈述、使用虚假官方文件，以及篡改民事登记资料以非法取得泰国国籍。警方说，四人接受调查时承认所涉罪行。
泰国地方行政当局此前在民事登记数据库发现异常并展开调查，特别是一些出生记录显示母亲为中国籍、父亲为泰国籍的情况。进一步调查发现，部分泰国男子被登记为父亲，但孩子的亲生父亲实际为没有泰国国籍的外国人。
泰国国家警察副总长森兰说，当局已锁定曼谷1483名儿童的相关记录进一步审查，目前确认两宗存在违规，导致这次四人被捕。警方将逐一检查其余记录，如证实通过欺诈取得国籍，将予以撤销。
当局也将检查全国超过470家私立医院，以调查医院是否有人协助伪造出生和亲子资料。泰国卫生服务支持厅说，涉案医护人员可能面对刑事、纪律及专业处分。
