华盛顿，2月28日（美联社）—— 美国总统唐纳德·特朗普在其社交平台Truth Social发布的一段约8分钟视频中宣布，美军已在伊朗展开“重大作战行动”。
他称此举旨在消除来自伊朗政权的“迫在眉睫”威胁，阻止其发展核武与可达美国本土的远程导弹，并摧毁伊朗的导弹工业与海军力量，切断并削弱其在黎巴嫩、也门、叙利亚和伊拉克扶植的武装代理人。
特朗普在发言中指责伊朗长期支持恐怖主义，列举1979年占领美驻德黑兰大使馆、1983年贝鲁特海军营炸弹案、2000年疑涉袭击“科尔”号等历史案列，并称去年6月“午夜铁锤行动”摧毁了德奥堡、纳坦兹和伊斯法罕的核设施。他重申“伊朗绝不能拥有核武器”。
他向伊朗革命卫队、武装部队与警察发出最后通牒：放下武器可获完全豁免，否则将面临“必死无疑”。并呼吁伊朗民众待在家中、避免外出——“炸弹将无处不在”——并在战事结束后接管政府。
特朗普警告此次行动可能造成美方伤亡，但称这是为保卫未来与美国利益所必需。他同时赞扬美军实力并为参战官兵祈祷，最后以“愿上帝保佑美国军人和美国”为结语。
