停工逾两年 中国阿根廷联合承建水电项目重启
阿根廷经济部星期四（3月5日）宣布，在停工两年后，由中国和阿根廷企业联合承建、位于圣克鲁斯省的塞佩尔尼克（Jorge Cepernic）水电站项目将恢复施工。
据阿根廷媒体《Buenos Aires Herald》等报道，阿根廷经济部长卡普托（Luis Caputo）当天披露，这一项目有望在2030年前完工，并预计这将为阿根廷国家互联电力系统提供1860吉瓦时（GWh）的发电量。
塞佩尔尼克水电项目的融资协议在2014年签署，由中国葛洲坝集团与阿根廷企业组成的联营体负责建设。
不过，由于历届阿根廷政府遗留下来的财务与技术问题，这一项目自2023年底起陷入停滞。据估计，塞佩尔尼克水电站工程已完成超过40%，而配套建设的基什内尔（Néstor Kirchner）水电站约完成五分之一。
今年1月，圣克鲁斯省政府宣布将从中国获得1.36亿美元（1.73亿新元）贷款，用于推进工程，但双方仍需就新的技术和经济条件达成一致。
阿根廷经济部在声明中称，这一工程最早于2013年启动招标，原计划于2023年完工，但“长期受到违约、未能根据成本上涨调整价格以及大量投诉等问题困扰”。
声明还说，要完成塞佩尔尼克水电站及基什内尔水电站，仍需约50亿美元投资。
圣克鲁斯省省长比达尔（Claudio Vidal）在社交平台X发文称，这一项目是“国家最重要的工程之一”。他透露，目前已有150名工人正在接受体检，准备在未来几天加入施工队伍。“这是一个能够创造真正就业、带动经济活动，并为当地民众带来新机遇的项目。”
