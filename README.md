俄阿穆尔州跨西伯利亚铁路立交桥将对重型车收费
据俄罗斯阿穆尔州交通与道路建设部消息，自7月1日起，该州新布列伊斯基镇跨西伯利亚大铁路立交桥将对重型卡车收取通行费，所收资金将用于偿还贷款及设施维护。
该立交桥长107米、引道总长3公里，于2025年12月通车，工程自2023年起开始建设。该桥距离黑河-布拉戈维申斯克新大桥180公里，是中俄贸易额增长过程中保障来自中国的货运的基础设施之一。该桥彻底消除了公路与铁路的平交道口，此前在西伯利亚大铁路通行能力不断增加的情况下，这种平交曾导致车辆延误数小时。
消息说：“单次通行费为250卢布，仅针对三轴及以上且高度超过2.6米的货运车辆收取，其中包括挂车。该设施是根据特许协议利用借贷资金建设的，收取的通行费将用于偿还所有借款并支付日常运营费用。”
阿穆尔州交通部指出，大型车辆收费系统计划于7月1日启用，目前不停车收费系统的测试工作已进入尾声。在车辆驶近该设施时，系统将自动识别并记录车牌号、车辆尺寸及车轴数量。
乘用车、巴士以及双轴货车不收费。

世界银行计划2031年前逐步停止向中国提供贷款
据世界银行周二提交理事会的方案，该行拟于2031年前停止向中国提供贷款。
据西方媒体援引一位消息人士的话报道，世行董事会将于7月20日当周审议该计划，尽管无需进行正式投票。该计划由世行与中国共同商定，是双方五年期“国别伙伴框架”的一部分。
这一调整意味着从现在起到2031年，这家多边开发银行对华贷款总额将被限制在20亿美元以内，此后将彻底终止放贷。
世行对华贷款规模已呈稳步下降趋势，从2017年的每年24亿美元降至2025年的7.5亿美元。早在2000年，中国就已不再符合世行旗下面向最贫困国家的国际开发协会（IDA）的贷款资格。中国自2007年起开始向该协会捐资，目前已是第五大捐助国。
西方媒体援引一位熟知内情的世行官员表示：“过去几十年间，中国在发展方面取得了显著成就。现在我们正步入双边关系的全新阶段，这也反映了当前的现实。”
鉴于中国经济实力不断增强，美国及其他国家长期以来一直敦促世行停止对华贷款。自特朗普首个任期以来，中国继续从世行及其他多边机构借款就一直是令美方感到恼火的问题。
美国财政部一位发言人称赞此举是“迈出了正确的一步”，并表示华盛顿期待其他多边机构效仿跟进。该发言人强调：“作为全球第二大经济体，中国不应再接受多边机构的援助。”
一位美国高级官员指出，鉴于中国庞大的经济体量，其不应再具备获得发展融资的资格，并呼吁亚洲开发银行、国际农业发展基金及联合国驻华机构等其他多边机构也停止对华援助。

据路透社报道，由标普全球编制、星期三（7月1日）发布的瑞霆狗（RatingDog）中国通用制造业采购经理人指数（PMI），从5月的51.8微降至6月的51.7，但仍高于分析师预期的51.6。PMI高于50意味着制造业活动扩张，低于50则代表收缩。

瑞霆狗创始人姚煜说，“总体而言，受新订单持续增长、成本压力缓解以及劳动力市场状况改善支撑，6月的制造业保持稳步扩张。”

经济学人智库高级经济学家徐天辰受访时说，6月制造业PMI回升，主要是“抢出口”，一些企业受美国关税政策不确定性影响，为规避风险提前出口。他认为，在AI需求驱动下，中国仍将维持强势出口。
根据瑞霆狗调查，产出连续第七个月增长，但增速放缓至三个月来的最低水平。
就业人数三个月来首次增长，新增就业规模创2023年8月以来最高水平。积压业务量也连续第五个月增加，显示尽管企业增加了人手，但仍面临不断增加的工作量。
调查也显示，整体新订单连续第13个月增长，追平了2018年以来最长的连续扩张纪录。不过，新出口业务连续第二个月下滑。出口价格继续上涨，但涨幅为自3月以来最低。

伊朗战争重塑中国石化贸易格局 助中企扩大出口
伊朗战争扰乱波斯湾油气供应后，中国生产商得以扩大出口，在一定程度上缓解塑料、橡胶和纺织等基础化工原料长期供过于求的问题。
据彭博社报道，由于波斯湾油气田原料供应减少，越南、印尼等亚洲国家的买家纷纷转向中国采购。相比依赖中东供应的生产国，中国不仅拥有庞大库存，也具备更多元的原料来源。
报道称，如果这些新建立的贸易渠道在冲突结束后得以延续，将为深受产能过剩困扰的中国化工行业提供重要出口市场。
市场研究机构ICIS基础化工与原料首席分析师指出，战争迫使许多聚烯烃等石化产品进口商转向中国供应商，也让中国石化企业得以建立覆盖亚洲市场的供应链网络。
聚乙烯等聚烯烃是塑料产业的重要基础原料。原油和天然气经过裂解形成烯烃后，再加工成食品包装、垃圾袋和玩具等各类塑料制品。
尽管霍尔木兹海峡航运受阻同样对中国市场造成影响，但ICIS指出，中国可通过动用库存、扩大煤制化工生产，以及增加美国乙烷进口来填补供应缺口。乙烷是天然气衍生产品，也是生产烯烃的重要原料。
ICIS数据显示，中国今年4月和5月首次由烯烃净进口国转为净出口国，相关贸易流向变化规模相当于全球需求的约10%；下游化工产品市场也出现类似波动。
分析认为，即使未来美伊关系缓和，贸易格局也未必完全恢复原状。霍尔木兹海峡作为全球能源运输关键通道的脆弱性已暴露无遗，促使贸易商加快分散供应来源。同时，在经济增速放缓背景下，中国近年来形成的大规模石化产能也迫切需要开拓海外市场消化。
ICIS预计，中国乙烯年产能将从2025年的约6000万吨增至2030年前后的8000万吨，占全球总产能近三分之一。

习近平：坚决打击台独势力 反对外部干涉
中共总书记习近平强调，要坚决打击“台独”分裂势力，反对外部势力干涉。
习近平星期三（7月1日）在庆祝中国共产党成立105周年大会上发表讲话。他在谈及台湾问题时说，解决台湾问题、实现中国完全统一，是中共矢志不渝的历史任务，是全体中华儿女的共同愿望。
习近平要求深入贯彻新时代党解决台湾问题的总体方略，坚持一个中国原则和“九二共识”，团结广大台湾同胞，深化两岸交流合作和融合发展，坚决打击“台独”分裂势力，反对外部势力干涉，坚定推进祖国统一大业。
习近平还说，促进香港、澳门长期繁荣稳定，是中华民族伟大复兴的内在要求。新征程上，要全面准确、坚定不移贯彻“一国两制”、“港人治港”、“澳人治澳”、高度自治的方针，落实“爱国者治港”、“爱国者治澳”原则，提升港澳依法治理效能，促进港澳经济社会发展，支持港澳更好融入和服务国家发展大局。
