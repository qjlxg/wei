泽连斯基：不希望因伊朗战争失去美国支持
乌克兰总统泽连斯基说，乌克兰不希望因美伊战争而失去美国对乌克兰的支持。
综合法新社和路透社报道，泽连斯基星期六（3月14日）告诉媒体说：“我们不想失去美国人（的支持）”，尽管“毫无疑问，他们目前正忙于处理中东事务”。
他补充说，乌克兰通过分享本国在无人机领域的专业经验，“表明我们愿意帮助美国及其在中东的盟友”，并强烈希望美国不会因为中东局势，而对俄乌战争问题置之不理。
泽连斯基也表明，乌克兰已向中东地区四个国家派遣专家，希望借此帮助中东国家防御伊朗“神风”（Kamikaze）无人机队，以换取金援和技术支持。　　
泽连斯基透露，乌方向中东派出的每支团队都由数十人组成，这些人员将能够开展专业评估，并演示无人机防御系统应如何运作。

解放军海陆空火箭军演练登岛抢滩
隶属解放军南海舰队的中国首艘075型两栖攻击舰“海南号”，日前在某海域展开多军兵种联合演练。军事专家指出，联合演练由空军部队、海军部队、陆军部队及火箭军部队共同执行登岛登陆任务，信息化的程度和协调配合的能力要非常强，一定要做到万无一失。
中国官媒央视新闻星期天（3月15日）发布消息，指海南号两栖攻击舰编队日前在某海域开展一场敌情威胁条件下、多军兵种立体投送演练。在联合作战大体系、信息、火力支撑下，编队成功将海、陆等兵力投送上岸。
报道引述军事专家曹卫东介绍，联合作战或联合演训难点很多，有空军部队、海军部队、陆军部队、火箭军部队，多个部队在执行同一个任务，即登岛、登陆任务。
在此次演练中，空军通过空中的对陆打击，协助完成这次任务。陆军乘坐两栖攻击舰，从海上航渡到岸边，抢占滩头阵地进行进攻。
火箭军的导弹并从陆地上，对陆或对海进行打击。海军则派出水下潜艇、水面舰艇，还有海军陆战队。
曹卫东说，多个军兵种、多个武器装备，要像10个指头弹钢琴一样协调，大家必须有一个统一的对战场态势的感知，敌我的区分，还有对重点目标的打击。他指出，信息化的程度和协调配合的能力要非常强，一定要做到万无一失。
曹卫东强调，075型两栖攻击舰是重中之重，既要和其他的军兵种进行协调，还要保证把登岛、登陆的部队和装备在尽量短的时间里，快速、大量运送上去。在这个过程中，战场态势是千变万化的，这是对075型两栖攻击舰或两栖作战部队最大的考验。

中美巴黎经贸磋商开始举行
中美高级官员星期天（3月15日）在法国巴黎开始为期两天经贸磋商，预计为三周后两国元首北京峰会铺路。
美国财政部长贝森特、贸易代表格里尔和中国副总理何立峰，星期天至星期一（16日）将在巴黎举行第六轮经贸磋商。
据香港《南华早报》报道，第六轮经贸磋商已开始，中美代表团分别由何立峰、贝森特率领，双方巴黎当地时间星期天抵达经济合作与发展组织（OECD）总部展开磋商。
中国官媒央视、新华社等也报道了，中美经贸磋商当地时间星期天上午开始举行的消息。　
新华社星期天发表题为“维护中美经贸关系健康、稳定、可持续发展”的评论文章，指新一轮经贸磋商是机遇也是考验，呼吁双方应算大账，多看合作带来的长远利益，拉长合作清单、压缩问题清单，争取更多积极进展，为中美关系打开新的合作空间。

特朗普要求多国派军舰 韩国回应将密切磋商
美国总统特朗普要求韩国等国向霍尔木兹海峡派遣军舰， 韩国总统府官员对此回应说，政府将同美方密切沟通，并慎重考虑后作出决定。
法新社报道，韩国总统府青瓦台官员星期天（3月15日）说：“我们正在密切关注特朗普总统在社交媒体上的言论，并将与美国方面密切磋商，慎重考虑此事。”
韩联社引述青瓦台相关人士报道指出，国际海上通道的安全和航行自由符合全体国家的利益，是受国际法保护的对象。因此，希望全球海运通道能早日恢复畅通。政府密切关注中东局势和有关国家动向，为确保韩国公民和能源运输线安全考虑多重方案。
美方尚未正式提出派兵要求。据分析，青瓦台将在今后一段时间了解美方具体意图和动向，同时关注中国、法国、日本、英国等被特朗普点名国家的动向，认真研究应对措施。
特朗普星期六（14日）在社媒平台发文说，希望受霍尔木兹海峡封锁影响的中国、法国、日本、韩国、英国及其他国家能派遣军舰到该地区。

IDC：中国产AI眼镜今年出货量预计占全球市场45%
中国国产人工智能（AI）眼镜销量持续走高，国际数据公司（IDC）预计，今年出货量将同比增长56%，在全球市场中的占比达到45%。
《科创板日报》星期天（3月15日）消息，在近日于上海举办的中国家电及消费电子博览会（AWE）上，国产AI眼镜成为一大焦点。
多名AI眼镜产业链人士表示，2025年以来，国产AI眼镜的销量持续高速增长，在全球市场中的占比逐步提升。在AWE上，中国科技巨头阿里巴巴旗下千问AI眼镜等产品迎来中国首秀。
IDC预计，2026年全球中国产AI眼镜市场出货量将达到2267万1000台，同比增长56.3%；其中，中国市场出货量预计为450万8000台，同比增长77.7%。增长主要集中在音频、音频拍摄眼镜及AR/ER眼镜领域。
中国厂商在全球智能眼镜市场的出货占比稳步提升，IDC预计2026年中国厂商出货量将占全球市场的45%，中国科技创新厂商在北美、欧洲和亚太地区表现活跃。
不过业内人士坦言，中国国产AI眼镜仍需在硬件轻量化、世界模型、生态建设等方面持续突破，属于这个行业的“iPhone时刻”，才会到来。
