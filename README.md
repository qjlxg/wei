张雪在台湾《联合报》星期六（7月11日）刊发的专访中说，他非常想在台湾开店，现在正在对接中，最晚不超过明年，“开幕时我一定要亲自到场”。
对于赴台开店，张雪说，“想要让门店真正站稳脚跟，车型的丰富度是关键，店里不能只有单一品类，得有复古车、跑车，也有踏板车，让不同喜好的车友都能找到自己的选择，这样门店的经营才能更稳。”
张雪承诺，未来台湾的车主将和大陆车主享受完全一样的售后保障服务，还会给大家配套赠送贴膜、车壳等周边福利，也随时邀请台湾的车友到大陆参观工厂。
另据中新社报道，张雪近日在重庆接受媒体访问时透露，将到台湾设门店的计划，并称跟有意向的台商对接代理事宜，各项筹备工作有序推进，初步计划“最晚不超过2027年”。

张雪介绍，两岸在机车产业上的合作早已开始。以张雪机车为例，部分车型轮胎是与台湾正新轮胎联合研发的成果，“我们一起攻关、优化测试、共同成长”，后期协同研发将持续深化。
张雪表示，若有机会赴台开展行业交流，正新轮胎总部将是他首站打卡地，同时也期待与台湾本土机车企业互通产业发展经验。

广西一动物园洪灾锁兽笼致狮子溺亡 PETA：不可接受
在台风“美莎克”引发严重洪涝期间，中国广西一家动物园为防止猛兽逃脱，将狮子等动物锁在笼中，最终导致部分动物被淹死。国际动物权益组织善待动物组织（PETA）批评这一做法“不可接受”，并呼吁动物园建立完善的灾害应对机制。
广西上星期六（7月4日）以来遭遇罕见的大范围强降雨，‌南宁、防城港、钦州、崇左、北海等地出现持续暴雨，境内多个水库发生险情。
据《红星新闻》报道，贵港动物园在洪水来袭前，为防止园内猛兽逃出伤人，将狮子、熊和狼等动物锁在笼中。动物园负责人殷飞飞星期三（7月8日）说：“洪水到来前，我们把猛兽区的笼门给锁死了。我们不能在洪水到来时再给国家添乱，让猛兽跑出去伤人。”
报道称，洪灾期间园区积水最深超过两米。洪水退去后，三只狮子死亡，另有100多只动物失踪。
根据动物园在社交媒体发布的寻物启事，失踪动物包括两匹斑马、三匹迷你马，以及鸵鸟、羊驼、浣熊和孔雀等。园方呼吁公众协助寻找和捕回被洪水冲走的动物。
中国官媒《环球时报》随后报道，其中一匹失踪斑马已被发现死亡；被锁在笼中的棕熊和狼虽然幸存，但因险些被洪水淹没，身体状况不佳。
PETA亚洲地区总裁贝克（Jason Baker）星期五（7月10日）在声明中说，贵港动物园的悲剧应为所有位于极端天气风险地区的动物园和野生动物圈养设施敲响警钟。
他说，在洪水上涨时让动物困在笼中“不可接受”；但另一方面，在灾害发生时直接放生圈养野生动物，同样是不负责任且危险的做法，既可能危及动物，也可能威胁公众安全。
贝克呼吁动物园建立完善的动物疏散和应急预案，并最终停止圈养野生动物。

试管胚胎样本错配案 港警“一哥”：暂未有人被捕
提供人工受孕服务的香港希愈生殖医学中心，因涉及错配试管胚胎活检组织样本，接受官方调查。香港警务处处长周一鸣出席活动后受访时说，警方正在调查此案是否涉及刑事成分，因此暂未逮捕任何人。
综合香港《明报》和网媒“香港01”报道，周一鸣星期六（7月11日）在出席香港警察学院结业会操后受访时说，警方在星期一（6日）收到相关部门转介上述案件，已交由中西区重案组调查。
他介绍，警方并已即时开始调查工作，包括到涉事中心了解整个运作流程、与负责人及其他员工等会面。
周一鸣称，警方需要理解案件是否涉及刑事成分或是人为失误，目前未发现有刑事成份，以及所违反的相关法例，因此暂未逮捕任何人。
据香港政府新闻网消息，香港人类生殖科技管理局（简称生殖局）星期二（7日）公布，该局6月17日分别接获涉事的希愈生殖医学中心，以及为其提供化验服务的香港中文大学威尔斯亲王医院植入前遗传学诊断中心通知，指希愈生殖送交诊断中心进行基因检测的胚胎活组织样本中，有样本怀疑被其他胚胎活检组织样本替代，诊断中心在基因检测过程中及时发现错误。
生殖局随即成立委员会调查事件，并在7月3日和星期一（6日）先后将事件通报卫生署和警方。
生殖局视察委员会成员杨树标在会见传媒时说，事件涉及两宗发生在5月的个案，。其中一宗个案送交检测的七个胚胎样本，有六个样本不属于个案中的父母；另一宗个案全部两个胚胎样本，均不属于个案中的父母。
杨树标称，经进一步检测证实，胚胎未被混淆，只是送往化验的胚胎样本出错。目前没有证据显示有任何胚胎被混淆或被错误植入人体。
生殖局主席梁宪孙表示，事件可能涉及人为因素和刑事成分，并强调这次事件严重，会严肃跟进。
希愈生殖医学中学目前已采取措施，加强处所内的处理胚胎和样本的存取控制。
据网媒“香港01”报道，希愈生殖医学中心母公司新风天域集团星期四（9日）在声明中重申，非常重视这次事件，正进行全面内部检讨，也已确保胚胎没有错配。
集团表示，香港人类生殖科技管理局、卫生署及警方已介入调查，会全力配合三个部门的调查工作，并提交一切所需资料。
