位於西北太平洋的熱帶氣旋巴威於過去24小時出現爆發增強，成為今年南海和西北太平洋範圍內，第二個被美國聯合颱風中心評為五級颱風的熱帶氣旋。
衛星雲圖可見，巴威風眼清晰，圍繞風眼的對流旺盛，短期內強度有繼續上升的可能。
巴威將於未來一兩日採取西北偏西路徑移向關島，對當地構成相當威脅。隨後巴威的路徑變化視乎下週中後期副熱帶高壓脊的強弱變化，下週需要前往台灣的讀者需留意巴威可能帶來的天氣影響。

香港以南海域持續有強雷雨帶發展並由南向北移動，先前未受大驟雨及狂風雷雨影響的其他地區亦將會受到影響。
與美沙克外圍雨帶相關的強雷雨帶呈南北走向，並正由南而北影響本港。因此，大驟雨、強陣風及雷暴的風險可能於下午維持一段時間，讀者應保持高度警覺。

雖然暴雨警告已經取消，但市民仍須警覺有關河道氾濫可能帶來的危險。

---

美莎克是今年首個影響本港的熱帶氣旋，期間間中為本港帶來大驟雨及猛烈陣風。
美莎克最終只在本港西南面590公里左右掠過，但在副熱帶高壓脊共同影響下，廣東沿岸的氣壓梯度較為緊密，即使在沒有雨帶影響期間，本港高地、西南部及新界東北離岸亦間中錄得強風。
在強雨帶影響期間，本港風力曾短暫上升，並錄得猛烈陣風。八個風球參考測風站中有三個錄得強風，其中長洲更一度錄得烈風，而西貢僅差一公里便達強風程度（該站其後於強烈季候風信號生效期間錄得七級強風）。除長洲外，位於高地的昂坪及「前哨站」橫瀾島亦錄得烈風，而昂坪的陣風更一度達颶風程度。
雖然技術上是次風暴可視為「八中三」，惟由於風勢並不持續，最終只維持一號戒備信號，情況類似2019年熱帶氣旋白鹿。

---

熱帶氣旋美莎克為本港帶來狂風驟雨後，先後登陸海南島和越南。風暴登陸越南前，一度迅速發展出完整眼壁，其風眼結構甚至在陸地上維持一段時間。
熱帶氣旋進入北部灣後，不時出現快速增強，主因是狹窄的海灣地形可化身為風暴加速器，有助加強地形輻合；此外，北部灣夏季海溫較高，亦為風暴提供更多能量。
美莎克登陸地沒有密集的氣象站觀測網絡，但鄰近廣西近岸有測風站錄得颶風。由於該測風站有一定海拔，數據可否參考，以及美莎克實際增強幅度仍有待氣象機構判斷。

---

強雷雨帶正影響大嶼山，本港以南海域亦有強雷雨區發展並往北移動。
未來數小時本港部分地區有機會受大驟雨、雷暴和強陣風影響。由於強雷雨帶的發展位置有機會往東移動，因此未能排除相關天氣擴展至其他地區的可能。

华为拟秋季发布Mate 90系列 搭载韬定律麒麟晶片
中国科技巨头华为计划于今年9月发布全新旗舰手机Mate 90系列，该系列将首发搭载基于“韬定律”设计的新一代麒麟晶片。
中国《科创板日报》星期一（7月6日）引述知情人士报道这一消息。
知名科技博主“智慧皮卡丘”透露，华为Mate 90系列目前正进行晶片装测。除了搭载新款晶片外，软件也将首发预装鸿蒙7正式版。
华为公司董事、半导体业务部总裁何庭波今年5月首次提出“韬定律”，这也是中国在半导体领域首次提出指导产业发展的新原则。
过去数十年里，全球半导体产业一直遵循摩尔定律，即通过几何微缩增加集成电路上的晶体管数量来提升性能。但随着晶片尺寸逼近物理极限，这一路径正面临技术与经济效益的双重挑战。
面对这一瓶颈，何庭波提出的“韬定律”以系统性降低时间常数（τ）为目标，通过逻辑折叠（Logic Folding）等技术，持续压缩信号传播时延，从而提升晶体管密度。
据何庭波7月3日发表的论文，与去年推出的麒麟9030 Pro晶片相比，即将于今年秋季面世的新款麒麟晶片晶体管密度提升约53.5%。在以往的摩尔定律下，如此幅度的提升通常需要三年的几何微缩才能实现。
科技之家报道，这款最新麒麟晶片的理论性能已与英特尔18A工艺持平，并接近台积电初代的三纳米晶片。
何庭波此前曾透露，基于韬定律，华为目前已设计并量产381款晶片，服务于移动设备、人工智能（AI）、汽车、工业和基础设施等市场。 
但她坦言，将韬定律描述为一个已完成的系统具误导性，目前该路径仍有几个问题待解决，包括工具链与方法论的完善、晶圆间工艺变化等。“前方的路线图要求很高，但方向是明确的。”

路透社报道，在消费者偏好不断变化的背景下，消费品企业面临加快创新与降低成本的压力，因此纷纷推动将AI整合进产品开发流程。欧莱雅（L’Oréal）自四年前开始在实验室使用AI，大众化产品部门总裁梅加班（Fabrice Megarbane）说，公司通过预测分子对皮肤和头发的影响，已为美容产品识别出新的分子。
他指出，公司近期的一项创新，是将原用于护肤产品的分子重新应用于一款含胶原蛋白的洗发水中，使头发更加蓬松有弹性。
梅加班6月底于消费品论坛全球峰会上说：“通过设想新的分子组合及潜在功效，确实可以大幅提升研发速度。”

亿滋国际首席信息与数码官卡塔拉诺（Filippo Catalano）说，在AI辅助下的人类产品创新已成为“改变游戏规则”的关键。
相关技术帮助拥有吉百利（Cadbury）和瑞士三角巧克力（Toblerone）等品牌的公司加快流程并重新构想配方。公司称，AI可以生成包括“跳脱框架”的配方创意，再由人类专家进行评估。
卡塔拉诺说：“你可以优化配方开发方式。”
他指出，这也有助于减少对单一供应来源的依赖，并能根据消费者口味变化调整配方。
他也说，公司的AI工具减少创新过程中通常需要生成的样品数量，并已用于开发无麸质版奥利奥饼干及改良版趣多多（Chips Ahoy）配方。

韩国5月邮轮访客首破20万 中国游客占近半数
得益于中日关系持续恶化，到韩国出游的中国旅客人数增加。官方数据显示，韩国5月邮轮访客首次突破20万，中国游客占近半数。
据韩联社报道，韩国观光公社旗下数据平台星期一（7月6日）发布的统计数据显示，5月乘坐邮轮到韩国的外国人数为20万6542人次，环比增加13.4%，同比增加12.2%，创下历史新高。
去年12月乘邮轮访韩游客为5万6000人次，今年1月略增至5万7000人次，2月翻一倍至11万2000人次，之后持续增加，到5月首次突破20万人次大关。今年1月到5月乘邮轮访韩游客累计达70万9000人次，同比增加5.9%。
按国籍分布来看，中国游客人数最多，达9万3000人，占比45.1%，其后依次为菲律宾、日本、印尼和印度。按年龄来看，30多岁年龄段游客最多，其后依次为20多岁、60多岁、40多岁等。
日本国家旅游局6月17日公布的数据显示，5月赴日中国大陆游客人数同比锐减60.4%，连续六个月低于去年同期。
日本首相高市早苗去年11月在国会发表“台湾有事”言论后，令日中关系降至冰点。北京随之采取反制措施，从官方至民间限缩与日本的交流和互动，也建议中国公民避免赴日旅游。
为促进观光业，韩国政府本月初宣布，将面向中国、越南、印度等六国团体游客实施的免签证费措施延长至今年12月31日。
韩国自去年9月29日起对中国旅行团实施阶段性免签政策，持续至今年6月30日。韩国政府去年3月公布这项试点计划时曾预计，访韩中国游客人数有望突破100万人次，并希望借此提振韩国经济，并改善韩中关系。
