国泰赴英航班与民航管制人员无联系 触发警报机制战机升空
香港国泰航空一架从香港飞往英国伦敦的客机，上星期六（7月4日）在飞越罗马尼亚领空时，因当地民航管制人员未能与客机取得联系，触发北约快速反应警报机制，与罗马尼亚接壤的匈牙利派出战机升空并作目视警告。香港民航处要求航空公司一周内提交调查报告。
罗马尼亚和匈牙利皆为北约成员国。据香港《明报》报道，在客机与地面恢复通讯后，匈牙利战机才撤离。匈牙利国防部长鲁辛-森迪（Ruszin-Szendi Romulusz）当天在社交平台脸书公布这起事件。
国泰星期三（8日）证实，CX257航班曾与途经的航空交通管制部门经历短暂通讯中断，有关当局按国际公认处理程序，作出拦截程序安排，航机通讯这时也恢复正常，并按原定计划前往目的地，机组人员当时立即通报事件。
公司称，该航班一直沿用经批准航道运作，航机或所有机上人员的安全均没有受到任何影响。
根据北约盟军空军司令部网页资料，若任何飞机飞近或进入欧洲北约领空，如未能与管制人员以无线电联络或提交飞行计划，就会触发快速反应警报机制，派拦截机跟进。按机制战机会近飞至客机左边，监察客机驾驶室状况，如仍无法联系或有怀疑，可采取迫降或驱逐等行动。
香港民航处说，航空公司已向民航处呈报，初步资料显示此事与通讯短暂失联有关。民航处高度关注，并按既定程序跟进，要求相关航空公司就此事在一星期内提交调查报告。
《星岛日报》引述业内人士报道，涉事的空中巴士A350-1000是国泰航空最新型号广体客机，设有多项备援的通讯系统，因此失联及需要战机升空拦截的情况十分罕见。

洪水侵袭广西一动物园 三狮溺亡
中国广西遭遇大范围强降雨，洪水侵袭当地一座动物园，三只狮子溺亡。动物园负责人估计，洪水侵袭所造成的损失达400万元（人民币，下同，76万新元）。
据中国新黄河报道，广西贵港动物园星期一（7月6日）凌晨突遭洪水侵袭，园区最高积水达3米多高，笼舍大面积坍塌损毁。园方为避免猛兽出逃伤及群众，提前将狮子、熊类等锁入加固后场。
贵港动物园负责人王女士提供的照片显示，动物园内多处笼舍坍塌，污泥遍地，树木弯折倒地。据她估计，此次洪水侵袭，致动物园损失约400万元。
王女士说：“我们赶紧往外跑，跑了50米，水已经没过腰，我们就爬到了3米高的集装箱上，但很快积水也涨到跟集装箱一般高。”员工的手机证件等已全部冲走，只有一部手机可以与外界联系。“洪水来得太快了，晚一步连人都会被冲走！”
王女士也说，园区共有五名工作人员，当天下午在武警官兵的救援下已脱离危险，目前园区还有狗熊、棕熊、狼、狮子、猴子等几种动物，“损失了三只狮子，一匹斑马，梅花鹿、羊驼、矮马等二十几种草食动物全被冲走了”。
中国多地近期遭遇极端天气，出现洪涝、山体滑坡等灾情，截至星期三（8日），至少有38人遇难。
受台风“美莎克”影响，广西上星期六（4日）以来出现罕见的大范围强降雨，‌南宁、贵港、防城港、钦州等地受灾严重‌。其中，南宁横州市六蓝、云表等多座水库出现漫坝、坝体缺口险情，下游村屯大面积被淹。

中国消费通胀放缓幅度超预期
随着上月伊朗局势缓和，大宗商品成本回落，中国消费通胀增速的放缓幅度超过预期。
根据中国国家统计局星期四（7月9日）发布的数据，6月居民消费价格指数（CPI）同比上涨1.0%，涨幅低于5月的1.2%。彭博社和路透社调查的经济学家此前均预计，6月CPI将同比上涨1.1%。
中国国家统计局的数据也显示，6月工业生产者出厂价格指数（PPI）同比上涨4.1%，涨幅连续第四个月扩大，并创下2022年7月以来最高水平。
彭博社报道认为，中国可能已在上季度摆脱持续三年的经济整体通缩，主要原因是人工智能（AI）投资激增，以及中东冲突引发的油价冲击。
尽管油价、晶片和金属价格上涨，但更广泛的物价回升前景仍存在疑问。由于消费支出疲弱，工厂难以将上涨的成本完全转嫁给消费者，盈利能力因而承压。
作为全球最大的制造业国家，中国价格上涨压力回升，可能引发全球通胀外溢效应。中国出口价格正以2023年初以来最快的速度上涨，扭转了此前连续多年几乎持续下跌的趋势。

中国外交部：欢迎更多美国民众赴华参访交流
中国外交部星期三（7月8日）表示，欢迎更多美国民众赴华参访交流。
中国外交部发言人毛宁星期三主持例行记者会。她在会上回应有关“鼓岭缘”中美青少年棒球友谊赛暨体育交流周的提问时说，鼓岭故事跨越百年，正焕发新的生机，续写出更多友好交往新篇。
“鼓岭缘”中美青少年棒球友谊赛暨体育交流周6月28日至7月2日在福州举行，活动由中国人民对外友好协会、国家体育总局和福建省政府共同主办。中美八支青少年棒球队、“鼓岭之友”、飞虎队友谊学校学生、友城代表等约400人参加。
毛宁说，近期中美青少年体育交流周、飞虎队青年领袖对话会、“跟着熊猫游中国”等交流活动成功举办，充分说明中美两国人民有着共通的情感，完全可以跨越差异、彼此欣赏、建立友谊，充分印证中美关系希望在人民、基础在民间、未来在青年、活力在地方。
毛宁也说：“我们欢迎更多美国民众来华参访交流，实地感知真实、立体、全面的中国，投身中美友好事业。”
另一方面，美国总统特朗普称9月24日前后与中国国家主席习近平会面，时间点恰逢常年联合国大会在纽约举行，而习近平自2012年就任以来只出席过一次联合国大会。
特朗普此前已表明有意到深圳参加亚太经合组织（APEC）峰会。APEC峰会将于11月18日至19日在深圳举行，即美国11月3日国会中期选举结束约两周后。
特朗普5月访问中国时，中美同意将“建设性战略稳定关系”作为两国关系的新定位。
