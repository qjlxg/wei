印度学生放弃“美国梦” 转赴欧洲深造
（金奈/阿姆斯特丹路透电）随着美国收紧移民规定，加上印度与欧盟的贸易协定为人员流动提供了便利，越来越多的印度年轻人放弃追求“美国梦”，改而前往欧洲留学。
路透社引述多名学生和教育顾问报道，来自金奈的22岁工程系毕业生什里尼凯坦就决定打破赴美深造的家族传统，到德国攻读硕士学位。
什里尼凯坦指出，赴德留学的费用远比美国或加拿大低，且德国大学与工业界的联系向来十分紧密。他说：“世界正变得越来越多极化。过去，大家谈论的只有美国。”
教育咨询公司KC Overseas说，欧洲的学费通常比美国低50%到80%，生活费也更低。
据报道，在欧盟寻求年轻人才以填补人口老龄化造成的劳动力短缺的背景下，欧洲正成为印度学生除北美之外的可行目的地。如果这一趋势持续，全球学生流动的格局或将重塑。
今年1月，印度和欧盟签署贸易协定，通过简化签证程序、明确毕业后的就业路线图、提高对印度学历的认可等措施，方便印度学生和技术人才进入欧洲。
与此同时，美国却进一步收紧移民政策，包括缩短外国学生的签证有效期。美国、澳大利亚和英国等主要留学目的地的教育费用也越来越高，从而降低了它们的吸引力。
教育咨询机构欧洲留学中心说，印欧达成贸易协定后，印度人赴欧留学的咨询增加了25%至30%。中心主任潘迪安形容，对那些因其他地区留学费用上涨、签证政策收紧而却步的家庭来说，欧洲已成为“炙手可热的目的地”。
留学平台Leap预计，未来三到五年，前往欧盟留学和工作的印度公民将激增60%至70%。
政府数据显示，目前有超过12万1000名印度公民在欧盟求学，人数虽仅为美国的一半左右，但正在稳步增加。
荷兰统计局数据显示，2025/26学年当地印度学生人数增至约3700人，高于2018/19学年的2630人，使印度成为荷兰第三大非欧盟国际学生来源地。
不过，印度学生对欧洲的兴趣能否转化为持续的入学数字，取决于印欧贸易协定能否顺利执行。
一些赴欧的印度学生发现，他们面对着语言障碍、阿姆斯特丹等大城市住房短缺、就业竞争激烈等挑战。另一些人则对欧洲极右翼政党崛起、部分地区对外国人态度日益强硬等情况感到担忧。
无论如何，印度学生对欧洲的看法已开始改变。考虑赴捷克或意大利留学的工程师苏蒂尔说：“在协定敲定前，我的家人总说欧洲不好。现在他们则改口说，欧洲很好。”
