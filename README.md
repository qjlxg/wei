美国商务部公布的修正数据显示，去年四季度美国实际国内生产总值（GDP）按年率计算增长0.7%，较首次预估值1.4%显著向下修正。
美国商务部星期五（3月13日）发布的数据显示，去年最后三个月的美国经济成长率数据调整，是受到出口、消费者支出、政府支出及投资额均下修的影响，不过进口额萎缩幅度未如原先估算严重。

美国消费者信心降至三个月低点 战争推高油价担忧
美国消费者信心降至三个月低点，因为近期对伊朗战争会影响汽油价格的担忧加剧。
密歇根大学星期五（3月13日）公布的数据显示，3月消费者信心指数初值从2月的56.6降至55.5。调查时间为2月17日至3月9日。彭博调查的经济师的预估中值为54.8。
彭博社报道，星期五的数据显示，消费者预计未来12个月的通胀率为3.4%，与前月持平。他们预计未来五至十年通胀率为3.2%，较前月略有回落。
展望未来，在美国与以色列对伊朗采取军事行动背景下，燃料价格上涨可能进一步打击消费者情绪，尤其是在生活成本高企的情况下。2月疲弱的就业报告反映出劳动力市场仍然脆弱，也可能继续压制信心。
另有星期五公布的数据显示，即便在伊朗战争爆发前，美国联邦储备局青睐的核心通胀指标1月环比上升0.4%，涨幅较为坚挺。
对于普遍预计将在下周政策会议上维持利率不变的美联储官员而言，通胀前景的不确定性可能限制未来几个月降息的空间。
该调查负责人Joanne Hsu在声明中说：“在伊朗军事行动前完成的访谈显示，信心较前月改善，但随后九天内的回落完全抹去这些初步涨幅。”
她指出，伊朗遭到空袭后，人们对汽油价格的担忧升温，促使3月对未来一年汽油价格的平均预期升至2022年以来最高水平。
消费者对自身财务状况和整体经济的展望均出现恶化。
