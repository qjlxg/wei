超強颱風巴威逐步逼近台灣、浙江、福建和八重山群島一帶。計劃前往或正逗留當地的讀者應留意當地天氣警示及航班安排。
巴威環流廣闊，且會維持較高強度，對台灣構成頗大威脅。按照現時預測，巴威將於週六近距離掠過台灣以北，屬對台灣北部影響最大的「西北颱」路徑，會帶來猛烈風勢、豪雨和風暴潮。
若巴威未登陸台灣，對浙江、福建威脅亦會相當大。然而，目前未能排除風暴近岸受地形影響，先登陸台灣的可能。
本港方面，受巴威外圍下沉氣流影響，週五起將迎接高溫天氣。天文台預料，市區週六氣溫高達35度，達極端酷熱水平。同時，高溫或觸發大驟雨及雷暴，但熱對流發展位置可以相當隨機。
未有外遊計劃的讀者，應做好防曬及防中暑措施，並多關心身邊慢性疾病人士及長者健康。計劃進行戶外活動時，應留意高溫及驟雨可能造成的影響。

一道低壓槽會在星期一稍後開始在華南沿岸徘徊，屆時本港天氣漸轉不穩，大驟雨及狂風雷暴的風險會在本週中期持續。
預料低壓槽內可能有低空渦旋發展。除非該渦旋發展位置顯著偏南，疊加副熱帶高壓脊相關的東風充分配合，否則其發展為熱帶氣旋的機會暫時偏低。
視乎低空渦旋掠過的位置，水氣有機會被加強抽入至沿岸抬升，並誘發本週中期強雷雨區發展。本港雨勢強弱、雨區發展時間和影響範圍仍有變數。
雖然低壓槽及低空渦旋會在本週後期消退，但緊接而來的是活躍西南氣流，因此本週後期天氣仍會不穩定。讀者本週外出前應留意最新天氣消息，並留意已安排的戶外活動可能受到影響。

天文台表示，南海北部廣闊低壓槽上有一個低壓區會逐漸發展，並靠近廣東沿岸。若該低壓區發展為熱帶低氣壓，會評估是否在今日傍晚至明早發出一號戒備信號。
天文台亦表示，低壓系統相關的對流雲團現時集中於南側，預料尚需一些時間整合，在較為接近廣東沿岸時，才能確定會否發展成熱帶低氣壓。
按照現時預測，該低壓系統明早最接近，會在本港100公里範圍內掠過。天文台預料明日本港會有大驟雨、雷暴及狂風，可能需要發出暴雨警告信號或局部地區大雨提示；離岸吹強風，部分低窪地區有機會出現水浸。

---

受南海北部低壓系統影響，預料本港明早（星期二）天氣轉差，有大驟雨和狂風。星期三、四天氣繼續不穩。
目前低壓系統對流集中在南側，而且尚未發展出明確和穩定的中心。然而，受近岸地形輻合影響，未能排除中心加快整合，導致低壓系統發展為熱帶氣旋的可能。
按照現時預測，低壓系統明早會近距離掠過本港，但實際路徑要視乎中心整合情況。由於低壓系統大風區集中在東側和南側，若在本港以西掠過，風力會較大；至於東登的話，風力會在低壓系統掠過本港以北時有所上升。
低壓系統靠近時，近岸地形輻合或誘發對流發展；此外，低壓系統登陸後會引進西南氣流，亦有利強雷雨區發展。因此，預料未來兩三日本港不時受大驟雨影響，強雷雨區掠過期間或有短暫風力上升，有機會伴隨雷暴。然而，強雷雨區發展位置和影響時間存在一定變數。
無論掛波與否，本港未來兩三日天氣亦會轉差，讀者要密切留意天氣情況，MET WARN 亦會跟進最新消息。

英国扩大社媒管制 16至17岁青少年将实施午夜“宵禁”
英国政府星期二（7月14日）宣布，将对16岁和17岁青少年实施社交媒体“宵禁”，规定他们每天午夜12时至清晨6时不得使用Instagram和Facebook等社媒平台。
这是首相斯塔默上月宣布禁止16岁以下儿童使用社交媒体后，进一步收紧未成年人使用社媒的措施。
法新社报道，政府也计划将关闭“无限滚动”等容易令人沉迷的功能设为16岁和17岁用户的默认选项。不过，用户可以自行更改这些设置，外界质疑措施成效，也担心执行难度较大。
英国科技部长肯德尔说，即使16岁青少年拥有更多自主权，政府仍应保护他们免受网络成瘾功能的影响，以帮助他们获得充足睡眠、专注学业，并有更多时间陪伴家人和朋友。
这些措施还包括对人工智能聊天机器人的使用进行监管，要求18岁以下用户在使用相关平台时定时休息。
英国上月宣布，自2027年初起，禁止16岁以下儿童使用Snapchat、TikTok、YouTube、Instagram和Facebook等社交媒体平台。
部分儿童权益团体欢迎新措施，认为早应加强保护未成年人；也有团体担忧，限制措施可能促使青少年转向更不安全的网络环境。
澳大利亚去年12月率先立法禁止16岁以下的人使用社交媒体；加拿大、阿联酋等国也宣布类似措施，印度尼西亚则于今年3月开始对16岁以下用户实施禁令。
