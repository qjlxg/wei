1. B-2 隐形轰炸机：每架造价超过10亿美元，是美国空军最强大的平台。B-2由四台喷气式发动机提供动力，可携带常规武器或核武器，具备洲际航程和空中加油能力。
2. LUCAS单向无人机：“史诗狂怒行动”标志着美国首次在战斗中使用这种无人机。低成本无人作战攻击系统（LUCAS）本质上是伊朗设计的Shahed 136无人机的仿制品，俄罗斯在对乌克兰的战争中大量使用了这种无人机。
3. 美国各类海军军舰：中央司令部表示，美国航空母舰和导弹驱逐舰已在战争中投入战斗。美国发布的视频还显示，导弹驱逐舰正在发射战斧导弹。
4. 爱国者和萨德导弹防御系统：爱国者和萨德（末段高空区域防御）导弹防御系统用于对抗来袭的伊朗无人机和弹道导弹。目前尚不清楚已发射的“爱国者”和“萨德”拦截导弹的数量。
5. 各类型战斗机：中央司令部表示，战争初期多种战斗机都投入了战斗。其中包括空军使用的F-16战斗机，以及海军和海军陆战队使用的F/A-18战斗机。空军、海军和海军陆战队装备的隐形F-22和F-35战机也参与了此次行动，但未透露具体任务。
6. EA-18G电子攻击机：EA-18G“咆哮者”电子攻击机以F/A-18战斗机为基础，配备干扰吊舱、通信对抗设备和雷达，用于识别和压制敌方电子威胁。这款双引擎喷气式飞机还可以携带追踪导弹，专门攻击雷达和通信中心等电子传输目标。
7. 空中预警与控制飞机：美国使用两种类型的预警机，空军的E-3哨兵和海军的E-2鹰眼。预警机的探测范围约为250英里（约400公里），能够识别和跟踪敌方飞机和舰船，并监控美军在战场上的详细情报。这些信息会与海上指挥中心和舰船共享。
8. 空中通信中继飞机：中央司令部没有给出具体细节，但在空袭发生前的几周，有人在中东地区看到了空军的EA-11 BACN（战场空中通信节点）飞机。
9. P-8A海上巡逻机：该双引擎海军喷气式飞机以波音 737 商用飞机为原型，被命名为“海神”，用于反潜作战以及情报、监视和侦察。
10. RC-135侦察机：RC-135可搭载30多名机组人员，包括电子战军官、情报人员，甚至还有机上维修技术人员，自越南战争以来一直是美国军事行动中的常客
11. MQ-9“死神”无人机：美国空军将MQ-9无人机描述为一种主要用于攻击“高价值、转瞬即逝且时间敏感的目标”的遥控单引擎涡轮螺旋桨无人机。它们携带“地狱火”导弹和制导炸弹，可用于打击敌方装甲车辆或人员，并在战场上空盘旋以进行侦察和情报收集。
12. 各类运输机：C-17“环球霸王”喷气式飞机和C-130“大力神”涡轮螺旋桨飞机将大量弹药和参与袭击伊朗的许多部队运送到中东。

政府经济部门正密切跟进并准备能源、金融與貿易等配套措施，工商会表示愿与政府紧密合作，深入评估行业影响并协助中小企业稳住营运。工商会呼吁企业与民众关注官方信息、谨慎经营，共同维护国家经济稳定。

泰国工商会主席博·阿兰瓦塔南（ดร.พจน์）表示，今日（3月2日）与总理会商，评估中东冲突对泰国经济与企业的影响，并制定全面应对方案。政府已邀请安全、经济与民间部门共同研判，以减轻对民众与企业的冲击。 
报告指出，霍尔木兹海峡为全球海运石油约每日14–20百万桶、约占海运石油的20%，亦是供应亚洲液化天然气的重要航道。一旦航运受限或保险与安全风险上升，将立即收紧能源供给，推动油价快速上涨并加重通胀压力。中国、印度、日本、韩国等亚洲国家受影响显著；对能源净进口的泰国而言，油气价格波动将传导为生产成本、电价、运输费与民生开支上升。 
工商会与物流业评估的具体影响包括：一、油价不可避免上涨，推高全球海运、空运与陆运成本，增加整体物流费用；二、对波斯湾航线的“战争风险保费”会显著提高，初步估计运费每航次上升约50%；三、对中东的出口可能短期延缓或暂停，承运减少订舱或改航线将拉长运期并增加库存与资金压力；四、即便对欧洲的航线已有改道绕行好望角，若油价继续上涨，运费与运输时间仍将进一步上升，最少延长约一月，影响双程箱运与出口竞争力。 
若油价与运费同时上升，将对全球贸易与泰国出口竞争力形成双重压力。工商会建议企业尽快审查运输合约、保险条款与能源成本，做好多情景规划并维持充足流动性；同时把握农业、食品加工及日用消费品等可能增长的出口机会。 
政府经济部门正密切跟进并准备能源、金融與貿易等配套措施，工商会表示愿与政府紧密合作，深入评估行业影响并协助中小企业稳住营运。工商会呼吁企业与民众关注官方信息、谨慎经营，共同维护国家经济稳定。

特拉维夫一名菲律宾护理员回忆：2月28日早上8点，她和两名子女被以色列政府发来的短信警报惊醒——“导弹将至，请就近避难”。48岁的Irene Dela Cruz Mohtaseb说，这类短信成为他们在以美对伊朗发动先发制击时判断何时入避难所的重要信号。
她与家人住在一所设有地下避难所的学校旁，来自世界各地的外籍劳工在警报期间聚集在地下室躲避空袭。首日警报频繁，每15分钟一响，连上厕所也成了惊慌的折返：一坐下警报又响，孩子们既习惯又害怕。
Irene说丈夫是以色列人，习惯警报选择留在家中；第二天空袭较首日缓和。她表示自己作为护理员被视为必要岗位，仍坚持上班，照顾依赖她的高龄病人。尽管有报道称一名菲律宾护理员在最新空袭中被弹片击中身亡，Irene暂无回国计划，许多外籍人士也不打算撤离。
她向菲律宾的家人报平安：“我们这里还算安全，不用担心，上帝会保佑我们，我们非常想念你们，希望很快见到你们。”
