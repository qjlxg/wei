位於菲律賓以東海域的潛在風暴預料較大機會週末在本港以東登陸，但其與本港的距離須視乎中心整合位置和速度，以及季風槽和副熱帶高壓脊的互動。
潛在風暴過去一兩日發展速度較慢，尚未建立和鞏固中心。人工智能模式預測潛在風暴短期內中心整合速度較快，繼而較早進入南海，預測登陸位置亦略為偏東。部分傳統電腦模式則預測潛在風暴仍須一段時間整合中心，因此起跑時間較遲，預測登陸位置亦略為偏西。
人工智能和傳統電腦模式對風暴初期中心整合的預測有別，在去年數個案例亦有出現，但現時未有足夠數據判斷誰優誰劣。
至於潛在風暴進入南海後的路徑北份量，則須視乎副高往南伸展的幅度。早前傳統電腦模式預測季風槽有殘餘渦度，因而阻礙副高南伸。然而，傳統電腦模式過去一兩日有加大副高南伸幅度，以致潛在風暴預測路徑往東修正的趨勢。此外，新一輪西風波動會在週末通過中國東北部，亦令副高短期內較難進一步往西伸展。因此，現階段預測潛在風暴東登的機會較大。
東登風暴須較接近本港，以及登陸後採取較為偏西的路徑，才會對本港造成顯著的風雨影響。相信潛在風暴中心完成整合後，本港週末風雨預測會更為明朗。

2026年7月23日上午，中共中央政治局委员、外交部长王毅在马尼拉会见埃及外长阿卜杜勒阿提。
王毅表示，今年是中埃建交70周年。70年来，中埃两国风雨兼程，走过了不平凡历程，树立了发展中大国团结合作典范，彰显了两国关系的历史性、特殊性和重要性，也为推动世界和平发展发挥了重要作用。近年来，习近平主席和塞西总统保持密切沟通，建立深厚互信，为双边关系发展提供了战略保障。中方愿同埃方一道，密切高层交往，深化经贸、军事、人工智能等领域合作，加强多边事务协作，共同构建面向新时代的中埃命运共同体。在埃方支持下，米娜宫饭店建立《开罗宣言》纪念碑，具有重要纪念意义，也体现了埃方尊重历史、弘扬正义、重视对华关系的积极态度。

俄罗斯国家杜马7月22日通过一项由政府提出的法案，自2027年1月1日起，将建立覆盖商品流通全链条的价格监测机制，对纳入俄罗斯联邦国家统计局消费者价格指数核算范围的商品和服务开展价格监测，覆盖从生产商或进口商到零售终端的各个环节，以分析价格上涨的动态及成因。据《俄罗斯报》报道，首批监测对象可能包括肉类、蔬菜、牛奶、谷物等社会重要商品，具体名单将由政府确定。
根据法案，俄罗斯政府将指定有权获取数据的部门，由俄罗斯联邦税务局提供企业销售量、收入、支出和利润等经营数据，信息来源包括纳税申报表、财务报表及税控收银设备记录。相关部门将在法定权限内利用这些数据分析价格走势及形成因素，除《俄罗斯联邦税法》规定的情形外，相关商业数据不得公开。
对于该法案，支持者认为，新机制有助于监管部门利用交易数据更准确分析价格上涨环节及其原因；部分学者和零售业专家则担忧，扩大政府获取商业交易数据的范围可能增加信息泄露风险，而物流、租金、人力等成本并不能完全从税务数据中反映，因此该机制对降低商品价格的实际效果仍有待观察。

（华盛顿综合电）美国参议院拟议推出一项法案，收紧条例禁止中国制造的车辆、软件和硬件进入美国。
参议院商业委员会的跨党派议员星期三（7月22日）达成协议推出一项网联车辆安全法案（Connected Vehicle Safety Act），拟议从明年1月起禁止与中国相关的无线网联汽车和软件进口、制造和销售，并从2030年起将禁令扩大至相关硬件。
网联车指的是附有互联网连接功能、感应器和无线科技的车辆，基本上涵盖所有具备现代化科技的车辆。
美国政府目前通过《贸易法》301条款对中国电动车征收100%关税，叠加在《贸易扩张法》232条款对所有进口车征收的25%关税之上。加上前总统拜登时期采取的科技进口管制措施，实际上阻断了中国车进入美国，主要品牌如比亚迪、奇瑞等都不在那里贩售。
这项由共和党参议员莫雷诺和民主党参议员斯洛特金联合推动的新法案，旨在将拜登时期的限制措施正式立法，并进一步阻止中国业者进军美国的轻型客货车市场。除了中国，法案也覆盖俄罗斯、伊朗和朝鲜三个美国敌对国家。
莫雷诺在声明中说，新措施将“阻止中共接管美国车的转向盘、刹车和加速器”，“我们是在阻止美国工业基础被绝对、彻底及全盘摧毁”。
然而，德国汽车制造商马赛地（Mercedes-Benz）却可能遭殃，因为法案附带条款限定在美销售的汽车制造商不可由中国实体持股超过15%，而马赛地的中国持股接近20%。
参院商业委员会主席、共和党参议员克鲁兹透露，美国通用汽车公司游说制定有关条款，试图将竞争对手挤出局。不过他说，绝不考虑禁止马赛地在美销售。

韩国地方选举投票率疑遭篡改 调查人员搜查选委会总部
（首尔法新电）韩国调查人员星期四搜查选举委员会办公室，调查选举官员涉嫌在6月的地方选举投票率数据造假的案件。
韩国6月3日举行地方选举，当时数十个投票站发生前所未见的选票短缺，至少有90名选民未能投票。 
这起争议导致中央选举管理委员会委员长卢泰岳辞职，并引发要求重新选举的示威。韩国总统李在明下令彻查此事。
韩国警察和检察官组成的联合调查组星期四（7月23日）说，针对事件进行更广泛的调查中，发现官员篡改了投票率数据，因此搜查选委会办公室。
联合调查组发言人说：“有迹象表明，中央选举管理委员会和京畿道选举委员会官员，在地方选举期间输入了虚假的投票率数据。”
调查组说，两名选举官员已因涉嫌伪造官方电子记录等罪名接受调查，并将进一步查明有关操纵行为是否涉及选举高层官员。

港媒：捷克众议长在北京称不会再有官方人员去台湾
香港媒体报道，捷克众议长冈村富夫在北京说，不会再有捷克官方人员访问台湾，他还称“不是中方人士首先提出台湾问题，都是我主动强调捷克遵守一个中国政策”。
据凤凰卫视报道，捷克众议长冈村富夫应邀于星期一至五（7月20日至24日）率团访华。这是捷克众议长时隔七年访问中国，也是过去七年来第一个到访中国的捷克官方代表团。
冈村富夫星期四（7月23日）接受凤凰卫视采访时说，不会再有捷克官方人员访问台湾，“不是中方人士首先提出台湾问题，都是我主动强调，捷克遵守一个中国政策”。
据新华社报道，中国全国政协主席王沪宁星期四在北京会见冈村富夫。
王沪宁表说，中捷两国人民友好交往源远流长。今年是中捷战略伙伴关系建立10周年，中捷合作潜力巨大，前景广阔。中方愿同捷方携手努力，尊重彼此核心利益和重大关切，加强务实合作，推动中捷传统友好重新焕发生机活力。
据报道，王沪宁介绍了中国在台湾问题和涉藏问题上的原则立场，表示中国全国政协愿为促进中捷关系健康稳定发展作出贡献。
根据中方新闻稿，冈村富夫说，捷克坚持一个中国政策，这是两国关系重要支柱。捷方愿以两国战略伙伴关系建立10周年为契机，同中方加强对话合作，推动捷中关系向前发展。捷克众议院愿为推进两国经贸、旅游、直航、人文等领域交流合作，增进人民友谊发挥积极作用。
据路透社报道，冈村富夫访华目的是要加强捷中两国之间在旅游和文化方面的联系。
捷克现任政府去年12月上台后，正寻求与中国建立更务实的关系，中捷两国外交关系近几个月来也有所回暖。捷克前任政府与台湾关系密切，曾导致捷克和中国大陆之间关系紧张。
