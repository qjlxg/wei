311地震為一次罕見且發生機制複雜的「連動型地震」。其震央雖然位於宮城縣對開海域，但整體的斷層破壞面卻異常廣闊，並由岩手縣一直延伸至茨城縣，合共長達約500公里，東西闊約200公里。在短短五分鐘內接連引發了至少三處大範圍的斷層錯動，累積釋放的能量極為龐大。
災後十五年，受災地區目前在安全性、基礎設施、觀光及出口等方面持續進行復興工作。然而，部分災民的心理創傷，以及福島第一核電廠為降低潛在輻射風險而持續推進的廢爐作業（如近年展開的取出核燃料碎片樣本、處理水排海及移除部分機組乏燃料），依然是這場災難留下的長遠挑戰。
而日本三陸近海未來仍然存在強震風險。三陸北部近海至北海道東部近海的岩盤可能已累積足以引發巨大地震的能量。最壞情況下地震規模可達9級或以上，北海道至東北可能出現超過20米的巨大海嘯。

马尼拉—众议院在二读通过一项法案，拟授权总统在国家或全球经济紧急情形下，暂时暂停或降低石油产品的消费税，以应对伊朗冲突后居高不下的油价。 
根据众议院法案8418号，总统得在发展预算协调委员会（DBCC）提出建议并与能源部长协调后行使此权力，触发条件为：一是迪拜原油月均价达到或超过每桶80美元；二是总统宣布国家紧急或灾难状态，且能源部长证实造成国内汽油等油品零售价格异常上涨。 
法案规定，可针对特定石油产品采取全部暂停或部分减征，单次措施最多有效六个月，国会可通过联席决议延长，但累计不得超过一年；该授权可适用至2028年12月31日止。 
同时，总统须经财政部长在发布暂停或减征令后15天内向两院提交事实依据报告，并就预计的未入税收入、对通胀、燃油价格及其他经济活动的影响每月报告一次。 
众议院税收委员会主席米罗·金博表示，下周休会前下院可望在三读通过该案。若参议院通过内容有异，双方将召开两院联席会议协调，最终版本经两院通过后再移送总统签署成为法律。

美国启动贸易调查 或对中国欧盟等征收新关税
美国特朗普政府星期三（3月11日）启动一项新的贸易调查，针对包括欧盟、中国大陆和台湾在内的16个主要贸易伙伴的工业产能过剩问题，为可能实施的新一轮关税措施铺路。
综合彭博社和路透社报道，美国贸易代表格里尔当天在记者会上说，这项依据《301条款》展开的“不公平贸易行为”调查，最快可能在今年夏天促使美国对中国大陆、欧盟、印度、日本、韩国和墨西哥征收新的关税。
被纳入调查范围的其他贸易伙伴还包括台湾、越南、泰国、马来西亚、柬埔寨、新加坡、印度尼西亚、孟加拉国、瑞士和挪威。美国第二大贸易伙伴加拿大则未被列为此次调查对象。
格里尔说，这些调查将重点针对存在结构性产能过剩和制造业过度生产的经济体，“例如长期出现较大贸易顺差，或产能利用不足，甚至存在闲置产能的情况”。
此举标志着在美国最高法院上月裁定特朗普政府去年对多国加征关税违宪后，政府迈出重建关税壁垒的重要一步。
彭博社指出，美国政府在总统特朗普计划与中国国家主席习近平举行峰会前数周启动这项调查，可能进一步加剧美中紧张关系；而将墨西哥列为调查对象之一，也可能影响重新谈判《美墨加协定》的进程。

台民调：大陆攻台美不协助 近六成民众愿不计代价抵抗
台湾中央研究院最新民间调查显示，若中国大陆攻台且美国不出兵援助，仍有近六成受访者愿意不计代价抵抗。
《联合报》等媒体报道，中研院欧美所星期三（3月11日）公布的“美国肖像”民调显示，如果大陆攻台但美国未出兵协防，仍有58.7%受访者表示愿意不计代价抵抗，其中41.2%“非常愿意”，17.5%表示“有点愿意”。
若美国出兵协助，34.4%的受访者“非常愿意”不计代价抵抗大陆，22.1%“有点愿意”，合计56.5%表达愿意抵抗。
参与上述计划的东吴大学社会学系副教授潘欣欣指出，无论是学术机构还是政府智库进行的调查，都呈现出相当一致的趋势，也就是绝大多数台湾人表示“愿意抵抗外敌、保卫台湾”。
对于大陆会不会以武力攻打台湾，受访者态度两极化。有46.4%倾向相信大陆会动武，有46.1%倾向不相信大陆会动武。
这份民调也显示，53.5%受访者支持政府将国防预算提高至台湾生产总额（GDP）的3%。至于台湾向美国购买武器，69.5%受访者表示支持。
潘欣欣说，几乎所有民进党支持者都支持购买美国武器，约三分之二的民众党支持者及不到一半的国民党支持者表示支持。
民调也指出，有22.4%的受访者“非常不同意”美国是讲信用的国家，这个比率高于去年民调显示的15.8%。另有54.5％的受访者倾向不相信美国的安全承诺。
至于大陆是否是讲信用的国家，有76.3%的受访者倾向不同意，仅有17%的受访者倾向同意。这个数据较去年稍为缓和，去年有82.4%的受访者倾向不同意、有11.7%的受访者倾向同意。
上述调查以满20岁台湾成年人为对象，访问期间为今年1月20日至1月26日，总共有1206个有效样本。

中国官方再发安全提示 建议“养龙虾”坚持最小权限原则
中国工信部发布有关防范代理式人工智能OpenClaw（“龙虾”）安全风险的六条建议，包括严控互联网暴露面、坚持最小权限原则等。
央视新闻报道，中国工业和信息化部网络安全威胁和漏洞信息共享平台星期三（3月11日）发布关于防范OpenClaw开源智能体安全风险的“六要六不要”建议。
第一，建议使用官方最新版本，并开启自动更新提醒；在升级前备份数据，升级后重启服务并验证补丁是否生效。不要使用第三方镜像版本或历史版本。
第二，要严格控制互联网暴露面。工信部建议，用户要定期自查是否存在互联网暴露情况，一旦发现立即下线整改。不要将“龙虾”智能体实例暴露到互联网，确需互联网访问的可以使用SSH等加密通道，并限制访问源地址，使用强密码或证书、硬件密钥等认证方式。
第三，坚持最小权限原则。要根据业务需要授予完成任务必需的最小权限，对删除文件、发送数据、修改系统配置等重要操作进行二次确认或人工审批。优先考虑在容器或虚拟机中隔离运行，形成独立的权限区域。
工信部提醒，不要在部署时使用管理员权限账号。
第四，谨慎使用技能市场，审慎下载ClawHub“技能包”，并在安装前审查技能包代码。不要使用要求“下载ZIP”“执行shell脚本”或“输入密码”的技能包。
第五，防范社会工程学攻击和浏览器劫持。要使用浏览器沙箱、网页过滤器等扩展阻止可疑脚本，启用日志审计功能，遇到可疑行为立即断开网关并重置密码。不要浏览来历不明的网站、点击陌生的网页链接、读取不可信文档。
第六，建立长效防护机制。要定期检查并修补漏洞，及时相关风险预警。中共党政机关、企事业单位和个人用户可以结合网络安全防护工具、主流杀毒软件进行实时防护，及时处置可能存在的安全风险。不要禁用详细日志审计功能。
OpenClaw能部署在用户的个人电脑、手机和云服务器上，整合调用不同大语言模型和应用软件。只要发布语音指令，就能自主执行文件管理和数据汇总等任务。
工信部早在2月5日已发布安全风险的预警提示，指OpenClaw可能因指令诱导、配置缺陷或被恶意接管，执行越权操作，造成信息泄露、系统受控。中国国家互联网应急中心星期二（10日）也提示OpenClaw默认的安全配置极为脆弱，攻击者一旦发现突破口，便能轻易获取系统的控制权。
