## 来源: metwarn

超強颱風巴威逐步逼近台灣、浙江、福建和八重山群島一帶。計劃前往或正逗留當地的讀者應留意當地天氣警示及航班安排。
巴威環流廣闊，且會維持較高強度，對台灣構成頗大威脅。按照現時預測，巴威將於週六近距離掠過台灣以北，屬對台灣北部影響最大的「西北颱」路徑，會帶來猛烈風勢、豪雨和風暴潮。
若巴威未登陸台灣，對浙江、福建威脅亦會相當大。然而，目前未能排除風暴近岸受地形影響，先登陸台灣的可能。
本港方面，受巴威外圍下沉氣流影響，週五起將迎接高溫天氣。天文台預料，市區週六氣溫高達35度，達極端酷熱水平。同時，高溫或觸發大驟雨及雷暴，但熱對流發展位置可以相當隨機。
未有外遊計劃的讀者，應做好防曬及防中暑措施，並多關心身邊慢性疾病人士及長者健康。計劃進行戶外活動時，應留意高溫及驟雨可能造成的影響。

---

一道低壓槽會在星期一稍後開始在華南沿岸徘徊，屆時本港天氣漸轉不穩，大驟雨及狂風雷暴的風險會在本週中期持續。
預料低壓槽內可能有低空渦旋發展。除非該渦旋發展位置顯著偏南，疊加副熱帶高壓脊相關的東風充分配合，否則其發展為熱帶氣旋的機會暫時偏低。
視乎低空渦旋掠過的位置，水氣有機會被加強抽入至沿岸抬升，並誘發本週中期強雷雨區發展。本港雨勢強弱、雨區發展時間和影響範圍仍有變數。
雖然低壓槽及低空渦旋會在本週後期消退，但緊接而來的是活躍西南氣流，因此本週後期天氣仍會不穩定。讀者本週外出前應留意最新天氣消息，並留意已安排的戶外活動可能受到影響。

---

天文台表示，南海北部廣闊低壓槽上有一個低壓區會逐漸發展，並靠近廣東沿岸。若該低壓區發展為熱帶低氣壓，會評估是否在今日傍晚至明早發出一號戒備信號。
天文台亦表示，低壓系統相關的對流雲團現時集中於南側，預料尚需一些時間整合，在較為接近廣東沿岸時，才能確定會否發展成熱帶低氣壓。
按照現時預測，該低壓系統明早最接近，會在本港100公里範圍內掠過。天文台預料明日本港會有大驟雨、雷暴及狂風，可能需要發出暴雨警告信號或局部地區大雨提示；離岸吹強風，部分低窪地區有機會出現水浸。

---

受南海北部低壓系統影響，預料本港明早（星期二）天氣轉差，有大驟雨和狂風。星期三、四天氣繼續不穩。
目前低壓系統對流集中在南側，而且尚未發展出明確和穩定的中心。然而，受近岸地形輻合影響，未能排除中心加快整合，導致低壓系統發展為熱帶氣旋的可能。
按照現時預測，低壓系統明早會近距離掠過本港，但實際路徑要視乎中心整合情況。由於低壓系統大風區集中在東側和南側，若在本港以西掠過，風力會較大；至於東登的話，風力會在低壓系統掠過本港以北時有所上升。
低壓系統靠近時，近岸地形輻合或誘發對流發展；此外，低壓系統登陸後會引進西南氣流，亦有利強雷雨區發展。因此，預料未來兩三日本港不時受大驟雨影響，強雷雨區掠過期間或有短暫風力上升，有機會伴隨雷暴。然而，強雷雨區發展位置和影響時間存在一定變數。
無論掛波與否，本港未來兩三日天氣亦會轉差，讀者要密切留意天氣情況，MET WARN 亦會跟進最新消息。

阿尔巴尼斯在悉尼大学发表讲话时表示：“今天，我宣布，为了把握人工智能带来的机遇并引导其发展，我国政府将制定澳大利亚人工智能标准。”

澳总理还宣布，将在总理和内阁部内设立人工智能办公室，负责协调政府在该领域的政策。
阿尔巴尼斯表示，版权保护将受到特别重视。他强调，未经版权所有者许可，任何公司都不得使用艺术作品、书籍、音乐或媒体内容来训练神经网络。
“澳大利亚的作家、音乐家、艺术家和记者必须保留对其作品的所有权和控制权。任何公司都不得在未经作者监督的情况下使用澳大利亚的书籍、音乐、艺术作品或新闻来开发或训练人工智能……否则就是盗窃。”澳总理说道。
他还指出，澳大利亚将成为首个将数据中心选址、电力供应和用水量要求整合到国家统一标准体系中的国家。

针对茂木敏充星期天（7月12日）重申中国拒绝接受裁决损害国际社会法治的日方立场，中国外交部亚洲司当天召见日本驻华使馆首席公使横地晃，提出严正交涉。横地晃则阐明日本政府立场，并予以反驳。
综合日本放送协会（NHK）和共同社报道，茂木敏充星期二（7月14日）在记者会上说：“我们已多次向中方阐明了日方的立场，指出仲裁裁决是最终裁定结果，对争议当事国具有法律约束力。”
被问及如何看待日中目前缺少对话机会时，茂木敏充说：“日本政府一贯坚持全面推进日中战略互惠关系、构建具有建设性和稳定性的双边关系这一方针。双方存在悬而未决的问题，正因为如此，加强沟通十分重要。日方愿与中方开展各种对话。”
茂木敏充强调，继续冷静、妥善地与中方进行沟通十分重要。
共同社指出，对于日本首相高市早苗去年11月涉台国会答辩引发日中关系恶化的局面，两国政府间的沟通机制陷入失效状态。由于无法准确判断对方意图，还可能带来安全保障方面出现突发事态的风险。尽管茂木敏充强调“对对话持开放态度”，但并未能勾勒出打开局面的前景。
报道称，能否找到改善关系的突破口，试金石将是11月在中国举行的亚太经合组织（APEC）峰会。若能实现高市早苗与中国国家主席习近平的对话，或许有可能推动关系缓和。然而，日中外交消息人士悲观地说：“并没有能展开建设性讨论的氛围，改善关系还需要相当长的时间。”

港澳办支持的港澳研究中心在北京揭牌 开展港澳重大问题研究
中共中央港澳工作办公室支持的港澳研究中心在北京正式揭牌。中国官方称，港澳研究中心将构建“一国两制”基础理论和学术话语体系，开展港澳重大问题研究。

俄媒：朝鲜劳工“身价暴涨” 俄地方政府因工资过高放弃引进
俄乌战争持续导致俄罗斯劳动力短缺加剧，朝鲜劳工在当地已不再是“廉价劳动力”。随着薪资水平大幅上涨，部分俄罗斯地方政府和企业被迫放弃引进朝鲜劳工。 
俄独立媒体《梅杜扎》（Meduza）星期一（7月13日）报道，俄罗斯南部奥伦堡市原计划引进朝鲜劳工，从事市政清洁及维护等公共服务工作，但双方因薪资分歧最终未能达成协议。
奥伦堡市长尤马迪洛夫说，市政府去年曾提出每名工人月薪5万5000卢布（约900新元）的条件，但朝方没有接受。“据我了解，朝鲜工人不会为了5万5000卢布来工作，他们现在要求的工资是这一水平的两三倍。”
按此推算，朝方期望的月薪约为11万至16万5000卢布（约1800至2700新元）。
尽管如此，尤马迪洛夫仍高度评价朝鲜劳工，称他们“像机器人一样工作”，纪律严明、效率极高，但以当地财政预算难以承担如此高的工资，最终未能签约。
他还说，俄部分地区已成功引进朝鲜劳工，但奥伦堡州未被列入合作名单。最终，当地改为引进来自西非塞内加尔的劳工，目前已有31人在公共服务部门工作。
韩国统一研究院今年发布的报告指出，根据俄企业者等受访时透露，俄方为朝鲜劳工支付的人均人工成本通常约为每月1000美元（约1290新元）。
分析指出，尤马迪洛夫所说的金额并非单纯工资，而是引进一名朝鲜劳工的整体成本。截至2025年底，包括机票、签证、保险及规避联合国对朝制裁所需的中介费用在内，单名劳工的用工成本已超过2300美元。
专家也对俄地方政府披露的薪资数字持保留态度。朝鲜经济专家世宗研究所沃德（Peter Ward）认为， 普通朝鲜劳工的平均工资绝不可能达到1400美元，更接近1400韩元（约1.21新元）。”
他认为，朝方往往不是根据劳工实际价值定价，而是依据对方的支付能力来开价。
研究俄罗斯经济的东西大学教授蒙代（Chris Monday）则认为，朝鲜政府及中介机构可能从海外劳工工资中大幅抽成，也不排除朝方利用俄劳动力严重短缺的机会，提高对俄方的报价。他形容，这更像是朝鲜中介商借机向俄罗斯抬价。

美官员证实 少量英伟达H200人工智能晶片已交付中国客户
（华盛顿综合讯）负责监管出口管制项目的美国官员证实，英伟达公司在获得美国政府批准后，少量H200人工智能（AI）晶片已交付给中国客户。
据彭博社报道，负责工业和安全事务的美国商务部副部长杰弗里·凯斯勒，在当地时间星期二（7月14日）说，运往中国的英伟达处理器“非常少量”，但他拒绝透露具体数量或买家信息。
凯斯勒在众议院外交事务委员会听证会上说：“对美国民众而言，最重要的是，根据许可证，H200及其同类产品的出货量非常少，所以微不足道。”
他说，每个H200晶片许可证申请者都必须满足严格的国家安全要求，包括确保晶片的实际性能达到预期水平，而不是超出预期。
美国总统特朗普去年12月批准英伟达向中国出口H200晶片，标志着华盛顿大幅放宽了旨在遏制中国人工智能发展而实施的出口限制。
美国商务部1月发布规定，将特朗普的上述决定制度化，对相关出口征收25%的美国关税，同时申请出口许可证的企业必须确认，这些晶片不会用于中国军事用途或最终流向中国军方。
尽管美国已批准向中国出口H200晶片，但北京也收紧了对企业进口晶片的审批，部分原因是担心会阻碍中国发展自主晶片产业的长期目标。 
在上述听证会上，美国共和党委员会主席马斯特敦促商务部采取进一步措施，阻止中国获得英伟达性能更强大的Blackwell系列晶片，呼吁将更多中国实体列入黑名单。
马斯特向凯斯勒施压，询问包括长鑫存储、长江存储、腾讯控股和阿里巴巴在内的中国科技巨头，是否应受到美国商务部的进一步限制。
凯斯勒没有明确回应是否正考虑采取更正式行动，但他表示同意五角大楼的观点，认为商务部在允许向这些公司出口时，必须“非常谨慎”。
美国科技媒体《信息报》（The Information）上星期三（7月8日）报道称，阿里巴巴、字节跳动和AI明星企业DeepSeek被中国官员告知，或很快获得进口部分H200晶片的许可，前提是需要说明晶片的数量及用途。
