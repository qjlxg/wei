潛在風暴過去一兩日發展速度較慢，尚未建立和鞏固中心。人工智能模式預測潛在風暴短期內中心整合速度較快，繼而較早進入南海，預測登陸位置亦略為偏東。部分傳統電腦模式則預測潛在風暴仍須一段時間整合中心，因此起跑時間較遲，預測登陸位置亦略為偏西。
人工智能和傳統電腦模式對風暴初期中心整合的預測有別，在去年數個案例亦有出現，但現時未有足夠數據判斷誰優誰劣。
至於潛在風暴進入南海後的路徑北份量，則須視乎副高往南伸展的幅度。早前傳統電腦模式預測季風槽有殘餘渦度，因而阻礙副高南伸。然而，傳統電腦模式過去一兩日有加大副高南伸幅度，以致潛在風暴預測路徑往東修正的趨勢。此外，新一輪西風波動會在週末通過中國東北部，亦令副高短期內較難進一步往西伸展。因此，現階段預測潛在風暴東登的機會較大。
東登風暴須較接近本港，以及登陸後採取較為偏西的路徑，才會對本港造成顯著的風雨影響。相信潛在風暴中心完成整合後，本港週末風雨預測會更為明朗。

王毅与鲁比奥会晤：筹备高层交往 要求美方恪守一个中国原则
中国外长王毅与美国国务卿鲁比奥在马尼拉举行会晤。王毅肯定中美两个大国在探索和平共处道路上取得的重要进展，但同时就美方“消极言行”阐明中方立场，要求美方尊重中国的核心利益，恪守一个中国原则。
美国国务卿鲁比奥与中国外交部长王毅星期三（7月22日）在菲律宾马尼拉出席一场区域会议期间，于场边举行了双边会晤。
据央视新闻，王毅表示，今年是中美关系的“大年”。两国元首在北京成功举行历史性会晤，确定了中美建设性战略稳定关系的定位，提出了双方共同努力的方向。这是中美两个大国在探索和平共处道路上取得的重要进展，符合两国人民根本利益，符合国际社会共同期待。
王毅说：“现在我们的责任是，沿着两国元首设定的轨道，排除干扰、克服障碍，确保将领导人的共识转化为全政府、各领域的共识和行动，以中美的战略稳定促进世界的和平安宁，以中美的建设性互动为国际合作提供重要动力”。
王毅还就近期美方的一系列消极言行阐明中方严正立场，要求美方尊重中国的核心利益，恪守一个中国原则，切实管控矛盾分歧，解决中方正当关切，让中美关系的机遇之年照进现实。
双方还就国际地区热点问题交换了意见。
中方文告也形容此次会晤务实、积极、富有建设性，同意共同落实两国元首达成的重要共识，发挥政治外交渠道作用，筹备好下阶段高层交往，推动中美建设性战略稳定关系取得实质进展。
据路透社，鲁比奥星期三向记者表示，在与王毅会晤时，就潜在的中国国家主席习近平计划访问美国一事进行深入交流。
鲁比奥称双方存在分歧，但两国的职责是管控这些分歧，并补充说，双方在一些领域具有潜在合作空间。
此次鲁比奥和王毅的会晤正值全球两大经济体维持脆弱休战之际。美国总统特朗普指责中国干预美国选举，可能令两国关系进一步复杂化。北京则称有关指控“纯属捏造”。

乌克兰再次袭击俄罗斯最大电商平台仓库
俄罗斯在线零售商Wildberries的老板星期三（7月22日）说，乌克兰再次袭击了公司另外两个仓库。
路透社报道，Wildberries联合创始人兼首席执行官塔季扬娜·金说，位于俄罗斯南部克拉斯诺达尔（Krasnodar）和涅温诺梅斯克（Nevinnomyssk）的仓库遭到袭击。
当地州长说，一名年轻女子在袭击中遇难，另有三人正在医院接受治疗。
Wildberries被誉为俄罗斯的亚马逊，是俄罗斯最大的雇主之一。
对Wildberries大型仓库的袭击标志着乌克兰对俄罗斯境内的深度打击进入新阶段。乌克兰正努力削弱莫斯科的战争能力，并将冲突带入俄罗斯普通民众的生活。
乌克兰总统泽连斯基星期三称，他们将打击为俄军提供无人机部件等装备的物流枢纽。
克里姆林宫否认Wildberries处理军需品。俄罗斯曾多次袭击乌克兰的物流枢纽、能源系统和电网。
此前，乌克兰无人机于7月18日发动袭击，造成Wildberries的八名员工死亡。

王毅会见菲律宾外长拉扎罗：中菲关系站在十字路口
中国外交部长王毅在马尼拉与菲律宾外长拉扎罗会见，向菲方阐明中方严正立场，对菲方人员在仁爱礁（菲律宾称阿云津礁）“冲撞中方执法船只的粗暴行径”表示强烈抗议，并表示中菲关系站在十字路口，何去何从需要菲方作出正确理智选择。
王毅星期三（7月22日）在马尼拉出席东亚合作系列外长会期间应约与菲律宾外长拉扎罗会见。此次会见正值中菲两国船只在仁爱礁海域爆发冲突、双方各执一词，争执不下之际。这也是两国外长两年来首次会晤。
王毅表示，过去几年，中菲关系持续面临困难挑战，这显然不符合双方利益，更有损菲律宾自身发展。“面对变乱交织的世界，我们本应共同维护本地区得来不易的和平发展局面，但菲律宾的做法却与其他地区国家背道而驰，成为地区和平稳定的干扰源。”
王毅说，事实证明，任由外部势力干涉介入，蓄意炒作扩大海上分歧，只会沦为被人利用的工具。如果试图借助域外国家挑衅滋事，最后只能吞下自己酿成的苦果。
“遗憾的是，每当中菲双方要开展对话时，菲国内就有人特别是军警部门一些势力，蓄意制造事端，破坏对话的推进。这些人居心不良，他们想的不是菲人民的福祉，而是执行外部势力的旨意，也是在谋取个人的私利。当前，中菲关系站在十字路口，何去何从需要菲方作出正确理智选择。”
王毅强调，发展中菲关系，关键要把握正确方向。他说，中国从来不是菲律宾的安全威胁，历史上从未侵略或殖民过菲律宾。中菲存在领土和海洋划界争议是客观事实。中方始终致力于通过双边对话协商管控分歧、稳定局势。但菲方却背离双边谈判渠道，在域外势力怂恿下，搭了个临时“仲裁庭”的草台班子，炮制炒作非法无效的所谓“裁决”，中方从一开始就不接受、不参与，更不会承认。
王毅表示，菲方应立即停止在涉海问题上的挑衅行为，恪守迄今对中方作出的承诺，为稳定双边关系作出切实努力。
中方文告称，拉扎罗表示，菲中交往历史悠久，是搬不走的邻居。菲方始终奉行一个中国政策，履行建交承诺，世界上只有一个中国，台湾是中国领土不可分割的一部分。菲方希望通过对话，寻找涉海问题的妥善解决办法，争取实现两国关系的改善。
