马尼拉—众议院在二读通过一项法案，拟授权总统在国家或全球经济紧急情形下，暂时暂停或降低石油产品的消费税，以应对伊朗冲突后居高不下的油价。 
根据众议院法案8418号，总统得在发展预算协调委员会（DBCC）提出建议并与能源部长协调后行使此权力，触发条件为：一是迪拜原油月均价达到或超过每桶80美元；二是总统宣布国家紧急或灾难状态，且能源部长证实造成国内汽油等油品零售价格异常上涨。 
法案规定，可针对特定石油产品采取全部暂停或部分减征，单次措施最多有效六个月，国会可通过联席决议延长，但累计不得超过一年；该授权可适用至2028年12月31日止。 
同时，总统须经财政部长在发布暂停或减征令后15天内向两院提交事实依据报告，并就预计的未入税收入、对通胀、燃油价格及其他经济活动的影响每月报告一次。 
众议院税收委员会主席米罗·金博表示，下周休会前下院可望在三读通过该案。若参议院通过内容有异，双方将召开两院联席会议协调，最终版本经两院通过后再移送总统签署成为法律。

礼来星期三（3月11日）在微信发布声明，称投资将着力布局首个申报注册的减重药物orforglipron的生产能力。该药目前正在中国进行监管审批，礼来已于2025年底向中国国家药品监督管理局提交orforglipron用于治疗2型糖尿病与肥胖症的上市申请。
为扩大产能，礼来也与中国本土药企康龙化成达成战略合作，预期投资2亿美元并支持其技术能力建设，未来视发展逐步扩大规模。
彭博社报道，康龙化成星期四（12日）港股股价一度上涨13%，创7月底以来最大涨幅，而其在上海上市的A股涨幅最高达9.1%。
礼来希望利用中国日益成熟的药物研发、生产与制造基础设施，服务数亿本地患者。公司迄今已在中国投资近60亿美元。
据礼来统计，中国约有1.48亿2型糖尿病患者，以及超过五亿人体重超标或肥胖。

日本政府据报警告援乌日企华为5G信息外泄隐患
日媒报道，日本政府已向积极支持乌克兰重建的日企发出警告，指出使用中国通信设备巨头华为的5G移动通信系统有技术或敏感信息外泄隐患。
据共同社报道，相关人士星期三（3月11日）透露这一消息。
去年，乌克兰大型移动运营商在部分城市与华为进行了5G实证试验。乌克兰政府将5G频段分配给了华为。日本乐天集团也报名参与当地5G业务，但未被纳入对象范围。
共同社获得的日本政府相关资料显示，考虑到基辅等主要城市引进华为5G的情况，日本政府认为日欧美对于乌克兰包括能源在内的重要基础设施进行投资风险很大。
此外，日本政府还担忧，即便中国企业不直接参与，也有土耳其企业经手的中国制品会流入市场。

环保团体：台湾能源韧性全球最差 人均进口LNG达大陆25倍
台湾社会对美伊冲突导致的能源危机焦虑升级，有环保团体指出，台湾的能源韧性全球最差，去年从霍尔木兹海峡进口的液化天然气（LNG），人均量是中国大陆的25倍。
美伊冲突将满两周，能源要道霍尔木兹海峡近乎停摆，台湾面临“断气”挑战。《联合报》报道，许多环保团体星期四（3月12日）在立法院召开记者会指，台湾不产天然气，但却是全世界最重度依赖霍尔木兹海峡进口油气的地区。
台湾健康空气行动联盟创会理事长叶光芃表示，从中东波斯湾霍尔木兹海峡进口LNG的全球30余个国家和地区中，台湾是依赖度最高、能源韧性最差的。
台湾健康空气行动联盟引用国际分析显示，2024年度，台湾从霍尔木兹海峡进口LNG总量为5903千吨，占台湾年度LNG进口总量的28%，其中25.9%来自卡塔尔、2%来自阿联酋，台湾从霍尔木兹海峡进口LNG的占比为全球第一。
叶光芃指出，比较2024至2025年期间通过霍尔木兹海峡进口LNG的人均量变化，韩国大幅下调25%的同时，台湾却逆势增长近33%。2025年度台湾从霍尔木兹海峡进口LNG的人均量是日本的10倍，更是中国大陆的25倍。
台湾天然气现行安全存量为11天，而伊朗封锁霍尔木兹海峡即将届满14天，显示台湾过度依赖燃气电力的模式脆弱度高。报道质疑，若中国大陆封锁台海，台湾能否应对断气危机。
