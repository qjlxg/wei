信达生物（Innovent）有望在2025年实现自上市以来首个全年盈利，彭博汇编的预测显示盈利约9.84亿元人民币，而2024年亏损为946.3万元。公司在2025年营收约119亿元，同比增长约45%，并在2月4日的公告中称这是“历史性里程碑”。其糖尿病与肥胖药物mazdutide今年可望贡献约24亿元销售额，且与武田与礼来达成的合作将带来约27亿元的分成收入（诺姆拉估计）。 
原名百济神州、2024年更名为BeOne的公司披露，自2010年成立以来首度实现全年盈利，2025年净利约2.8693亿美元，而上年则亏损6.4479亿美元。其血液癌症药物Brukinsa全球销售达39亿美元（美国市场28亿美元），超过阿斯利康的Calquence（35亿美元）。 
美国咨询机构SAI MedPartners及其子公司Idea Pharma的创新指数将信达与BeOne列为将候选药转化为临床与商业化产品的领先者，榜单上亦包括恒瑞与豪森等企业。

多家在港上市的中国内地制药公司在经历多年研发投入后，随着药品销售回暖和与国际制药商的授权合作兑现，正逐步回到全年盈利轨道。 
麦格理资本亚洲医疗研究主管Tony Ren表示，尽管国内面临药价压力等挑战，创新药的业绩在中国2025年仍应表现良好。 
信达生物（Innovent）有望在2025年实现自上市以来首个全年盈利，彭博汇编的预测显示盈利约9.84亿元人民币，而2024年亏损为946.3万元。公司在2025年营收约119亿元，同比增长约45%，并在2月4日的公告中称这是“历史性里程碑”。其糖尿病与肥胖药物mazdutide今年可望贡献约24亿元销售额，且与武田与礼来达成的合作将带来约27亿元的分成收入（诺姆拉估计）。 
原名百济神州、2024年更名为BeOne的公司披露，自2010年成立以来首度实现全年盈利，2025年净利约2.8693亿美元，而上年则亏损6.4479亿美元。其血液癌症药物Brukinsa全球销售达39亿美元（美国市场28亿美元），超过阿斯利康的Calquence（35亿美元）。 
美国咨询机构SAI MedPartners及其子公司Idea Pharma的创新指数将信达与BeOne列为将候选药转化为临床与商业化产品的领先者，榜单上亦包括恒瑞与豪森等企业。 
野村预计恒瑞2025年第四季度净利将增长32%至23亿元，2026财年净利可望增30%至112亿元，超出彭博估值约13%，增长动力来自新药上市及与葛兰素史克（GSK）就慢阻肺口服药的授权费。豪森预计下半年营收同比增27%至约73亿元。 
与此同时，多起早期药物的出海许可交易为内地药企带来新增长点且热度不减。杭州赛维（Sciwind）近日将其GLP‑1注射剂ecnoglutide在中国内地的独家销售权授予辉瑞，交易包括最高4.95亿美元的首付款、监管及销售里程碑款。 
麦格理的Ren指出，与中国生物医药企业合作已从机会性行为转变为全球研发战略的重要组成部分，投资者对中国医疗板块兴趣浓厚，部分对冲基金甚至将资金从日本投向中国。

浙江如何缩小科技中心的收入差距——避免硅谷式的两极化？
浙江作为国家推进“共同富裕”的试验田，统计显示：2021至2025年城镇与农村人均可支配收入比由1.96降至1.81，成效显著。该省今年发布草案，提出到2030年在过去五年基础上继续缩小城乡差距，并使人均GDP和收入“接近”发达经济体水平；更远目标到2035年实现物质文化生活丰富、治理现代化和更均衡的区域发展。
浙江既是阿里巴巴等巨头及DeepSeek、Unitree Robotics等创新企业的聚集地，也是人工智能、机器人、游戏和脑机接口等前沿产业基地。草案提出鼓励企业推行灵活股权激励和员工持股，盘活闲置农房，优化收入与财富分配，同时警惕“好高骛远”。
外部经验令人警醒：研究显示硅谷0.1%居民掌控71%财富，少数家庭集中大量财富，另有大量家庭几乎无资产。学者熊万生指出，技术革命带来资本密集与替代就业的风险，且我国社会保障在养老金、失业救助和家庭政策等方面仍有不足；浙江学者何文炯建议评估科技对就业影响、加强职业教育、保障公共服务公平，甚至研究普遍基本收入以维护基本生活与发展权利。
国家统计局数据显示，2025年全国城镇-农村收入比为2.31；2024年全国收入基尼为0.465（反映收入不含资产），省级基尼尚未公开。浙江的挑战在于如何在保持科技活力的同时，通过制度设计实现更广泛的共享发展。
