习近平今年2月在与特朗普通话时，重申对台湾问题的底线，并要求美国慎重处理对台军售。分析认为，北京意在避免4月习特会前出现更多对台军售案，但特朗普很清楚“台湾牌”的价值，只要对“让美国再次伟大”（MAGA）有利，就会继续做自己的事。
《纽约时报》上月引述多名美国官员报道，特朗普政府推迟了一笔价值数百亿美元的对台军售案，以确保特朗普访华之行取得成功，并称这项包括防空导弹在内的军售案自1月获美国资深议员批准后，便在国务院搁置至今。
路透社的最新报道显示，特朗普政府计划维持或甚至增加对台湾的军事援助。
知情者称，最新一轮的对台军售案，主要涵盖爱国者3和美国先进地对空两类防空系统。“一切已准备就绪，只要总统批准，对台军售案将正式对外公布。”
知情人士说，另一笔价值60亿美元的不对称作战防卫配套也有待核批，将与有关联或后续军售案一并公布。
