人工智能正迅速渗透军事实践，从情报分析、目标甄别、侦察监视到信息与电子战，机器正在改变战场运作模式。无人机、智能武器与具备自主导航的无人载具，使AI能参与作战规划、武器推荐与情报管理，攻击速度已超越人类决策节奏。 
但AI在战争中的局限明显：系统难以可靠区分平民与战斗人员，算法偏见和数据误差可能导致严重平民伤亡并违反国际人道法。责任归属不清——战时错误究竟应由指挥官还是算法承担——成为迫切问题。 
现实中，AI已从理论走入实战。以色列-美国等在中东冲突中部署AI辅助系统；Palantir的Maven自2024年起整合Anthropic的Claude，用于卫星与监控影像情报提取并生成实时目标库，据称能在数秒内识别并定位大量目标。

调查显示，贾洛因试图加入恐怖组织“伊斯兰国”，于2016年被判入狱，直到2024年才获释。事发时，贾洛闯入一间教室，先询问是否正在进行“后备军官训练团”（ROTC）课程后，高喊“真主至大”并开枪。
袭击造成三人受伤，其中两名伤者为美军人员，另一人送院后伤重不治。贾洛随后被训练员现场制服并死亡，当局称他并非死于枪击，具体死因仍在调查中。
欧道明大学位于诺福克市，距离世界最大海军基地诺福克海军基地仅数公里。学校数据显示，近30%的学生与军方有关。当局已将这起枪击案列为恐怖事件调查。
同日晚些时候，密歇根州底特律发生另一起严重袭击。一名卡车司机加扎利（Ayman Mohamad Ghazali）驾驶卡车强行冲撞一间犹太教堂，并试图沿走廊向建筑内部推进，随后与安保人员发生激烈交火。
美国国土安全部证实，加扎利出生于黎巴嫩，2016年取得美国公民权。美国媒体引述邻居的话报道，加扎利近期精神状态极度不稳定，原因是他位于黎巴嫩家乡的村庄在10天前遭到以色列空袭，导致至少四名亲人丧生。
尽管教堂内的职员和孩童幸免于难，但袭击引发的火灾导致30名执法人员因吸入过量烟雾送院。目前起火原因不明。
联邦调查局已将此案列为“针对犹太社群的暴力袭击”。
美国总统特朗普称，这起袭击事件“非常糟糕”且“难以置信”。他说：“我想向密歇根州的犹太社区以及底特律地区所有民众送去我们的关爱。”
两起连环暴力事件发生之际，正值美伊冲突导致美境内犹太与回教社区关系高度紧张。
联调局的数据显示，自2024年2月以来，全美记录了5300多起涉及宗教动机的仇恨犯罪，其中近三分之二针对犹太人。
目前，全美各地的犹太与回教社区均已显著加强安保。

日本大阪市区下水道施工管道从地面“冒出” 扰乱交通
日本大阪市政府正在调查一处下水道施工现场，一个巨型钢管突然从地面冒出，并上升到四层楼的高度。
一名市政府官员星期五（3月13日）告诉法新社记者，他们星期三接到报告称，这个用于挡土的钢制外壳“从大阪高速公路附近的地面冒起”。
日本共同社报道，星期三早晨6时55分前后，大阪府警方在大阪市北区茶屋町的闹市区路上，发现铺设在下水道施工现场地下的管道隆起，抬升至地面十几米高。所幸未发现人员受伤。
这个直径约3.5米的钢管一度上升到13米高，但到了星期五早上，工人们紧急向其内部注水使其沉没后，钢管已下沉至约1.6米的高度。
现场位于阪急大阪梅田站附近。423号国道（新御堂筋）的高架路从上方经过，但隆起的管道并未与之接触。包括高架路在内的部分路段进行交通管制，造成交通堵塞。
一名大阪市官员说，附近两条通往市中心的道路自星期三起封闭，造成交通拥堵。其中一条道路于星期五下午重新开放，另一条道路也将“很快”开放，市政府官员正在考虑切除剩余的裸露在路面的圆柱体。
他补充说：“我们仍在调查事故原因。”
此前，大阪市上个月宣布收到一笔不同寻常的捐赠——21公斤黄金，用于支付老旧供水系统的维护费用。
大阪市长横山英幸在新闻发布会上说，这笔价值360万美元（459.43万新元）的捐赠来自一位此前已向市自来水公司捐赠3300美元现金的捐赠者，捐赠时间是去年11月。
据当地媒体报道，大阪这座拥有280万人口的城市，其水管更换工程遭遇了阻碍，实际成本超出了预算。
此前，东京附近去年出现一处巨大的塌陷坑，吞噬了一辆卡车和司机，这凸显了日本老化的基础设施所带来的风险。

广东外贸今年首两月“开门红” 规模创历史同期新高
中国外贸大省广东今年前两个月进出口1.64万亿元（人民币，下同，3056亿新元），规模创历史同期新高，比去年同期增长22.1%。进出口双向发力，其中出口1.04万亿元，同比增长22.6%，进口5920.1亿元，同比增长21.3%。
据海关总署广东分署星期五（3月13日）发布上述最新外贸数据，今年前两个月，广东有进出口实绩的企业数量近13万家，同比增加32.9%。其中，民营企业进出口值1.09万亿元，同比增长29.2%；外商投资企业进出口4784.3亿元，同比增长12.8%。
高技术产品、绿色产品出口表现强劲。今年前两个月，广东出口机电产品7189.1亿元，同比增长25.9%。
其中，工业机器人、无人机、数控机床、3D打印机等高技术产品出口分别同比增长32.3%、66.7%、78.5%、172.3%。
风力发电机组、“新三样”（电动载人汽车、太阳能电池、锂电池）等绿色低碳产品出口分别同比增长38.6%、54.6%。
今年前两个月，广东对主要贸易伙伴进出口均实现增长，其中亚细安、香港、欧盟为广东前三大贸易伙伴，进出口分别同比增长18%、36.9%、23.1%；对美国同比增长1%，对台湾同比增长6.9%。对拉美、中东、中亚五国、非洲等进出口分别同比增长15.7%、24.8%、25.3%、41.8%。
进口方面，今年前两个月，广东进口集成电路、电脑及其零部件、铜矿砂等主要生产资料分别同比增长31.1%、54.4%、135.4%。进口大米、牛肉、食用植物油等民生消费品分别同比增长65.1%、90.8%、228.2%。
