中国两会登场 官方承认经济面临老问题新挑战
面对外部不确定因素加剧和国内供强需弱挑战，经济课题成为中国今年两会（全国人大、全国政协年会）备受瞩目的焦点，出席首场发布会的官员表明，中国经济还面临“老问题、新挑战”，但长期向好基本趋势没有变，且会坚持对外开放。
本次中国全国政协会议新闻发布会星期二（3月3日）下午在北京人民大会堂举行，这是今年两会的首场发布会。
据全国政协会议新闻发言人刘结一介绍，今年全国政协会议将在星期三（3月4日）下午开幕，下星期三（3月11日）上午闭幕，会期七天，比去年多一天。
今年是“十五五”（2026至2030年）规划开局之年，外界关注中国今年经济增长目标以及未来五年的经济方针。中国将在星期四（3月5日）开幕的全国人大会议上，公布今年国内生产总值（GDP）增长预期目标。
中国去年GDP增长5%，达到预期目标。虽然工业生产和出口持续增长，但国内消费疲弱、楼市依旧低迷，整体经济呈“K型”分化，不少分析师预计中国官方可能下调今年经济增长目标到4.5%至5%区间。

她呼吁各方克制，通过对话与外交途径维护地区安全，并指出阿联酋是200多国籍人士的家园，此次袭击不仅针对阿联酋，也是对国际准则的挑战，必须如此对待。
