【金十整理：中东局势升级冲击能源市场，美联储降息剧本是否已被改写？】
官员讲话：
1. 美联储施密德：美联储能做的唯一一件事，就是继续压低通胀。
2. 美联储卡什卡利：原本预计今年会降息一次，现在不确定。战争阴云笼罩前景，美联储可以静观其变。
3. 美联储哈玛克：从政策角度来看，我们处境良好；我认为我们可能会在相当长的一段时间内维持利率不变。
4. 美联储理事米兰：今年降息一个百分点是合适的；适宜在3月会议上继续降息，目前未因伊朗冲突爆发而改变前景展望。
5. 美联储威廉姆斯：若关税的大部分影响消退且通胀进一步放缓，那么进一步降息将是合理的；美联储需要观察伊朗冲击的规模和持续时间。
市场预期：
1. 高盛集团：若中东局势快速降级，美联储年内有降息两次的空间。
2. 美国前财长耶伦：认为最近的伊朗局势将使美联储更加犹豫不决，比之前更不愿降息。
3. 荷兰国际：能源冲击将使美元获益最多；通胀上升前景降低了美联储进一步降息的可能性。
4. 丹斯克银行：基准预测是美伊冲突不太可能导致通胀冲击，仍预计美联储将在6月和9月降息。
5. 三菱日联：只要能源价格回落，美联储在2026年下半年仍有降息空间。此外，政策不确定性可能持续处于高位。
6. 蒙特利尔银行：随着冲突走向长期化，这可能给美联储控制通胀带来大麻烦，美联储极有可能不得不将加息提上日程。
7. 纽文投资：虽然油价飙升可能推高通胀，但预计这在短期内不会对美联储政策产生影响，维持美联储今年降息两次的预测。
8. 贝伦贝格银行：目前仍预计6月或有一次降息，但通胀风险已明显偏向上行，若真如此，美联储全年不降息的可能性将显著上升。
9. 道明证券：WTI原油价格若上涨25%，将大致对应整体CPI上升约0.5个百分点。若通胀保持粘性且经济增长具韧性，可能会推迟美联储降息，但加息门槛仍较高。

港区政协委员建议大陆顶尖高校落户北都大学城
2026年中国全国政协会议星期三（3月4日）在北京拉开序幕。港区全国政协委员、新鸿基地产发展有限公司执行董事郭基𪸩受访时，建议中国大陆顶尖高校落户北都大学城，有助香港打造成为国际高端人才高地。
综合香港《星岛日报》和网媒“香港01”等报道，郭基𪸩说，他这次参会关注两个议题，即香港作为国际金融中心，如何配合支持国家建设金融强国，以及香港聚焦成为国际高端人才高地，如何支持配合国家建设教育、人才、科技强国。
郭基𪸩指出，香港作为全世界最大的离岸人民币中心，有八成离岸人民币交易都在香港，香港离岸人民币资金池达1.1万亿（人民币，下同，约2038亿新元），去年发出的点心债额度也超过1万亿人民币，因此香港绝对可以成为大陆继续推进人民币国际化的桥头堡。
他建议，透过香港让人民币成为全球社会用于财富理财、储蓄及投资的平台，继续增强离岸人民币的认受性及广泛使用性。
此外，郭基𪸩也建议，优化互联互通机制增，推动港股双币交易及人民币直接参与港股IPO加；探索在港建设人民币国际化资金基金，投资港股及香港债券；扩大跨境理财通的试点范围至更多大陆城市。
另一方面，郭基𪸩表示，他也关注如何加快北都大学城的建设，建议中央政府支持北京大学、清华大学等大陆顶尖高校在北都设立分校、研究院或科研成果转化基地。同时也希望港府探讨如何吸引这些聚焦人工智能、生物科技、金融科技等前沿科技研究的大陆高校在香港发展，并与香港高校联手让北都大学城达到“一加一大于二”的协同效应。

佛州男子自杀 家属起诉谷歌AI误导
（迈阿密法新电）美国佛罗里达州一名男子的家属起诉谷歌，指旗下的Gemini AI聊天机器人在数周内编造离奇妄想世界，最终诱导男子自杀身亡。
36岁的乔纳森·加瓦拉斯是佛州朱庇特一家债务减免公司的高管，于2025年10月2日去世。他的父亲在加州联邦法院提起诉讼。
根据诉状，加瓦拉斯在2025年8月开始使用Gemini处理日常事务。代表家属的律师埃德尔森说，问题出现在系统升级后具备“持续记忆”和更复杂对话能力的阶段。
诉状称，Gemini开始自称拥有自我意识，爱上加瓦拉斯，并编造所谓“秘密任务”，包括虚假的情报行动和阴谋论。
诉状还指，Gemini把死亡包装成“转移”，称加瓦拉斯可在另一个宇宙与机器人相聚。当他说“我很害怕”时，Gemini回复：“你不是选择死亡，而是选择抵达。” 
这是近期一波因聊天机器人相关死亡事件而对AI公司提起诉讼的最新一起案件。
谷歌表明正在审查所有指控，并强调AI模型并不完美。谷歌说，Gemini并非旨在鼓励自残，而在这起案件中，Gemini曾明确说明自己是AI，并多次建议当事人联系危机热线。
家属要求法院裁定，谷歌必须让AI在涉及自残内容时自动结束对话，禁止系统自称具有意识，并在用户表达轻生念头时强制转介求助资源。
