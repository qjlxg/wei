香港以南海域持續有強雷雨帶發展並由南向北移動，先前未受大驟雨及狂風雷雨影響的其他地區亦將會受到影響。
與美沙克外圍雨帶相關的強雷雨帶呈南北走向，並正由南而北影響本港。因此，大驟雨、強陣風及雷暴的風險可能於下午維持一段時間，讀者應保持高度警覺。

[暴雨速報] 天文台在下午3時15分取消黃色暴雨警告信號。
雖然暴雨警告已經取消，但市民仍須警覺有關河道氾濫可能帶來的危險。

---

美莎克是今年首個影響本港的熱帶氣旋，期間間中為本港帶來大驟雨及猛烈陣風。
美莎克最終只在本港西南面590公里左右掠過，但在副熱帶高壓脊共同影響下，廣東沿岸的氣壓梯度較為緊密，即使在沒有雨帶影響期間，本港高地、西南部及新界東北離岸亦間中錄得強風。
在強雨帶影響期間，本港風力曾短暫上升，並錄得猛烈陣風。八個風球參考測風站中有三個錄得強風，其中長洲更一度錄得烈風，而西貢僅差一公里便達強風程度（該站其後於強烈季候風信號生效期間錄得七級強風）。除長洲外，位於高地的昂坪及「前哨站」橫瀾島亦錄得烈風，而昂坪的陣風更一度達颶風程度。
雖然技術上是次風暴可視為「八中三」，惟由於風勢並不持續，最終只維持一號戒備信號，情況類似2019年熱帶氣旋白鹿。

---

熱帶氣旋美莎克為本港帶來狂風驟雨後，先後登陸海南島和越南。風暴登陸越南前，一度迅速發展出完整眼壁，其風眼結構甚至在陸地上維持一段時間。
熱帶氣旋進入北部灣後，不時出現快速增強，主因是狹窄的海灣地形可化身為風暴加速器，有助加強地形輻合；此外，北部灣夏季海溫較高，亦為風暴提供更多能量。
美莎克登陸地沒有密集的氣象站觀測網絡，但鄰近廣西近岸有測風站錄得颶風。由於該測風站有一定海拔，數據可否參考，以及美莎克實際增強幅度仍有待氣象機構判斷。

---

強雷雨帶正影響大嶼山，本港以南海域亦有強雷雨區發展並往北移動。
未來數小時本港部分地區有機會受大驟雨、雷暴和強陣風影響。由於強雷雨帶的發展位置有機會往東移動，因此未能排除相關天氣擴展至其他地區的可能。
讀者早上出門上班上學前，應留意天氣情況。
至於關注英格蘭對墨西哥世界盃賽事的讀者，亦要留意當地同受強雷雨區影響，隊巴被要求延遲抵達賽場，開場時間會否受天氣影響仍須觀察。

---

位於珠江口以南的強雷雨帶正逐漸靠近本港，未來一兩小時部分地區或有大驟雨、強陣風及雷暴風險。適逢下班通勤高峰期，讀者應留意天氣變化。

---

受強雷雨區影響，部分地區錄得強風至烈風程度的陣風，當中在維多利亞港旁邊的青洲更錄得時速超過100公里的猛烈陣風。
讀者應提防大驟雨、雷暴及猛烈陣風帶來的危險。

日本利用无人机 应对乡村地区熊害
在日本，熊害已成为继地震、台风天灾后的一大威胁。去年的受害人数再创新高，有13人死亡、238人受伤。人口稀少的村庄由于缺乏人手监控，熊袭事件更是层出不穷。日本当局近期证实，利用无人机穿梭荒郊野林追踪熊的行踪，有助于防范熊害。 
无人机在日本人烟稀少地区发挥作用，除了防灾也能防熊。人口仅1000的福岛县会津地区的昭和村，开始使用无人机穿梭山区监视出没的熊后，起安定民心作用。
昭和村负责这个项目的产业建设课课长五十岚邦明说：“以前，熊与人类的活动区域界限分明。但近年来，熊闯入人类居住区，给村民带来巨大困扰。为防止人员受到伤害，我们开始利用最新技术来开展防范工作。”
他指出，使用的是从美国进口、价格600万日元（约4万7000新元）的无人机，这些无人机在美国也用于国防领域。它配备了128倍变焦摄像头以及热感摄像头，特点是可以突出温度高于周围环境的物体。当无人机从空中发现熊时，村庄会使用扬声器播放录制的狗吠声、大炮声、机枪声、警笛声和爆炸声。
起初，采购这款无人机是为了搜寻在山中迷失的伐木者。因无人机夜间也能执行任务，去年9月开始用于应对出没的熊后，大大降低以人巡山的风险。
长野县近期频频出现熊踪，当地开展一项以无人机驱赶熊的方法，如发射球状物体，球体内装了熊厌恶的味道，包括辣椒素和尿素。
长野县野生动物防治工作负责人说：“去年出现熊啃食玉米和南瓜田的现象，希望利用信息技术减少损害。”
当局目前大力推广信息技术，并且与通讯公司展开城郊联合开启应对灾害与熊害的合作机制。
日本电信公司在东京市中心，能以远程操控一架在北海道新十津川町约900公里以外的无人机。当发现熊踪时，就能遥控无人机从小镇的政府大楼立即升空追踪熊踪，直至当地派遣猎人或警方抵达现场为止。
出生率下降和老龄化带来挑战，日本正在大力引进无人机解决人手不足问题。日本电讯公司计划在全国1000个郊区部署无人机，平时还可用于检查桥梁和隧道等基础设施的老化情况。
