【中东冲突升级致全球航空运输陷入混乱 多条航线票价飙升】金十数据3月6日讯，中东冲突升级，导致全球航空运输陷入大混乱。路透社称，中东地区大部分商业航班停飞导致全球航空运输陷入自新冠疫情期以来“最大混乱”。多名行业专家认为，即使冲突方立即宣布停火，航班秩序也难以马上恢复正常，因为航空公司需要时间重新调配飞机和机组人员、制订航班时刻表并获得安全飞行许可。受多国或地区空域关闭影响，全球多家航空公司为规避风险纷纷调整航线，选择绕行更远路线，加之油价上涨，此举直接推高航空运营成本。美联社5日报道称，目前，近一周内多条航线已出现票源售罄、余票票价畸高的情况，远期预订票价暂未大幅飙升。但专家警示，一旦冲突持续恶化，票价走势将随时发生剧变，旅客出行成本将持续承压。

针对中国去年录得创纪录贸易顺差，下一步如何平衡进口和出口的问题，中国商务部长王文涛表示，关注到顺差、贸易伙伴的想法和观点，下一步会推动贸易平衡发展，稳出口的同时扩进口，并强调中国是全球第二大进口市场，潜力很大。

王文涛表示，中国将实行进出口的平衡。“贸易的平衡发展，主要就是在稳出口的同时，扩进口。把两者结合起来，发挥我们的超大规模市场的优势，扩大农产品、优质消费品、现金的技术设备、和关键零部件的进口。”
他也提到要利用好进博会这样的以进口为主题的展会。“同时中国也主办了共享大市场，出口中国系列活动，这是站在贸易伙伴的立场上，鼓励他们出口到中国，拓展进口渠道，推动更多优质产品和服务进口中国市场”。
