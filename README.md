中国国家发展和改革委员会主任郑栅洁表示，中国初高中及高等教育学龄人口将分别于今年、2029年及2032年排浪式达峰，未来五年将重点建设普通高中和扩容优质本科。

美企首例 五角大楼列Anthropic为供应链风险
（华盛顿综合电）美国五角大楼已正式通知人工智能（AI）公司Anthropic，这家公司及其AI产品已被认定为供应链风险，这进一步升级了双方围绕AI安全措施的激烈争议。
这是首次有美国企业被列为供应链风险，这项认定此前都是针对中国等对手国家的企业如华为。
正式被指定为供应链风险意味着，国防供应商和承包商必须证明他们在与国防部的合作中，没有使用Anthropic的Claude模型，这可能对Anthropic产生广泛影响。
不过，据多家美国媒体报道，美国军方在上周末对伊朗发动的袭击中使用了Anthropic开发的Claude模型，目前也还在继续使用。
Anthropic首席执行官阿莫代伊指出，企业仍可在与五角大楼无关的其他项目中使用Claude，并称这些限制仅适用于五角大楼合同。
阿莫代伊星期四（3月5日）也说，五角大楼的做法在法律上站不住脚，公司别无选择，只能诉诸法院。
这场争端源于Anthropic坚持它的技术不应用于大规模监控或全自动武器系统，这激怒了战争部长赫格塞斯。
华盛顿方面反驳称，五角大楼的行动完全合法，合同供应商无权干涉产品的使用条款。
