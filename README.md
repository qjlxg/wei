彭博：破纪录毕业生叠加AI冲击 中国青年失业率或达20%
随着创纪录数量的高校毕业生步入劳动力市场，叠加人工智能（AI）加速取代初级岗位的冲击，彭博社预计，中国青年失业率可能将逼近20%的高位。
中国国家统计局星期一（7月20日）公布的数据显示，6月份16至24岁劳动力城镇调查失业率录得14.9%。尽管这一数据延续了季节性回落态势，但仍创下官方两年多前调整统计口径以来的同期新高。
在中国官方将高校在校生剔除出统计样本前，中国青年失业率曾一度突破20%。
彭博社分析指出，中国青年失业率通常在6月份触底，并随着更多毕业生在夏末加入劳动力市场而急剧攀升。中国教育部数据显示，今年将有创纪录的1270万名高校毕业生步入职场，较2025年增加近4%。
澳新银行经济学家邢兆鹏预计，随着劳动力供过于求加剧，未来两个月青年失业率将逼近20%。他指出：“由于周期性和结构性挑战，就业压力仍在持续增加。”主要影响因素包括内需疲软，以及职场对AI技术的广泛应用。
在第二季度经济增速超预期放缓并跌破年度目标的背景下，青年群体黯淡的就业前景，进一步凸显官方面临的严峻挑战。在消费和投资持续低迷之际，AI的普及正引发职场变革。花旗集团预测，这最终可能将威胁中国约7000万个工作岗位。
汇丰控股分析师也指出，中国的AI普及率为亚洲最高，甚至超越美国。随着AI技术引发的就业不确定性上升，北京在最新的五年规划罕见地没有设定城镇新增就业人数具体数字目标，为1990年代以来首次。
6月份，中国官方制造业采购经理指数（PMI）中的从业人员指数进一步恶化，延续了自2023年3月以来的收缩态势。长江商学院针对以民企为主的调查结果也显示，6月未来雇佣预期指数已降至九个月来最低点。
此外，旨在协助失业人员再培训并支付部分薪资的失业保险基金，在今年1至5月的支出激增至881亿元人民币（约168亿新元），规模已与2020年疫情期间相当。
中国人力资源和社会保障部星期三（22日）公布，今年上半年，全国城镇新增就业695万人，同比持平，城镇调查失业率平均值为5.2%，就业形势保持总体平稳。

独立书店被查后 香港物流商台湾分司拒运违规书籍
香港警方国安处接连查处多家独立书店后，物流企业嘉达国际物流台湾分公司通告称，为配合《香港国安法》，即日起拒运“违反法规的书籍”，同时拒绝承运曾遭香港特区政府调查的独立书店相关货物。
据香港《明报》报道，嘉达国际物流台湾分部星期二（7月21日）发布通告宣布上述新规。根据通告，客户出货前必须先提供书单，待确认无违法嫌疑后才会安排运输，否则一律拒收。
通告特别点名拒绝承运曾被特区政府调查的书店产品，包括田园书屋、留下书舍、一拳书馆及猎人书店。
嘉达台湾分公司职员受访时说，鉴于香港独立书店接连遭警方国安处调查、甚至有负责人被捕，公司此举旨在提醒客户不要订购及运送“标题耸动”、“批评中国政府”及“让香港政府敏感”的书籍。
当被问及是否受香港当局要求时，该职员称，自《香港国安法》实施以来，港府“动作很大”，公司只是提防万一，整体做法以“安全、平安”为主。
至于如何审视书籍是否合规，职员称公司内部并无“禁书名单”，香港当局也未提供，“但检视权在港府手上”，若货品抵达香港海关受查时才发现有问题将“比较麻烦”。
有物流界从业员透露，香港海关由过往的不定期随机抽查，近日因应书展已改为每周加强抽查。香港海关回应查询时重申，若发现具煽动意图的物品，将依法扣查，并视情况转交警方国安处跟进。
香港警方国家安全处此前以涉嫌出售具“煽动意图”刊物为由，接连查处了两家独立书店。香港保安局局长邓炳强上周强调，书店经营者有责任确保所售书籍不危害国家安全，并表示港府无意设立“禁书”名单。

香港游客在日自驾连出车祸　汽车业者：经验不足过度自信
日本北海道接连发生两起香港自驾游客的车祸，造成一人死亡多人受伤，汽车业者指出，香港游客对在日本驾驶缺乏经验、过度自信。
据网媒“香港01”报道，在日本从事租车行业的港人阿Chun表示，香港游客普遍缺少驾车经验，对车的性能不了解，也不熟悉日本的交通规则。
他指出，香港与日本的驾驶规则有所不同，日本住宅区的道路较窄，道路中间没有白色线分隔两边行车，需要驾驶者之间互相礼让。另外，日本人在经过平交道前，会先停车约三秒，确保没有火车经过且信号灯没有闪烁的情况下，才会通过。这些都是要注意的地方。
阿Chun说，旅客在到外地驾驶时，要具备一定的驾驶经验，建议打算自驾游的乘客，最好先在香港找驾车师傅练习，熟习车的性能，行前也可通过网上影片，先了解日本驾驶规则。
香港汽车会永远荣誉会长李耀培指出，不少港人觉得港日两地同为右舵，因而掉以轻心，但香港车程多为短途，而日本点到点动辄要一个半至两小时，加上人生路不熟分心观看导航、旅游时夜睡等，都会令出车祸的风险大增。
李耀培预计，虽然日本租车公司未必会针对港人加价，但可能会因意外率较高而提高保险费，甚至在热门旅游日期将港人租车的优先级排至较后。他建议计划自驾游的港人，应熟读路线、多用中途休息站，并尽量安排后备司机同行，协助留意导航及提醒路况，减少意外发生。
据“中国驻日本大使馆”微信公号，北海道富良野市某铁路道口上星期四（7月16日）发生一起香港游客自驾轿车同列车相撞事故，五名游客受伤；富良野市星期一（20日）发生一起两辆轿车相撞事故，一名香港自驾游客死亡，五人受伤。
