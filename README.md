彭博社星期一（3月9日）引述知情者称，亚洲私募股权投资公司太盟集团已与潜在顾问展开初步沟通，以评估出售山东凤祥的可行性。
知情人士称，山东凤祥在潜在交易中的估值可能达到7亿至9亿美元（8.99亿至11.55亿新元）。相关讨论仍在进行中，最终未必达成交易。
太盟投资集团代表拒绝对上述报道置评。
除了中国市场，山东凤祥的鸡肉产品还销往亚洲、中东和欧洲等海外市场。该公司于2020年7月在香港上市，并在去年由太盟投资集团通过菁裕企业发展（山东）有限公司实施私有化。
据中金公司官网消息， 太盟投资集团（简称PAG）境内子公司以吸收合并方式完成对山东凤祥股份有限公司（简称凤祥股份）的私有化交易，凤祥股份于去年7月31日从香港联交所退市。
PAG是一家领先的亚太地区另类资产投资公司，拥有不动产、信贷与市场以及私市股权三大核心业务，为约300家全球机构投资者管理超过550亿美元的资产。PAG私市股权投资业务管理的泛亚洲收购基金及成长基金的在管资产逾210亿美元。
凤祥股份是中国最大的白羽肉鸡出口商及领先的鸡肉食品零售企业之一，主要产品包括深加工鸡肉制品、生鸡肉制品、鸡苗及其他。

中国油价今将调整 大概率上调
中国油价将在星期一（3月9日）午夜12时调整，本轮调价大概率将迎来上调。
红星新闻报道，据卓创资讯数据，截至上星期四（3月5日）收盘，中国第九个工作日参考原油变化率为11.35%，预计汽、柴油上调每吨500元（人民币，下同，92.86新元），远超出每吨50元的上调红线。
按照目前的幅度计算，折合升价后，92#汽油、95#汽油、0#柴油将分别上调0.39元/升、0.41元/升、0.42元/升。
南方网报道称，这已为年内最大涨幅。
中国成品油价格定价不随国际原油“秒更”，而是遵循“十个工作日”原则。只要前十个工作日国际原油均价较上一轮调价时上涨超过调价红线，中国成品油限价就会顺势上调。
按照惯例，本轮油价调整的落地时间为星期一午夜12时，最终调整幅度以中国国家发展改革委官方宣布为准。
由于中东局势没有缓和迹象，油价已飙升至每桶100美元（约128新元）以上水平。

台外长出访图瓦卢 回程过境澳洲悉尼
台湾外交部长林佳龙上周访问南太平洋邦交国图瓦卢（台称吐瓦鲁），回程过境澳大利亚悉尼，并与澳洲新南威尔斯州议员麦克德莫特会面。
据台湾外交部上星期六（3月7日）发布的新闻稿，林佳龙应图瓦卢政府的邀请，以台湾总统特使身份于上星期三（3月4日）至星期四（3月5日）率团访问图瓦卢。访团此行受到图瓦卢政府及人民热切欢迎，行程结束后，已在上星期六（3月7日）清晨经澳洲顺利返抵台湾。
台湾外交部称，此次赖清德指派林佳龙担任特使访图，彰显对台图情谊的珍惜与重视。
台湾外交部也提到，台图建交至今，双方在诸多领域长期深入合作，成果有目共睹。未来台湾也将在“总合外交”的理念下，推动“荣邦计划”之进一步落实，促成公私协力，加强布局台湾友邦国家。
民视新闻网报道称，林佳龙访问图瓦卢，回程罕见过境悉尼，造访悉尼歌剧院及悉尼会展中心。他还特地在悉尼歌剧院外，为世界台湾商会联合总会活动录制开幕致词视频。
麦克德莫特（Hugh McDermott）星期一（3月9日）在脸书发文说，上星期五（3月6日）他很高兴与林佳龙会面。
麦克德莫特也说，澳洲和台湾拥有共同的民主原则，并同样关注一个开放、包容，以及稳定的印太地区。
