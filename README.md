古特雷斯出席世界人工智能大会
联合国秘书长古特雷斯抵达上海，准备出席世界人工智能大会。
据中国央视新闻报道，古特雷斯星期四（7月16日）乘机抵达上海浦东国际机场。他将出席7月17日至20日在上海举行的2026世界人工智能大会暨人工智能全球治理高级别会议。
古特雷斯曾多次呼吁强化人工智能全球治理。近年来，联合国设立全球人工智能治理对话和国际人工智能科学小组两项机制，为各国加强沟通、凝聚共识、拓展合作提供了重要平台。
在中美科技竞争日益加剧的背景下，中国国家主席习近平将首次出席在上海举行的世界人工智能大会，显示北京对人工智能的重视程度日益提升。
中国官方早前宣布，习近平将出席大会开幕式并发表主旨讲话。
中国外交部发言人林剑星期一（13日）在例行记者会上介绍，今年的世界人工智能大会主题是是“智能伙伴 共创未来”，习近平将在大会开幕式致辞，全面系统阐述中方对于人工智能发展和治理的政策立场和理念主张。
林剑表示，中方期待以这次大会为契机，为各方增进互信、凝聚共识，深化合作搭建平台，共同推动人工智能健康、安全、有序发展，使这次大会成为人工智能发展史上的一座里程碑。
据彭博社报道，习近平出席这次世界人工智能大会，标志着会议规格较去年有所提升。中国总理李强去年在大会开幕式发表主旨讲话时，对AI垄断提出警告。北京当时还宣布成立一个合作组织，与美国争夺对全球AI标准影响力。
为加强世界人工智能大会期间低空空域安全管控，上海市全域星期三（15日）至下星期一（20日）期间，未经批准禁飞无人机。
中国官媒发文示警“人工智能铁幕”与技术垄断，称中国将促进AI成果共享，反对任何垄断AI技术的企图，预示习近平首度出席中国最高级别科技峰会的基调。

空置中间座位 美联航抢攻愿多花钱买舒适旅客
美国联合航空公司（United Airlines）周二宣布，将在旗下全新空中巴士A321XLR的“经济舱Plus”推出取消中间座位的崭新配置，改设为靠窗与走道乘客共用的桌面，让两侧乘客享有更宽敞的乘坐空间。
美联航说，将是全美首家提供此类座位配置的航空公司，未来不排除推广至其他机型。
近年航空业积极推出多元化升级服务，以满足旅客需求。德国汉莎航空（Lufthansa）与芬兰航空（Finnair）等欧洲业者已在商务舱提供空座保留服务。部分航空公司也允许乘客付费保留隔壁座位，不与陌生人相邻而坐。
今年3月，美联航还宣布明年起将于部分波音广体客机推出“Relax Row”，经济舱乘客在起飞后可将三人座位变成可躺卧休息的空间。
业界趋势显示，航空公司正通过更多座位选项、加购服务与分级票价，抢攻愿意多花钱换取舒适度的客群。然而，服务内容差距日渐扩大也引发部分旅客不满，质疑过去免费的服务如今为何需要额外付费。

全球首款植入式脑机接口产品首例手术在中国完成
中国媒体报道，全球首款植入式脑机接口产品首例手术在中国进行。
中国《新京报》引述上海市相关部门消息报道，由博睿康医疗科技（上海）有限公司自主研发的植入式脑机接口手部运动功能代偿系统（NEO）获批上市后，首例商业化临床手术星期一（7月13日）在上海复旦大学附属华山医院完成。
术中测试显示，系统采集的硬膜外脑电信号稳定、质量良好。患者术后生命体征目前平稳。
NEO在3月获得中国三类医疗器械注册证，成为全球首款获批上市的植入式脑机接口产品。产品适用于因颈段脊髓损伤导致四肢瘫痪、手指无法完成抓握动作的患者，通过微创植入实现大脑意念控制手部运动，填补了长期未被满足的临床康复需求。
据不完全统计，中国现存脊髓损伤患者超370万人，每年新增约9万人，受伤时年龄在50岁以下的患者占比70%以上。《新京报》的报道称，脑机接口展现出广阔的应用场景。
报道也引述未具名机构预测数据称，中国脑机接口市场规模在2027年将达到55.8亿元（人民币，下同，10.63亿新元），年均增长率20%。业内普遍认为，未来脑机接口产业将进入稳步增长阶段，医疗康复将成为主要驱动力，并逐步向更多应用场景延伸。
去年，中国工信部为脑机接口定下“2027年关键技术取得突破、2030年产业创新能力显著提升”的时间表，今年两会（全国人大、政协年会），中国首次把发展脑机接口写入政府工作报告，明确为培育发展的未来产业。
强烈政策信号带动下，相关企业融资加速。博睿康2月办理上市辅导备案，6月9日辅导完成。以非侵入式方式控制假肢的强脑科技，今年1月也以保密形式向港交所递交上市申请，两家公司争夺“脑机接口第一股”。
据《上海证券报》，今年前五个月，脑机接口融资超33宗，总额突破46亿元，超过2025年全年。
