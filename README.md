然而，并非该地区所有国家都愿意轻易放弃中国的投资，以换取与美国的合作。报告作者总结道：“通常情况下，无论是关税还是直接威胁，都无法说服其他国家改变立场。”

在台风“巴威”来袭前，台湾总统赖清德到中央灾害应变中心进行视察，了解各部会、地方政府及台军的防救灾整备工作情况时，一时口误将同行的行政院长卓荣泰叫成“部长”。

广西政府主席乘皮划艇察看洪灾灾情：“乡亲们大家受苦了”
中国广西发生严重洪灾。广西壮族自治区主席韦韬乘坐皮划艇实地察看灾情，官媒报道形容“韦韬动情地”说：“乡亲们，大家受苦了，我代表自治区党委、政府看望慰问你们。”
受台风“美莎克”与水库溃坝的影响，广西多地遭遇严重洪灾。根据官方通报，洪灾已致全区至少39人死亡、九人失联。
据中共广西壮族自治区委员会机关报《广西日报》报道，广西首府南宁市宾阳县、横州市连日来遭遇持续极端强降雨袭击，部分乡镇出现不同程度险情。韦韬星期四（7月9日）前往宾阳县、横州市实地察看灾情，看望慰问受灾民众和一线抢险救援人员，调研督导防汛救灾工作。
报道称，受洪水侵袭，宾阳县甘棠镇街道、村庄被淹，水电、网络、道路大面积中断，是该县受灾最严重的乡镇。韦韬来到甘棠镇，听取宾阳县防汛救灾情况汇报，现场协调解决实际问题；走进敬老院，察看老人照护、物资调配等情况，与老人交流，询问他们的身体状况、生活保障。
横州市云表镇也是受灾最严重的乡镇之一，韦韬前往该镇中心广场，沿209国道实地察看灾情，来到民众家中和医疗点，了解民众受灾和就医救治情况；乘坐皮划艇前往校椅镇六蓝村独田屯，察看受灾和物资保供应情况，慰问消防等一线救援人员；前往横州市镇龙乡，实地督导防汛救灾工作。
报道写道：“‘乡亲们，大家受苦了，我代表自治区党委、政府看望慰问你们。’韦韬动情地说，这次广西遭遇历史罕见强降雨，造成了较大范围灾情，希望大家坚定信心、共克时艰，在党委、政府和社会各界的大力支持下，我们一定会渡过难关。”
在横州市校椅镇前线指挥部，韦韬主持召开现场指挥部工作会议。他强调，当前，防汛救灾和灾后恢复重建还有大量工作要做，时间紧任务重，要抢抓黄金时间，抓好防汛救灾和生产生活恢复各项工作；聚力攻坚重点灾区和地区，全面排查摸清受困群众底数，强化生活物资保障，抓紧抢修水利、电力、道路、通信等受损基础设施，开展环境清淤消杀，严密防范次生灾害，维护社会大局和谐稳定。
韦韬也要求同步做好灾损评估汇总和灾后恢复重建方案，强化资金筹措，做实各项工作，帮助民众早日重建家园。

据中国央视新闻报道，这些科研失信行为案例涉事者来自山西白求恩医院、江西省人民医院、江西赣州市人民医院等等，其中来自江西省人民医院的张萍，以通讯作者或第一作者的身份，出现在五起案例。
针对张萍的处理措施为：科研诚信诫勉谈话；一定范围内公开通报；终身禁止承担或参与科技计划（专项、基金等）项目等财政性资金支持的科技活动；终止或撤销利用科研失信行为获得的财政性资金支持、相关学术奖励和荣誉；10年内取消申请或申报科技奖励、科技人才称号、评审专家等资格；记入科研诚信严重失信行为数据库；追回该论文涉及的科研奖励及版面费资助；责令撤稿。
此外，另一个名叫邵靓的涉事者，则以其他作者和通讯作者的身份，出现在四起案例。在其中两起以其他作者身份的案例中，邵靓被认定不存在科研失信行为，予以科研诚信警示教育。
至于另外两起以通讯作者身份的案例，针对邵靓的处理措施为：科研诚信诫勉谈话；一定范围内公开通报；终身禁止承担或参与科技计划（专项、基金等）项目等财政性资金支持的科技活动；终止或撤销利用科研失信行为获得的财政性资金支持、相关学术奖励和荣誉；10年内取消申请或申报科技奖励、科技人才称号、评审专家等资格；记入科研诚信严重失信行为数据库；追回该论文涉及的科研奖励及版面费资助。

佛州棕榈滩机场正式改名“特朗普”
（棕榈滩综合电）美国再有公家机关改名“特朗普”，在州长德桑蒂斯正式签字批准下，佛罗里达州的棕榈滩国际机场星期四（7月9日）正式改名为“特朗普国际机场”。
为庆祝这一刻，特朗普总统的次子埃里克当天搭乘家族私人飞机“特朗普一号”（Trump Force One）在机场降陆。
他接受《福克斯新闻》访问时说：“我想不到还有谁比特朗普总统更能与棕榈滩甚至整个佛罗里达州画上等号。”
特朗普家族经营的海湖庄园，离机场只有短短车程。埃里克受访时还说，通往机场的主要道路已称为“特朗普总统大道”。
美国联邦航空管理局也从星期四起，将机场代号从“PBI”更改为“DJT”（特朗普英文全名缩写）。不过机场当局说，乘客仍须以PBI来订购机票，直至8月18日。
机场也透露，这次更改名字的花费是550万美元（约710万新元）。
搭客对机场改名的反应褒贬不一，有人称特朗普“显然取得足够成就配得上机场名字”；也有人说：“棕榈滩的品牌更响亮，人们其实不大喜欢特朗普，看起来品味有点差。”

中国对氦气实施临时禁止出口管理
中国商务部、海关总署发布公告，对氦气实施临时禁止出口管理。
中国商务部网站星期五（7月10日）公告，依据《中华人民共和国对外贸易法》有关规定，商务部、海关总署决定对氦气实施临时禁止出口管理，自公布之日起执行，后续相关调整将另行公告。
氦气属于战略资源，用于半导体制造、磁共振（MRI）超导磁体、火箭燃料加压等航天和国防领域。
中国长期是全球最大的氦气进口国之一，出口量有限，2024年从中国进口这一类稀有气体较多的市场包括欧盟、日本、法国、韩国和美国。将氦气纳入管理，反映出中国不断扩大对关键战略物资的出口管理范围。

伯纳姆几乎笃定出任工党党魁接任英国首相
（伦敦综合电）英国议会下院议员、前大曼彻斯特市长伯纳姆在工党党魁选举的首日，获绝大多数工党议员正式提名，已基本锁定党魁职位，有望接任英国首相。
工党星期四（7月9日）启动党魁提名程序。根据选举规则，候选人需获得包括自己在内的至少81名工党下院议员提名，才可进入下一阶段竞选。
目前，56岁的伯纳姆（Andy Burnham）距离党魁职位仅一步之遥。除他之外，尚无其他工党议员获得提名。
他已获得403名工党议员中322人的提名，只需再获得一名议员提名，其他潜在竞争者将无法获得参选所需的81份提名，伯纳姆将成为唯一候选人。
尽管获得压倒性支持，但根据选举程序，伯纳姆仍需待7月17日完成相关程序后，才会被正式宣布当选工党党魁。预计他将于下一个工作日，即7月20日，接受国王任命并出任英国首相。
伯纳姆在确认提名得票数后，对党内同事的支持“深表感谢”，并称“这反映了大家的共同信念，即英国需要一种新的政治方式”。
伯纳姆因连续三次赢得大曼彻斯特市长选举，被昵称为“北方之王”。他誓言要“推动我国有史以来最大规模的权力再平衡”。
他承诺将严格执行财政纪律，并削减不断膨胀的福利支出。此前，他已承诺遵守政府目前的借贷上限，以稳定市场。
不过，伯纳姆将面临与斯塔默近两年的首相任期中遭遇的相同挑战，即经济增长乏力、生活费上涨，以及美国总统特朗普的不可预测性。这些挑战最终导致斯塔默上个月辞职。 
伯纳姆接受英国《卫报》独家采访时说，英国必须就以色列在加沙冲突中的行动，向以色列施加更大压力。
他指出，许多人认为，斯塔默政府在应对以色列于2023年10月哈马斯武装分子袭击加沙后对加沙发动的军事行动时“处理不当”。
