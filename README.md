泰国新版《土地分配法》（第3号）2568年修订版将于2569年3月1日起生效，副政府发言人拉利达于2月28日宣布。此次修法旨在加强对分配土地买方的保护，明确分配方的监管责任，符合宪法第26、37、40条关于以法律为依据并具必要性的限制权利自由的要求。
主要修订包括：
1）强化公共设施管理：将道路、绿地、儿童游乐场等纳入为买方利益的役权，分配方须维持设施标准，并须向银行或金融机构提供维护担保；
2）明确解除维护义务的条件：分配方仅在买方成立小区法人接收资产或将财产捐作公用并移交担保金后方可免责；
3）赋权买方自行动手：当分配方不履责时，已售地块过半的买方可申请成立小区法人；
4）调整公摊费计收原则：可依土地用途或面积大小差别定费；
5）加重执法与处罚：违令者可处5万至10万泰铢罚款，并按日罚款直至改正。
发言人表示，修法将厘清分配方责任、减少小区争议，为购地民众提供更实质的保障，政府致力提升土地分配制度的透明、公平与民众权益保护。

泰国新版《土地分配法》（第3号）2568年修订版将于2569年3月1日起生效，副政府发言人拉利达于2月28日宣布。此次修法旨在加强对分配土地买方的保护，明确分配方的监管责任，符合宪法第26、37、40条关于以法律为依据并具必要性的限制权利自由的要求。
主要修订包括：
1）强化公共设施管理：将道路、绿地、儿童游乐场等纳入为买方利益的役权，分配方须维持设施标准，并须向银行或金融机构提供维护担保；
2）明确解除维护义务的条件：分配方仅在买方成立小区法人接收资产或将财产捐作公用并移交担保金后方可免责；
3）赋权买方自行动手：当分配方不履责时，已售地块过半的买方可申请成立小区法人；
4）调整公摊费计收原则：可依土地用途或面积大小差别定费；
5）加重执法与处罚：违令者可处5万至10万泰铢罚款，并按日罚款直至改正。
发言人表示，修法将厘清分配方责任、减少小区争议，为购地民众提供更实质的保障，政府致力提升土地分配制度的透明、公平与民众权益保护。

卫士理事会由12名成员组成（6名由最高领袖任命的神职人员），负责审核选举候选人并有权否决议会通过的法律，从而间接控制选举结果。总统和议会虽经选举，但候选人须经卫士理事会审查、其通过的法律亦可被否决。 
专家议会名义上能选举或罢免最高领袖，但实践中从未公开挑战其权威。和解理事会由最高领袖任命，裁决议会与卫士理事会之争。伊斯兰革命卫队（含海外的圣城部队）直接向最高领袖负责，在政治、经济与对外行动上影响巨大。最高国家安全委员会负责国防与安全协调。危机时决策通常收窄至宗教领导层与军方的核心圈，这将决定伊朗下一步的应对。
