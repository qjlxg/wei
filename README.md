中国全国人大代表哽咽建议 提升农村70岁以上老人养老金
（广州／郑州／成都综合讯）中国全国人大代表毕利霞在两会（人大、政协年会）期间哽咽发言，呼吁提高农村70岁以上老人的养老金标准，相关建议已引发多个部门关注。
综合《南风窗》和河南广播电视台旗下大象新闻报道，毕利霞上星期六（3月7日）在湖北代表团小组审议中建议，将农村70岁以上老人养老金提高至每月400元（人民币，下同，74新元），并免除其居民医保个人缴费。
毕利霞哽咽道，中国老一辈农民为国家农村改革发展奉献了一辈子，“如今老了，肩不能挑、手不能提”，每月仅有100多元的养老金，生活十分艰难。
毕利霞的建议引起相关部门关注，她在会后证实，中国财政部、民政部等部门就她的建议通过电话与她进行沟通。
中国农村养老金全称为城乡居民养老保险，由基础养老金和个人账户养老金两部分组成。其中，基础养老金由中央和地方财政承担，在2025年的国家最低标准为每人每月143元。
今年的中国政府工作报告提出，2026年城乡居民基础养老金每月最低标准提高20元至163元。
城乡居民养老保险待遇偏低，长期以来是中国社会保障的短板。今年全国两会上，有10多名代表委员建言提高农民养老金水平。
红星新闻引述中国社会保障学会会长郑功成称，养老金制度需要理性建制与理性发展，补偿高龄农民的年龄设定要根据财政实力来定，如果财力有限，可先从80岁及以上的农村老年人开始，后续根据财力再逐步提高标准。

伊朗战事挤压飞行空域 中东航空枢纽地位面临考验
（香港 / 格但斯克综合电）波斯湾地区机场数十年来致力于成为全球航空枢纽，但当前伊朗战争导致中东空域缩小，航班锐减，迪拜、阿布扎比和多哈等长途航程关键中转站都受到打击。 
2022年俄罗斯入侵乌克兰后，许多航空公司开始避开俄罗斯领空，如今中东战争切断飞越伊朗和波斯湾的航线，中东空域进一步受限，航班只能绕道改飞格鲁吉亚和阿塞拜疆上空的一条狭窄空域。
冲突也导致航班时刻表混乱，数万名旅客和数百架飞机滞留。
航空数据分析公司Cirium说，2月28日至3月11日间，往返中东的4万6000多个预定航班取消。尽管航空公司正努力恢复运营，但还不清楚他们能多快恢复正常航班。
不过，这也为欧洲和亚洲航空公司提供难得的机会，可夺回部分被阿联酋航空、卡塔尔航空、阿提哈德航空等公司抢走的业务。
随着更多长途旅客寻求绕飞中东，英国航空和德国汉莎航空增加了往返亚洲和非洲的航班，法荷航空从东京、上海、孟买等亚洲城市起飞的班机改用更大型客机，印度航空和国泰航空也因机票迅速售罄而增加航班。
但飞行限制拉长飞行时间，对整体航空业而言，是一大挑战。以阿联酋航空为例，公司大多数航班因绕飞波斯湾、伊朗和伊拉克领空，飞行时间增加至少一小时，耗油量也增加。
自美以对伊朗开战以来，原油价格上涨约三成，航空燃油价格也随之上涨，但涨幅却远超原油，达到一倍之多，航空公司的利润被显著压缩。 
航空专家埃尔纳斯告诉路透社，历史数据显示，中东危机等动荡事件会导致燃油成本持续数月保持在高位。
美国和中国的主要航空公司都没有订立燃油保值合约对冲油价波动风险，因此将全面受到燃油价格波动的影响。
摩根大通预测，如果燃油价格上扬10%，将导致法荷航空、汉莎、英航等欧洲航空公司今年的营业利润下降3%至10%。
分析员指出，亚洲航空公司大多没有对冲措施作为保护，或是仅签订保值合约对冲原油而不是航空燃油的价格波动。唯有新加坡航空公司和维珍澳洲航空例外，采取了对冲措施，预防燃油价格上涨。
为维护利润，航空公司已开始调高机票价格，连不飞中东航线、已对冲油价波动至今年6月的新西兰航空和澳洲昆达士航空也上调了票价。
