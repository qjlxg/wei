一道低壓槽會在星期一稍後開始在華南沿岸徘徊，屆時本港天氣漸轉不穩，大驟雨及狂風雷暴的風險會在本週中期持續。
預料低壓槽內可能有低空渦旋發展。除非該渦旋發展位置顯著偏南，疊加副熱帶高壓脊相關的東風充分配合，否則其發展為熱帶氣旋的機會暫時偏低。
視乎低空渦旋掠過的位置，水氣有機會被加強抽入至沿岸抬升，並誘發本週中期強雷雨區發展。本港雨勢強弱、雨區發展時間和影響範圍仍有變數。
雖然低壓槽及低空渦旋會在本週後期消退，但緊接而來的是活躍西南氣流，因此本週後期天氣仍會不穩定。讀者本週外出前應留意最新天氣消息，並留意已安排的戶外活動可能受到影響。

---

天文台表示，南海北部廣闊低壓槽上有一個低壓區會逐漸發展，並靠近廣東沿岸。若該低壓區發展為熱帶低氣壓，會評估是否在今日傍晚至明早發出一號戒備信號。
天文台亦表示，低壓系統相關的對流雲團現時集中於南側，預料尚需一些時間整合，在較為接近廣東沿岸時，才能確定會否發展成熱帶低氣壓。
按照現時預測，該低壓系統明早最接近，會在本港100公里範圍內掠過。天文台預料明日本港會有大驟雨、雷暴及狂風，可能需要發出暴雨警告信號或局部地區大雨提示；離岸吹強風，部分低窪地區有機會出現水浸。

---

受南海北部低壓系統影響，預料本港明早（星期二）天氣轉差，有大驟雨和狂風。星期三、四天氣繼續不穩。
目前低壓系統對流集中在南側，而且尚未發展出明確和穩定的中心。然而，受近岸地形輻合影響，未能排除中心加快整合，導致低壓系統發展為熱帶氣旋的可能。
按照現時預測，低壓系統明早會近距離掠過本港，但實際路徑要視乎中心整合情況。由於低壓系統大風區集中在東側和南側，若在本港以西掠過，風力會較大；至於東登的話，風力會在低壓系統掠過本港以北時有所上升。
低壓系統靠近時，近岸地形輻合或誘發對流發展；此外，低壓系統登陸後會引進西南氣流，亦有利強雷雨區發展。因此，預料未來兩三日本港不時受大驟雨影響，強雷雨區掠過期間或有短暫風力上升，有機會伴隨雷暴。然而，強雷雨區發展位置和影響時間存在一定變數。
無論掛波與否，本港未來兩三日天氣亦會轉差，讀者要密切留意天氣情況，MET WARN 亦會跟進最新消息。

自民党制定AI安全法则的议员平昌明说，AI已成为国家安全和经济发展极其重要的问题，在使用上有必要加强政府的参与，并且避免过度依赖特定国家或公司。为此，日本必须加强AI自主研发。
据悉，美国AI技术公司Anthropic最近发布具备发现网络安全漏洞和高性能攻击能力的旗舰模型Claude和Mythos，促使当局着重于AI的安全问题。 
据报道，日本当局优先考虑金融领域的应对措施，以及石油天然气和电力公司等关键行业的AI使用，不但要加强这些基础设施的检查，还将建立风险评估体系，增强应对网络攻击能力。

这项协议是首个旨在覆盖欧盟所有成员国和企业的无人机协议。欧盟执行机构委员会说，双方的伙伴关系将在2028年前扩展至反弹道导弹领域。
欧盟委员会在一份声明中说，将与参与这份协议的19个创始合作伙伴开展合作。这项新倡议将打破官僚主义壁垒，制定共同标准，使乌克兰和欧盟企业更容易开展合资项目。
泽连斯基在仪式致辞中表示：“我们将竭尽所能，整合欧洲所有反弹道导弹能力，构建欧洲反弹道导弹系统。”
这反映出，欧洲政界人士和企业日益渴望利用乌克兰在俄乌战争中积累的专业知识，以增强自身抵御莫斯科的能力。
冯德莱恩对泽连斯基说：“你们在无人机和反无人机系统操作方面积累的知识独一无二。”
“我们深知欧洲在这一领域面临的威胁，因此我们必须共同利用好这些知识。”
