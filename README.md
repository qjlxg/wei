朝方公开的照片显示，舰尾垂直发射系统（VLS）连续发射疑似“箭”系列战略巡航导弹，127毫米主舰炮、舰空导弹和近程防御系统（CIWS）也同步进行测试。
专家指出，相较于首舰崔贤号，姜健号的试验周期明显缩短。
韩国统一研究院高级研究委员洪珉说，崔贤号曾历经数月完成单项武器、传感器、指挥控制及战略武器联动等验证，姜健号则一次完成目标探测、情报处理、综合火力和实弹射击等测试，显示朝鲜已将首舰经验应用于二号舰，新型驱逐舰的试验和形成战斗力程序正逐步标准化。
韩国国防安全论坛事务总长申钟宇认为，朝鲜正试图借姜健号尽快服役，扩大东西海海域的海上核打击能力。此次不仅展示战略巡航导弹，还首次公开多种近程防御武器组成的密集火网。
公开画面显示，舰体两侧部署30毫米级近程防御炮和14.5毫米自动机关炮等武器，反映朝鲜希望通过多层近程防御体系，弥补舰艇中短程防空能力不足。
分析认为，崔贤号和姜健号相继形成战斗力，显示朝鲜正推动海军作战范围由沿岸向远海延伸，加快由近岸防御型力量向远洋作战力量转型。
不过，专家指出，朝鲜距离真正具备远洋舰队作战能力仍有不小差距。大型驱逐舰持续运作不仅需要舰艇本身，还需专用码头、维修补给、舰员训练、指挥控制和远海航空支援等配套体系。
金正恩此次再次要求建设大型、多功能舰队基地并扩大造船能力，也反映朝鲜已意识到基础设施建设滞后于舰艇发展速度。“有驱逐舰、缺基地”，仍是朝鲜海军现代化的现实瓶颈。
韩国媒体猜测，姜健号最快可于今年9月朝鲜建政纪念日前后正式服役。
若朝鲜再于10月建党纪念日前后推动第三艘同级舰下水，其大型水面舰艇规模将进一步扩大，但“海军核武装化”能否形成真正的海上核威慑，仍取决于基地、后勤和指挥体系等基础能力能否同步完善。

中国国家一级文物永乐款菩萨像上现TCL广告字样
中国工艺美术馆展出的国家一级文物菩萨像上，被指出现科企“TCL”广告字样。馆方称已将相关情况和现场照片反馈主办方，问题引起单位领导高度重视。
潇湘晨报报道，有参观者发现，近日展出的“高原丝路 瞿昙之光——青海丝路文物与瞿昙寺壁画艺术展”核心展品——永乐款鎏金铜观音菩萨像展柜内，从造像下方仰观，观音裙摆、脚踝区域能清晰看见“TCL”字样。
中国工艺美术馆工作人员星期天（7月5日）核实情况后回访表示，确实能透过展柜玻璃看到“TCL”相关文字，类似报纸印刷画面。
工作人员说，上述展览由青海省博物馆主导承办，中国工艺美术馆仅提供展出场地，展柜开启、文物近距离查验等操作权限归属青海文博专业人员，展馆现场工作人员无法自行开柜核查、清理。
馆方已将观众反映的情况、现场取证照片同步反馈青海省博物馆，单位领导高度重视该问题。
青海省博物馆工作人员则称，针对观众看到的广告字样问题并不清楚。

谴责大陆新法提案遭蓝白否决 民进党批封杀讨论是“价值选择”
台湾在野党在立法院凭借人数优势，否决民进党针对中国大陆《民族团结进步促进法》施行所提出的谴责案。民进党批评蓝白两党联手封杀讨论，并直指此次表决是一次“价值的选择”。
中国大陆《民族团结进步促进法》7月1日起正式生效，规定将依法追究实施破坏民族团结、制造民族分裂行为的境外组织和个人。该法在台湾引发高度警惕，舆论担忧这可能将为北京提供新的法律依据，借以惩治被界定为“台独”分子的民众。
综合台湾《联合报》《自由时报》报道，民进党星期五（7月3日）在立法院提案，拟对上述新法的施行表达谴责，但遭到国民党和民众党以人数优势否决。
民进党中国部星期天（7月5日）在脸书发文称，《民族团结进步促进法》不是一般国内法，是企图将中国大陆的政治控制扩张到全世界，让海外人士、港澳居民，甚至台湾民众都可能因为言论或立场成为追究对象。
民进党中国部称，面对这样一部引发国际高度警戒的法律，民进党团提出谴责案，希望立法院向国际社会表达立场，谴责中国大陆进行跨国镇压、侵害人权，守护台湾人民安全与自由。“没想到，蓝白60名立委联手投下反对票，让谴责案连讨论机会都没有，程序阶段遭到封杀。”
民进党中国部批评国民党替中国大陆找理由开脱，甚至不惜站在国际民主社会的对立面；不仅如此，国民党、民众党联手阻挡立法院讨论，“蓝白为何那么害怕讨论中国恶法”。
民进党中国部称，台湾民众应该记住这次表决，“当民主国家共同反对跨国镇压时，蓝白却选择让台湾的立法院不讨论。这次表决，不只是一次程序投票，更是一次价值选择。”
