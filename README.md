一道低壓槽會在星期一稍後開始在華南沿岸徘徊，屆時本港天氣漸轉不穩，大驟雨及狂風雷暴的風險會在本週中期持續。
預料低壓槽內可能有低空渦旋發展。除非該渦旋發展位置顯著偏南，疊加副熱帶高壓脊相關的東風充分配合，否則其發展為熱帶氣旋的機會暫時偏低。
視乎低空渦旋掠過的位置，水氣有機會被加強抽入至沿岸抬升，並誘發本週中期強雷雨區發展。本港雨勢強弱、雨區發展時間和影響範圍仍有變數。
雖然低壓槽及低空渦旋會在本週後期消退，但緊接而來的是活躍西南氣流，因此本週後期天氣仍會不穩定。讀者本週外出前應留意最新天氣消息，並留意已安排的戶外活動可能受到影響。

日本改革离婚子女监护制度 擅自带走孩子仍难遏止
日本允许离婚父母实行共同亲权的《民法》修正案今年4月生效，打破过去离婚后只能由一方拥有亲权的制度。
不过，法律界人士及多名与子女失去联系的家长认为，新法执行机制不足，父母在离婚前率先带走孩子、借此取得抚养优势的现象恐难消失。
美国有线电视新闻网（CNN）报道，日本家事法院过去处理争议时，往往倾向维持孩子与现有照顾者共同生活的安排。
一些家庭律师因此会建议当事人在离婚程序开始前先带走孩子，以确立主要照顾者地位。这种做法在一些国家可能被视为父母拐带儿童，但在日本一般不被当作刑事案件处理。
日本法务省说，新法要求离婚父母从子女最佳利益出发，共同承担养育责任。若一方擅自带走孩子，法院日后审理亲权争议时，可能作出对这一方不利的判断。
不过，共同亲权并非自动适用于所有离婚家庭。
家庭律师基塔卡（Masami Kittaka）指出，共同亲权只意味着父母双方对教育、医疗等重大事项拥有同等决定权，并不保证共享育儿时间或探视权。
国际律师本田（Masayuki Honda）也认为，由于率先带走孩子的一方通常不会受到实质惩罚，新法仍不足以消除这种诱因。
美俄双重国籍女子明科娃（Anastasiya Minkova）说，丈夫去年趁她出国探亲时带走2岁儿子。她上一次见到儿子已是六个月前，当时只能在福利机构工作人员陪同下相处30分钟。
美国公民佐藤（Emily Sato，化名）也说，丈夫2022年带走女儿后，法院把孩子与父亲共同生活的安排视为不宜打破的稳定环境。她自2025年11月以来再也没有见过女儿。
日本法务省回应说，当局一直依据国际条约和《海牙国际儿童诱拐民事方面公约》妥善处理跨国儿童带离案件。
法务省称，2017年至2025年，东京和大阪家事法院一审案件中发出儿童返还令的比例为70%，上诉案件为63%，均高于59%的国际平均水平。
共同亲权也在日本引起争议。部分反对者担心，新制度可能迫使家暴受害者继续与前伴侣保持联系。不过，多名律师说，新法允许法院在涉及家暴或虐待儿童的案件中，维持单独亲权或安排双方不接触。
