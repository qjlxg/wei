受伊朗战争影响 中国买家为加拿大原油支付溢价
由于伊朗战争导致中国在中东主要原油来源的供应受限，中国买家正为加拿大原油支付溢价。
据彭博社报道，ATB Cormark Capital Markets董事总经理奥罗克说，由于油轮租赁费用上涨，加上中东保险费用因战争风险上升，加拿大原油目前在亚洲市场具有每桶2至3美元（2.54至3.81新元）的价格优势。
据彭博社汇编的航运报告显示，过去两周至少有五艘邮轮被初步预订，计划将加拿大阿尔伯塔省油砂原油运往中国。这些邮轮由中国贸易商联合石化美洲、壳牌和埃克森美孚等公司租用。
这些预订表明，托运人愿意在成本更高的现货运费市场上运输原油。彭博社的计算显示，自本月初以来，运费已飙升超过70%。
美伊冲突持续超过一周，虽然伊朗官方否认已全面封锁霍尔木兹海峡这个全球五分之一原油运输的关键通道，但航运不确定性导致国际油价剧烈波动，星期一（3月9日）一度接近每桶120美元（152新元）。
另据彭博社星期二（10日）报道，短短几年内，原油已成为加拿大最大港口的主要出口商品之一，而中国成为其最热衷的买家。
尽管加拿大绝大部分石油仍然销往美国，但美国总统特朗普2025年大部分时间里加征关税，并声称要把加拿大变成美国的一个州，导致加拿大致力于贸易伙伴多元化并减少对美国依赖。

台陆委会副主委称北京试图改变台海现状 吁台各界认清风险
台湾政府的大陆委员会副主委沈有忠谈及两岸关系时预计，北京未来将持续加大在印太地区的军事演训与灰色地带胁迫，试图通过军事施压改变台海现状，并呼吁台湾各界认清相关风险，不要配合北京操作。
据陆委会官网消息，沈有忠星期二（3月10日）在“中共‘两会’后对外情势及区域安全前瞻”座谈会上说，中国今年军费预算为约1.9万亿元人民币（3528亿新元），同比增长7%，规模已超过台湾的11倍。这显示北京在面对经济发展挑战之际，仍优先将资源投入军事扩张，对台湾及周边国家持续形成威胁。
他指出，这也预示北京未来将持续加强在印太地区，特别是针对“第一岛链”的军事演训与灰色地带行动，试图通过军事施压改变台海及区域现状。
沈有忠并从经济、国防、外交与两岸交流四个方面提出建议。在经济方面，他呼吁台商审慎评估中国大陆当前的经济环境，认为赴陆投资将面临经济与政治的双重风险；在国防方面，他呼吁民众团结支持国防预算，强化台湾的自我防卫能力。
在外交方面，沈有忠呼吁社会支持政府外交政策，持续深化与美国、日本等理念相近国家的实质合作，而非以“不选边”立场削弱国际社会对台湾的支持，并称台湾永远站在自由民主的一边。
在两岸交流方面，沈有忠呼吁台湾任何团体、个人或政党在进行两岸交流时，应看清北京加大对台统战与分化的意图，并称北京坚持的“九二共识”是“消灭中华民国的主张”，任何政党或个人都不应配合北京的政治操作。
沈有忠指出，台湾政府从未放弃在对等尊严基础上与北京对话，但不会以削弱防卫能力、甚至让步妥协的方式，“把国家主权交由他人决定”。他认为，只有展现自我防卫的决心，加强国防并深化与民主伙伴的合作，台湾才能在国际局势变化中立足，确保世世代代的台湾民众继续享有民主自由的生活方式。
