“网民在微信公众号发布‘官员简历’不属于编造谣言或者泄露国家秘密行为，公安机关暂无法律依据对相关网民开展立案侦查或予以行政处罚。此外，对夸大落马官员贪腐情节、编撰绯闻轶事等行为，公安机关同样无权率先辟谣。‘警方不掌握相关落马官员的具体犯罪事实，因此无法认定网民发布的信息为谣言。’”
官方目前对此类现象持严厉打压态度。文章将此类“听床”信息定性为“国家重要的政治信息”，并称这种炒作“破坏了相关工作的严肃性”。

据澎湃新闻报道，这项“新影像·新影响”2026全民摄影大赛共收到近400件投稿，经专业评审团评选出18件获奖作品。其中，4月月赛优秀奖作品《风雨路》因画面中车牌乱码等疑似AI生成痕迹，引发网民质疑。
对此，惠州市文化馆工作人员星期五（7月17日）回应称，经核对作者提供的原图，确认照片为3月30日在高速公路上用手机实地拍摄，但作者承认在后期制作中使用AI添加了一道闪电。
工作人员指出，本次赛事规则并未明确限制AI使用，因此认可实拍素材搭配轻度AI修改的作品，评选阶段也未能通过肉眼识别出AI处理痕迹。主办方称，目前暂未收到市民投诉；若后续引发公开争议或收到有效反馈，将组织摄影评委重新评议，研判是否对作品作出处置。
根据比赛征稿要求，参赛作品虽可进行后期处理，但不得改变原始影像，也不接受无摄影元素、纯粹依靠电脑合成或拼贴的作品。
这并非中国摄影赛事首次卷入AI风波。此前，内蒙古呼和浩特一项摄影比赛一等奖作品被证实为AI生成，主办方宣布暂停赛事并全面整改；全国古树名木保护摄影大赛一件入围作品被指涉嫌AI篡改，作者也被永久取消参赛资格。

科威特消防部门通过社交媒体X发布消息说，他们正在处理两处不同地点发生的火灾，火灾造成多名消防员和一名工人受伤。
科威特国家航空公司指出，由于火箭弹和无人机袭击，科威特国际机场的空中交通暂时中断，公司正在重新安排大部分航班。

中国外交部：密切关注英钢国有化 必要时将采取维权措施
英国政府通过立法将中国企业旗下的英国钢铁公司收归国有后，中国外交部说，正密切关注此事进展，必要时将采取维权措施。
中国外交部星期六（7月18日）在官网以发言人答记者问形式说，英国如何处理此事将直接影响中国投资者对英投资环境的看法，影响中国民众对英国政府信誉的看法。
发言人提到，中英签有投资保护协定，投资者的正当合法权益必须依法得到充分保障，并敦促英国切实尊重市场原则和契约精神，找到包括如何赔偿等双方都能接受的解决方案。
发言人还说，中国支持企业运用法律手段维权，正密切关注事件进展，必要时将采取维权措施。
中企敬业集团2020年收购英钢，但在英国政府去年介入这家陷入困境的企业后，敬业集团已失去对公司的控制权。
敬业集团此前指出，位于英国斯肯索普的工厂已不具备财务可行性，导致这座英国最后一家能够从原料开始炼钢的工厂面临关闭风险。
英国政府随后在2025年4月介入并接管工厂运营，并在今年5月宣布将推动立法，实现英钢国有化。
在英国政府星期四（7月16日）宣布完成相关立法后，中国商务部同日回应称，中国敦促英国政府遵守相关国际规则，切实履行中英投资保护协定有关义务，公平公正对待在英中资企业，充分保障正当合法权益。

德国安全警戒升至高威胁层级 防范攻击风险
基于近期频繁收到的恐袭报告与情报，德国政府已正式将全国的安全警戒级别从先前的“潜在威胁级别”，提升至最高防范的“高威胁级别”。德国内政部长警告，针对德国境内的袭击计划已“显而易见”，民众必须时刻保持高度警惕。
据路透社报道，德国内政部长多布林特（Alexander Dobrindt）星期六（7月18日）接受德国报纸《星期日世界报》（Welt am Sonntag）访问时表明，由于安全局势恶化，当局决定上调警戒级别。
他说：“提升警戒级别意味着在德国，我们必须时刻警惕袭击风险。目前针对我国基础设施、个人和机构的袭击计划，已经是显而易见的了。”
德国内政部目前尚未就具体情报内容以及即将部署的安保细节作出正式回应，也未公开更多细节。
德国近年来曾频繁遭遇致命的恐怖袭击与暴力事件，社会安全防线屡受冲击。其中一起引发国际社会关注的严重案件发生于2024年圣诞节前夕，一名在德国执业的沙特阿拉伯医生驾驶租来的宝马轿车，在东部城市马格德堡（Magdeburg）一个历史悠久的圣诞集市疯狂冲撞人群，造成六人死亡、数百人受伤。 
在另一起极端暴力案件中，一名受到极端组织伊斯兰国（IS）思想蛊惑的叙利亚公民，在西部城市索林根（Solingen）举行的建城庆祝活动期间突袭行凶，疯狂持刀砍杀现场民众，造成三人死亡、10人受伤。德国法院已在去年裁定凶手罪名成立并收监。
