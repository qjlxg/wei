香港建筑地盘周五起全面禁烟
香港建筑地盘工地将从星期五（7月17日）起全面禁烟。
综合香港电台和网媒“香港01”报道，香港新界大埔宏福苑去年11月发生致命大火，造成168人遇难后，引起社会关注建筑地盘吸烟问题。
香港特区政府之后向立法会提交关于建筑地盘全面禁烟的三项法例修订，其中两项将以“先订立、后审议”的程序推行。如果获得立法会通过，将从星期五起刊宪生效。
其中《2026年吸烟（公众卫生）条例（修订附表2）令》指定建筑地盘为禁止吸烟区，包括大厦维修工程地盘。
《2026年定额罚款（吸烟罪行）（指明主管当局及公职人员）（修订）公告》则指明劳工处职业安全主任可依据《定额罚款（吸烟罪行）条例》，执行所有涉及在建筑地盘吸烟罪行的执法工作。
任何人在建筑地盘吸烟即属违法，职安主任可向违例者发出定额罚款通知书，定额罚款3000港元（493新元）。地盘承建商也须确保没有人在地盘内吸烟，违规承建商可罚款40万港元。若地盘有人吸烟而构成严重火警风险，劳工处会按相关条例起诉涉事的东主或雇员，东主可判处罚款300万港元及监禁六个月，雇员则可判处罚款15万港元及监禁六个月。
上述修订法例决议案星期三（15日）在立法会获得议员通过，将从星期五起实施。
对此，香港劳工及福利局局长孙玉菡说，新法规将于星期五刊宪后正式生效，意味着执法工作立即展开，包括不事先通知日常进行的突击巡查，也不会有缓冲期和警告期。
他也说明，举证方面，如果有工人违规吸烟的相片、影片作为证明，能辨别违规者的样貌身份、事发地点和时间，例如地盘出入纪录等，以上证据齐全，港府就会执法。
