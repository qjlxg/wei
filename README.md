多年摩擦后 印度批准有限放宽对华投资限制
印度政府星期二（3月10日）宣布，已批准在部分领域放宽对中国投资的限制，以缓解资本紧张，显示在经历六年摩擦后，中印经济关系逐步回暖。
综合路透社和彭博社报道，印度政府发布的声明显示，总理莫迪领导的内阁已批准修改外国直接投资（FDI）规则。此前相关规定限制来自中国及其他与印度陆地接壤国家的投资；调整后，将允许中国资本进入电子、资本设备和太阳能电池等领域。
声明称，只要企业始终由印度居民保持多数股权，中国在上述领域的投资申请将在60天内完成审批。
2020年，中印两国军队在喜马拉雅边境发生冲突，造成20名印度士兵和四名中国士兵死亡，两国关系随之恶化。此后，印度政府加强对中国企业投资的审查，中国实体投资印度公司必须获得由印度内政部和外交部组成的小组的安全许可。
由于相关限制，一些交易被迫搁置。例如，中国车企比亚迪在2023年提出、计划投资10亿美元在印度设立电动车合资企业的方案最终未能推进。
印度当天还宣布，允许中国持股比例不超过10%的投资者，在符合相关行业投资上限的情况下，通过“自动审批路径”进行投资。
声明称，新指导方针将有助于吸引更多外国直接投资、引入新技术，并加强印度与全球供应链的整合。
报道称，在美国关税引发全球产业链调整的背景下，印度正考虑以更审慎的方式调整对华关系，以稳定供应链并吸引更多投资。
莫迪去年8月访华，并在与中国国家主席习近平会面时讨论改善双边关系的途径。此后，印中两国恢复直航航班，新德里也简化了中国商务人士的签证程序。路透社上月报道，印度已放宽国有电力和煤炭企业采购中国设备的限制。

“养龙虾”爆火 中国官方发布风险提示
“养龙虾”近期在中国爆火，中国国家互联网应急中心发布风险提示指出，由于有关默认的安全配置极为脆弱，攻击者一旦发现突破口，便能轻易获取系统的完全控制权，并呼吁在部署和应用“小龙虾”（OpenClaw）时，采取安全措施。
近两个月，“养龙虾”成为中国的一大热词。这是指一款名为OpenClaw的AI智能体（AI Agent）软件，标志是一只龙虾，中国网民便把安装并使用它的过程称为“养龙虾”。
据国家互联网应急中心微信公众号星期二（3月10日）消息，近期OpenClaw应用下载与使用情况火爆，中国主流云平台均提供了一键部署服务。此款智能体软件依据自然语言指令直接操控计算机完成相关操作。为实现“自主执行任务”的能力，该应用被授予了较高的系统权限，包括访问本地文件系统、读取环境变量、调用外部服务应用程序编程接口（API）以及安装扩展功能等。
国家互联网应急中心指出，然而，由于其默认的安全配置极为脆弱，攻击者一旦发现突破口，便能轻易获取系统的完全控制权。
对此，国家互联网应急中心建议相关单位和个人用户在部署和应用OpenClaw时，采取四项安全措施。首先，强化网络控制，不将OpenClaw默认管理端口直接暴露在公网上，通过身份认证、访问控制等安全控制措施对访问服务进行安全管理。对运行环境进行严格隔离，使用容器等技术限制OpenClaw权限过高问题。
第二，加强凭证管理，避免在环境变量中明文存储密钥；建立完整的操作日志审计机制；第三，严格管理插件来源，禁用自动更新功能，仅从可信渠道安装经过签名验证的扩展程序；最后，持续关注补丁和安全更新，及时进行版本更新和安装安全补丁。
由奥地利工程师施泰因贝格尔（Peter Steinberger）创建的OpenClaw今年1月对外开源，被形容为“真正能干活的AI”，主打不只能对话，还可替用户执行任务，包括收发邮件、管理日程、预订机票、运营社媒等。
