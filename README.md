专家和地方族团表示，这与中国经济社会变迁有关：自1978年改革开放以来，约8亿多人脱贫，农村人均可支配收入从1978年的134元增至2023年的逾2.1万元；全国人均可支配收入在2020年达32,189元，国内生产总值同年突破100万亿元，生活水平显著提升。 
族团人士指出，过去侨胞常为重建祠堂、修路筹款，如今许多乡亲已自给自足，寻根行程更多聚焦家史与文化认同。新加坡中华宗乡总会副秘书长傅志远（Francis Phua）与厦门大学庄国土教授亦认为，新一代移民更像“全球华人”，与祖居地保持多元往返、非单向依赖的关系。

联邦调查局局长卡什·帕特尔在社交媒体X网站上发表声明说：“多亏一群勇敢的学生挺身而出制服了枪手，他现在已经死亡。”
贾洛于2017年入狱，并于2024年获释。
联邦调查局诺福克分局的特工多米尼克·埃文斯告诉记者，嫌疑人在袭击前高喊“真主至大”（Allahu akbar）。她说，嫌疑人“想要发动一起类似于得克萨斯州胡德堡（Fort Hood）枪击案的恐怖袭击”，她指的是2009年由一名美国陆军精神科医生发动的大规模枪击事件，事件造成13人死亡，30多人受伤。
与此同时，美国密歇根州底特律一座犹太教堂也在星期四遭到袭击，一名男子驾驶卡车冲撞一所犹太教堂，被安全人员开枪击毙。媒体发布的一段视频显示，事发地点附近有大量警力部署，犹太教堂部分区域冒出滚滚黑烟。
联邦调查局称案件为“针对犹太社群的暴力袭击”。美国总统特朗普在白宫受访时称案件发生“是很糟糕的事”。

刘国梁卸任WTT董事会主席
世界乒乓球职业大联盟（WTT）宣布领导层核心变更，刘国梁卸任董事会主席。
WTT官网和微信公众号星期五（3月13日）公布上述变动。
据公告，2026年首届WTT世界乒联董事会会议3月1日在新加坡召开，会议任命了核心领导层。
根据国际乒联执行委员会新的职责划分，国际乒联主席佩特拉·索林和执行副主席瓦希德·奥肖迪代表国际乒联出任WTT世界乒联董事会成员。
随后在WTT董事会会议上，佩特拉·索林获任命为WTT董事会主席，接替刘国梁此前担任的相关职务。
公开资料显示，现年50岁的刘国梁曾是中国奥运冠军和世界冠军，然后在2003年6月开启教练生涯，先后出任中国乒乓球男队主教练、中国乒乓球队总教练，带领中国乒乓球队获得三十多项国际赛事冠军，直至2017年4月卸任总教练和男队主教练。
之后，刘国梁回到中国乒乓队管理层，2018年12月当选中国乒协主席，直至2025年4月辞任。刘国梁在于2020年当选WTT理事会主席，之后在2022年当选WTT董事会主席。2021年，刘国梁当选国际乒联执行副主席（任期2021年至2025年），并于2022年获任国际乒联第一副主席。去年11月，他再获佩特拉·索林任命为国际乒联第一副主席。
