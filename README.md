位於菲律賓以東海域的潛在風暴預料較大機會週末在本港以東登陸，但其與本港的距離須視乎中心整合位置和速度，以及季風槽和副熱帶高壓脊的互動。
潛在風暴過去一兩日發展速度較慢，尚未建立和鞏固中心。人工智能模式預測潛在風暴短期內中心整合速度較快，繼而較早進入南海，預測登陸位置亦略為偏東。部分傳統電腦模式則預測潛在風暴仍須一段時間整合中心，因此起跑時間較遲，預測登陸位置亦略為偏西。
人工智能和傳統電腦模式對風暴初期中心整合的預測有別，在去年數個案例亦有出現，但現時未有足夠數據判斷誰優誰劣。
至於潛在風暴進入南海後的路徑北份量，則須視乎副高往南伸展的幅度。早前傳統電腦模式預測季風槽有殘餘渦度，因而阻礙副高南伸。然而，傳統電腦模式過去一兩日有加大副高南伸幅度，以致潛在風暴預測路徑往東修正的趨勢。此外，新一輪西風波動會在週末通過中國東北部，亦令副高短期內較難進一步往西伸展。因此，現階段預測潛在風暴東登的機會較大。
東登風暴須較接近本港，以及登陸後採取較為偏西的路徑，才會對本港造成顯著的風雨影響。相信潛在風暴中心完成整合後，本港週末風雨預測會更為明朗。

【非官方預測：MET WARN 最新評估】
MET WARN 目前評估，天文台就熱帶氣旋紅霞發出三號強風信號的機會為「中等至高」。
【預測：紅霞相關的強風區靠近珠江口以東】
傳統模式及人工智能預報過去一日將預測路徑稍微往西修正。按照現時預測，紅霞會以颱風級或以上的強度，在本港以東200公里內的廣東東部沿岸登陸。
預計紅霞相關的強風區會在星期六稍後靠近珠江口以東一帶，屆時本港西北風會逐步增強；風暴登陸後，本港會轉吹更為當風的西南風，潛在強風威脅會在週日大部分時間維持。
【變數：未能完全排除較遠距離東登】
紅霞轉向西北移動的位置及角度，會影響風暴實際與本港的距離。基於現階段集成預報，未能完全排除紅霞以更遠距離東登的可能。在此情境下，紅霞相關的強風區未必能持續影響本港。
在本港東面登陸的風暴，初期帶來的西北風較受地形屏蔽；轉吹西南風時的風力增強幅度，則取決於雨帶發展和低空風力。因此，東登風暴路徑的細微變化，會對本港最終風雨程度有顯著影響。
註：「中等至高」代表基於現有資料，MET WARN 評估天文台較可能發出相關信號，但有一定機會出現「唔掛波」的例外情況。我們會視乎最新資料，判斷須否更新風球概率評估。

中国汽车流通协会：授权模式价格内卷严重 经销商盈利承压
中国汽车流通协会指出，今年是中国汽车的“产品大年”，但终端市场并未如期回暖。供需错配导致经销商在授权分销模式下爆发恶性价格竞争，利润严重承压。
据澎湃新闻报道，中国汽车流通协会副秘书长郎学红星期四（7月23日）在2026中国汽车论坛上指出，车企对市场预判趋于乐观，纷纷在今年大量推出新车型以抢占先机。但市场并未给予积极响应。目前中国车市面临约20%的销量下行压力，汽车销售渠道整体呈现收缩和结构性调整。
郎学红分析，目前中国汽车销售渠道可归为分销（授权经销）与直销两大体系。数据显示，传统的授权分销模式仍占主导，比重约50%；小鹏、小米等采用的代理模式约占三分之一；而特斯拉等重资产直营模式占比不足20%。
她指出，当前车市的价格战主要集中在授权分销领域。在这一模式下，经销商为了完成车企下达的批发任务并缓解库存资金压力，普遍采取低价走量的销售策略，引发经销商之间的恶性竞争。反观直营与代理模式，由于库存风险由厂家承担且终端价格受统一管控，渠道秩序相对稳定。
目前，海外市场已率先开启渠道变革，欧洲车企正全面从授权模式向代理制切换。这种轻资产模式既规避了纯直营的资金压力，又能帮助车企直面用户。
但现阶段，代理制在中国市场的进程缓慢。郎学红说，转型遇阻源于车企与经销商的双向阻力。对车企而言，经销商是承接新车库存的蓄水池，厂家需要借此转移短期的经营压力；对经销商而言，转为代理制意味着企业营收规模将大幅锐减。
中国汽车流通协会去年发布的数据显示，2025年上半年，仅有29.9%的经销商实现盈利，亏损面达52.6%。郎学红透露，协会的初步调研显示，相较于去年，今年上半年汽车经销商的亏损面将继续扩大。

制造工艺波动导致空气弹簧缺陷 小鹏召回逾3.3万辆电车
因制造工艺波动导致前空气弹簧气密性下降，中国车企小鹏将召回超过3万3000辆X9型电动汽车。
中国国家市场监管总局网站星期五（7月24日）消息，肇庆小鹏新能源投资有限公司日前向国家市场监督管理总局备案了召回计划。
公告称，小鹏汽车将自2026年8月28日起，召回2023年8月8日至2025年8月11日生产的部分小鹏X9汽车，共计3万3473辆。
据介绍，这次召回范围内部分车辆，由于制造工艺波动，导致前空气弹簧气密性下降，在高温高湿等环境长时间使用后，前空气弹簧可能缓慢漏气，进而触发故障灯点亮及文字提醒。极端情况下，影响车辆的操控性，存在安全隐患。
小鹏将为召回范围内的车辆免费更换改进后的前空簧滑柱总成，以消除安全隐患。因部分车辆此前已更换改进后的零部件，本次召回此部分车辆无需再次更换。
综合每日经济新闻、观察者网等报道，此前，多名小鹏X9车主反映车辆在行驶过程中“趴窝”（抛锚），包括在高温天气状况下，空气悬架出现突发失效、车身单侧或整体塌陷等问题。
报道称，空气悬架属于汽车的核心安全部件，行驶过程中如突发塌陷会直接影响操控，极易引发追尾、侧翻等严重交通事故。

摩洛哥停发签证 台外交部：摩方尚未回复
摩洛哥驻日本大使馆发通知，自星期四（7月23日）起暂停受理台湾人签证申请，台湾外交部就此回应说，已通过驻外馆处向摩方了解情况，目前暂未收到回复。
综合台湾FTNN新闻网、民报和联合报等报道，由于台湾与摩洛哥没有正式外交关系，台湾民众无法直接在台申请摩洛哥签证。
依外交部领事事务局说明，想前往摩洛哥旅游的台民众，须先向摩洛哥驻日本大使馆申请入境签证函（Visa Confirmation Letter），待摩洛哥政府完成审核并核发文件后，持签证函入境。实际可停留天数，需视入境时摩洛哥机场警察人员在签证函上加盖的许可停留期限章戳而定，一般仅给予14或15天，之后须在期限内离境。
由于申请流程需跨境送件，整体审查时间通常约需一至两个月以上，加上程序繁琐，因此台湾旅客赴摩洛哥大多是透过旅行社代办签证。
在摩洛哥驻日使馆发布上述通知后，台湾雄狮旅游星期四在官网公告，根据摩洛哥驻日本大使馆发布的通知，该旅游社也从即日起暂停受理摩洛哥观光申请入境签证函代办业务。
雄狮星期五（24日）再发布声明称，受影响旅客可全额退还已缴订金，也可改订其他行程并享有转团优惠。
台湾外交部发言人萧光伟星期五受询时说，已透过相关驻外馆处向摩洛哥了解上述举措的情况，但尚未获得回复，建议近期规划前往摩洛哥从事商务或旅游的台民众，应审慎评估行程规划，并密切留意外交部发布的最新信息。

逾300名乘客杭州被困机舱超10小时 阿联酋航空致歉
原计划飞往上海的一架阿联酋航空航班，因天气原因备降浙江杭州，300多名乘客据报困在机上超过10个小时。阿航致歉的同时辟谣称，网传边检未让乘客下机的信息不属实。
据荔枝新闻、潇湘晨报等报道，由迪拜飞往上海浦东的阿联酋航空EK302航班，星期三（7月22日）因天气原因，于当晚9时许备降杭州萧山国际机场。
多名这趟航班上的旅客反映，被困机舱一整晚。有旅客称在机舱等待长达10个多小时，因空调故障机舱闷热，有人在机舱里晕倒。
萧山机场工作人员解释说，航班备降杭州后，原执飞机组执勤时长已超限，无法继续执飞，亟需更换新机组。因本地无适配机组，只能从上海调派专属机组。
其次，从上海调派的新机组在赶赴杭州途中遭遇道路限速，行程大幅耽搁。而在新机组好不容易抵达机场、准备开展飞行工作时，工作人员又排查出飞机存在技术故障，最终导致航班持续延误、乘客长期滞留机舱。
机场方面说，国际航班备降后，旅客的下机权限主要受制于国际航班备降的出入境管制相关规定。
阿联酋航空星期四（23日）晚间发布致歉信称，EK302航班因恶劣天气条件备降杭州，随后因技术故障进一步长时间延误。
航司说，所有乘客均已下机，且公司已根据乘客意愿安排了餐食、酒店住宿以及返回上海的班车服务。客户事务团队将与受影响的乘客联系处理。
