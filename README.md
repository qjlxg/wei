3月3日（佛历2569年），พีระพันธุ์ สาลีรัฐวิภาค（皮拉潘·萨里拉特维帕克）——รวมไทยสร้างชาติ党党首就中东局势引发的能源风险向政府提出应对建议。 他呼吁政府尽快采取措施，应对伊朗关闭霍尔木兹海峡可能导致的燃油供应中断，建议“至少一个月内”临时冻结油价，防止商家趁机涨价并减轻对民生与经济的冲击。 
在短期方案上，他指出，能源部拟动用《燃油贸易法》（พ.ศ.2543）禁止销售以应对事态，但该法仅赋予部长“禁止销售”的权力，并不涵盖“禁止出口”，不足以应对燃油短缺。为迅速有效处置，他建议启用《防止燃油短缺法令》（พ.ศ.2516），该法授予总理在紧急情况下一人决断的权力，可宣布禁止进出口燃油并下令临时定价管制。 
在长期对策方面，党提出建立国家石油储备（Strategic Petroleum Reserve，SPR）模型，改以实物燃油储备取代依赖易引发公债负担的燃油基金。该模式由商业渠道按低价入库为国家储备，国际油价高涨时即可投放市场救助民众。 皮拉潘指出，目前国内约有60天的商业流通燃油储量，一旦严重断供将迅速耗尽；若设立国家储备，能够再增加约90天的应急能力。 
他表示，党已完成《国家石油储备法》草案，愿提交政府审议以共同强化国家能源安全，切实保障国民利益。
