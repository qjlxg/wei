官方：民族团结进步促进法顺应中华民族从多元凝聚为一体趋势
针对即将审议的民族团结进步促进法草案，中国全国人大会议新闻发布会发言人娄勤俭说，制定有关法律就是要顺应中华民族从历史走向未来、从传统走向现代、从多元凝聚为一体的发展大趋势。
全国人大会议星期三（3月4日）在北京人民大会堂举行新闻发布会。娄勤俭回应有关民族团结进步促进法草案的提问时说，中共十八大以来，以总书记习近平为核心的党中央高度重视民族工作，鲜明提出铸牢中华民族共同体意识，形成了习近平关于加强和改进民族工作的重要思想。
娄勤俭称，立足新时代党的民族工作的历史方位，制定民族团结进步促进法，就是要顺应中华民族从历史走向未来、从传统走向现代、从多元凝聚为一体的发展大趋势。
他进一步说，通过国家立法方式，保证党对民族工作的全面领导，健全铸牢中华民族共同体意识制度机制，支持民族地区加快融入国家发展大局，更好保障民族团结、维护各族群众合法权益，进一步增强中华民族的凝聚力和向心力。
娄勤俭说，这部法律注重与民族区域自治法等其他法律法规协调和衔接，侧重聚焦铸牢中华民族共同体意识、推进中华民族共同体建设，调整全国范围内的民族关系。
娄勤俭提到，草案明确了民族团结进步事业的总体要求、重要原则以及相关主体的职责要求，规定了构筑共有精神家园、促进交往交流交融、推动共同繁荣发展等方面的具体举措。
他还说，待法律颁布后，将做好贯彻实施，以中华民族大团结促进中国式现代化。
