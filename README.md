危机警戒系统是启动紧急遣返与出入境限制的依据，适用于疫情、医疗危害及治安或外部攻击等情况。各级别对应不同建议与政府措施：
一级（预防期）：出现不稳迹象或威胁限于小范围，建议采取防范、避免非必要外出，仍可正常旅行但需谨慎。
二级（限制期）：暴力或不稳扩散或外部攻击趋近，建议就地避险、远离高风险区域；使领馆启动应急计划。
三级（建议自愿撤离）：动荡或外部威胁显著升高时，建议民众自愿撤离或遣返，强烈不建议前往受影响国。
四级（强制撤离）：发生大规模内乱或全面外部攻击时，实施政府主导的强制撤离与全面应变，禁止前往受影响国。

日本拟在离东京2千公里无人岛建首个核废料埋置场
在进口能源成本飙升之际，日本首相高市早苗极力重启核电满足需求。由于日本发展核能还没有一个可永久存放高放射性废料的设施，当局计划在无人岛“南鸟岛”建设首个核废料埋置场。
日本核电设施运转的放射性废料，目前暂存放在青森县和茨城县的原子能设施内，还未进行最终处置。日本经济产业部长赤泽亮正星期二（3月3日）在记者会上说，当局拟在隶属东京的小笠原村管辖的“南鸟岛“建设首个埋置场。当局将在这个岛上展开环境评估调查，然后由东京政府审批后才进行实地探勘。
被问及为何选择南鸟岛时，赤泽说：“科学特征图显示，该区域相对而言具备有利特征。而且，整个岛屿均为国有土地。我们希望尽快为村民举办信息说明会，提供详细信息，以便村民能够理解并配合。”
他也认为南鸟岛不仅是无人岛，适合成为核废料埋置场外，另一个因素是它位于太平洋板块地质稳定地区。
此前，因为地理位置远，长途运输和挖掘工程被视为难题。但是，近期它因周围含稀土量高而受到关注。
南鸟岛距离东京约1950公里处，它面积约1.5平方公里，地势平坦，呈三角形。 二战期间，这个岛曾是重要的军事要塞，战后逐渐荒废。目前，只有一个小型军事基地进驻25个自卫队员以及科研设施工作人员。
掌管这个无人岛的东京都知事小池百合子对中央的要求持保留立场。她说：“我会关注实施的细节，同时也会密切关注民众的反应。”
距离南鸟岛1200公里的父岛，岛民们得知日本政府要以离岛作为核废料埋置场心情复杂。岛上一名68岁的老人指出，普通垃圾和核废料最终都要通过埋置处理，但这两者的危险性显然不同。
因福岛核事故，日本曾对核设施叫停。在积极重启核电厂之际也要增设核废料埋置场，引起民间反弹。据了解，当局之前也对北海道寿都町、上内町和佐贺县的玄海町等进行调查。据悉，核废料埋置场须确保安全达10万年，因此在有人居住的本岛选址极为不易。

菲律宾危机警戒级别：海外菲律宾人需知
迪拜讯（2026年3月3日）——记者 Tricia Gajitos
菲律宾外交部（DFA）在美以与伊朗冲突升级之际，加强与中东各地使领馆协调，保护海外菲律宾劳工（OFWs）。DFA表示，相关使领馆“仍在当地并保持高度警觉”，并会根据各国安全情势检视并更新危机警戒级别，以便提供必要协助。
危机警戒系统是启动紧急遣返与出入境限制的依据，适用于疫情、医疗危害及治安或外部攻击等情况。各级别对应不同建议与政府措施：
一级（预防期）：出现不稳迹象或威胁限于小范围，建议采取防范、避免非必要外出，仍可正常旅行但需谨慎。
二级（限制期）：暴力或不稳扩散或外部攻击趋近，建议就地避险、远离高风险区域；使领馆启动应急计划。
三级（建议自愿撤离）：动荡或外部威胁显著升高时，建议民众自愿撤离或遣返，强烈不建议前往受影响国。
四级（强制撤离）：发生大规模内乱或全面外部攻击时，实施政府主导的强制撤离与全面应变，禁止前往受影响国。
阿联酋目前未被列入任何警戒级别，官员称该国拥有世界领先的防空与反导系统，呼吁居民以官方与经核实的信息为准，保持冷静与警觉。
