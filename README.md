受台风影响 中国多地机场取消航班
受今年第9号台风“巴威”影响，中国多地机场通行能力下降，纷纷取消航班。
据央视新闻报道，记者星期天（7月12日）从中国民航局了解到，上海虹桥机场取消航班100余班，上海浦东机场取消航班400余班，温州机场取消航班100余班，宁波机场取消航班100余班，杭州机场取消航班300余班，南京机场取消航班200余班。
预计午后，台风对华东地区主要机场影响逐渐减弱，航班运行将逐步恢复。
台风夜间在浙江接连两次登陆后，强度目前已减弱为强热带风暴，中心最大风力降至11级，浙江和福建多地桥梁公路逐步恢复通行。

港新民党成立15周年：继续支持港府依法施政
香港建制派政党新民党的党主席叶刘淑仪称，将继续支持港府依法施政，维护香港的安全稳定，也会继续参加选举，争取更多市民支持。
新民党星期六（7月11日）在脸书发文说，新民党当天在香港会议展览中心举行15周年党庆典礼。党主席、行政会议召集人叶刘淑仪致辞时说，新民党在2011年创立时，香港经历尖锐的政治斗争，建制派需要能够获得中产、专业人士和外籍人士支持的力量，来扩阔建制派的光谱，因此该党一直积极参与不同选举，为建制派争取更多市民支持。
叶刘淑仪回顾过去15年，香港经历“违法占中、黑暴等严峻国安事件，在宪制权威被挑战的时候，自己和新民党都没有退缩，坚定为正义发声，反驳抹黑国家和香港的声音”。
她称，新民党区议员、社区发展主任、关爱队坚定在地区深耕，聆听市民的意见。至于立法会方面，她也经常提醒立法会议员要花时间阅读政府的文件，学习怎样审议条例草案、拨款建议。
她表示，新民党即将踏出下一个15年，未来将继续为市民服务，支持特区政府依法施政，维护香港的安全稳定，维护国家的主权安全和发展利益，也会继续参加选举，争取更多市民支持。
出席新民党党庆的还有行政长官李家超、政务司司长陈国基、律政司司长林定国、立法会主席李慧琼等港府高官。

不丹颁布生育补贴政策 应对人口下降
由于出生率急剧下降，加上越来越多年轻人为寻求发展机会而选择移居国外，不丹政府正在向家庭提供现金奖励，鼓励他们多生育子女。
法新社报道，过去十年间，不丹的年出生率下降超过四分之一。
据不丹国家广播公司报道，首相策林·托杰多次就人口骤减发出警告，称其为“生存危机”。
托杰在政府于6月推出的“第三胎及以上”计划介绍中写道：“证据非常明确——不丹的生育率已下降到接近或低于更替水平。”
这项计划为每个第三胎及以后的孩子提供每月105美元的补贴，直至孩子年满三岁。
报告显示，自2020年以来，不丹第三胎及以上出生的比例下降27%。
联合国估计，不丹的生育率已降至每名妇女约生育1.8个孩子，低于人口更替水平，而65岁及以上人口的比例预计将从约6%上升到2050年的17%。
首相说：“这些并非抽象的统计数据，它们对不丹的劳动力、财政可持续性以及全国各地社区的社会结构构成了切实且日益加剧的压力。”
政府担心，出生率下降加上持续的人口外流，将导致劳动年龄人口过少，难以支撑经济发展和应对人口老龄化问题。
专家认为，不丹家庭规模缩小的原因很复杂，从生活和育儿费用的上升到生活优先级的变化，不一而足。
