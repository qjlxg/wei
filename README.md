台国防部长批国民党版军购案三不合理 吁朝野理性审查
台湾朝野围绕对美军购案持续僵持，台国防部长顾立雄指国民党版“3800亿+N”军购案有三处不合理，包括所设2028年底的执行期限等于实质上封杀五个军购案。
台湾在野的国民党星期四（3月5日）拍板“3500亿元（新台币，下同，141亿新元）＋Ｎ”的国防特别预算条例版本，不过在稍后发布的新闻稿中又调涨数字，修改为“3800亿+N”。
“+N”的意思是，除了已公布的军购内容外，只要美国启动下一阶段对台军售，正式提出新的军售报价书，主管机关应立即研拟第二阶段特别条例草案，送请立法院审查，立法院会从速完成三读程序。
综合《自由时报》、“Newtalk新闻”等媒体报道，台湾立法院会星期五（6日）将朝野国防特别条例草案付委审查。国防部长顾立雄受访时直言，国民党版草案设定3800亿元上限，条文中并不存在“+N”的概念，因为条例规定若要增加，须另提特别条例，意味着上限金额就是3800亿元，“+N”的逻辑完全不存在、也不确实。
此外，顾立雄质疑，起草国民党版条例的人没看过国防部专案报告，把已纳入年度预算的项目也列入军购条例。
第三，顾立雄说，条例执行的年限只到2028年12月31日，包括精准火炮、远程精确的海马式、托式、标枪及反装甲的无人系统不可能在那之前交运完毕。因此草案相当于实质封杀这五项军购案，顾立雄三次强调“不可能执行”。
顾立雄称，行政院提出的1.25万亿元版本，缺了任何一块都会产生重大缺口及下行。他呼吁，希望行政院版付委后，不分朝野共同支持国防预算，一起理性面对、合理审查。
