中使馆：中菲联手捣毁电诈窝点 遣返六名通缉在逃者
中国驻菲律宾大使馆星期四（7月16日）在社交平台公布，中菲联手捣毁电诈窝点，并遣返六名犯罪嫌疑人，六人均为中国通缉在逃人员。
中国使馆发文通报，中菲两国执法部门紧密协作，成功捣毁一个盘踞在菲电诈窝点。该团伙涉案金额近千万元（人民币，190万新元），六名犯罪嫌疑人悉数落网，均为中方通缉在逃人员，涉嫌从事电信诈骗、洗钱及掩饰、隐瞒犯罪所得等多项罪名。
行动中，执法人员在窝点现场查获大量电脑、手机、假印章等作案工具，同时缴获手枪、毒品等违禁物品。
使馆也说，菲国移民局官员星期二（14日）将六名嫌疑人押解遣返回中国。“中国驻菲使馆将持续深化执法合作，坚决打击跨境违法犯罪活动，切实守护同胞生命财产安全。”
中国驻菲使馆也在星期三（15日）通报，中菲两国执法部门合作抓获并遣返一名涉嫌组织跨境赌博、电信诈骗犯罪的林姓逃犯。
林姓逃犯所属跨境赌诈犯罪集团在菲律宾、柬埔寨等地设立网络赌博及电信诈骗窝点，借助赌博网站及色情诱导APP，实施跨境赌博、电信诈骗违法犯罪活动，涉案金额数亿元人民币。
