升级是战略还是地缘困境？海湾处于全球变局核心
美以2月28日对伊朗的联合打击，将长期处于“影子对抗”状态的中东带入公开军事对抗的新阶段，不仅重塑区域安全，也考验全球能源与外交秩序。此次升级发生在全球本已受经济脆弱、气候危机与多地长期冲突牵动的大背景下，令世界从疫情后复苏的希望再度受挫。
人道与经济代价持续扩大。中东多年来累积的战争、流离失所与治理薄弱，使任何新一轮冲突都进一步侵蚀制度韧性与公众信任。海湾作为全球能源供应要地，霍尔木兹海峡的安全一旦受威胁，立即引发油气价格波动、保险与供应链成本上升，进而加剧全球通胀与贸易分裂压力。
更为危险的是，冲突的报复范围已超出主要交战方。尽管阿拉伯海湾国家普遍采取谨慎外交、呼吁降级并积极斡旋，伊朗将报复对象扩大到与海湾国家关联的目标，可能削弱调解渠道，迫使这些国家调整安全策略并深化防御联盟，加剧对抗与地缘扩散。
此轮对抗不只影响军事层面，还冲击气候合作、反恐与可持续发展。不断上升的军费开支将挤占社会发展与气候适应资金，推动不平等与流离失所循环，并为极端主义提供温床。
结论上，单靠军事威慑难以维系长期稳定。遏制进一步溢出、保障能源通道与保护平民的出路在于重回外交与区域互信机制，通过共同安全与合作框架化解风险，否则海湾及全球将长期承受高昂代价。
作者：Rashid AlZaabi，TRENDS研究与咨询研究员

企业普遍担忧，政府未必会主动退还过去10个月已征收的逾1700亿美元（约2163亿新元）关税。
彭博社报道，多家知名企业已加入追讨行列。联邦快递（FedEx）星期一（3月2日）入禀法院，随后戴森（Dyson）、Dollar General、博士伦（Bausch & Lomb）、Brooks Brothers等公司也提出诉讼，要求退还进口关税。
联邦快递称，如获得退款，将把款项返还给最初承担费用的托运人和消费者。

美国国际贸易法院此前因最高法院审理而暂停相关案件。尽管正式结案可能仍需数周，企业已敦促法院尽快恢复审理，以推进退款程序。
法律界普遍认为，短期内难有明确结果，退款之争或将演变为一场持久战。

升级是战略还是地缘困境？海湾处于全球变局核心
美以2月28日对伊朗的联合打击，将长期处于“影子对抗”状态的中东带入公开军事对抗的新阶段，不仅重塑区域安全，也考验全球能源与外交秩序。此次升级发生在全球本已受经济脆弱、气候危机与多地长期冲突牵动的大背景下，令世界从疫情后复苏的希望再度受挫。
人道与经济代价持续扩大。中东多年来累积的战争、流离失所与治理薄弱，使任何新一轮冲突都进一步侵蚀制度韧性与公众信任。海湾作为全球能源供应要地，霍尔木兹海峡的安全一旦受威胁，立即引发油气价格波动、保险与供应链成本上升，进而加剧全球通胀与贸易分裂压力。
更为危险的是，冲突的报复范围已超出主要交战方。尽管阿拉伯海湾国家普遍采取谨慎外交、呼吁降级并积极斡旋，伊朗将报复对象扩大到与海湾国家关联的目标，可能削弱调解渠道，迫使这些国家调整安全策略并深化防御联盟，加剧对抗与地缘扩散。
此轮对抗不只影响军事层面，还冲击气候合作、反恐与可持续发展。不断上升的军费开支将挤占社会发展与气候适应资金，推动不平等与流离失所循环，并为极端主义提供温床。
结论上，单靠军事威慑难以维系长期稳定。遏制进一步溢出、保障能源通道与保护平民的出路在于重回外交与区域互信机制，通过共同安全与合作框架化解风险，否则海湾及全球将长期承受高昂代价。
作者：Rashid AlZaabi，TRENDS研究与咨询研究员
