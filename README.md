政府工作报告的边际变化
一、核心经济发展预期目标变化
GDP下调：从5%左右下调至4.5%-5.0%。
CPI、城镇调查失业率、城镇新增就业、粮食产量、居民收入增长、国际收支等预期与去年一致。
二、财政政策与债务工具核心变化
1. 政策基调与基础规模
基调延续：两年均明确实施更加积极的财政政策，2026年更强调“存量政策和增量政策集成效应”，政策协同性要求提升。
中央预算内投资：中央预算内投资安排从去年的7350对比今年的7550亿元同比增加200亿元，政府直接投资力度小幅提升。
2. 地方政府专项债券
2025年：拟安排4.4万亿元，重点用于投资建设、土地收储和收购存量商品房、消化地方政府拖欠企业账款等。
2026年：核心变化为用途优化聚焦，重点转向支持重大项目建设、置换隐性债务、消化政府拖欠账款，弱化了土地收储、商品房收购相关投向，化债与稳投资的双重定位更突出。
3. 超长期特别国债
两年发行总规模均为1.3万亿元，但资金投向拆分与侧重发生显著调整：
2025年：3000亿元专项支持消费品以旧换新，剩余资金重点支持“两重”建设。
2026年：资金投向更细分、撬动作用更精准，8000亿元用于“两重”建设、“两新”工作，2500亿元支持消费品以旧换新，2000亿元专项支持大规模设备更新；以旧换新资金规模较2025年缩减500亿元，新增设备更新专项安排，实现消费提振与产业升级的双向发力。
4. 新增财政工具
2026年新增设立1000亿元财政金融协同促内需专项资金，2025年无此项安排，填补了财政与金融协同促内需的专项工具空白，形成扩内需的增量政策支撑。
三、货币政策核心变化
政策基调延续：两年均明确继续实施适度宽松的货币政策，保持流动性充裕、社会融资条件宽松的核心导向不变。
政策性金融工具大幅扩容：2025年新型政策性金融工具全年发行规模为5000亿元，2026年拟发行规模提升至8000亿元，同比扩容60%，撬动社会资本参与投资的力度显著增强，对重大项目、重点领域的资本金支持力度进一步升级。
四、民生保障、国防建设变化
民生保障：城乡居民基础养老金月最低标准，两年均明确再提高20元，增幅维持。
国防支出：2025年国防支出预算同比增长7.2%，2026年同比增长7%，增速小幅回落0.2个百分点，仍维持7%左右的稳健增长节奏，与经济发展水平相适配。
五、核心政策导向
目标设定更具弹性。GDP增速从固定值转为区间目标，兼顾稳增长与高质量发展，更适配复杂的内外部环境。
政策工具更协同多元。从单一财政、货币政策发力，转向“财政+金融”协同发力，新增专项促内需资金、扩容政策性金融工具，形成扩内需、稳投资的组合拳。
资金投向更精准聚焦。债务资金从兼顾地产纾困、土地收储，转向聚焦重大项目建设、化债、产业升级、消费提振，对设备更新、以旧换新的专项支持更细化，撬动产业升级与内需释放的双重效应。
绿色转型更深化。从能耗管控转向直接的碳排放强度管控，目标更贴合双碳战略，约束性更强，绿色发展的导向更突出。

变革往往是渐进的，尤其在我们这里，很多变化不易察觉，甚至有时朝坏处发展。人们常以文化或个人喜好解释我们选出的领导人，但真正的问题在于体制与选择机制长期未改。我们需要一种“扰动”来打开改变的窗口，关键在于能否把这次扰动变成真正的制度性改革。
可以从是否真正拥有民主开始反思。四十年前的人民力量（EDSA）被称为“革命”，理应带来根本性重置，但两代人过去后，民生与政治结构并未根本改善。流行与名气越来越能左右选举，但国家更需要有能力、有资格的领导人，而非仅靠表演或形象当选。
回顾马科斯（老）时代，尽管其下台备受争议，但其政权在早期确实推动过重大改革，组建了一批技术型内阁成员，并尝试重新设计宪政架构（1971年制宪、1973年宪法、1976年第六号修正案）。若这些制度化的改革成功并延续，菲律宾或许能如韩国、台湾、新加坡般实现发展；可惜实施中出现严重问题，承诺未兑现，腐败与压迫最终引发民怨。
因此，1986年的人民力量既是对承诺破灭的回应，也反映出民众对真正变革的渴望。问题仍然是：那次运动是否真正完成了制度性的革命？答案需回到对历史、政治体制与人才培养机制的深刻反省，才能把一次“扰动”转为长期的政治革命与国家治理提升。

中国人大代表建议鼓励大学生加入家政行业

美议员对英特尔测试盛美半导体设备提出安全担忧
美国国会跨党派议员针对半导体巨头英特尔测试盛美半导体生产的晶片制造设备提出国家安全担忧。
据路透社报道，盛美半导体是一家晶片制造设备生产商，公司总部位于加州，但与中国有深厚渊源，它的两家海外子公司曾被列为美国制裁目标。
由民主党参议员沃伦（Elizabeth Warren）和共和党参议员科顿（Tom Cotton）牵头的跨党派议员星期三（3月4日）致信英特尔首席执行官陈立武，称盛美半导体的客户包括长江存储（YMTC）和中芯国际（SMIC）等受制裁的中国企业，因此担忧盛美是否可能将英特尔的敏感信息传递给中国客户。
