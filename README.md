以军星期五（3月13日）称，在空袭中打死超过380名黎巴嫩真主党武装人员。黎巴嫩卫生部稍早时候说，自2日以来，以色列空袭导致黎巴嫩773人死亡、1933人受伤。
13日晚间，以色列国防军说，12日晚在黎巴嫩利塔尼河以北地区发动袭击，打死黎巴嫩真主党两名负责向以色列发射火箭弹的“关键指挥官”。
以色列国防军总参谋长扎米尔说，他已指示向以军北方司令部增派兵力，以加强“战备能力”。根据以军声明，以军将向北部边境增派一个师、几个旅和工兵营，还将动员预备役人员，做好现役部队轮换。
新华社记者13日在以色列北部与黎接壤的边境地区，看到大量以军坦克和装甲车辆向该地调动集结，无人机和武装直升机从天上飞过。在以黎边境附近一座城市，多轮防空警报拉响，听到巨大爆炸声，并看到边境两侧均有烟雾升起。
美国和以色列2月28日对伊朗发起大规模军事行动，伊朗对以色列以及美国在中东地区的军事基地等目标发起还击。作为伊朗的主要地区盟友，黎巴嫩真主党2日晚开始向以色列北部发射火箭弹，以色列则对黎巴嫩南部、东部和首都贝鲁特等地发起猛烈空袭，并在黎南部展开小规模地面行动。扎米尔12日说，针对真主党的行动“不会很快结束”。

时隔六年 中国国航本月底重启北京至平壤直航
时隔六年，中国国航本月底重启北京至平壤直航。
韩联社星期五（3月13日）引述韩国外交部和中国航空业界人士消息报道，中国国航北京至平壤直飞航班（CA121）将从3月30日起恢复运行。
报道称，上述航班还将于4月6日、13日、20日、27日等日期执飞，为每周一班的定期航班。3月30日航班将于北京时间当天上午8时05分从北京首都国际机场起飞，飞行1小时55分钟后，于平壤时间上午11时抵达平壤顺安国际机场。
机票目前已可中国国航和携程等旅游平台订购。国航官网信息显示，3月30日经济舱票价由2630元人民币起（490新元），携程网站显示的经济舱票价为3650元人民币。
中国国航北京至平壤航线自2020年1月起因冠病疫情防控措施暂停运行。朝鲜高丽航空已于2023年8月重启平壤至北京航线，每周执飞两班。
北京至平壤的双向国际旅客列车也在星期四（12日）恢复运行。
中国国家铁路集团星期二（10日）晚在官方微博发文称，自星期四起，往来于北京、丹东至朝鲜平壤之间双向开行国际旅客列车，每周运行四次。
中国外交部发言人郭嘉昆当天下午在例行记者会上应询时说，中国和朝鲜是友好近邻，保持常态化客运列车运营对促进双方人员往来便利化具有重要意义。
重启后的首班列车从北京出发，途经天津、山海关、沈阳、丹东，进入朝鲜新义州后再抵达平壤，全程逾24小时，车上乘客包括政府官员、记者、企业人士。
从平壤始发的列车也在星期五（13日）上午抵达北京。这是因冠病疫情中断六年后，中朝跨境铁路客运再次运行。由于朝鲜旅游签证尚未开放，目前中国观光客无法购票前往平壤。

广州32岁程序员猝死 认定为工伤
中国广东省广州市32岁程序员高广辉在过度劳累后猝死，引发广泛关注。据中国媒体报道，高光辉死亡情形已被认定为工伤。
去年11月29日，广州视源电子科技股份有限公司高广辉周末突然在家中晕倒，送医抢救无效。他被曝出抢救过程中还被拉入工作群，死亡后还收到工作消息，引发社会关注。
高广辉妻子杨华思12月29日通过丈夫所在公司，向广州市黄埔区人力资源和社会保障局提交工伤认定申请。
澎湃新闻星期五（3月13日）从家属处获悉，今年2月，广州黄埔区人力资源和社会保障局作出认定：高广辉死亡的情形视同为工伤。
《认定工伤决定书》显示，考虑高广辉的工作时间，工作岗位等因素，综合多方证据，高广辉死亡的情形，符合《工伤保险条例》第十五条第一项之规定，现予以视同为工伤。
报道称，高广辉于1993年在河南出生，并于2012年考入广州软件学院。2019年，高广辉通过社招进入广州视源电子科技股份子公司工作。生前，高广辉担任公司软件二部的部门经理。
杨华思称，视源电子公司实行弹性工作制，工作强度大，办公室常备行军床。生前，高广辉承担了巨大的工作量，经常需要加班。出事当天是星期六，高广辉早起开工时觉得肺有些难受，回床上躺了会儿又继续工作。之后，高广辉晕倒，后经抢救无效死亡。
此外，杨华思也就丈夫的生命健康权、丈夫的遗物被私自处理、丈夫的加班费三个案由，向广州市黄埔区人民法院及黄埔区劳动仲裁委提交立案申请，目前正在待审核阶段。
