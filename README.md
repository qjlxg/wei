 以前低保边缘家庭、支出突然困难的家庭救济不太明确，现在把救助对象扩大了，还建了分层分类的体系。

 用人单位必须按时足额发工资、少安排加班、岗位强度要合适，还得给上工伤保险。影响最大的是现在很多返聘老人和灵活就业的群体

西班牙旅游热点附近野火蔓延 近万人避险
数百名西班牙消防员正在奋力扑灭地中海热门旅游胜地布拉瓦海岸附近不断蔓延的野火，火势已迫使数以千计居民居家避险。
法新社报道，这场大火于星期五（7月3日）上午在东北部拉比斯瓦尔-德尔安普尔丹（La Bisbal d’Emporda）附近烧起，距离平日挤满避暑游客的布拉瓦海岸约20公里。
由于强风助长了火势，加泰罗尼亚地方当局要求10个市镇居民居家避险，其中包括著名海滩度假胜地普拉加德阿罗（Platja d’Aro）。
地区应急部门说，约150人已被疏散，其中包括一个度假营约70名儿童。
加泰罗尼亚消防部门的视频显示，直升机俯冲向烈火洒水，四周山丘林地间升起滚滚的灰色和橙色浓烟。
另据新华社报道，加泰罗尼亚大规模林火的过火面积约2300公顷。近万名居民被疏散或被要求居家避险。，多条公路临时封闭。强风阻碍灭火，还引发了二次火灾，火势蔓延迅速，已波及当地一处自然保护区。
消防部门出动超400名消防员，出动多架直升机和灭火飞机扑救。西班牙军事应急部队也派出200人和60辆车增援。
赫罗纳的消防行动负责人加西亚说，这场火灾已超出他们能力范围，接下来的目标是尽可能控制火势、减小损失。
西班牙多家媒体引述当地警方消息称，一人因涉嫌纵火被拘留。报道也指出，火灾可能是公路施工设备产生火花所致，具体原因仍在调查中。

纽时：董郁玉申请保外就医被拒
美国媒体报道称，因间谍罪被判刑七年的中国官媒《光明日报》原评论部副主任董郁玉申请保外就医被拒。
美国《纽约时报》星期四（7月2日）引述董郁玉的儿子董一夫称，医生在董郁玉的肺部发现一个疑似恶性肿瘤，董郁玉于是申请保外就医，希望能在北京设备更好的医院接受手术，但被官方拒绝。
董一夫称，当局提出从外部请外科医生到监狱医院主刀，但董郁玉认为监狱医院没有进行这类手术的专业设施，且担心外科医生的经验可能不足。
董一夫说，监狱官员表示，如果不同意他们开出的手术条件，董郁玉可能从监狱医院转出，送回牢房。
据路透社报道，现年64岁的董郁玉2022年2月在北京与一名日本外交官共进午餐时，被北京警方拘留，然后一直被拘押。自2023年7月的一次闭门听证会后，董郁玉一直被关押在北京一所监狱。
北京法院2024年11月判处董郁玉间谍罪成并入狱七年。董郁玉的家人称判决“严重不公正”，指法院在没有证据的情况下判处董郁玉七年有期徒刑，“向世界宣告了中国司法系统的破产”。
董郁玉随后提出上诉，法院去年11月驳回上诉，维持原判。家属坚称董郁玉无罪，批评法院判决是“迫害行为”。

伊朗为哈梅内伊举行六天公众告别仪式
伊朗星期六（7月4日）上午开始为已故最高领袖哈梅内伊举行为期六天的公众告别仪式。
大批民众前往位于首都德黑兰的伊玛目霍梅尼清真寺，向哈梅内伊遗体致哀。送葬巡游活动也将在伊拉克举行。
法新社报道，在哈梅内伊灵柩抵达伊玛目霍梅尼清真寺前，数千名手持象征复仇的红色旗帜的哀悼者聚集在清真寺庭院，高呼“打倒美国”和“复仇”等口号。
哈梅内伊于2月28日在美国和以色列发动大规模空袭首日身亡，终年86岁。在首日空袭中受伤的哈梅内伊次子穆杰塔巴，于3月9日当选新最高领袖。
穆杰塔巴至今未公开露面，仅发表书面声明。
