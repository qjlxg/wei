伊朗表示，2月28日美以空袭导致大阿亚图拉阿里·哈梅内伊遇害后，将迅速选出新最高领袖，称伊斯兰共和国体制已准备好填补这一最高职务。
伊朗现行体制自1979年伊斯兰革命以来结合神权与民选元素，最高领袖终身制，由民选的专家会议（Assembly of Experts）选出并理论上受高级宗教机构监督，实则掌握国内外政策最终决定权并兼任最高统帅。专家会议由88名法学家组成，每八年选举一次；迄今仅在1989年霍梅尼逝世后经历过一次领导层更替。
官员称按宪法将尽快作出决定。卫士委员会发言人3月1日表示，法律要求尽快确定领袖，战争状态下会在最短时间内完成。外长阿巴斯·阿拉格奇在3月1日对半岛电视台称，可能“在一两天内”产生新领袖；他此前在接受采访时援引1989年专家会议在霍梅尼去世后不到24小时就选出新领袖的先例。
伊朗当局已宣布成立三人过渡领导委员会负责国事，包括法学家阿里雷扎·阿拉菲（卫士委员会12人之一）、总统马苏德·佩泽什基安及强硬派司法长官戈拉姆侯赛因·穆塞尼·埃杰伊。佩泽什基安3月1日在国家电视台播放的视频中称，过渡委员会已开始工作，并表示将“继续沿霍梅尼设定的道路前进”。
回顾历史：1980年代霍梅尼曾最初支持侯赛因·阿里·蒙塔泽里为接班人，但临终前将支持转向哈梅内伊；当时86岁的哈梅内伊虽曾一度婉拒提名，但终被宗教界一致推选接任，并长期执掌要职直至身亡。

但他指出，伊朗随后进行了持续报复性攻击，其导弹击中了包括有英籍人士下榻的机场和酒店在内的目标。海湾地区的盟友要求英国提供更多防护，保护英国人生命是政府的职责。英国战机已参与协同防御行动并拦截了伊朗的打击，斯塔默称，要根除威胁必须摧毁导弹的来源——储存库或发射器。
他强调，接受美国请求是基于“长期盟友间的集体自卫”和保护英国人生命，并符合国际法。 来源：路透社

伊朗表示，2月28日美以空袭导致大阿亚图拉阿里·哈梅内伊遇害后，将迅速选出新最高领袖，称伊斯兰共和国体制已准备好填补这一最高职务。
伊朗现行体制自1979年伊斯兰革命以来结合神权与民选元素，最高领袖终身制，由民选的专家会议（Assembly of Experts）选出并理论上受高级宗教机构监督，实则掌握国内外政策最终决定权并兼任最高统帅。专家会议由88名法学家组成，每八年选举一次；迄今仅在1989年霍梅尼逝世后经历过一次领导层更替。
官员称按宪法将尽快作出决定。卫士委员会发言人3月1日表示，法律要求尽快确定领袖，战争状态下会在最短时间内完成。外长阿巴斯·阿拉格奇在3月1日对半岛电视台称，可能“在一两天内”产生新领袖；他此前在接受采访时援引1989年专家会议在霍梅尼去世后不到24小时就选出新领袖的先例。
伊朗当局已宣布成立三人过渡领导委员会负责国事，包括法学家阿里雷扎·阿拉菲（卫士委员会12人之一）、总统马苏德·佩泽什基安及强硬派司法长官戈拉姆侯赛因·穆塞尼·埃杰伊。佩泽什基安3月1日在国家电视台播放的视频中称，过渡委员会已开始工作，并表示将“继续沿霍梅尼设定的道路前进”。
回顾历史：1980年代霍梅尼曾最初支持侯赛因·阿里·蒙塔泽里为接班人，但临终前将支持转向哈梅内伊；当时86岁的哈梅内伊虽曾一度婉拒提名，但终被宗教界一致推选接任，并长期执掌要职直至身亡。

以色列再次空袭德黑兰并引发反击：继伊朗最高领袖阿里·哈梅内伊被击毙后，以色列于3月1日对德黑兰发动新一轮空袭，伊朗随即以更多导弹反击。美以行动与伊朗报复已对航运、航空与石油等行业造成冲击，外界警告海湾地区及全球贸易可能出现能源涨价和商业中断。
美国表态与伊朗新领导层动向：美国总统唐纳德·特朗普称此次打击旨在阻止伊朗获得核武、遏制其导弹项目并消除对美及盟友的威胁。他在接受采访时表示，美以行动造成48名伊朗“领导人”死亡，并称伊朗新领导层愿意与他对话，他已表示同意。
阿联酋关闭驻伊朗使馆：阿联酋3月1日宣布关闭驻德黑兰大使馆并召回大使，原因是伊朗在海湾地区的导弹与无人机打击已造成4人死亡、多人受伤。阿联酋称这些针对民用设施的“敌对袭击”是严重且不负责任的升级，危及平民安全。
德州酒吧枪击案：美国得克萨斯州奥斯汀一处酒吧发生枪击，袭击造成至少两人死亡、14人受伤，凶手随后被警方击毙。监测极端组织的SITE情报组织称，嫌犯为塞内加尔裔美籍人士Ndiaga Diagne，社交媒体显示其表达“亲伊朗政权”立场。
英超：阿森纳险胜切尔西并稳住榜首：3月1日，阿森纳在德比战中以2比1险胜切尔西，尤里恩·廷伯下半场头球建功，使枪手将对曼城的领先优势恢复至5分。曼联在老特拉福德逆转水晶宫2比1，重返联赛前三位。

耶路撒冷/德黑兰讯 — 以色列在3月1日再次对德黑兰发动新一轮空袭，伊朗以导弹和无人机回击。此次攻防发生在最高领袖阿里·哈梅内伊被击毙一天后，引发中东与全球市场剧烈动荡，航运、航空和石油价格面临冲击。
以美军和以色列的打击据称已摧毁伊朗多处情报、安保与军方指挥中心。美国总统特朗普在社交平台上称行动旨在阻止伊朗发展核武、遏制导弹项目并消除对美及盟友威胁，誓言行动将持续至“实现所有目标”。美国中央司令部称已打击逾千处伊朗目标，并确认出现首批美军伤亡。
伊朗革命卫队称已击中3艘美英油轮并袭击科威特、巴林军事基地，霍尔木兹海峡一度关闭，多艘船只抛锚，油价预期飙升；迪拜等中东重要机场关闭，国际航班严重受阻。
伊朗国内面临建制最大危机，代理总统马苏德·佩泽什基安表示由其本人、司法长官及守护委员会一名成员组成的领导委员会暂代最高领袖职责。哈梅内伊在2月28日被炸死，死者还包括其多名亲属，消息引发伊朗境内外不同反应，巴基斯坦和伊拉克亦爆发哀悼与示威冲突。
以方声明将继续打击以促使伊朗政权瓦解，俄中对哈梅内伊之死予以强烈谴责。局势走向及伊朗能否重建领导层仍充满不确定性。

伊朗发动袭击进入第二天 阿联酋报告3死58伤
阿布扎比——阿联酋国防部表示，自2026年2月28日伊朗发动攻击以来，阿联酋空军与防空系统共应对165枚弹道导弹、2枚巡航导弹和541架无人机。
国防部称，在第二天清晨，武装部队摧毁了20枚弹道导弹，另有8枚落海；同时击落并摧毁2枚巡航导弹及311架无人机，但有21架无人机击中民用目标。自袭击开始共探测到165枚弹道导弹，已击毁152枚，13枚落海；探测到2枚巡航导弹并全部摧毁；追踪到541架伊朗无人机，拦截摧毁506架，35架落入境内造成物质损失。
事件已造成3人死亡（巴基斯坦、尼泊尔、孟加拉国籍），58人轻伤，伤者国籍包括阿联酋、埃及、埃塞俄比亚、菲律宾、巴基斯坦、伊朗、印度、孟加拉、斯里兰卡、阿塞拜疆、也门、乌干达、厄立特里亚、黎巴嫩和阿富汗等。
国防部表示，部分拦截残骸散落各地，导致若干民用建筑轻中度损毁，武装部队保持高度戒备，确保公民、居民与访客安全为首要任务，并敦促公众从官方渠道获取信息，避免传播谣言或未经证实的消息。

阿联酋关闭驻伊朗大使馆并召回大使，回应伊朗在海湾地区发动的导弹和无人机报复行动，造成多地4人死亡、数十人受伤。阿联酋外交部称这些袭击针对民用目标，包括住宅、机场、港口和服务设施，是严重且不负责任的升级。
阿联酋2022年曾在与伊朗关系缓和时重开驻德黑兰大使馆并派驻新大使，沙特随后效仿。德黑兰的连番打击引发对更大规模冲突的担忧，海湾长期的相对稳定受到冲击。法新社记者连日听到迪拜、多哈、麦纳麦和利雅得等地多次强烈爆炸声。
阿布扎比一处有多个外方使馆的建筑群（包括以色列使馆）也成为目标，埃蒂哈德塔两人被弹片击伤。阿布扎比国防部称自2月28日以来，阿联酋发现165枚弹道导弹，摧毁152枚，拦截两枚巡航导弹，并击落506架（共541架）伊朗无人机；另有三名在阿联酋死亡者为巴基斯坦、尼泊尔和孟加拉国籍。
科威特卫生部报告1人死亡、32人受伤；巴林机场受损轻微。阿曼的杜克姆港和一艘油轮遭无人机袭击，船员撤离、4人受伤，阿曼外长与伊朗外长通话呼吁停火并重启对话。海湾合作委员会将通过视频连线协调统一回应。
受击目标包括机场、海港、住宅和酒店，迪拜棕榈岛和帆船酒店等地曾起火。卡塔尔称遭65枚导弹与12架无人机袭击，大部分被拦截，但8人受伤、1人伤势危殆。沙特方面称拦截了针对利雅得国际机场和驻美军的苏丹王子空军基地的导弹。
分析人士警告，海湾国家已处于战争前线，若伊朗继续升级，沿岸国家将难以坐视不管。
