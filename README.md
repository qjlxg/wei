前外交部长G.L.皮里斯教授敦促政府就中东危机阐明立场
2026年3月3日 11:08
皮里斯在致总统阿努拉·库马拉·迪萨纳亚克（Anura Kumara Dissanayake）的信中，代表关切公民警示，中东局势波动可能给斯里兰卡带来严重后果，要求政府尽快向国内外说明其立场及应对措施。
他指出四大直接影响：
（一）国际油价上涨将迅速传导至国内，推高生活成本，进一步加剧民众的经济困难；
（二）出口收入可能下降，尤其是茶叶等对受影响地区依赖较大的商品，因市场准入受阻；
（三）酒店业已有大量来自欧洲、北美和海湾地区游客的取消，受受影响空域航班中断所致；
（四）在海湾及周边地区务工的斯里兰卡劳工汇款的可持续性面临风险。
皮里斯表示，关切公民愿在国家利益范围内协助政府，相关方面正就缓解危机影响的实务措施进行讨论，并将在近期举行公开简报。他呼吁政府公开基本信息，以便公众在知情基础上参与。
