日本氣象廳對菲律賓以東熱帶低氣壓發出烈風警報，預料該系統24小時內增強為熱帶風暴。
若該系統獲日本氣象廳升格為熱帶風暴，預料將命名為「紅霞」。紅霞是朝鮮提供的名字，顧名思義是指紅色的天空。

天文台表示，會在明日稍後考慮發出一號戒備信號。
天文台會視乎熱帶氣旋的強度變化、其強風區與本港的距離及本地風力情況，評估是否需要在星期六稍後改發更高熱帶氣旋警告信號。
天文台指，現時評估風暴較大機會在惠州至汕頭一帶登陸，但與本港的距離仍存在變數。

預料本港週六驟雨增多，局部地區或受強雷雨區相關的雷暴和強陣風影響，加上會有湧浪，原定週末有戶外或水上活動的讀者須留意天氣帶來的潛在影響。
菲律賓以東熱帶低氣壓會在週五晚間至週六凌晨進入南海，隨後靠近廣東東部沿岸。由於副熱帶高壓脊未穩定控制珠江口一帶，加上受風暴外圍環流和日間升溫影響，本港附近週六強對流風險有所增加。然而，此類強對流的發展時間、位置和覆蓋範圍均較隨機。
現時預測風暴週日凌晨登陸廣東東部沿岸，但週六至週日的實際風雨程度，仍須視乎風暴登陸位置。只要東登風暴與本港距離稍有差異，本港風雨程度已足以有較大變化。MET WARN 會繼續跟蹤此風暴的動向。

【非官方預測：MET WARN 最新評估】
MET WARN 目前評估，天文台就熱帶氣旋紅霞發出三號強風信號的機會為「中等至高」。
【預測：紅霞相關的強風區靠近珠江口以東】
傳統模式及人工智能預報過去一日將預測路徑稍微往西修正。按照現時預測，紅霞會以颱風級或以上的強度，在本港以東200公里內的廣東東部沿岸登陸。
預計紅霞相關的強風區會在星期六稍後靠近珠江口以東一帶，屆時本港西北風會逐步增強；風暴登陸後，本港會轉吹更為當風的西南風，潛在強風威脅會在週日大部分時間維持。
【變數：未能完全排除較遠距離東登】
紅霞轉向西北移動的位置及角度，會影響風暴實際與本港的距離。基於現階段集成預報，未能完全排除紅霞以更遠距離東登的可能。在此情境下，紅霞相關的強風區未必能持續影響本港。
在本港東面登陸的風暴，初期帶來的西北風較受地形屏蔽；轉吹西南風時的風力增強幅度，則取決於雨帶發展和低空風力。因此，東登風暴路徑的細微變化，會對本港最終風雨程度有顯著影響。
註：「中等至高」代表基於現有資料，MET WARN 評估天文台較可能發出相關信號，但有一定機會出現「唔掛波」的例外情況。我們會視乎最新資料，判斷須否更新風球概率評估。

根据法案，俄罗斯政府将指定有权获取数据的部门，由俄罗斯联邦税务局提供企业销售量、收入、支出和利润等经营数据，信息来源包括纳税申报表、财务报表及税控收银设备记录。相关部门将在法定权限内利用这些数据分析价格走势及形成因素，除《俄罗斯联邦税法》规定的情形外，相关商业数据不得公开。

沙特改以苏伊士运河出口石油 海运时间增1个月
（伦敦综合电）船舶追踪数据显示，星期四通过霍尔木兹海峡的油轮只有一艘，是自5月7日以来最少的。
由于霍尔木兹海峡和曼德海峡这两条主要石油出口通道分别受到伊朗和也门胡塞武装的干扰，沙特阿拉伯如今不得不改以苏伊士运河出口石油。
虽然沙特过去曾利用苏伊士运河出口部分原油，但过去数十年从未把这条航线作为主要出口通道。
沙特大部分原油买家如今在亚洲，经由苏伊士运河的油轮必须绕行整个非洲大陆，才能把原油运到亚洲，航程将增加约一个月。
根据海运贸易信息公司Kpler和伦敦交易所集团（LSEG）的数据，一艘油轮从沙特西部红海港口延布经曼德海峡驶往亚洲，通常仅需19天。经由苏伊士运河、地中海、直布罗陀海峡，再绕过非洲好望角至亚洲的航线则需要48天。
这意味着，单是燃料成本就将从126万美元增至约287万美元（163万新元增至约370万新元），增加一倍以上。此外，通过苏伊士运河还须额外支付约100万美元的通行费。
早在2月下旬，当美伊冲突干扰了经由霍尔木兹海峡的运输时，沙特就已将其大部分石油出口路线从波斯湾改道至红海。然而，本周胡塞武装在红海袭击了船只，使得这一替代路线变得不再安全，从而促使沙特考虑通过苏伊士运河运输石油。
苏伊士运河位于红海北端，连接地中海，是全球少数具备大型商船通行能力的无船闸运河。
能源研究咨询机构Energy Aspects的数据显示，受苏伊士运河水深限制，大型油轮需半载通过，进入地中海后再进行补给。
为实现这一操作，沙特可将部分原油通过苏伊士－地中海输油管道输送至地中海。该管道全长320公里，绕过苏伊士运河，连接埃及红海侧的苏赫奈泉港与埃及地中海侧的西迪克里尔港，日输油能力可达250万桶。
伊朗战事爆发前，沙特日均石油总出口量约700万桶。

日本连续四天出现“酷暑日” 追平历史纪录
日本中部岐阜县观测到40摄氏度高温，这是日本连续四天出现40摄氏度以上的“酷暑日”，追平历史纪录。
新华社报道，日本气象协会星期五（7月24日）发布数据显示，当天午后在岐阜县美浓市观测到40摄氏度的高温。这是日本连续第四天出现“酷暑日”，也是继2013年8月10日至13日、2025年7月30日至8月2日之后，日本第三次连续四天观测到40摄氏度以上高温。
日本气象厅今年决定将40摄氏度及以上高温天气称为“酷暑日”。21日，岐阜县郡上市测得40摄氏度高温，这是日本今年首次出现“酷暑日”。22日和23日，日本又有多地观测到气温达40摄氏度或以上。
日本各地连日来出现35摄氏度以上高温天气。日本总务省消防厅数据显示，仅7月13日至19日的一周内，就有10857人因中暑紧急送医。
日本媒体引述专家的话报道说，日本连日来受高气压覆盖，形成了称为“热穹顶”（heat dome）的天气现象，日本列岛如圆顶体育馆一般积蓄热量，这是造成此轮热浪的重要原因之一。通常情况下，天气炎热时，陆地上空会形成上升气流，冷空气补充进来并形成环流，但目前由于强劲的高压控制，冷空气难以进入。
