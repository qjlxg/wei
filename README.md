国际能源署指出，伊朗战争正给石油市场带来前所未有的动荡，影响全球7.5%的供应，以及甚至更大比例的出口。
彭博社引述能源署（IEA）星期四（3月12日）发布的月度报告说：“中东战争正在造成全球石油市场历史上最大规模的供应干扰。”此前一天，能源署成员国同意史无前例地从紧急储备中释放4亿桶石油，以应对这场混乱局面。
这场战争对能源市场最直接的影响，是霍尔木兹海峡的封闭，切断了这条贸易要道上的石油和天然气运输。由于波斯湾主要产油国无法从该地区出口原油而被迫减产，冲突正在侵蚀原本全球石油供应过剩的局面。这也意味着，即便未来航道重新开放，其影响仍将持续显现。
能源署估计，本月，战争将使全球石油供应每日减少800万桶，合总计接近2.5亿桶。能源署还说，目前通过该海峡的流量已下降逾90%。
能源署指出，由此引发的油价飙升、航班取消和经济不确定性也正冲击着需求，因此将今年全球消费增长预期下调约25%，至每日64万桶。这是自去年4月发布2026年预测以来最低的预期值。

赖清德4月或许出访非洲友邦斯威士兰
台湾总统赖清德上任快两年，仅于2024年11月出访过太平洋友邦马绍尔群岛等国。台外交部星期四（3月12日）证实，收到非洲友邦斯威士兰邀访，赖清德若有相关规划，总统府会适时对外说明。        
受访学者认为，美国总统特朗普今年可能与中国国家主席习近平会面四次，赖清德今年出访欲过境美国本土不容易，与其想过境美国，不如专心出访斯威士兰等友邦，巩固邦交，而非坐困愁城。
赖清德2024年上任，同年底出访马绍尔群岛、图瓦卢及帕劳等三个南太平洋邦交国，往返过境美国夏威夷、关岛。
去年8月一度传出赖清德拟访问南美巴拉圭，并过境美国纽约和达拉斯，但外媒报道特朗普拒绝让赖清德过境纽约。台湾总统府则以考量南部风灾复原等为由，否认有上述出访计划。故赖清德就职600多天以来，从未过境美国本土。
据TVBS报道，赖清德这次可能出访的缘由，是4月中适逢斯威士兰国王姆斯瓦蒂三世登基40周年暨58岁生日庆典，斯威士兰已向台湾提出邀请。
外界关切赖清德若出访，是否会过境欧洲或直飞。前总统蔡英文2018年曾出访斯威士兰，当时是由专机直飞13个小时，未曾过境他国或转机。
政治大学国际关系研究员严震生接受《联合早报》访问时直言，台湾在非洲就只剩下一个邦交国，斯威士兰赖清德应该去，至于过境，因为斯威士兰位于非洲南部而非西非，“要过境欧洲，真的是绕路……想转机南非，南非也不会让赖过境”。
严震生认为，去年特朗普考量中美贸易谈判，不让赖清德过境纽约。今年特朗普3月底访问北京后，习近平是否会在7月美国庆祝250周年国庆时回访？加上年底11月特朗普要去深圳参加亚太经合会议（APEC），12月G20会议又在美国迈阿密举行，“赖清德可能一路到年底都没法过境美国”。
他说： “即使美国国务卿鲁比奥对台湾不错，但现在一切都以特朗普为依归，特朗普又不想节外生枝，赖清德就不要再想过境美国了；台湾只剩下12个邦交国，即使美国不让（过境），加个油也得去，才不会让太平洋和加勒比海等友邦觉得他们只是过境美国的工具。”

伊朗新领袖伤势不明 传战争武器自动运转
（伦敦综合电）据报道，伊朗新任最高领袖穆杰塔巴虽在美以的第一波袭击中受了伤，但仍被推上高位，可见伊斯兰革命卫队急于扶持它属意的人选登上权力巅峰，也显示它有信心即使穆杰塔巴无法亲自掌舵，伊朗的战争机器也自能运转。  
英国《卫报》星期三（3月11日）的分析报道指出，穆杰塔巴的具体伤势依然不明，但据称他至少腿部和面部受伤。伊朗最高国家安全委员会秘书拉里贾尼说，穆杰塔巴的伤势“据说并不危急”，这说法意味着，他也未亲眼见到穆杰塔巴。 
伊朗专家会议星期天（8日）宣布，推举已故最高领袖哈梅内伊之子穆杰塔巴为第三任最高领袖。四天后的星期四（12日），穆杰塔巴才首次通过国家电视台发表声明。然而，他仍没有在视频中露面或公开现身，这不免引发诸多揣测。 
在海外的伊朗反对派早前称，穆杰塔巴陷入昏迷，在医院接受治疗。向来受严格审查的伊朗媒体也设法探知新领导人的状况。一家媒体星期二（11日）问外交部发言人，穆杰塔巴是否已接手最高领袖职务，发言人没有正面回答，只说：“那些应该接收指令的人已经收到了指令。”
反对派称，伊斯兰革命卫队挟穆杰塔巴为傀儡，借助他的名望操控国家。革命卫队不仅是一支军队，更是一个商业帝国，投资遍布伊朗的多个经济领域。
牛津大学伊朗历史与政治副教授阿勒姆扎德指出，伊朗的政治与权力结构经过特别设计，领导层可轻易替换，这股韧性依赖伊斯兰革命卫队、巴斯基民兵和其他部门组成的网络，而美以军事行动未能打击这个网络。
彭博社报道，美以空袭削弱伊朗生产“见证者-136”（Shahed-136）自杀式无人机的能力，但库存仍在，要增产也不需复杂的零部件。
据彭博社估算，伊朗这十多天发射超过2100架这款无人机，袭击波斯湾周边的目标，破坏石油基础设施、摧毁军事设施以及迫使机场关闭。尽管这款无人机速度慢、容易被发现，但它凭借数量优势，消耗对手的拦截导弹库存。

大豆采购料成习特会焦点 分析：特朗普中期选举欲巩固农民选票
中国国家主席习近平和美国总统特朗普预计本月底、下月初在北京举行会晤，中国何时恢复向美国采购大豆，料是两国元首会谈的重点之一。
受访学者指出，美国农民是特朗普重要支持者，美国关注大豆贸易，与特朗普今年底面对中期选举的政治利益息息相关；中国则希望美国针对台湾问题，作出一些更利于中国的承诺。
美国白宫早前透露，特朗普将在3月31日至4月2日访华，中国方面则还未公布任何细节。美国财政部长贝森特、贸易代表格里尔和中国国务院副总理何立峰，预计本周末在巴黎会面，为习特会做准备。
据彭博社星期四（3月12日）报道，大豆采购很可能成为中美贸易官员会晤的议题，此次会谈或许能揭示北京何时计划真正恢复向美国采购大豆。
大豆是中美贸易关系的核心，凸显两国农民之间的相互依存关系。中国农民需要进口大豆作为饲料喂养牲畜，美国农民则依靠出口维持生计，而他们一般上是特朗普的支持者。
特朗普去年重返白宫再发起贸易战后，中国停止向美国进口大豆。双方后来在10月达成为期一年的贸易休战协议，北京也履行了向美国购买1200万吨大豆的承诺。但在这之后，采购陷入停滞，何时恢复尚不明朗。
根据休战协议，从今年起的三年内，中国将每年向美国进口2500万吨大豆。在完成了最初的1200万吨承诺后，贸易商认为中国买家可能会等到9月新豆上市后再采购，因为届时价格通常最具吸引力。
特朗普今年2月在他的社媒平台Truth Social发文称，中国正考虑将这一季的大豆采购增至2000万吨。市场对中国可能提前采购更多大豆有了强烈预期，但中国并未对特朗普的发文作出任何回应。
独立农业市场信息公司No Bull Ag分析师斯特劳德告诉彭博社，3月至今，美国仍未从中国获得任何善意的大豆采购，“这种情况我们在过去的谈判中也曾遇过”。
复旦大学国际问题研究院院长兼美国研究中心主任吴心伯接受《联合早报》采访时说：“特朗普关心的还是做生意。中方采购美国的农产品、能源产品，还有飞机等，其中农产品最关键，因为跟他的中期选举是相关联的，有非常重要的政治利益在里面。”
不过，吴心伯认为，中国目前最关心的还是台湾问题，这可从今年2月两国领导人的通话中看出来。中国特别呼吁美国“务必慎重处理”对台军售，也希望美国明确表态反对台独；这将是观察美国能不能照顾中国关切的两个指标。 
吴心伯说，如果美国不在中国的核心利益台湾问题上照顾中国的关切，要中国购买美国农产品、波音飞机和能源产品等，“美方恐怕很难如愿以偿”。
新加坡南洋理工大学拉惹勒南国际研究院副教授李明江分析，此次两国领导人会晤，中国的期望高于美国。中国至少期望通过这次访问能让双边关系有某种形式的突破，包括经贸关系更加长期稳定，以及美国在台湾问题能有一些更有利于中国的承诺。
李明江指出，美国视中国为全方位的战略竞争对手，要美国在安全、经贸或科技方面做出任何的大让步，比较困难，“双方能重申、巩固已有的一些共识，已算是很好的结果”。
他预计，此次会晤可能取得的一些成果，包括双方延长贸易休战，中国采购更多大豆等美国农产品、美国民航飞机，以及为美国企业在中国提供更多市场便利等。此外，美国若愿意在科技出口限制方面做出一些让步，对中国也是利好。

航运危机加剧 欧洲天然气价格追随油价上涨
随着中东航运危机加剧，市场为中断情况将持续数月做准备，欧洲天然气追随油价上涨。
彭博社报道，基准期货价格连续第二天上涨，一度飙升7.7%并收复本周前些时候的部分失地。阿曼主要石油出口码头疏散船只，两艘油轮在伊拉克水域遭到袭击，进一步反映出全球能源市场承压。
在液化天然气（LNG）领域，随着市场供应收紧，亚洲买家正在准备为未来两个月增加采购。
摩根士丹利上调了欧洲天然气年内价格预测，因为该地区需要在夏季大量囤积LNG以补充枯竭的燃料库存。根士丹利预计，卡塔尔LNG生产将至少中断一个月，2026年原本料将出现的全球燃料过剩局面因此彻底无望。
阿姆斯特丹时间星期四（3月12日）上午8时26分（新加坡时间傍晚5时26分） ，作为欧洲天然气基准的荷兰近月期货上涨3.4%，至每兆瓦时51.67欧元（76.27新元）。
