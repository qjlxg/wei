瓦拉武特·西拉帕查（Warawut Silpa-archa）履历揭要
瓦拉武特生于1973年7月11日，是已故前总理班汉·西拉帕查（Banharn Silpa-archa）与妻子詹赛的最小儿子。早年就读圣嘉布遣学校，后赴英国伦敦大学学院取得工程学学士，继而在美国威斯康辛大学麦迪逊分校取得金融方向MBA，具备国际化教育背景与系统化思维。

复出后，他以Chartthaipattana党身份出任总理巴育（Prayut Chan-o-cha）内阁的自然资源与环境部长，推动一次性塑料禁用与国际气候行动。随后在社会发展与人类安全部任职，主攻弱势儿童、妇女与老人等社会基础建设。
私生活方面，他与苏瓦娜结婚，育有三名子女；长女“Tems”为泰国花样滑冰国家队选手，展现家族在政务与体育领域并重的形象。

新华社报道，科学家对化石进行分析后认为，这是一种长颈食草蜥脚类恐龙，体长可达约20米，是迄今在巴西所发现体型最大的恐龙之一。科学家以化石的发现地托坎廷斯河东岸为其命名，称它为托坎廷斯龙（Dasosaurus tocantinensis）。
参与研究的科学家还发现，新确认的恐龙与在西班牙发现的加伦巴巨龙（Garumbatitan morellensis）有极高的相似度。这表明，这种恐龙可能源于今天的欧洲，其祖先在约1.2亿年前、大西洋尚未完全形成时经由北非迁徙至现在的南美洲。
这项研究由巴西圣弗朗西斯科河谷联邦大学研究员埃尔弗·迈耶领衔，得到其他多家研究机构专家的合作。研究结果发表在古生物期刊《系统古生物学杂志》上。

空袭击中伊朗小学疑数据过时所致 美军调查升级
（华盛顿综合电）对于伊朗小学在美以首发空袭时被摧毁、造成逾百名学生死亡，美国军方说，有关调查已提升到更高层级。
2月28日美国和以色列向伊朗发动战事，导弹击中伊朗一所女子小学，据报有180名学童死亡。此前有媒体报道称，调查显示美军很可能用了过时数据，制定错误的打击坐标而酿成惨祸，因此须对此负责。如果证实是美军过失，这将是数十年来，美方在中东军事行动中导致平民死亡最严重的事件之一。
美国战争部长赫格塞斯星期五（3月13日）在五角大楼的记者会上，拒绝对有关调查的初步发现置评。“我们不会让媒体报道主导我们，或迫使我们透露发生了什么事。”
针对军方启动更高级别的调查，他说调查由中央司令部以外的军官主导，以确保调查有更大的独立性。“指挥部的调查工作将须要花多长时间就花多长，务必查清与这起事件相关的一切。”
美国三名匿名官员告诉路透社，赫格塞斯所指的调查是一项名为“15-6”的行政调查。这类调查通常包括相关人员的宣誓陈述或面谈，若有必要也可作为采取纪律处分的依据。
在这次打击伊朗的行动中，五角大楼被批对外沟通不良，信息量少，缺乏细节，且对主流媒体持对立态度。
过去，在伊拉克和阿富汗战争以及打击伊斯兰国组织期间，媒体记者几乎每天都能获得详细简报，甚至随军采访；可是，美国军方对眼前这场打伊行动的披露方式明显有异。许多关键问题，如美国是否必须对伊朗小学的致命打击负责，以及华盛顿对战后伊朗有何计划等，都未得到回应。
拥有20年公共服务经验的前美军上校沃伦说：“五角大楼发布的信息比以往任何时候都少，而且这些信息……内容也不够充分。”
佐治亚大学传播学教授斯塔尔指出：“如今真正在五角大楼周围工作的记者已大幅减少，透明度已经成为一个严重的问题。”
他说，这意味着“那些不光彩的事实即使最终会曝光，也得等上更长的时间。政府在提供连贯的理由和计划方面将面临较小的压力。”
对于伊朗问题，斯塔尔说：“特朗普政府一直完全不透明。”
他还说：“这是第一次在发动重大行动时，美国政府连向国会和民众解释都懒得做。在发动袭击后，它也没有真正尝试提供一个合理的说法。”
