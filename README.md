联合战斗机项目停滞 德法举行关键防务会谈
在旗舰联合战斗机项目搁置后，德国与法国领导人举行关键会谈，讨论两国防务合作的未来。
星期五（7月17日）上午，德国总理默茨在科隆附近的诺尔韦尼希空军基地（Noervenich air base）迎接马克龙。
两名领导人从象征法德成功合作的超级美洲狮（Super Puma）直升机走下来，随后在法国“阵风”战斗机（Rafale）和德国空军主力机型“台风”战斗机（Eurofighter Typhoon）旁主持联合防务与安全会议。
法新社报道，两国政府全体内阁成员齐聚科隆附近的一座城堡举行会谈。1962年，法国总统戴高乐（Charles de Gaulle）和德国总理阿登纳（Konrad Adenauer）正是在这里达成共识，缔结两国之间的友好条约。
预计双方将在会谈中成立一个联合“指导小组”，以深化在雷达系统、“纵深打击能力”和导弹防御方面的合作。
法国总统马克龙认为，在明年春季的总统大选之前，迫切需要在这一领域取得进展。极右翼领导人勒庞是此次大选的热门人选，有可能成为他的继任者。
马克龙星期四说，他希望此次会谈能为防务合作注入“新的活力”，助力打造一个“团结我们力量的强大欧洲”。
由德国与法国和西班牙联合发起的未来空中作战系统（FCAS）战斗机项目上个月破裂，此次会谈旨在走出阴影，“重振旗鼓”。这个项目因空中客车公司和法国达索公司之间的争执而告吹。

涉捏造小米SU7碰撞后断电 中国博主获刑20个月
涉发布测评视频捏造中国科企小米的SU7电动汽车碰撞后断电，中国一名有近百万粉丝的博主一审获刑一年八个月，并被罚款10万元（人民币，1.9万新元）。
中共北京市委机关报《北京日报》旗下“长安街知事”星期五（7月17日）引述其查阅的判决书称报道，北京市海淀区人民法院近日对上述高姓博主损害商品声誉一案依法作出判决，认定其因“捏造虚假事实”“主观故意损害他人商品声誉”等，被判犯损害商品声誉罪。
报道称，这是中国国家网信办、市场监管总局今年6月联合印发《网络测评活动规范》后少有的标志性“网络测评”案例。
据悉，2024年8月，汽车自媒体高姓博主发布了一段汽车碰撞评测视频，在社媒上的播放量迅速突破300万次。
这名博主在视频中称，小米SU7因设计缺陷导致碰撞后全车断电、紧急救援呼叫系统失效。碰撞后工作人员拉不开车门、自动紧急呼叫未见工作痕迹，中控电子屏不亮。
视频引发争议，有网民发现有的画面小米SU7车身上有小电瓶，有的画面中没有，因此怀疑是博主摘了小电瓶后进行拍摄，故意营造碰撞后断电的情况。
公开资料显示，小米公司于2024年11月报警，2025年通过官方微博公布，相关博主及其同伙“已经被依法缉拿归案”，直至今年7月案件宣判。
根据判决书，高姓博主及团队在制作测试视频过程中，隐瞒小米SU7小电瓶的电源线在碰撞之前被人为拆装的事实，使用了被叉车损坏的小电瓶照片特写，并将损坏原因直接描述为在碰撞中损坏，在未经核实的情况下宣称小米SU7的紧急救援呼叫系统功能遭碰撞后没有正常运作，致使小米汽车声誉受到严重影响。
