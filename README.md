特朗普接受福克斯电台访问时被问及此事，他说：“这款鞋子不错。”
79岁的特朗普说：“我会留意，有谁老是穿着一双不合脚、不舒服的鞋子。”有时他会当面猜测他的部长穿几号鞋子。
特朗普说：“很有趣，当他们来告诉我，他们遇到一个难题时，我会说，‘让我买双鞋子送给你’。这招似乎很管用。现在他们看起来精神焕发，又帅气。这是来自唐纳德·特朗普的礼物。”
特朗普也说，他“不喜欢内阁成员穿运动鞋走来走去”。
《华尔街日报》称，穿Florsheim品牌鞋子的高级政府官员包括副总统万斯、国务卿鲁比奥、战争部长赫格塞斯。
有一张照片显示鲁比奥好像穿同一款鞋子，但他的鞋跟显得有点大。
在Florsheim官方网站上，与鲁比奥照片中的鞋子最相似的一款鞋子是中国制的皮革正装鞋，每双售价145美元。

意大利政府据报将与耐倍力及投资者商讨中国控制权争议
路透社星期五（3月13日）引述两名知情人士透露，意大利政府据报将于下星期一（16日）与耐倍力和两大主要投资者的代表会面，商讨解决围绕中国股东中化集团对公司影响力引发的争议。
据报道，意大利轮胎制造商倍耐力（Pirelli）两大股东中国国有企业中化集团和Camfin分别持有34%和25.3%股权，是这场公司控制权长期争议的核心单位。
Camfin以及倍耐力方面均呼吁，限制中化集团的持股地位，并称如若不然将使倍耐力在美国这一高端轮胎业务关键市场的扩张计划复杂化。
意大利政府使用黄金权利保护属于国家利益的资产，正考虑解决相关争议的选项，包括将中化集团的股权转移至保密信托，冻结集团的投票权。
其中一名知情者星期五称，中化集团倾向一个临时方案，即允许集团长期持有耐倍力的股权，但治理权利受限。
上述两名知情者称，意大利政府预计将于4月中就耐倍力的治理权作出裁决。
另两名知情人士对路透社说，中化集团也考虑发行可转换债券，将暂时中断集团对耐倍力的持股。
意大利通讯社Radiocor最先报道，意大利政府下星期一与耐倍力和两大主要投资者的代表会面的消息。意大利总理梅洛尼办公室官员将参与会面。
中化集团今年1月提交一份提案，以解决耐倍力的治理问题，以及美国潜在的监管担忧。
彭博社去年9月引述知情者报道，美国商务部工业和安全局7月中旬致函意大利官员，称意大利在2023年为了限制中化影响力而对倍耐力动用的否决权，不足以保护公司免受美国限制。
据彭博社去年4月报道，倍耐力董事会在接到监管要求后，作出了将中化治理权降级的决定。
