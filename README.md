曼谷第6選區第74投票站重新計票結果與2月8日相同，人民黨以198票領先
2026年3月1日12:47 報導／記者報告，泰國選舉委員會（กกต.）在曼谷丁達能區（Din Daeng）宣布對第6選區第74投票站進行重新計票，計票僅針對粉色的政黨名單（บัญชีรายชื่อ）選票。
