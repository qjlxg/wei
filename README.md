沙赫德与LUCAS：两款“自杀式”无人机解读
沙赫德‑136（伊朗）与美制LUCAS——在美以联盟与伊朗冲突中成为焦点的两种近似自杀式滞空弹药。美方媒体称，LUCAS系直接逆向工程并规模化生产的沙赫德克隆版本，用于反制伊朗。
概况对比：沙赫德由Shahed Aviation研制，机长约3.5米、翼展约2.5米，巡航速约185公里/时，射程可达约2000公里，有约50公斤（部分型号可达90公斤）弹头，单价在1万–5万美元（出口价可高达19.3万美元）。LUCAS由SpektreWorks与美军合作，外形与性能相近，单价约3.5万美元，具备自主与联网能力，可用弹射、移动发射台或舰载发射。
规模决定战果：沙赫德并非技术领先，优势在于低成本、结构简化、易大量生产，从而能以数百枚饱和攻击压垮防空。美方同样转向“可消耗化”武器，LUCAS支持编队与互联协同。
首战表现：冲突前六天，伊朗被报道曾发射数百架沙赫德攻击海湾目标。优点是数量大、射程远、低空难以雷达侦测；但推进慢、噪音大，防空系统（如爱国者、铁穹等）有较高拦截率。
LUCAS在海湾已短期部署并从舰艇发射，主要用于打击导弹发射点、雷达与实施编队打击。
核心结论：今日战争的关键不在于单机优劣，而在于能否大规模生产并运用廉价自杀无人机。能制造成万架级别的一方，或将掌控战场主动权。

美国“不担心”俄罗斯协助伊朗锁定美军目标
美国战争部长赫格塞斯说，美国对有关俄罗斯向伊朗提供美军部署和动向情报的报道“并不担心”。 
《华盛顿邮报》星期五（3月6日）引述三名美国知情官员称，俄罗斯正在向伊朗提供有关美国军事资产的位置信息，包括美军在中东地区的军舰和飞机位置。
尽管赫格塞斯拒绝予以证实，但他当天接受哥伦比亚广播公司（CBS）《60分钟》采访时说：“我们密切关注一切。”
赫格塞斯说：“美军司令对一切了如指掌，我们拥有世界上最好的情报。我们知道谁在与谁通话。”
“我们并不担心。我们会在需要时采取应对措施。”
白宫也淡化有关俄罗斯向伊朗提供驻中东美军目标信息的报道。白宫新闻秘书莱维特告诉记者：“这显然没有对我们在伊朗的军事行动产生任何影响，因为我们正在彻底摧毁他们。”
《华盛顿邮报》称，中国似乎未向伊朗提供防御援助。
美国驻科威特基地3月1日遭无人机袭击，造成六名美军士兵死亡；位于沙特阿拉伯利雅得的中央情报局（CIA）站点也遭到袭击。

中国今年将开展补贴性职业技能培训逾1000万人次
中国官方披露，今年将深入开展大规模职业技能提升培训，计划开展补贴性培训逾1000万人次。
中国人力资源社会保障部部长王晓萍星期六（3月7日）在民生主题记者会上说，当前技术变革日新月异，技能更新速度加快，学习成为这个时代的必然要求。
他说，中国将根据劳动者不同阶段的个性化需求，构建贯穿职业生涯全过程的终身职业技能培训体系，努力实现人人皆学、处处能学、时时可学。
针对不同群体，中国将开展差异化培训。王晓萍说，对新成长劳动力，开展预备制培训、通用技能培训；对企业职工，开展学徒培训、在岗培训；对转岗人员，开展新技能培训、创业培训。
王晓萍指出，中国今年将深入开展大规模职业技能提升培训，补贴性培训1000万人次以上，重点围绕低空经济、新能源汽车、人工智能技术、康养服务等组织专项培训，更好紧贴产业、服务就业。

中国预计今年高校毕业生人数将达1270万
中国官方预计，今年高校毕业生人数将达到1270万，比去年的数据增加48万。中国去年应届高校毕业生预计人数达到1222万。
中国人力资源社会保障部部长王晓萍星期六（3月7日）在全国人大会议民生主题记者会上强调，“十五五”时期（2026年至2030年）中国经济长期向好的支撑条件和基本趋势没有变，高质量发展前景广阔，加上强大的制度优势，实现就业大局稳定、态势向好，“我们有信心、有能力”，但坦言不确定难预料因素同时增多，稳就业面临新变化新挑战。
王晓萍也说，中国将深入实施就业优先战略，强导向、扩渠道、保重点，不断推动高质量充分就业取得新进展。
王晓萍列出重点做好的三项工作，其中包括构建就业友好型发展方式。“我们正在会同有关部门编制‘十五五’就业专项规划，谋划一批务实管用、含金量高的重大就业政策、行动计划等，提高发展的就业带动力。”她也说：“我们也在研究相关政策，积极发挥人工智能在创造新岗位和赋能传统岗位方面的作用，推动实现技术进步与民生改善相协调的包容性发展。”
另一项重点做好的工作为实施稳岗扩容提质行动。王晓萍介绍，“稳岗”方面，加大对外贸、建筑、住宿餐饮等劳动密集型行业的扶持，在促进行业发展中稳岗位。“扩容”方面，挖掘数字经济、高端制造、现代服务等领域就业潜力，在推动产业升级中扩容量。“提质”方面，落实好最低工资标准调整机制，规范人力资源市场秩序，全面落实农民工工资支付保障制度等，在强化权益保障中提质量。
中国官方也将重点做好强化精准施策，促进重点群体就业的工作。王晓萍说：“我们将深挖各渠道就业潜力，拓展基层就业空间，帮助青年人在各行各业施展才干。抢抓春招和离校前的时间窗口，推动就业公共服务提前进校园，开展大规模就业见习和能力实训，引导青年人涵养职业精神，更好融入职场、融入社会。”
对于大龄劳动者，王晓萍说，中国官方将推出一批适合他们的技能培训项目，深化劳务协作，拓宽农民工就业路子，对防止返贫致贫对象加强常态化就业帮扶，对困难群体开展就业援助，帮助他们稳定就业增收。
王晓萍也指出，今年以来，中国官方组织开展春风行动暨就业援助季活动，已举办3万1000场招聘会，发布2200万个岗位。从春节后调研情况看，目前企业开工率、劳动者返岗率都达到九成以上，劳动力市场实现温暖开局。接下来还将陆续推出系列服务活动，努力实现全年招聘不打烊、送岗不停歇。同时，用好用足稳岗返还、社保补贴、创业担保贷款贴息等惠民惠企政策，用政策服务的力度提升民生保障的温度。

伊朗称向美国航空母舰“林肯”号发射导弹
伊朗军方称，伊朗导弹部队星期五（3月6日）晚向美军“林肯”号航空母舰发射了导弹。
伊朗塔斯尼姆通讯社报道，伊朗伊斯兰革命卫队说，6日继续推进代号为“真实承诺－4”的系列军事打击行动，启动第24波攻势，针对以色列及美军相关目标实施远程打击。
伊朗伊斯兰共和国通讯社引述伊朗伊斯兰革命卫队的声明说，第24波攻势是在前一波打击后立即展开的，导弹发射和无人机攻击已覆盖多个军事目标，对以色列特拉维夫发射的三枚导弹均命中预定目标，“打击行动取得成功，敌方重要目标被有效压制”。
报道还说，在第23波攻势中，伊朗伊斯兰革命卫队动用新一代固体、液体燃料导弹，打击位于中东多国的美以军事基地，以及以色列南部城市贝尔谢巴的先进技术中心、网络安全中心和军事支援中心。

中东战事升级，英国首相斯塔默与沙特阿拉伯领导人通电话，表明英国准备好协助沙特保卫家园。

中国民政部：争取十五五末建成县乡村三级养老服务网络
中国民政部长陆治原针对养老服务改革表示，争取到“十五五”末基本建成县乡村三级养老服务网络。
陆治原星期六（3月7日）在全国人大经济主题记者会上说，民政部会同有关部门狠抓《关于深化养老服务改革发展的意见》的落地落实，已初步建成覆盖城乡的养老服务“一张网”，大力发展村（社区）支持的居家养老服务等。
他还说，在养老服务产品供给方面，发挥政府主导作用，强化政策支持，发展普惠性养老服务；并推进政府引导、基层群众性自治组织支持、老年人自愿互助、社会广泛参与的互助性养老。
陆治原也提到，民政部指导各地科学规划养老服务设施布局，制定养老服务国家标准和行业标准，大力发展智慧养老。加强养老服务人才培养培训，推动职业技能等级评定，将“养老服务师”纳入新职业。
他强调，将进一步完善深入实施积极应对人口老龄化国家战略的政策措施，推动养老服务业扩容、提质、增效。扩容方面，未来五年持续建设县级综合养老服务管理平台，拓展区域养老服务中心功能，因地制宜扩大村和社区养老服务设施站点的覆盖面，争取到“十五五”末基本建成县乡村三级养老服务网络。
提质方面，将加大养老机构设备更新力度，加强养老服务人才专项培训，大力发展养老科技，加强养老服务质量安全管理。增效方面，将持续扩大普惠性养老机构供给，培育专业化连锁化品牌化养老服务经营主体，大力发展村（社区）为依托的居家养老服务，支持发展互助性养老服务等。
