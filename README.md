（韩国投资证券，蔡敏淑，金延俊）
- 从6月22日至26日，连续五天与美国投资者会面，进行了存储半导体相关会议
- 虽然对利润预测大幅上调存在担忧，但大多数投资者认同估值相对于利润水平仍处于较低状态
- 对SK海力士ADR的关注度很高，也有许多投资者询问三星电子是否有可能发行ADR
- 除了DRAM，对NAND的关注也超出预期，主要问题是AI服务器需求将如何惠及NAND价值链
- 投资者们认同，由于投资缩减的影响，NAND的供应也进入了结构性受限阶段
- 从第三季度开始，PC和智能手机客户公司将难以进一步提高价格，而服务器客户公司将反映LTA价格，因此下半年DRAM ASP涨幅预计将较上半年放缓
- 预计2027年HBM价格将比2026年上涨100%以上，在模型中反映HBM4为3.0美元，HBM4e为3.3美元水平
- 市场期待更高的价格，但鉴于HBM特性，利润率难以达到通用DRAM的水平
- 黄仁勋访韩期间，英伟达与SK海力士签订了包括HBM在内的LTA，预计三星电子和美光也将与英伟达签订LTA
- 即使因LTA而价格上涨受限，但DRAM合同市场不同于现货市场，更优先考虑数量和长期伙伴关系，因此从供应商角度来看难以视为损失
- 客户公司们在了解2028年以后新Fab产能的情况下，首先提出长期合同，这被解读为供应短缺不会在短期内结束的信号
- 对美光SCA在与市场分享合同细节方面给予积极评价，并了解到其条件与三星电子和SK海力士的LTA相似
- 五家存储供应商均以相似条件签订了3至5年的LTA，这显示存储业务模式本身正在发生变化
- NAND供应商们被判断为优先考虑盈利性而非份额扩张，从而制造供应短缺局面
- 有限的资本支出首先分配到以HBM为中心的DRAM投资，因此NAND扩产相对缓慢
- Sandisk HBF是弥补NAND带宽限制的有趣尝试，但不同于HBM，由供应商主导开发，因此预计需要时间才能确立为标准
- 控制器、接口、软件优化等技术难题仍需解决，因此短期量产预计不易实现
- CXMT和YMTC等中国存储企业的扩产是中长期风险，但增加的产能大部分被中国国内AI需求吸收，因此影响被评估为较过去更有限
- 由于美国出口管制，存储供应链逐渐呈现美国中心和中国中心分化的迹象
- 三星电子代工厂将TSMC先进工艺供应不足和边缘AI扩散视为机会，以4纳米客户基础为跳板，预计能逐步扩大
- 不过在先进封装领域，TSMC和英特尔领先，因此被指出为三星电子必须追赶的部分
- 对于半导体板块，维持买入和权重扩大意见，即使短期波动存在，中长期估值重估可能性仍有效

亚洲人进门脱鞋有道理 实验发现鞋底藏致病细菌
在许多亚洲家庭，进门脱鞋是再自然不过的事；但在欧美一些家庭，穿着鞋进屋也很常见。这个看似普通的生活习惯差异，最近被英国广播公司（BBC）拿到实验室做了检测。结果发现，鞋底可能带着致病细菌，让人忍不住感叹：“比我想象中脏多了。”
BBC 近日前往英国布莱顿大学的微生物实验室，请微生物学家皮特（Sarah Pitt）检测一双穿过的鞋。皮特用棉签分别在鞋面和鞋底取样，再把样本涂到琼脂培养皿中。24小时后，培养皿里长出了不少细菌，其中就包括金黄色葡萄球菌（Staphylococcus aureus）。
皮特说，金黄色葡萄球菌可能相当危险。即使是健康人，如果皮肤有抓伤，细菌进入伤口，也可能引发脓肿或化脓性感染；它还可能导致肺炎等严重感染，并在免疫功能较弱者身上造成血液感染。
据报，进屋该不该脱鞋的问题，在不同地区和互联网上都引发分歧。美国女子索菲亚和塞尔维亚丈夫佩贾此前就在社交媒体上分享过两人对此的不同看法。索菲亚觉得进屋脱鞋很奇怪，佩贾则说，自己从小在家就是要脱鞋的。
皮特指出，鞋上发现的细菌可能在家中存活一段时间，甚至长达数天，并有机会感染家中其他人；地毯这类安全、温暖的环境尤其有利于细菌停留。她也提醒，世界各地还可能存在不同细菌、真菌和寄生虫，因此建议不论身处何地，进屋后都应尽快脱鞋。

塔利班新法令变相承认童婚 联合国批侵蚀女童权利
阿富汗塔利班政权颁布有关离婚程序的新法令，因其中多项条款被指变相承认童婚，引发联合国、人权组织和法律学者批评。新法允许监护人为未成年人安排婚姻，并未设定最低结婚年龄。
美国全国公共广播电台（NPR）报道，根据塔利班司法部颁布的新法令，第五条规定，儿童监护人有权为未成年人安排婚姻，相关婚姻契约被视为合法；第七条则允许把已进入青春期的处女保持沉默，视为同意结婚。
联合国阿富汗援助团秘书长副特别代表加尼翁（Georgette Gagnon）指出，这项法令进一步将歧视制度化；若与限制女童受教育和女性参与公共生活的措施结合，将进一步巩固一套剥夺阿富汗妇女与女童自主权、机会和司法救济渠道的体系。
阿富汗人权律师兼人权观察研究员阿巴西（Fereshta Abbasi）说，旧民法典规定女童适婚年龄为16岁，父母同意下可在15岁至16岁之间结婚；但新法令没有列明任何最低年龄限制。
她批评，这样的法律赋予男性大量权力，几乎剥夺女性所有保护，并违反《消除对妇女一切形式歧视公约》和《儿童权利公约》。
塔利班2021年重新掌权后，禁止女性在法律系统工作，女律师和女法官也被排除在外，使女性面对家暴或强迫婚姻时更难寻求法律支持。法令虽允许儿童在进入青春期后向法院申请撤销婚姻，但没有明确界定“青春期”；相关申请还需要证人，女性前往法院也须有男性监护人陪同，使司法救济困难重重。
除法律保护倒退外，女童失学和经济危机也加剧童婚风险。
报道说，估计近85%的阿富汗人艰难求生，一些家庭为取得男方支付的“聘金”维持生计，因而更早安排女儿结婚。阿富汗妇女历史转型运动创办人之一萨伊（Roqia Saee）批评说，塔利班把童婚纳入正式法律结构，使其得以延续和扩大。

强化长效约束 中国券商密集修订薪酬管理制度
（上海/北京综合讯）中国证券行业正推动薪酬制度改革，逾40家券商今年陆续公布或修订了薪酬制度，主要集中在绩效薪酬占比、薪酬递延支付、追索扣回机制等，以强化对行业的激励约束体系。
综合《国际金融报》与中国《金融时报》报道，中信证券、国泰海通、中金公司、华泰证券等12家券商上星期五（6月26日）集体发公布了新版薪酬管理制度。中国银河、信达证券等八家券商星期一（29日）晚间公告，相关薪酬管理制度获股东会决议通过。
截至目前，约42家中国券商公布或修订了相关薪酬管理制度，其中30余家已获股东会通过。
此次中国券商修订薪酬制度，是监管部门整改要求下的集中落地。中国证监会修订发布的《上市公司治理准则》1月实施，提出上市公司应当建立薪酬管理制度，中国证券业协会4月发布修订后的相关指引，引导券商规范相关制度。
各家券商本次修订的主要内容，包括绩效薪酬占比、递延支付、薪酬止付、追索扣回机制等。
关于绩效薪酬占比，约37家券商提出高级管理人员或董事薪酬的浮动比例要求，大多数券商将绩效薪酬占基本薪酬与绩效薪酬总额的比例，设定为不低于50%。
多家券商对高管、核心业务人员、风险岗位人员等建立绩效薪酬递延支付机制，递延支付比例普遍不低于绩效薪酬的40%，递延期限一般不少于三年，首次支付不早于绩效薪酬归属年度后的第二年。
此外，多家券商提出，若相关人员存在违法违规、重大失职等情形，公司可对其绩效薪酬进行止付、减少支付，甚至追索扣回已支付薪酬，中信证券、中信建投等券商还提出，追索扣回范围适用于离职和退休的责任人员。
