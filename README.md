 这个比较猛，重点是铸牢中华民族共同体意识，对境外破坏民族团结的行为有了明确法律追责。影响最大的地方在于民族地区和涉及民族事务的工作，以后相关宣传、教育和跨境管理会更严。
3. 《中华人民共和国社会救助法》
 以前低保边缘家庭、支出突然困难的家庭救济不太明确，现在把救助对象扩大了，还建了分层分类的体系。
4. 《超龄劳动者基本权益保障暂行规定》
 用人单位必须按时足额发工资、少安排加班、岗位强度要合适，还得给上工伤保险。影响最大的是现在很多返聘老人和灵活就业的群体
😂

 “即买即退”时间延长到28天，还搞无纸化、小额抽检，退税商店覆盖面扩大。影响最大的是来中国旅游购物的人，尤其是境外游客，消费更方便。

协会在信中说：“虽然有针对性的政策有助提升国内供应韧性，但那些扭曲价格或产能决策的干预措施，反而可能导致需求下滑持续更长时间。”
它补充说，目前业界正通过扩大美国制造业投资，以及与客户签订更多长期供货协议，来应对储存晶片供应紧张的问题。

网信办整治网络娱乐团播乱象 包括内容低俗不良等六类问题
中国中央网信办开展专项行动，整治娱乐团播违规PK刺激打赏、团播场景内容低俗不良等六类问题。
中央网信办今日星期五（7月3日）发布通知，在全国范围内开展为期两个月的“清朗·网络娱乐团播乱象整治”专项行动。
通知称，专项行动期间，重点围绕娱乐团播整治六类问题，包括团播账号注册使用违规，团播之间或团播成员间违规PK刺激打赏，运用不当团播玩法诱导打赏，团播场景内容低俗不良，侵害未成年人权益，以及MCN机构管理失范。
当前团播之间或团播成员间存在违规PK刺激打赏，例如用PK结果操控团播成员行为，进行低俗恶俗、带有人身侮辱性质的惩罚或进行危险动作。
通知也列出团播场景当前也存在内容低俗不良的场景，例如直播间入口页面使用暧昧猎奇等违规不良图片进行引流，诱导用户点击；直播间模拟布置酒吧、夜店等场景，刻意设计镜头机位和道具位置，采用特殊运镜和特定拍摄手法，进行性暗示；或穿着情趣、透视、破洞等服饰，暴露或凸显隐私部位，摆出顶胯、抖胸等低俗浮夸动作。
通知也列出侵害未成年人权益的现象，如招募不符合法律规定情形的未成年人加入团播，在直播中穿着童装、校服、书包等明显具有未成年人特征的服饰，或模拟未成年人语气，模仿未成年人直播等。
“团播”即“团队直播”或“团体直播”，是一种多位主播同时在同一直播间内互动的演艺形式，观众可以通过弹幕、给不同成员送礼、投票等方式参与。
中国演出行业协会发布的报告显示，2025年团播市场规模预计突破150亿元（人民币，28.5亿新元），日均开播直播间超过8000个。
今年1月，中国行业协会发布首份团体直播管理标准，要求不浓妆艳抹或过度磨皮美颜。
