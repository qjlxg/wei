为此，墨西哥客车与卡车制造商正呼吁采取“双管齐下”的解方：一方面，对来自未与墨西哥签署自贸协定国家（主要是中国）的重型车辆征收高达50%的关税（目前关税区间为5%至20%）；另一方面，呼吁美国将墨西哥制造的卡车视为北美工业基础的一部分，而非安全威胁。
