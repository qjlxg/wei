巴拿马港口公司批巴拿马政府搜查无视法治侵占私人财产
巴拿马港口公司批评，巴拿马政府搜查当地办公室，是无视法治侵占私人财产行为。
据路透社报道，香港长和集团旗下的巴拿马港口公司（PPC）星期五（2月27日）在声明中说，巴拿马政府“无视法治，正如其昨天（26日）采取的最新行动，入侵并没收PPC的财产那样”，并补充称，公司和投资者保留就此事件相关的一切适用权利。
巴拿马港口公司星期五说，公司已多次反对巴拿马政府所谓的非法接管港口和扣押公司财产的行为，其中包括机密且受法律保护的信息。

公司强调，巴拿马过去一年，尤其在过去一周里，已经（用行动）表明外国投资者不能依赖巴拿马国家的法律或合约框架。

路透社引述知情者称，此次对巴拿马港口公司的搜查，与政府此前撤销长和在巴拿马运河两个港口经营权的决定无关。

中国官媒：维护和平关键在于以行动阻击日本右翼狂飙
中国商务部决定将20家日本实体列入出口管制管控名单后，中国官媒《人民日报》发文指日本再军事化和拥核企图，已对地区安全稳定构成严重威胁，并强调维护和平的关键在于以行动阻击日本右翼的狂飙。
日本首相高市早苗去年11月发表“台湾有事论”触怒北京，日中关系跌至谷底。中国从经贸、军事及人文交流等领域反制，要求高市撤回相关言论，但她并未撤回。
中国商务部星期二（24日）公告，决定将三菱造船株式会社等参与提升日本军事实力的20家日本实体，列入出口管制管控名单。商务部也决定，将斯巴鲁株式会社等无法核实两用物项最终用户、最终用途的20家日本实体列入关注名单。
《人民日报》星期六（2月28日）发表署名“钟声”、题为《以实际行动阻击日本“再军事化”狂飙》的文章中称，中国此举旨在维护国家安全和利益、履行防扩散等国际义务，制止日本再军事化和拥核企图，完全正当、合理、合法，彰显了法治精神与负责任大国担当。
该报续称，中方已于1月6日宣布决定加强两用物项对日本出口管制，此次“点名”具体企业，是将此前禁令转化为精准、可执行的实体管控，以切实维护国家安全与地区和平稳定。
文章指三菱重工、IHI株式会社、川崎重工等多家日本企业，长期活跃于防卫产业，生产舰船、战斗机、导弹等装备，并举例称，三菱重工旗下的多家企业参与了日本高超音速武器系统“岛屿防御用高速滑翔弹”的研制；三菱造船株式会社建造的舰船“不仅服务于日方在钓鱼岛方向的海上侵权，还为菲律宾海警部门建造用于在南（中国）海方向侵权的多功能响应舰，对亚太地区和平稳定构成威胁”。该报称，中国有关举措正是精准遏制日本发展进攻性军力、坚定维护国际法和战后国际秩序的有力体现。
文章称，日本推进再军事化有实际路线和行动，一个新型的军工复合体正在加速形成，并说首相高市早苗执政以来，日本再军事化进程明显提速，备战被提升至国家绝对优先地位，对军工利益集团的扶持进一步加码。该报也说，在推进“再军事化”进程中，日本右翼的拥核野心日益膨胀。
文章强调，中国“依法出台管控措施，正是以实际行动防范两用物项流入日本扩军备武的链条，坚决遏阻军国主义死灰复燃”。中国将同所有爱好和平的国家一道，坚决捍卫战后国际秩序，共同维护地区安全稳定。该报也告诫日本右翼势力应当尽早清醒，“重走穷兵黩武的老路是一条自取灭亡的不归路”。

海牙讯——国际刑事法院（ICC）周五结束对菲律宾前总统杜特尔特的指控确认听证会第四日也是最后一日。审判庭将在60日内审议并作出是否进入正式审判的裁决。
辩方首席律师尼古拉斯·考夫曼称，第2项指控涉及的“高价值目标”（HVT）只是执法上的优先级分类，不能证明存在杀人共谋，主张逐案审查并指出多起事件有武装交火、缴获枪支与弹痕、并有司法搜查令支持。辩方反驳检方过度依赖内部证人、断章取义的警察报告及与杜特尔特关系的牵强关联，并否认除Kian外有证据显示未成年人被害。
检方代表朱利安·尼科尔斯强调证据显示协调与指挥，称杜特尔特任命涉嫌共犯担任要职，言辞为“中和”（neutralize）常被证人解释为“杀死”，并指其言论旨在掩饰非法行动。受害者法律代表吉尔伯特·安德烈斯称案件广泛且系统，要求确认全部指控并移送审判。
庭审结束后，法庭进行了杜特尔特移送至ICC后首次年度羁押复审。检方与受害者律师指出其否认法院管辖并称被“绑架”，有逃亡风险；辩方表示在相关上诉未决时不反对继续羁押。审判庭将依法在规定期限内就继续羁押作出裁定。
