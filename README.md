位於菲律賓以東海域的潛在風暴預料較大機會週末在本港以東登陸，但其與本港的距離須視乎中心整合位置和速度，以及季風槽和副熱帶高壓脊的互動。
潛在風暴過去一兩日發展速度較慢，尚未建立和鞏固中心。人工智能模式預測潛在風暴短期內中心整合速度較快，繼而較早進入南海，預測登陸位置亦略為偏東。部分傳統電腦模式則預測潛在風暴仍須一段時間整合中心，因此起跑時間較遲，預測登陸位置亦略為偏西。
人工智能和傳統電腦模式對風暴初期中心整合的預測有別，在去年數個案例亦有出現，但現時未有足夠數據判斷誰優誰劣。
至於潛在風暴進入南海後的路徑北份量，則須視乎副高往南伸展的幅度。早前傳統電腦模式預測季風槽有殘餘渦度，因而阻礙副高南伸。然而，傳統電腦模式過去一兩日有加大副高南伸幅度，以致潛在風暴預測路徑往東修正的趨勢。此外，新一輪西風波動會在週末通過中國東北部，亦令副高短期內較難進一步往西伸展。因此，現階段預測潛在風暴東登的機會較大。
東登風暴須較接近本港，以及登陸後採取較為偏西的路徑，才會對本港造成顯著的風雨影響。相信潛在風暴中心完成整合後，本港週末風雨預測會更為明朗。

法国6月至7月初热浪致5764人超额死亡
法国公共卫生署说，6月17日至7月2日席卷法国的热浪期间，法国记录了至少5764例全因超额死亡，比正常预期水平增加36%。
新华社报道，超额死亡是指特定地点和时期的实际死亡人数，超出没有健康危机情况下预期死亡人数的部分；“全因”则指不区分具体死因，涵盖所有死亡病例。
法国公共卫生署星期三（7月22日）发布报告说，这次热浪影响法国92个省，覆盖法国本土98.5%的人口。超额死亡主要集中在6月25日至27日，约占总数的56%。
除15岁以下人群外，其他年龄组均出现超额死亡，其中45岁以上人群的死亡增幅较为明显。75岁及以上人群至少有3719例超额死亡，占暂定统计总数约三分之二。
从地区来看，巴黎所在的法兰西岛大区受影响最严重，超额死亡接近2000人，增幅达81.2%。医院、住所和养老机构等各类场所的死亡人数均有所增加。
法国公共卫生署指出，超额死亡反映极端高温的强度，以及长时间暴露在高温环境下对公众健康造成的显著影响。
现有数据来自约5000个市镇的民事登记资料，覆盖全国约84%的死亡情况，不包含医学死因信息，仅用于估算全国层面的结果，法国将在夏季监测期结束后发布完整评估。
报告说，这次热浪持续时间长、发生时间早，强度甚至超过2003年8月热浪。
6月24日和25日成为法国有记录以来最热的两天，24小时全国平均气温首次达到30摄氏度。波尔多、巴黎等多地最高气温超过40摄氏度。

中国指菲律宾公务船侵闯黄岩岛海域 海警依法驱离
中国海警称菲律宾两艘公务船侵闯南中国海有主权争议的黄岩岛（菲称斯卡伯勒浅滩）海域，已通过跟监外逼等措施予以驱离。
中国海警在微信公众号发布，海警局新闻发言人姜略称，菲律宾3012、3018两艘公务船星期四（7月23日）不顾中方一再劝阻和警告，执意侵闯中国黄岩岛管辖海域。
发言人称，中国海警依法对菲船采取跟监外逼、拦阻管制等必要措施予以驱离，现场操作专业规范、正当合法。
姜略敦促菲方立即停止侵权挑衅行径，并称中国海警将依法持续在黄岩岛海域开展维权执法活动。

美FCC禁止销售含高风险中企关键零部件设备
美国联邦通信委员会（FCC）星期三（7月22日）表决通过新规，禁止在美国销售含有被认定危害国家安全的中国企业关键硬件设备。
据路透社报道，FCC列有一份对国安构成威胁的企业清单，华为和中兴通讯等中国电信设备商均在其中，相关企业的设备已被禁止在美国销售。
自2022年以来，华为等被列入清单的企业已无法为设备申请新的销售许可，但含有华为零部件的设备仍可获准进入美国市场。
新规则禁止含有任何华为制造、具备逻辑处理功能的硬件零部件的设备进入美国市场。FCC主席卡尔（Brendan Carr）说，机构将“彻底堵上零部件漏洞”。
曾在美国前总统拜登任内担任白宫国家安全委员会官员的麦圭尔（Chris McGuire）说，受到入侵或存在安全隐患的零部件，特别是半导体或通信设备，可被用来破坏整台设备。
目前，美国总统特朗普政府已全面收紧对中国科技设备的限制。FCC上月宣布，自7月16日起禁止进口更多由一批中国制造商生产的设备，并在星期二（21日）提议禁止进口大多数军用级无人机。
此外，FCC近期也禁止进口新款外国无人机和路由器，并提议禁止美国电信运营商与被视为构成国安风险的中国电信企业互联互通。
FCC还考虑禁止在美国拥有数据中心或网络接入点的中国电信企业与其他公司互联。这些接入点通常位于互联网交换节点。如果措施落实，相关中国电信企业实际上将被迫停止运营在美国的数据中心。
