工业部表示，鉴于中东冲突导致油价及物流成本上扬，已提出四项紧急措施——改善生产流程、降低生产成本、支持资金来源与税费等优惠，并同步制定中长期产业重组计划，以提高能源使用效率、减少对进口原料依赖，增强制造业韧性。
工业部长塔纳空·旺本空查纳（ธนกร วังบุญคงชนะ）称，2月28日霍尔木兹海峡一度关闭，若冲突持续，国际油价可能升至每桶150美元（3月6日油价为79美元/桶），将推高燃料与海运成本，并使保险及运费上涨约50–140%。
工业部评估识别出高耗能产业面临较大风险，包括水泥、玻璃、瓷砖、石油天然气、纺织与造纸等，石化业在原料面亦受冲击；部分产业如电子零件、珠宝、硬盘、家电与整车短期影响较小，但需持续监控。
冲突亦带来部分利多：食品饮料、電動車/混合動力車、国防业、天然橡胶及纸质包装等需求可能上升。
业者向政府反映两大需求：一是风控与原料供应多元化，二是业务转型与降本增效。为此，工业部将推四项紧急支持：推动生产设备更新与原料备援；协助厂区取得清洁能源（屋顶太阳能、生质能、以糖蜜/棕榈油生产可持续航空燃料SAF）；提供中小企业优惠信贷与绿色生产贷款；对符合节能减排与可持续发展者给予税惠。
中长期则着手产业结构重整，联动公私部门拟定目标与行动方案，推动在地原料与Made in Thailand政策，降低能源与原料进口依赖，提升产业竞争力。塔纳空强调，将同步短中长期措施，帮助业者因应全球经济动荡，保持竞争力。

交通部已成立情监中心（War Room），密切追踪中东事态并监管各类运输营运者，呼吁业者在国内柴油调价尚被政府管制期间协助维持票价稳定。
各单位被要求立即落实以下管控：
- 陆运：加强对公车、公共运输与各类出租营运的查核，确保按法定票价收取；
- 水运：督促客货轮船业者不得超收，并紧盯国际海运运价；
- 航空：监管航司票价与货运费，必要时研议暂时性票价冻结措施；
- 铁路：评估能源成本风险，尤其柴油机车，提前筹划风险管理对策。
交通部并要求各单位汇整运输业能源使用资料，连续通报运输警示与应变中心，以备充足燃料供应并研究对业者的后续协助方案。
披帕特强调，政府优先照顾民众利益，交通部将严防违规涨价，确保公共运输持续、公平地服务人民。

泰国总理签署命令，自今日起暂时停止出口燃料以防止供应短缺
泰国官报于2569年3月6日（公元2026年3月6日）公布总理第2/2569号命令。总理安努廷（Anutin）以应对中东冲突升级、海上航道受限可能影响国内燃油供给为由，命令自公布日起生效，暂时禁止以下燃料出口，直至另行通知：成品油（汽油、酒精汽油/基础汽油、轻柴油、航空用Jet A-1）及石油液化气（LPG）。
豁免情形包括：对老挝人民民主共和国和缅甸联邦的出口；存放于保税仓或保税区、用于转出口的燃料；以及质量不符合国内标准、无法在国内销售的燃料。
另规定自2569年3月31日起，国内燃料经营者须备储国产成品油1.5%，自2569年4月30日起提高至3%。关于储备量计算、储存场所审批及相关条件，按现行燃油贸易法定程序执行。若经营者能以书面证明确有困难或无法按规定备储且将遭受不当损害，能源商务署署长经能源部长同意可酌情给予临时豁免或减量，并可附加条件。

台湾国防部已经连续七天没有侦获任何解放军军机在台海周边活动，罕见的“空中静默”现象引发关注。台湾军事专家称，这显示中国大陆内部整肃未止，企图营造和平假象。

台湾国防部强调，面对台海动态，台军将持续严密监控，确保台湾安全与区域稳定。

中国半导体业领军人物联名吁：举国之力打造中国版阿斯麦
（北京综合讯）中国全国人大与政协“两会”召开之际，中国半导体行业的九名领军人物集体撰文，呼吁在2026至2030年的“十五五”规划期间，举全国之力打造中国版的光刻机巨头阿斯麦，突破美国的技术封锁。

文章呼吁由国家层面建立整合机制，让“被集成者”跳出“名利”的藩篱，统一调度资金和人力资源，锻造出真正可以与强手对垒的头部企业。
文章最后强调，未来的五年是中国晶片产业卧薪尝胆的五年，敦促行业丢掉幻想、准备斗争，尊重科学规律，潜心基础研究“磨一剑”。
