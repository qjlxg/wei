涉版权纠纷 字节跳动据报暂缓新AI全球上线
知情人士透露，涉版权纠纷问题，中国科技巨头字节跳动已暂停其新一代视频生成模型Seedance 2.0的全球发布计划。
美国科技媒体The Information星期六（3月14日）引述两名知情人士报道，Seedance 2.0因卷入与好莱坞主要工作室和流媒体平台之间的版权纠纷，而被字节跳动暂缓全球上线。
路透社暂时无法独立核实该报道，字节跳动也未立即回应置评请求。字节跳动上月曾表示，包括迪士尼在内的美国影视公司威胁采取法律行动后，公司将采取措施，防止Seedance 2.0未经授权使用知识产权。
2月13日，迪士尼向字节跳动发函，指控其在训练和开发Seedance 2.0模型时未经许可使用迪士尼作品，并要求停止侵权。那之前，由Seedance 2.0人工智能（AI）生成的一段视频在中国互联网走红，其中包括好莱坞影星汤姆克鲁斯（Tom Cruise）和布莱德毕特（Brad Pitt）打斗的画面。
报道称，字节跳动原计划在3月中旬向全球客户发布Seedance 2.0，但目前已暂停相关计划。
报道也称，字节跳动的法律团队正着手识别并解决潜在法律问题，工程师也在为模型增加防护措施，以防模型生成可能引发进一步知识产权侵权的内容。
字节跳动2月起在旗下的即梦、豆包、小云雀等产品中，小范围内测Seedance 2.0。官网信息显示，Seedance 2.0可根据文本或图像创建电影级视频。用户只需编写详细的提示或上传一张图片，系统即可在60秒内生成带有原生音频的多镜头序列视频。

特朗普：伊朗愿意谈判但我不想 因为条款不够好
美国总统特朗普说，伊朗方面愿意达成协议，结束战争，但他尚未准备好这么做，因为“条件还不理想”。 
特朗普星期六（3月14日）接受美国全国广播公司（NBC News）近30分钟电话访谈时表明，现阶段不愿与伊朗达成结束战争的协议。
“伊朗想达成协议，但我不想，因为条件还不够好”。
特朗普补充说，任何协议条款都必须“非常牢固”。他拒绝透露协议可能包含什么条件，但表明伊朗承诺彻底放弃核野心会是其中一项。
这次电访话题广泛，特朗普还预告美国在伊朗军事行动的后续可能走向。他说：“他们（伊朗）仅有的能力就是投放水雷或发射短程导弹，而且这种能力可以被相对迅速地切断。但等我们处理完海岸线，他们连这种能力也不会再有了。”
他还表明：“我们已摧毁伊朗大部分导弹、大部分无人机。我们已基本摧毁了他们的导弹和无人机生产能力。两天之内，这些能力会被彻底摧毁。”
自美伊冲突2月28日爆发以来，已有13名现役美军人员丧生，其中包括上周在伊拉克执行空中加油任务时坠机身亡的六名机组人员。
过去两周，特朗普政府官员就美军在伊朗的目标和冲突持续时间发出矛盾信号。特朗普有时声称可能需要一个月或更长时间，有时又说“我们远远超前于计划”，且“几乎已无目标可打”。
特朗普在采访中说，他对伊朗决定用袭击其他中东国家来回应美以军事行动感到惊讶，并称周六美国对伊朗哈尔克岛的空袭“彻底摧毁”岛上大部分区域，但“我们可能会再打几次，纯粹为了好玩”。
