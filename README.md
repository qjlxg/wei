2016年3月3日，商务部长素帕吉·苏塔潘表示，受中东局势影响，国际能源与航运市场出现波动，已对泰国出口产生潜在冲击。她通报，3月2日13:30在政府大楼召开评估会，由总理阿努廷·占威鲁主持。会议认为，尽管泰国与冲突方的直接贸易比重有限，但地区性扩散对全球经济、能源与海运稳定具有重要影响，需密切关注。 
素帕吉指出，泰国与中东直接贸易并不高——2015年对该地区出口额约1,247.56亿美元，占出口总额约3.67%，尚未出现大规模取消或推迟订单的明显迹象。但海运路线受限和安全风险已导致改航、航程延长，致使运费、保险费和物流成本上升，同时部分航线出现集装箱与航班紧张，影响出口商。国际能源价格波动亦可能传导至生产与运输成本，形成间接冲击。 
为应对，商务部提出六项主动措施：
(1) 强化商品价格监管，防止囤积与不当涨价，监测能源成本向品价的传导；
(2) 协调进口商检视库存，分散采购来源并鼓励使用国内原料；
(3) 支持出口商并优化物流管理，评估运费与保费上升影响，提供成本管理与交付条款咨询；
(4) 与船公司及物流服务商紧密联动，跟踪航线、港口拥堵与供应链连续性；
(5) 发挥驻外商务参赞作用，及时报告贸易状况与进口商信心并为企业提供风险管理建议；
(6) 分析能源与运费上涨对通胀及价格稳定的影响，提出针对性政策建议。 
她表示，政府将整合相关单位与民间力量，落实价格、成本与出口支持措施，维持泰国经贸与价格稳定，应对外部地缘政治不确定性。

另有多名政协委员同时被撤，包括涉国防工业并正遭反腐调查的Zhang Kejian。此次清洗延续了近期对军队和高官的整肃：上周共有19名高层被撤职（其中9人为军方），今年1月亦有高级将领Zhang Youxia被免，理由为“严重违纪违法”。
自2012年上台以来，习近平持续推进反腐运动，尽管外界质疑其可能被用作打击政治对手，但近期动作显示军队仍是其内部权力整顿的主要目标。 来源：BBC

2016年3月3日，商务部长素帕吉·苏塔潘表示，受中东局势影响，国际能源与航运市场出现波动，已对泰国出口产生潜在冲击。她通报，3月2日13:30在政府大楼召开评估会，由总理阿努廷·占威鲁主持。会议认为，尽管泰国与冲突方的直接贸易比重有限，但地区性扩散对全球经济、能源与海运稳定具有重要影响，需密切关注。 
素帕吉指出，泰国与中东直接贸易并不高——2015年对该地区出口额约1,247.56亿美元，占出口总额约3.67%，尚未出现大规模取消或推迟订单的明显迹象。但海运路线受限和安全风险已导致改航、航程延长，致使运费、保险费和物流成本上升，同时部分航线出现集装箱与航班紧张，影响出口商。国际能源价格波动亦可能传导至生产与运输成本，形成间接冲击。 
为应对，商务部提出六项主动措施：
(1) 强化商品价格监管，防止囤积与不当涨价，监测能源成本向品价的传导；
(2) 协调进口商检视库存，分散采购来源并鼓励使用国内原料；
(3) 支持出口商并优化物流管理，评估运费与保费上升影响，提供成本管理与交付条款咨询；
(4) 与船公司及物流服务商紧密联动，跟踪航线、港口拥堵与供应链连续性；
(5) 发挥驻外商务参赞作用，及时报告贸易状况与进口商信心并为企业提供风险管理建议；
(6) 分析能源与运费上涨对通胀及价格稳定的影响，提出针对性政策建议。 
她表示，政府将整合相关单位与民间力量，落实价格、成本与出口支持措施，维持泰国经贸与价格稳定，应对外部地缘政治不确定性。

能源部发言人呼吁民众勿恐慌囤油，称国内燃油供应充足，准备动用燃油基金机制稳住价格。 
能源部副常务秘书暨发言人威拉潘表示，能源部下属商业能源局已紧急召集油品业者开会，要求加速备货并改从非中东地区如美洲、西亚非洲、马来西亚等地持续采购，以应对当日伊朗宣布封闭霍尔木兹海峡、影响进口的局面。 
截至3月1日，国内燃油及法定储备为4,877百万升，海上在运原油为2,783百万升，合计7,660百万升（约76.6亿升），在无法立即补充的极端情形下可供约60天使用；但实际仍可继续从其他来源进口，惟价格可能随国际市场上升。 
对于对老挝等邻国的油品出口，能源部说明属于先前签订的合约，并指出泰国与老挝在能源上互有往来（泰国亦自老挝进口电力）。部门将严管不得再向第三国转出口，如情势持续恶化，商业能源局可视情势宣布停止出口以保障国家能源安全，同时兼顾对邻国的影响。 
威拉潘重申，切勿大量囤油——既可能违法亦有储存安全风险；能源部将密切监测情势，必要时采取一切以保障国民利益为优先的措施。
