研究团队提出了名为Pointer-CAD的开源框架，基于阿里巴巴的Qwen 2.5模型，能够帮助设计师在三维对象上精确选择边缘或面，从而提高计算机辅助设计（CAD）在工程、制造和建筑等领域的效率与准确性。

中国驻法国大使馆星期六（3月7日）在官网发布通稿称，有记者提问，法国外交部副发言人星期三（3月4日）就中国对日本的出口管制措施说，法国反对任何形式的经济胁迫以及将出口管制武器化的做法，并对中国的决定表示遗憾。中国使馆对此有何评论？

田轩：中国居民消费率未来五年升至45%较为合理
清华大学国家金融研究院院长田轩认为，未来五年中国居民消费率提升至45%，是比较合理且“够得着”的目标。
中国官方星期四（3月5日）发布的政府工作报告提到，“促进居民消费率明显提高”被列为“十五五”（2025年至2030年）规划重大战略任务之一。
田轩星期五（6日）接受媒体访问时说，目前中国居民消费率（居民消费占GDP比重）不到40%，实际上远低于国际上的居民消费率，如美国的68%、日本的55%。
他说，若“十五五”时期中国居民消费率能提升到45%，“那么对于提振消费的作用非常大”。
田轩也说，长期以来，中国老百姓爱把钱存起来。这也和官方目前重视民生福祉、社会保障的提升有很大的关系。
他说：“如果我们不能把养老、医疗、教育这些问题解决得很好，老百姓的储蓄倾向还是比较高，还是会把钱存起来，进行预防性储蓄。”
