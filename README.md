美国顶级基金增持腾讯阿里 称中国AI企业估值优于美国同行
美国新兴市场基金正增加对中国主要人工智能“超级云计算商”的投资，认为他们的估值比正大举扩张的美国科技巨头更具吸引力。
据彭博社，总部位于纽约的资产管理公司Pzena的首席执行官Caroline Cai表示，她管理的39亿美元（49.8亿新元）基金近来增持了腾讯控股和阿里巴巴集团等数字平台企业。
随着人工智能逐渐成为科技主导权竞争的新战场，投资者正权衡中美两国在这一领域的发展前景。在美国，几家科技巨头正投入数千亿美元建设数据中心和算力基础设施，竞相开发最先进的模型。相比之下，中国互联网公司投入规模较小，更侧重于将人工智能应用嵌入现有平台。

彭博汇编的数据显示，Pzena的新兴市场价值型基金过去五年表现优于97%的同类基金，今年以来的表现也超过90%的同行。该基金的主要持股包括三星电子、台积电和阿里巴巴。

与中国监管沟通后 苹果公司调降“苹果税”
与中国监管部门沟通后，苹果公司宣布降低有“苹果税”之称的中国应用商店（App Store）的佣金率。
据央视新闻报道，苹果公司宣布，根据与中国监管部门的沟通，将自这个星期天（3月15日）起，调整适用于中国大陆应用App Store的iOS及iPadOS佣金率。
其中，苹果应用商店内购买及付费App的标准佣金率将由目前的30%改为25%。
这一佣金率适用于年收入超100万美元或应用内购买的开发者，是苹果手机在应用商店通过抽成的方式，获得游戏用户充值或购买虚拟数字服务等的利润。
而针对小型企业计划和迷你应用合作伙伴，相关佣金率以及第一年后的自动续费订阅佣金率将由目前的15%改为12%。
苹果公司表示，致力于将iOS以及iPadOS打造成为中国开发者最佳的App生态系统和巨大的商业机会。“我们承诺对所有开发者保持公平透明的条款，并始终为在中国分发App的开发者们提供不高于其他市场整体费率水平的具有竞争力的App Store费率“。
苹果应用商店在中国收取的佣金率偏高，一直存在争议。
据《证券日报》报道，去年10月20日，55名中国消费者联名向监管部门递交《关于苹果公司滥用市场支配地位的举报信》，指苹果在华仍维持30%全球最高佣金率、封锁第三方支付与第三方应用商店，并首次在行政投诉层面提出“零佣金、全开放”的诉求。
报道引述业内人士称，“苹果税”的本质是数字时代的“过路费”，其合理性已在全球范围内受到质疑。数据显示，2024年中国消费者缴纳的“苹果税”总额达64.4亿美元（82.3亿新元），占苹果中国区营收的10%，而同期美国和欧洲的占比分别为8.8%和4.6%。
中国央视新闻去年12月也报道称，一边是全球多国推动苹果降税、开放，一边是中国市场仍维持30%佣金率的最高标准，调整中国“苹果税”的呼声日益高涨。

据《纽约时报》统计，自美国和以色列对伊朗发动军事行动以来，至少有16艘油轮、货轮及其他商船在波斯湾遭袭击。
