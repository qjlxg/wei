就未来五年健康、公共卫生领域的工作，中国国家卫健委主任雷海潮星期六（3月7日）在全国人大民生主题记者会上说，中国的基本医疗卫生服务覆盖了14亿城乡居民，但服务能力还需进一步提升，一些专业发展还不够平衡，比如儿科、精神卫生、全科、康复护理等服务，这些需要在“十五五”期间进一步发展。
雷海潮还说，“十五五”期间将进一步推进深化医药卫生体制改革，并实施医疗卫生强基工程，谋划1000个左右的紧密型县域医共体建设。
在强化公共卫生安全能力领域，他提到，要维护国家整体安全，必须要做好传染病防控工作，要提升传染病的监测预警、风险评估、检验检测、医疗救治、应急处置等各方面的能力。加强公共卫生体系能力建设将是着力的重点方向。同时加强急救急诊体系建设也是要致力提升的方面，并将加强血液供应和安全保障。
雷海潮说，健康是全人类共同追求，中国将加强与国际社会交流合作，把中国传统中医药文化更多地推向国际社会，同时促进传统医药之间的互学互鉴。提出“新时代神农尝百草”行动，目的就是能够在全球找到更丰富的药用植物资源，丰富中医药的治疗手段，应对全球共同的公共卫生挑战。
他最后强调，希望各方面共同努力，沿着“十五五”规划擘画的蓝图和目标，进一步提高居民的健康水平，努力到2030年让居民人均预期寿命达到80岁。

美议员促政府收紧对华为美国子公司限制
美国国会议员敦促特朗普政府堵住监管漏洞，限制被列入黑名单的中国科技巨头华为在美子公司继续自由运作。
美国众议院美中战略竞争特别委员会主席、共和党众议员穆勒纳尔（John Moolenaar）与首席成员、民主党众议员康纳（Ro Khanna）星期四（3月5日）致信美国政府，要求对华为在美子公司Futurewei Technologies实施与母公司华为相同的限制措施。
此前，美国官员认定华为构成重大网络安全风险，并指控公司大规模窃取知识产权。华为因此面临出口管制、金融与证券相关限制，并被列为与军方有关的企业。华为一直以来否认相关指控。
委员会称，要求限制华为在美子公司是源于一项调查。调查中公司提交的文件显示，华为对Futurewei的财务、运营和人员拥有最终所有权和控制权，使华为在美国政府对它采取行动后仍能保持在美国的业务存在。
信中称，Futurewei向委员会表示，公司由华为全资拥有，不通过向公众销售产品或服务获利，而是向母公司提供研发服务，包括协助华为申请美国专利。
根据Futurewei提交的一份人员预算计划，截至2023年，该公司共有172名员工。
委员会在去年9月向Futurewei发出信息请求时，还对多项问题表达关切，包括Futurewei在晶片巨头英伟达硅谷园区租用办公空间长达10年的情况，这一租约已于2024年结束。
委员会当时指出，鉴于有关华为曾利用Futurewei从事企业间谍活动的指控，这种“共址办公”的安排“令人深感担忧”。英伟达发言人回应称，公司会确保“办公室、员工和知识产权安全”。

联黎部队营地遭导弹袭击 两名加纳维和人员重伤
联合国维和部队和加纳军方发表声明说，黎巴嫩南部的一个联合国维和营地星期五（3月6日）遭到袭击，三名联合国维和人员受伤。黎巴嫩总统指责以色列把他们视为袭击目标。
综合法新社和新华社报道，此次袭击发生时，以色列和伊朗在黎巴嫩的代理人真主党正在交火。
联合国驻黎巴嫩临时部队（联黎部队）发表声明说：“在黎巴嫩南部考扎的营地内，三名维和人员在激烈的交火中受伤。伤势最重的人员已被送往贝鲁特的医院接受治疗。”
加纳武装部队说，联黎部队加纳营总部遭两次导弹袭击，对营地设施造成严重破坏，其中军官餐厅被直接击中，引发火灾后完全被烧毁。
加纳政府已向联合国提出正式抗议，谴责此次袭击并呼吁追究相关责任，同时敦促冲突各方履行国际义务，确保联合国维和人员的安全。
