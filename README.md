对于官方将今年国内生产总值（GDP）增速目标定在4.5%至5%，中国国务院研究室主任沈丹阳说，区间目标既为应对各种不确定因素留有空间，也有利于各地因地制宜制定本地区增长目标；另外，今年经济运行中不确定难预料因素可能比预料多，提区间目标也是为应对不确定性留有余地。
中国全国人大会议星期四（3月5日）上午在北京召开，中国总理李强做政府工作报告时说，今年发展主要预期目标是经济增长4.5%至5%，在实际工作中努力争取更好结果。
中国国新办随后举行吹风会，也是《政府工作报告》起草组负责人的沈丹阳回答提问时说，今年的经济增长目标实际上是两句话，即“经济增长4.5%-5%、在实际工作中努力争取更好结果”。这个“两句话的目标”综合考虑了国内经济运行和外部环境变化，兼顾了需要与可能，“是一个‘既跳起来摸高、又稳得住步伐’的积极务实的目标”。
他进一步阐释，一方面，这个目标既充分考虑今年要为调结构、防风险、促改革留出空间，又与2035年愿景目标总体衔接，体现了“十五五”时期夯实基础、全面发力的要求。另一方面，这个目标与中国增长潜力匹配，“国内外有很多经济组织、经济专家、智库都在测算，总体上跟我们的预期和目标是吻合的，在世界主要经济体中增速也差不多是最高的”。
沈丹阳强调，这个目标是个区间目标，既为应对各种不确定因素留有空间，也有利于各地因地制宜制定本地区增长目标，有利于引导各方面把精力聚焦到高质量发展上来。
他也提到，今年中国经济发展的外部环境更趋复杂多变，经济运行中不确定难预料因素很可能比预料的多，提出区间目标也是为了应对这些不确定性留有余地。
沈丹阳最后强调，“两句话的目标”包含“在实际工作中努力争取更好结果”，展现了积极进取的目标导向和政策取向，也有两方面考虑，一是中国经济具有强大韧性和活力，结构持续向优向好，新质生产力加快成长。二是今年继续实施更加积极有为的宏观政策，同时还将深入实施一系列重点领域改革，势必进一步释放经济增长潜力。
“在这样的情况下，我们认为，只要把各种有利条件用好用足，完全可以争取更好结果，实现质的有效提升和量的合理增长”。

中国未来五年要抢占科技发展及人工智能产业应用制高点

具体实施方面，政府将加强原始创新和关键核心技术攻关，全面提升基础研究水平；提高体系化创新能力，提高创新资源配置效率，构建自主完备、开放高效的国家创新体系；以及强化企业科技创新主体地位，落实企业在技术创新决策、研发投入、科研组织和成果转化应用中的主体地位，推动创新资源向企业集聚。完善促进企业创新的政策体系。
与此同时，政府将一体推进教育科技人才发展。深化教育科技人才一体改革，建立健全一体推进的协调机制，协同推进创新型人才培养。联动推进激励评价机制创新。
此外，中国政府也将加快数智技术创新，深化拓展“人工智能+”，赋能经济社会发展和治理能力提升，促进生产方式深层次变革和生产力革命性跃迁。
具体来说，这包括强化算力算法数据高效供给，统筹推进算力设施建设、模型算法发展和高质量数据资源供给；全面实施“人工智能+”行动，加强人工智能同科技创新、产业发展、文化建设、民生保障、社会治理相结合，抢占人工智能产业应用制高点，全方位赋能千行百业等。
摘要指出，中国政府也将加强数据基础制度规则建设和人工智能治理，营造有益、安全、公平的发展环境，并拓展数智领域国际合作。

大陆军机在台周边活动大幅减少 习特会或是原因之一
近几周来，中国大陆军机在台湾周边的活动大幅减少，过去一周更是完全没有飞行。这与此前几乎每天都有的军事行动相比出现骤降，可能表明北京正在重新调整对台施压。
路透社报道，根据研究机构安全台湾学会（STA）汇编的台湾政府数据，今年以来大陆已向台湾防空识别区派遣460架军机，包括战斗机和无人机，较去年同期减少46.5%。
今年2月，台湾探测到190架大陆飞机，这是自台湾国防部2022年开始公布相关每日数据以来的最低月度数字。台北方面认为，这是北京持续对台施压行动的一部分，而这一行动也不断磨练其攻台能力，包括通过军事演习。
路透社引述两名不具名的台湾官员称，大陆国家主席习近平可能正在降低公开施压力度，以在月底与美国总统特朗普会晤前营造更好的氛围。
一名台湾高级安全官员向路透社说，面对特朗普，“北京可能试图营造一种假象：我爱好和平，我正走向和平，所以你们应该停止向台湾出售武器。”

特朗普2月4日与习近平通话后表明，很期待在4月访华。路透社此前引述白宫官员称，特朗普将于3月31日至4月2日访问中国，与习近平会晤。中国外交部没有确认消息，按照惯例，中国一般在访问前几日公布行程安排。

早些时候，马国首相安华出席与参观了农粮部主办的斋戒月复兴计划，并与部门高层举行会议，就农粮部发展方向及主要议程进行交流。
