新上市公司只要市值排在当前Nasdaq-100前40名，仅需15个交易日就能直接进指数。标普500指数也类似，都完全跳过正常“seasoning期”（通常要等1年）。

据法新社报道，这是首起依据该罪行定罪的案例。男被告切尔戈（Alexander Csergo，59岁）是一名营销管理人员，从中国上海返回澳大利亚不久后，在2023年于悉尼被澳大利亚警方逮捕。
切尔戈被控接受中国情报人员的现金，撰写有关澳英美三国安全伙伴关系（AUKUS）核潜艇合作项目、关键矿产等主题的报告。
澳洲悉尼法庭获悉，被告通过职业社交平台领英（LinkedIn）接触到两名中国特工“肯”（Ken）和“伊芙琳”（Evelyn），并在上海与他们会面。
切尔戈在上海空荡荡的咖啡馆与两人见面，他怀疑这些场所已被安排用于三人会面。
被告2023年带着一份“购物清单”返回澳大利亚，清单列出“肯”希望能收集到的主题信息。清单是在切尔戈抵达澳大利亚，并与家人在邦迪海滩同住三周后，被警方发现。
法庭也获悉，切尔戈在2021年至2023年间，与两人交换数千条微信信息，并接受装在信封里的现金。
切尔戈是首名面对鲁莽外来干涉罪控状的人士，该罪最高可判处15年监禁。法院人员告诉法新社，陪审团星期五裁定切尔戈鲁莽外来干涉罪名成立。

英国国防部长暗示普京在幕后为伊朗“助力”
英国国防部长希利暗示，俄罗斯正在幕后操控以无人机反击美国和以色列猛烈空袭的伊朗。
法新社报道，希利星期四（3月12日）视察位于诺斯伍德（Northwood）的英国军事总部时告诉记者，俄罗斯总统普京可能是伊朗在中东采取某些战术的幕后“军师”。
英国位于塞浦路斯的阿克罗蒂里（Akrotiri）空军基地3月1日遭伊朗制无人机袭击。希利说，调查人员正在分析那架无人机，“调查是否存在俄罗斯或其他外国零部件的证据”。
希利指俄罗斯是伊朗的亲密盟友，两国去年同意互相帮助以应对“共同威胁”。
他说：“一有结果，我们会即时向大家汇报并予以公布。”
“但我认为，普京在幕后为伊朗制定战术，甚至为他们提供某些作战力量，并不令人意外。因为普京是目前受益于油价飙升的世界领导人之一。”
美国总统特朗普上周说，没有任何迹象表明俄罗斯在这场战争中为伊朗助力，但即便有，他认为也“没有起到多大作用”。
英军联合行动司令佩里（Nick Perry）告诉希利，“确凿”迹象显示俄罗斯和伊朗之间存在关联，包括伊朗借鉴俄罗斯使用无人机的经验。
3月1日，阿克罗蒂里空军基地的一个机库遭无人机袭击，所幸没有人员伤亡。当天飞向该基地的另外两架无人机被英国战机击落。

中越外交国防公安“3+3”首次部长级会议将在越南举行
中越外交、国防和公安“3+3”战略对话机制首次部长级会议将在越南举行。
中国外交部发言人星期五（3月13日）在例行记者会上宣布，应越南邀请，中共中央政治局委员、外交部长王毅与中共中央书记处书记、公安部长王小洪，国防部长董军将于3月15日至17日赴越南出席中越外交、国防、公安“3＋3”战略对话机制首次部长级会议。
在越期间，王毅还将同越南领导人共同主持中越双边合作指导委员会第十七次会议。
中共总书记、中国国家主席习近平去年4月在北京与到访的越共总书记苏林举行会谈时，就深化中越命运共同体建设提出六点举措，其中包括将两国外交、国防、公安“3＋3”战略对话机制确定为部长级，增进战略协作。
中越双边合作指导委员会第十六次会议2024年12月在北京举行，王毅与越南副总理兼外长裴青山共同主持。
王毅指出，双方要筑牢安全根基，发挥好中越外交、国防、公安“3+3”战略对话作用，坚定团结协作，捍卫共同利益。要促进互利共赢，扩大双向贸易和投资，构建稳定畅通的跨境产业链供应链，提升基础设施互联互通和贸易投资便利化水平，加强人工智能、电子商务、数字经济、绿色发展等新兴领域合作。
会议听取了中越外交、国防、公安“3+3”战略对话机制首次会议情况的汇报。在双边合作指导委员会会议前，中越双方召开外交、国防、公安“3+3”战略对话机制首次会议，就中越全面战略合作和共同关心的国际地区问题坦诚、深入交换意见，达成广泛共识。
双方认为，中越“3+3”战略对话机制的建立，充分体现了中越关系的高水平和战略性，为两国加强战略沟通提供了平台，为周边国家增进政治互信树立了典范。同意坚持两党两国高层战略引领，深化外交、国防、公安等战略领域机制化合作，持续提升政治安全合作和多边协作水平，为地区乃至世界和平、稳定、合作与发展作出贡献。
