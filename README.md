台海周边大陆军机数量骤降引发质疑
台湾官方过去10天里有九天未在台湾海峡周边侦获任何中国大陆军机，令专家对解放军活动数量骤降感到不解。
据法新社报道，北京近年持续对台湾施加军事压力，几乎每天都出动军机和军舰绕行台湾海峡。
不过，法新社根据台湾国防部每日发布的数据统计显示，自2月28日以来，当局24小时内在台海周边仅侦获两架大陆军机，比去年同期的86架显著减少。这是法新社2024年开始统计相关数据以来，大陆军机零侦获的最长时段。
统计也显示，当局过去10天在台海日均侦获六艘大陆军舰，与去年同期持平。
根据统计，今年1至2月在台海周边活动的大陆军机数量较去年同期下降42%，大陆军舰数量则较一年前减少4.5%。
专家预测，大陆显著减少派遣军机绕台，可能与中国全国两会正在北京举行，以及解放军高层近期被免职有关。其他可能原因，则包括美国总统特朗普即将在本月底访华，以及美伊冲突等。
新加坡南洋理工大学拉惹勒南国际研究院高级研究员唐安竹（Drew Thompson）上星期五（3月6日）在线上原创媒体平台Substack写道，“我没想过为解放军暂停在台海周边活动感到担忧，但缺乏合理的解释令人不安”。
总部位于美国华盛顿的国际战略研究中心的中国实力项目研究员贺博然（Brian Hart）则对近期在台海周边活动的大陆军机数量骤降不感意外。
他上星期六（7日）在X平台写道，在每年两会期间，大陆军机飞入台湾防空识别区的数量几乎降至零或接近于零。如果这种情况持续至两会结束后，那就显得不寻常。“我不觉得现在有任何显示不寻常的证据”。
台湾媒体上星期四（5日）引述台湾军事专家苏紫云分析，这是中共内部整肃未止，且企图透过营造和平假象和氛围，影响台湾支持国防特别预算民意的“心机城府”。　　
一名台湾国安官员对法新社说，北京或许在尝试制造一个假象，即大陆降低对台湾的威胁，以蒙骗美国让他们减少对台湾安全的支持。“我们不能松懈防备”。

中国传媒大学砍掉翻译等16个本科专业
中国传媒大学去年砍掉了翻译、摄影等16个本科专业和方向。这家知名学府的中共党委书记廖祥忠说，未来将是人机分工时代，大学正在做“课堂革命”。
综合央视新闻、南方+、快科技、IT之家报道，廖祥忠在中国全国两会期间说，去年中国传媒大学砍掉了翻译、摄影等16个本科专业和方向。
廖祥忠进一步说，他把未来定位为人机分工时代，“我们应该怎么做？我现在正在做课堂革命，我跟大家讲，我说怎么讲好一堂课，一个形式要改变，内容改变，还要改变思路。”
他阐述，作为老师必须清楚，“这堂课这门课为什么要上？这门课在整个知识体当中，它承担什么作用，什么角色，这是第一要掌握”。
其次，要掌握这堂课的支点在哪里，难点在哪里，和未来对接点在哪里，和传媒产业、传媒业的发展的连接点在哪里。最后，要解决破解之道，剩下的交给人工智能（AI），让学生去学习。
廖祥忠也提到，今年Seedance 2.0出现以后，他感到非常震惊，“以后何去何从”。
今年2月，中国科技巨头字节跳动推出新的AI视频生成模型Seedance 2.0。这个模型支持文字、图片、音频、视频四种模态输入，用户只需输入提示词或上传参考图，即可快速生成多镜头视频，显著降低专业创作的门槛与成本。
据中国传媒大学官网公示，2025年度学校向教育部拟申请了增设游戏科学与技术、电子竞技、品牌学三个本科专业。
IT之家查询中国传媒大学官网公告发现，该校2025年拟撤销本科专业包括国际经济与贸易、社会学等七个专业。

中国大陆反对日本在台湾问题打擦边球
针对台湾行政院长卓荣泰上周末赴东京观赛，中国大陆外交部回应时，指他鬼祟跑到日本搞谋独挑衅小动作，北京反对日本在台湾问题打擦边球，并称日本将为此付出代价。
在中国大陆外交部星期一（3月9日）例行记者会上，有记者就台湾相关官员访日本一事提问。
外交部发言人郭嘉昆回应时说，“你提到的这个人心怀鬼胎，偷偷摸摸、鬼鬼祟祟跑到日本搞谋‘独’挑衅的小动作，这种见不得人的勾当和伎俩令人不齿”，北京高度警惕，坚决反对日本妄图在台湾问题上打“擦边球”“搞突破”，日本的纵容挑衅和恣意妄为必将付出代价，由此造成的一切后果必须由日本承担。
郭嘉昆说，大陆将坚定不移打击“台独”分裂势力，坚决反对外部势力干涉台湾问题，坚决捍卫国家主权安全和领土完整。
卓荣泰上星期六（7日）赴日本观看世界棒球经典赛中华台北队对垒捷克队赛事，成为台日1972年断交后首名赴日的时任行政院长。 
台行政院称，这是卓荣泰私人行程，但传出此行是由华航专派包机载送，引起执政党私人行程由纳税人买单的质疑。
扣除观赛时间，据报道卓荣泰还在东京多停留近五个小时，不排除他在这期间会晤了日本政要。
台湾海洋委员会主委管碧玲星期天（8日）在脸书发文称，卓荣泰在这趟东京行，必定与日本政要会面，并指这是“台湾外交关系上的重大突破”。
法新社引述日本外务省说，卓荣泰此行是自1972年以来台湾在任行政院长首次访日，时任行政院长游锡堃2004年因台风被迫在日本中途停留的情况除外。
卓荣泰此行正值日中关系急剧恶化之际。日本内阁秘书长木原稔向记者说，卓荣泰没有与日本政府官员接触。
