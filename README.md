按照宪法，下一任最高领袖由由民选但受审查的88席专家会议选出，该机构有权任命或免去领袖，并可在必要时组建临时领导委员会。若要避免动荡，预计会快速做出决定。 
尽管专家会议形式上有终审权，实权常掌握在已扎根的权力中心，尤其是影响力巨大的伊斯兰革命卫队（IRGC）。在不确定时刻，卫队的态度可能左右接替人选或促成过渡委员会。 
暂无明晰继承者，但被视为可能人选者包括：被指在幕后有影响力的长子穆杰塔巴·哈梅内伊；政府资历深厚但与安全体系联系有限的阿里雷扎·阿拉菲；具有革命和宗教血统但影响力受限的哈桑·霍梅尼；以及被视为务实保守派的阿里·拉里贾尼。 
哈梅内伊任内国内抗议不断，经济与世代冲突加剧，外部与美以对峙亦持续。过渡能否平稳，将取决于宗教精英、安全高层与强硬派之间能否迅速达成一致；若无法形成统一领导，意识形态权威与军权的博弈将面临自1989年以来最严峻考验，地区风险亦可能上升。

阿亚图拉·阿里·哈梅内伊，伊朗最高领袖，掌权逾三十六年，于2026年2月28日在美以联合空袭中遇害，享年86岁。伊朗官方媒体周日凌晨宣布其死讯；美国总统唐纳德·特朗普数小时前亦称哈梅内伊在袭击中被击毙。半官方法尔斯通讯援引未具名消息称，其一名女儿、女婿、孙辈及儿媳在袭击中遇难。伊朗当局宣布全国哀悼40天，并放假七天。 
哈梅内伊1939年4月19日生于圣城马什哈德，早年在什叶派重镇库姆从事宗教学习并追随霍梅尼，曾参与反巴列维活动，多次被逮捕与流亡。1979年参与伊斯兰革命并入掌要职，1981—1989年任总统，1989年被选为第二任最高领袖，随后通过修宪降低宗教资格限制确立统治。

（联合国综合电）联合国研究人员警告，世界正面临不可逆转的水资源“破产”，数十年来不可持续的用水量消耗了含水层、冰川、土壤、湿地和河流生态系统中蕴藏的水资源，加上污染加剧，全球水资源“已处于危机后的崩溃状态”。
根据联合国大学水、环境与健康研究所日前发布的报告，全球30亿人口和超过一半的粮食产量集中在水资源储量本就不稳定或持续下降的地区，而盐碱化也已导致超过1亿公顷的农田退化。
超过1亿7000万公顷或大于伊朗面积的灌溉耕地在水资源方面“高度紧张”或“极为紧张”；土地退化、地面水枯竭和气候变化每年造成的经济损失逾3000亿美元（约3850亿新元）。
报告警告，全球水资源“破产”时代已降临。这是联合国首次用“破产”一词来形容缺水的严重程度，以往的用词“水资源紧张”或“水危机”意味着水资源尚存恢复的可能性，但水资源“破产”重点在于其不可逆性，许多地下水层已被永久压缩、湿地消失、冰川退缩，这些破坏已严重至无法逆转。
研究人员指出，目前解决水资源问题的方法已不再适用，当务之急不是“恢复常态”，而是制定一项旨在最大限度减少损害的全新“全球水资源议程”。
报告第一作者、联合国大学水、环境与健康研究所所长迈达尼说，许多地区水资源已被透支，许多关键的供水系统也已“破产”，各国须承认水资源破产的事实，作出保护人民、经济和生态系统的艰难选择。
报告呼吁敦促各国政府放弃短期的应急思维、创建追踪水资源的全球监督系统、采取手段防止对水资源的更多不可逆破坏、减少并调整用水需求、改造水密集型行业、打击非法取水和污染水资源等。

解放军启用预备役人员证 分析：便于战时应召参战
中国解放军预备役人员提升身份认证，启用更加权威和规范的预备役人员证，有分析指出，新的身份凭证便于预备役人员战时应召参战，也便于他们享受福利待遇。
央视军事消息，经中共中央军委批准，全军星期天（3月1日）起，制发启用《中国人民解放军预备役人员证》。
据介绍，《预备役人员证》为本芯证件式样，具有较高的防伪性和通识度，是证明预备役人员身份的有效证件，证件号码为系统生成的唯一身份号码，发给《中华人民共和国预备役人员法》颁布施行后在中国人民解放军依法服预备役的预备役军官、预备役军士和预备役兵。《中国人民解放军预备役军官证》即行废止。
中国国防大学联合作战学院教授张耀奎接受央视新闻访问时指出，新证件的制发和启用，标志着预备役人员有了统一编入预备役部队服预备役的证明，是国家和军队对预备役人员身份的正式认定。
张耀奎解释说，新证件便于预备役人员战时应召参战，平时被召集参加军事训练、担负战备勤务、执行非战争军事行动任务，也便于预备役人员依法依规享受有关福利待遇、抚恤优待等。
解放军预备役人员拥有“亦兵亦民”的双重身份，他们平时活跃在社会各行各业，是普通劳动者。一旦有需要，他们就要穿上军装，履行军人的职责。
张耀奎也提醒，只有本轮改革后选拔补充的新型预备役人员才拥有该证件，改革前曾服预备役的人员不发放新证件。以往制发的《中国人民解放军预备役军官证》即行废止。

印度气象局：未来三个月高温 或对居民健康构成威胁
（新德里彭博电）印度未来三个月的气温预计将高于往年同期水平，这可能推高电力需求，并对居民的健康构成严重威胁。　　
印度气象局局长穆鲁蒂扬杰星期六（2月28日）在线上新闻发布会上说，从3月到5月31日的三个月内，全国许多地区出现更多热浪天气的可能性增加，而大部分地区的最高气温很可能高于往年同期水平。
这种情况可能推高印度电力需求，促使政府更多依赖燃煤发电来应对用电高峰。气温持续攀升还将威胁农村收入，加剧食品价格压力，使政策制定者控制通货膨胀的努力面临更大挑战。
不过，气象局指出，3月份许多地区的最高气温可能仍将处于正常或低于正常水平，除西北部一些地区外，大部分地区的最低气温预计将高于平均水平。
3月份气温的任何异常升高都可能对冬播作物构成风险，因为冬播作物正处于萌芽成长阶段，过高的温度会抑制产量并降低品质。
持续高温天气也可能使印度的小麦产量低于上个季度的纪录，从而削弱政府为确保充足供应所做的努力，特别是为政府的各项福利计划，包括每月向约8亿人免费分发一定数量的粮食。

伊朗最高领袖阿里·哈梅内伊去世，且未留下正式接班人，令伊斯兰共和国进入自1979年革命以来最关键的权力过渡期，外界担忧国内稳定与地区局势可能因此升级。 
按照宪法，下一任最高领袖由由民选但受审查的88席专家会议选出，该机构有权任命或免去领袖，并可在必要时组建临时领导委员会。若要避免动荡，预计会快速做出决定。 
尽管专家会议形式上有终审权，实权常掌握在已扎根的权力中心，尤其是影响力巨大的伊斯兰革命卫队（IRGC）。在不确定时刻，卫队的态度可能左右接替人选或促成过渡委员会。 
暂无明晰继承者，但被视为可能人选者包括：被指在幕后有影响力的长子穆杰塔巴·哈梅内伊；政府资历深厚但与安全体系联系有限的阿里雷扎·阿拉菲；具有革命和宗教血统但影响力受限的哈桑·霍梅尼；以及被视为务实保守派的阿里·拉里贾尼。 
哈梅内伊任内国内抗议不断，经济与世代冲突加剧，外部与美以对峙亦持续。过渡能否平稳，将取决于宗教精英、安全高层与强硬派之间能否迅速达成一致；若无法形成统一领导，意识形态权威与军权的博弈将面临自1989年以来最严峻考验，地区风险亦可能上升。

阿亚图拉·阿里·哈梅内伊，伊朗最高领袖，掌权逾三十六年，于2026年2月28日在美以联合空袭中遇害，享年86岁。伊朗官方媒体周日凌晨宣布其死讯；美国总统唐纳德·特朗普数小时前亦称哈梅内伊在袭击中被击毙。半官方法尔斯通讯援引未具名消息称，其一名女儿、女婿、孙辈及儿媳在袭击中遇难。伊朗当局宣布全国哀悼40天，并放假七天。 
哈梅内伊1939年4月19日生于圣城马什哈德，早年在什叶派重镇库姆从事宗教学习并追随霍梅尼，曾参与反巴列维活动，多次被逮捕与流亡。1979年参与伊斯兰革命并入掌要职，1981—1989年任总统，1989年被选为第二任最高领袖，随后通过修宪降低宗教资格限制确立统治。 
在位期间，他强化治权与伊斯兰革命卫队影响，镇压多次国内抗争（包括1999年学生运动、2009年“绿色运动”和2022年“妇女·生命·自由”抗议），并扶持地区代理武装、推进有争议的核计划，形成长期与美以对抗的格局。哈梅内伊之死被视为伊斯兰共和国一个时代的终结。

2016年3月1日，总理府副发言人拉莉达·佩里斯维塔娜表示，泰国陆运局将设立“一站式服务中心”（One Stop Service），为通过手机应用提供服务的租赁车（即网约车）办理合规注册，目标在2016年3月31日电子交易发展局（ETDA）相关公告生效前完成整顿与登记。
该中心提供从登记或审批、车辆年检、资料核验、变更登记类别到收取费用与税款的全流程服务。服务地点为曼谷陆运局第五辖区车辆年检大楼（4号楼，乍都乍区），开放时间为3月2日至31日，每日08:30–18:00（含公休日）。
陆运局已通知应用平台业者尽快带车办理登记，若为团体同时办理可事先预约以加快流程，并已与各省陆运局统一摩托车营运登记（รย.17）与汽车营运登记（รย.18）的办理标准。拉莉达强调，此次设中心是为便民并促使车主合法化，提升服务标准与乘客安全，政府并非取缔行业，而是规范管理，确保公平与安全。
