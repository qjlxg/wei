于泽远： 中国军队正补齐两大战略装备短板
中国新型核潜艇的研制近期引发外界关注。美国国会美中经济与安全审查委员会副主席薛瑞福（Randall Schriver）3月2日在该委员会举办的听证会上表示，中国正在推进一支日益壮大的潜艇舰队，这些潜艇、无人水下航行器、海底传感器和先进的海洋测绘项目的性能，也在不断提升。虽然美国在水下领域仍然拥有显著优势，但“如果不加以维护，这些优势将会逐渐消失”。
据彭博社报道，美国海军情报局局长布鲁克斯（Mike Brooks）少将在听证会上表示，中国目前大约拥有60艘常规动力潜艇和六艘攻击核潜艇、六艘弹道导弹核潜艇。到2035年，中国将拥有超过40艘核潜艇，整个潜艇舰队规模将到达80艘。同时，中国095型攻击核潜艇和096型战略核潜艇可能在2030年前服役。这种史无前例的发展会让美国“难以在亚太地区行动”。
