 用人单位必须按时足额发工资、少安排加班、岗位强度要合适，还得给上工伤保险。影响最大的是现在很多返聘老人和灵活就业的群体

世界气象组织：厄尔尼诺现象增强 气候极端概率料升高
联合国世界气象组织发布的月度报告指出，厄尔尼诺现象（El Nino）预计在7月至9月间迅速发展成强烈事件，极端气候发生概率将因此升高。
法新社报道，世界气象组织每月发布的《全球季节性气候最新通报》指出：“厄尔尼诺现象已在热带太平洋地区形成，预计未来几个月将迅速增强，这将增加世界许多地区发生极端天气事件的可能性。”
气象组织把厄尔尼诺事件分为极强、强烈、中、弱四个等级，本次厄尔尼诺现象预计达到次高的“强烈”等级。
总部位于日内瓦的气象组织说，全球主要气候中心利用不同模型做出的预测均显示，赤道太平洋中部和东部的海面温度将持续显著升高，“关键监测区域的季节平均海面温度预计将比常年平均值高出2摄氏度以上”。
新华社报道，通报显示，预计厄尔尼诺将在北半球秋季继续增强，影响全球许多区域。
温度方面，南纬60度至北纬60度之间的大多数陆地区域气温极有可能高于常年均值，几乎涵盖了极地以外的所有人口稠密地区。
降雨方面，赤道太平洋中部和东部降雨量高于正常水平的可能性更大，而热带印度洋的部分地区、印度次大陆和澳大利亚大部分地区的降雨低于正常水平的可能性更大。
据世界气象组织介绍，厄尔尼诺事件的典型特征为赤道太平洋中部和东部海面温度高于平均水平。通常每两至七年发生一次，持续约9到12个月。通常在3月至6月间开始发展，并在11月至次年2月间达到顶峰，对全球温度的影响通常在形成后的第二年最为显著。

香港正式进入夏季流感季 冠病病毒活跃程度也升
踏入7月，香港正式进入夏季流感季节，冠病病毒活跃程度也持续上升。当局呼吁高风险群体尽快接种疫苗，做好个人防护。
香港卫生防护中心星期四（7月2日）透露，在上周收集的呼吸道样本中，季节性流感病毒阳性比率为6.69%，超越4.94%的基线水平。同时，公立医院流感相关入院率由5月首周每万名人口0.16起，升至上周每万名人口0.38起，高于0.27的基线水平，意味着香港已步入流感季节。
香港卫生防护中心传染病处主任欧家荣星期三在电台节目中指出，在目前录得的香港流感阳性样本中，主要以甲型H3流感为主，占整体约59%；其次为甲型H1流感及乙型流感。他列举数据说，这显示香港流感严重个案在短期内录得较大升幅。其中，成人严重个案由今年5月平均每周的8.6起，激增至上周的23起，当中高达七成患者为65岁或以上的长者。
儿童方面，本星期香港目前录得两起严重个案，今年至今累计10起儿童严重个案。此外，学校及院舍为高危地方，近期录得的爆发个案中，达七成发生在学校，当中七成为小学。
欧家荣说，虽然现阶段难以精准预测流感高峰期，但估计病毒活跃的情况仍会持续三四周。“以往夏季流感季节一般持续约8至12周，我们预计未来数个星期流感活跃程度可能进一步上升。”
此外，冠病病毒的整体活跃程度自5月起也持续上升，呼吸道样本对冠病病毒检测阳性比率，由5月初的0.45%升至6.02%。欧家荣预计，冠病病毒的整体活跃程度也会在未来数周进一步上升，并出现新一轮活跃周期。
至于今年夏季会否出现流感与冠病“双高峰”夹击，欧家荣认为有可能，但目前两者均刚步入活跃期，现阶段较难预测两者的病毒高峰时间会否完全重叠。
欧家荣提醒市民，如果家中有成员出现呼吸道疾病，应尽量避免密切接触并彻底清洁家居。市民在暑期外游也应小心防范各类蚊传疾病。
香港亲子教育顾问梁永乐接受《联合早报》访问时指出，全球冠病疫情结束至今已经三年，许多香港家长对冠病和流感的重视程度不如以前，没有积极为子女接种疫苗，以致香港幼小儿童的疫苗复盖率不高，情况并不理想。
他认为，冠病和流感病毒经常变种，打疫苗有助减低重症风险，随着香港的流感和冠病高峰期再次来临，港府应该多宣传，并与学校合作在校内为学生接种疫苗。

行政院的草案规划五年五个月2100亿元预算规模，采用特别预算编列。但在野党坚持为防止弊端，反对用特别预算，主张回归年度公务预算审查，国民党团版本匡列六年2400亿元预算；民众党团未规划预算上限。
台湾总统赖清德指责在野党提出上述条例草案，违反宪政分立原则、侵犯行政权，尤其无人载具技术发展日新月异，军方建军备战更具有高度时效性。 
但在立法院占有主导优势的在野党，于星期五将行政院和民进党立委林楚茵等人连署的《国防自主无人载具采购特别条例草案》以及在野两个版本，一并交付委员会审查，唯独否决民进党立委钟佳滨的《无人载具产业创建条例》草案。
钟佳滨的草案预计在五年内匡列5500亿元预算来扶植无人机供应链。在野党认为，钟佳滨口头喊五年5500亿元，但草案内无明文说明，形同无上限地编列特别预算购买无人机，会掏空财政。
民进党发言人李坤城指责，国民党早已是侵犯行政权、破坏权力分立的累犯，此次国民党版也未依《预算法》征询行政院意见，呼吁朝野共同支持行政院的版本。
不过，民进党籍退除役官兵辅导委员会前副主委李文忠肯定国民党版，认为整体而言比行政院版“更完整且高明”。
