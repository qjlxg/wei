四级（强制撤离）：发生大规模内乱或全面外部攻击时，实施政府主导的强制撤离与全面应变，禁止前往受影响国。

2万美元无人机消耗400万美元拦截弹 战局或看谁先耗尽弹药
美国与伊朗的军事冲突进入第四天，战况已呈现出典型且严酷的消耗战特征。伊朗正通过一波又一波的廉价无人机，对美国及盟友的防空系统持续施压。专家指出，这场战局的走向，或将取决于哪一方先耗尽弹药。
据彭博社分析报道，伊朗继续动用“见证者-136”（Shahed-136）自杀式无人机打击中东多地的目标。这种小型无人机被视为巡航导弹的廉价替代品。自2月28日美国和以色列对伊朗发动密集空袭以来，伊朗频繁出动这些无人机袭击美军基地、美国在中东的盟友的石油基础设施及民用建筑。
阿联酋方面说，美国爱国者防空系统在拦截伊朗导弹与无人机方面表现良好，拦截率超过90%。然而，军事成本的不对称性引发西方军方的忧虑：一枚爱国者拦截导弹成本约400万美元（约512万新元），而击落的只是价值约2万美元的无人机。
美国智库史汀生中心高级研究员格里科（Kelly Grieco）分析说：“伊朗的盘算是让防御方的拦截弹率先耗尽，让波斯湾国家的政治意志出现裂缝，进而向美以施压，促使美国在伊朗库存耗尽前停止行动。”
这种高强度的消耗节奏令多方感到吃紧。彭博社审阅的一份内部分析显示，按当前消耗速度，卡塔尔的爱国者拦截弹库存可能只能再维持四天。尽管卡塔尔国际媒体办公室随后发表声明称库存“仍然储备充足”，但消息指多哈已在私下敦促冲突尽早结束。
据彭博经济研究国防负责人瓦瑟（Becca Wasser）分析，伊朗在去年与以色列冲突后，估计仍拥有约2000枚弹道导弹。自今年冲突爆发以来，德黑兰已发射超过1200枚飞行物，其中绝大多数为廉价无人机。
军事专家认为，这意味着伊朗可能在刻意保留更具破坏性的弹道导弹，以应对更长期或更深层的打击需要。与此同时，伊朗深知美国总统特朗普的支持者普遍反对卷入长期且昂贵的战争，因此试图通过提高美国的战争代价来达成战略目标。

乌克兰称俄罗斯袭击破坏本国输油管道
乌克兰能源部长什米加尔星期二（3月3日）告诉国际文传电讯社乌克兰分社，1月底俄罗斯袭击乌克兰后，向匈牙利和斯洛伐克输送俄罗斯石油的“友谊”（Druzhba）输油管道的乌克兰支线遭到严重火灾损坏。
路透社报道，自1月27日以来，这条主要由俄罗斯运营的输油管道的石油输送一直处于暂停状态。基辅方面称，俄罗斯袭击了乌克兰西部的输油设施。
什米加尔告诉国际文传电讯社：“输油管道内部的大部分设备，包括各种传感器和其他设备，都因高温而受损。”
路透社无法核实损失程度。
“这条（友谊输油管道）的损坏从外部无法察觉。”什米加尔补充说，维修成本和所需时间将在缺陷检查完毕后才能计算。
欧盟委员会发言人说，欧盟委员会主席冯德莱恩很可能在星期二与乌克兰总统泽连斯基的电话会议中讨论友谊油管道的问题。
莫斯科指责基辅阻止俄罗斯石油通过该管道输送，威胁欧洲的能源安全。
总部位于基辅的石油咨询公司ExPro说，2025年该管道的石油运输量降至10年来的最低点，仅为970万吨。
ExPro称，斯洛伐克接收了490万吨石油，而运往匈牙利的石油为435万吨。

郑丽君：心中只有“关税”两字无选举念头
被视为可能代表民进党参选台北市长的台湾行政院副院长郑丽君，星期二（3月3日）强调现在心中只有“关税”两个字，这是她最重要、最优先要做的事情，要在后续变动情势中，为台湾与产业争取最大利益。
郑丽君是台湾负责对美关税谈判代表，在美国最高法院判决总统特朗普的对等关税违法前，一般认为郑丽君是民进党挑战台北市长蒋万安连任的最佳人选。关税情势生变，在野的国民党立委王鸿薇在立法院询问，郑丽君是否会在关税底定前辞职投入选举，她作上述表示。 
台湾在野党质疑台美对等贸易协定的效力，行政院长卓荣泰星期二率谈判团队在立法院作专案报告时表示，期盼确保台湾已取得的最佳待遇。而美方也承诺，完成内部行政作为与法律程序途径后，将依序通知已签约国家。
郑丽君在回应一些立委质询时指出，目前美国用1974年《贸易法》第122条课征关税是过渡措施，也有可能启动《贸易法》301条款的调查，届时将成《国际紧急经济权力法》（IEEPA）的替代性法律工具。
郑丽君说，不论美国用什么工具，都要确保巩固台湾既有的谈判成果。在与美方签投资备忘录（MOU）时，已写了一项规定，在既有已制定的项目税率，以及半导体的关税下的最优惠待遇，未来对等关税项目若有移至攸关半导体供应链的《贸易扩张法》232 条款，会持续与美国磋商，以为因应。
王鸿薇在质询时指出，122条款的有效期限有五个月，除了第122条，后续还可能有301条款、232条款调查，至少还需半年以上时间。她询问，在台美关税谈判底定前，郑丽君是否会离开副院长职位投入选举？
郑丽君有点无奈地说，“我心中没这问题，为什么我要回答这问题?”，“我现在生活中就是‘关税’两字”。她强调，作为谈判代表之一，目前最优先要做的事情，就是在后续法律变动中，为台湾与产业争取最大利益。
卓荣泰此时插话说，王鸿薇好像很怕郑丽君去选举。王鸿薇否认，强调台美关税充满变数，应以谈判为重。

危机警戒系统是启动紧急遣返与出入境限制的依据，适用于疫情、医疗危害及治安或外部攻击等情况。各级别对应不同建议与政府措施：
一级（预防期）：出现不稳迹象或威胁限于小范围，建议采取防范、避免非必要外出，仍可正常旅行但需谨慎。
二级（限制期）：暴力或不稳扩散或外部攻击趋近，建议就地避险、远离高风险区域；使领馆启动应急计划。
三级（建议自愿撤离）：动荡或外部威胁显著升高时，建议民众自愿撤离或遣返，强烈不建议前往受影响国。
四级（强制撤离）：发生大规模内乱或全面外部攻击时，实施政府主导的强制撤离与全面应变，禁止前往受影响国。

市场机构预计今年车市承压：德银预测整体汽车销量下跌5%，瑞银预计降2%。中国汽车产量（含商用车）在2025年达3450万辆，年产能约5千万辆；但销量增长放缓、补贴退坡与成本上升仍给利润带来压力。

为拉动需求，行业普遍采取超长免息贷款等促销手段（特斯拉五年免息，约20家国产车企最长可达七年）。国家市场监管总局警告继续低于成本销售的法律风险。高盛认为竞争将进一步压缩利润，但海外扩张可能加快。瑞银估算，锂价上涨或令一辆中型纯电动车成本增加约3800元，插混增加约2000元。
