昆明检测列车撞人致11死 官方调查后惩处34人
中国官方公布昆明一检测列车撞人酿11死两伤事故调查结果，对34名有关人员给予党纪政务处理。
中国国家铁路局网站星期五（7月3日）公布中国铁路昆明局集团有限公司洛羊镇站“11·27”重大铁路交通事故调查报告。
报告称，2025年11月27日，云南省昆明市呈贡区洛羊街道辖区内，中国铁路昆明局集团有限公司55537次检测列车行至洛羊镇站内时，与昆明南工务段施工作业人员相撞，造成11人死亡、两人受伤，直接经济损失1452万元（人民币，约276万新元）。
调查将事故定性为因施工过程中违章指挥、冒险作业，现场未建立有效安全防护体系，企业日常安全管理不严不实，行业监管责任落实不到位等造成的重大铁路交通责任事故。
调查组查明，事故直接原因是昆明南工务段在洛羊镇站实施施工作业时，施工负责人在未接到准许施工的调度命令，明知不具备施工条件的情况下，违章打开作业门并带领作业人员进入作业区域，且未与工地防护员共同确认列车运行情况，违章指挥作业人员开展施工作业；驻站联络员未按规定掌握列车运行情况并及时准确向工地防护员发出预报和确报信息，未提醒和督促现场下道避车，致使作业人员被正常通过的55537次检测列车撞击，造成重大人员伤亡。
调查认定，昆明局集团公司及相关单位对事故发生负有责任，建议由成都铁路监督管理局对有关单位存在的违法行为依法给予行政处罚，并责令国铁集团向国务院作出书面检查，昆明局集团公司和成都铁路监督管理局分别向国铁集团和国家铁路局作出书面检查。
调查组对涉嫌重大责任事故罪的驻站联络员移送司法机关处理，将事故调查过程中发现的有关部门和单位的公职人员履职方面的问题线索及相关材料按管理权限，分别移交中央纪委国家监委驻交通运输部、国铁集团纪检监察组，对负有责任的昆明局集团公司、成都铁路监督管理局等单位的34名有关人员给予党纪政务处理。

台海巡署证实大陆海警两船现身花莲港附近海域
台湾海巡署星期六（7月4日）通报证实，中国大陆海警两艘舰艇上午现身台湾花莲港附近水域。
综合台湾《自由时报》《联合报》报道，海巡署称，运用联合情监侦手段，严密掌握大陆海警2305“秀山舰”、1401“崇明舰”动态。
海巡署通报，两艘海警船星期六上午9时25分许均在花莲港东方54海里（限制水域外27海里）往外海航行，海巡署已派遣八里舰及花莲舰并航监控。
海巡署也批评中国大陆“假执法、真扩权”等认知作战，并说强调台湾在东部专属经济海域享有主权权利与管辖权，中国大陆对台湾周边海域没有主权权利。
大陆海警局同日上午在微博公布，海警在台湾以东海域开展常态化执法巡查，并说海警秀山舰编队接替岱山舰编队，继续在台湾以东海域依法开展执法巡查。

中国亚洲经济发展协会会长权顺基任上被查
学会协会领域持续反腐，中国亚洲经济发展协会会长权顺基在任上被查。
中共中央纪委国家监委网站星期五（7月3日）通报，中国亚洲经济发展协会会长权顺基涉嫌严重违纪违法，目前正接受中央纪委国家监委驻中央社会工作部纪检监察组纪律审查和山东烟台监委监察调查。
公开资料显示，权顺基今年66岁，吉林人，朝鲜族。曾任吉林龙潭机械厂厂长、吉林江城民族商社社长、吉林白雪宾馆总经理、吉林省信息总汇总经理、中国民族经济发展总公司副总经理、北京江城商社社长等职务。
2019年，权顺基担任中国亚洲经济发展协会会长，直至此番被查。
中国亚洲经济发展协会简称亚经协。协会官网称，协会是由中国外交部指导成立、国家民政部登记的全国一级社团法人机构；协会前身是1993年成立的“中韩经济发展协会”，2009年8月更名为“中日韩经济发展协会”，2016年5月变更为现名。
协会宗旨是组织和协调会员企业积极地开展同亚洲各国工商企业界的交往，开展双方乃至多方的经济、文化交流合作，增进彼此之间的了解和友谊。
据《中国新闻周刊》报道，亚经协官网一篇文章提到，2019年11月22日，亚经协第二届会员代表大会召开，选举并通过了亚经协第二届理事会。在第二届理事会的第一次会议上，权顺基当选为新一届会长，会议还选举产生了七名常务理事、25名常务副会长、75名副会长。
换言之，在上述会议上，这家协会一次性选举出了100名副会长。
此前，中共二十届中央纪委五次全会在今年1月将“学会协会”纳入反腐重点领域。此后，陆续有时任中国制药装备行业协会副秘书长遆倩鹤，中国纺织工业联合会党委常委、副会长端小平，中国国际跨国公司促进会常务副会长张笑宇等人被查。
