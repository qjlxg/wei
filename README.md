彭博：中国采取措施限制银行和政府机构使用OpenClaw
“养龙虾”在中国爆火之际，中国官方迅速采取行动，限制国有企业和政府机构在办公电脑上运行OpenClaw的人工智能（AI）应用，以消除潜在的安全风险。
近两个月，“养龙虾”成为中国的一大热词。这是指一款名为OpenClaw的AI智能体软件，标志是一只龙虾，中国网民把安装并使用它的过程称为“养龙虾”。
彭博社星期三（3月11日）引述知情人士报道，包括大型银行在内的政府机构和国有企业已收到通知，出于安全考虑不得在办公设备上安装OpenClaw软件。部分知情人士称，相关机构还要求员工如已安装相关应用程序，应向上级报告，以便进行安全检查并可能予以卸载。
据部分知情人士透露，包括国有银行和部分政府机构在内的一些员工，也被禁止在办公电脑以及连接单位网络的个人手机上安装OpenClaw软件。另有知情人士称，这项禁令也扩大至军人家属。
知情人士说，另一些通知并未要求全面禁止使用OpenClaw软件，只是规定在使用前需要事先获得批准。
“养龙虾”安全风险已引发中国政府部门警惕，中国工信部、国家互联网应急中心本周接连发布针对OpenClaw的风险提醒，社媒平台小红书星期二（3月10日）也宣布，即日起将整治AI托管类账号。
由奥地利工程师施泰因贝格尔（Peter Steinberger）创建的OpenClaw今年1月对外开源，被形容为“真正能干活的AI”，主打不只能对话，还可替用户执行任务，包括收发邮件、管理日程、预订机票、运营社媒等。

中国中东问题特使：停火止战才是根本出路
随着中东局势持续升级，中国政府中东问题特使翟隽说，地区生战生乱不符合任何一方利益，实现停火止战才是走出当前困境的根本出路。
据央视新闻报道，翟隽星期二（3约10日）访问阿联酋，与阿联酋副总理兼外长阿卜杜拉会面时说，中国一贯认为，海湾地区各国的主权、安全和领土完整都应得到尊重，任何攻击平民和非军事目标的行为都应受到谴责。
翟隽还说，冲突爆发后，阿联酋沉着冷静应对，采取保持克制的负责任态度，中国对此表示赞赏，愿同阿联酋加强沟通协调，共同做有关各方工作，推动早日实现停火止战。他也希望阿联酋继续全力保障在阿中国人员和机构安全。
美国和以色列2月28日对伊朗发动大规模军事打击，伊朗随后对以色列及美国在中东地区的军事基地等目标展开报复，目前局势仍未出现降温迹象。
