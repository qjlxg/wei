桑泰承认部分人可能觉得不妥，为触及敏感群体表示歉意，但仍坚持类比用于证明观点，称该类比有效地引起关注。他认为要求其参加性别敏感度训练是不必要且断章取义的，强调自己过去的言行显示对女性的尊重，绝无恶意。
部分议员则呼吁桑泰公开道歉，并接受培训，指其言论物化女性，不应在应维护司法与问责的场合出现。政党代表佩尔西瓦尔·森达尼亚并敦促桑泰向安妮·柯蒂斯及所有受冒犯的女性道歉，强调立法者应率先维护社会安全空间与对女性的尊重。
