中国大陆人大会议闭幕会星期四（3月12日）在北京举行，通过包括《民族团结进步促进法在内的11项表决。台湾政府的大陆委员会对此回应指，这部法律适用范围广泛，未来北京可凭此法任扣罪名。

梁文杰称，陆委会对这部法律抱持警惕，因为此法不仅适用于大陆境内，条文也提到，对于大陆境外不符合法律规定的个人与组织，也可依法制裁。换言之，影响范围可能扩及美国、日本与台湾等地的历史、宗教与文化界。
