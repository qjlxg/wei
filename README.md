官员们预计不会立即见到政治反弹，而是会出现“慢燃效应”，取决于战争持续时间、报复规模、美军伤亡和油价影响。路透/益普索民调显示，仅四分之一美国人赞成此次击杀伊朗领导人的行动，约半数认为特朗普过于轻率使用武力（民调在美军首批伤亡公布前结束）。

白宫模型显示，众多摇摆众议院选区更易受冲突影响，部分原本脆弱的共和党众议员将被迫面对战争授权等敏感投票。共和党内一名资深操盘手称，除非行动失败，否则中期选民通常不关心外交，但外交泥潭会造成伤害。
此前抓捕委内瑞拉领导人马杜罗的突击行动几乎没有政治代价，但特朗普支持率自1月已从42%降至39%。分析人士指出，若战争短促且迫使伊朗放弃核计划并更换领导层，公众接受度会高于长期冲突和大量美军伤亡的情形。

白宫发言人卡罗琳·利维特称，目前白宫首要与五角大楼及相关机构合作，确保“史诗怒火行动”持续并最终成功。多位顾问原本建议总统在中期选举前把重心放在医疗与可负担性等经济议题上，但突发军事行动使这一策略受挫。
白宫模型显示，众多摇摆众议院选区更易受冲突影响，部分原本脆弱的共和党众议员将被迫面对战争授权等敏感投票。共和党内一名资深操盘手称，除非行动失败，否则中期选民通常不关心外交，但外交泥潭会造成伤害。
此前抓捕委内瑞拉领导人马杜罗的突击行动几乎没有政治代价，但特朗普支持率自1月已从42%降至39%。分析人士指出，若战争短促且迫使伊朗放弃核计划并更换领导层，公众接受度会高于长期冲突和大量美军伤亡的情形。
部分特朗普支持者表示虽对介入持保留，但仍愿支持总统的强硬做法；亚特兰大83岁选民BJ Moore称，虽不想卷入战争，但支持特朗普的行动。

巴黎讯 — 3月2日，法国上诉法院在涉2020年教师被斩首案中，对三名被告的刑期作出减刑裁定。 
被害人为47岁的历史与地理教师萨穆埃尔·帕蒂（Samuel Paty），2020年10月被一名18岁车臣裔极端分子阿卜杜拉赫·安佐罗夫（Abdoullakh Anzorov）斩首；行凶者随后在与警方交火中被击毙。 
两名被指在行凶前为安佐罗夫提供交通并协助获取武器的同伙——法国籍奈姆·布达乌德（Naim Boudaoud）和车臣裔俄罗斯人阿齐姆·埃普西尔哈诺夫（Azim Epsirkhanov），上诉后原判16年监禁分别降至6年和7年。 
另案中，曾指称帕蒂在出示《先知》漫画前要求穆斯林学生离开教室的13岁女孩之父布拉希姆·什尼纳（Brahim Chnina），其原判13年减为10年；该女孩当时并未在教室内，且在一审时已向遇害教师家属道歉。 
法院维持法裔摩洛哥伊斯兰激进分子阿卜杜勒哈基姆·塞夫里欧伊（Abdelhakim Sefrioui）15年刑期不变。四人均为2024年被判为导致仇恨氛围的八名被告之一。帕蒂因在课堂上用漫画讨论言论自由而成为言论自由象征。 来源：法新社（AFP）

以色列总理内塔尼亚胡3月2日表示，美以对伊朗的战争“可能需要一段时间”，但不会持续多年。
美以对伊朗的空袭始于2月28日对德黑兰的打击，击毙伊朗最高领袖阿里·哈梅内伊，并引发伊朗对以色列的报复及对中东有美军基地的阿拉伯国家的导弹袭击。
特朗普最初预计战争会持续四到五周，随后称可能更久，并为一场范围更广、没有明确期限的对伊战争辩护。内塔尼亚胡在福克斯“汉尼蒂”节目中反驳战争会拖延多年，称“可能需要一段时间，但不会是无休止的战争”。
他表示，这场冲突也可能为中东长期和平创造条件，包括以色列与沙特之间的和解。路透/益普索民调显示，仅有四分之一的美国人支持对伊朗的打击；长期的伊拉克和阿富汗战争使美国公众对海外干预持怀疑态度。
内塔尼亚胡称，美以正在为伊朗民众最终推翻本国政府创造条件，但最终由伊朗人民决定。特朗普的表态自冲突初期已出现变化：2月28日他曾鼓励伊朗人“夺回你们的国家”，暗示推翻政权；3月2日则未提推翻，称战争目的是阻止伊朗发展核武（德黑兰否认）并遏制其远程弹道导弹项目。
以色列被广泛认为是中东唯一拥有核武的国家，华盛顿亦拥有核武。 路透

德黑兰讯——尽管美以在新一轮冲突中对伊朗发动空袭，网络安全分析师表示，伊朗被视为可畏的黑客力量迄今未能在这场战争中发挥重大作用，显示其线上攻击和信息战能力已显衰退。
2月28日空袭后，曾出现若干宣传性网络行动：流行祷告应用BadeSaba据报被劫持，向用户发布“援军已到”并劝军人投降；亲政府媒体账号遭入侵，电视台被用于播放特朗普与内塔尼亚胡的视频。美方称美军网络司令部协助扰乱伊朗通信并在战事初期发挥作用。
但安全公司Recorded Future的分析师亚历山大·莱斯利指出，伊朗军事与情报黑客迄今作用极小，曾与伊朗合作的代理黑客也大多沉寂。曾有少数亲伊团体声称攻破以色列关键系统，但尚未核实。谷歌威胁情报首席分析师约翰·赫特奎斯特称，亲伊黑客常夸大战果以扩大心理影响。
国土安全部此前披露，2023年哈马斯袭以后，疑与伊斯兰革命卫队相关的攻击者入侵美国污水、能源与食品制造的工业控制系统，并发起窃密并泄露以传播政治信息。专家称，除少量虚假宣称与短暂失败的破坏外，亲伊团体“几乎完全沉寂”。
西方政府与企业仍警惕报复性网络攻击与分布式拒绝服务风险。社交平台X报告阻挡了一波伊朗机器人账号。海事情报公司Windward称，自战争开始以来中东约1100艘船只受GPS干扰影响。分析认为伊朗国内互联网封锁是黑客行动减少的主要原因之一。
观察人士回顾，2025年伊朗曾遭受重大网络打击（包括对国有银行与金融服务的破坏），但当时也未见到有效反击。学者兰斯·亨特认为，当前事态凸显现代战争中网络、常规军事与信息战的融合，并将持续成为冲突形态的重要组成。

华盛顿消息 — 3月2日，美国国务院、财政部和卫生与公众服务部宣布停止使用Anthropic的AI产品，加入此前已转向竞争对手（如OpenAI）的国防部行列。
国务院在给路透的备忘录中称，其内部聊天机器人StateChat将由Anthropic模型切换为OpenAI的GPT‑4.1；国务院发言人托米·皮戈特表示，正在立即落实总统关于取消Anthropic合同的指示以确保合规。
财政部长斯科特·贝森特在社交平台X上表示，财政部将终止包括Claude在内的所有Anthropic产品使用；HHS也通知员工改用ChatGPT、Gemini等其他AI平台。联邦住房金融局局长威廉·普尔特宣布，其机构及房利美、房地美将停止使用Anthropic产品。
2月27日，总统下令对使用Anthropic产品的国防部等机构实施为期六个月的逐步淘汰，国防部并将其定性为供应链风险。此举源于政府与Anthropic在技术防护措施及是否允许军方和情报机构利用其技术进行武器目标选择或国内监控等问题上的分歧。
竞争对手OpenAI宣布已与国防部在机密网络部署技术达成协议；CEO山姆·奥特曼在X上表示，将修改与国防部的协议，明确禁止“刻意用于对美国人或本国公民进行国内监视、追踪或监控”。（路透）
