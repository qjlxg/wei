新上市公司只要市值排在当前Nasdaq-100前40名，仅需15个交易日就能直接进指数。标普500指数也类似，都完全跳过正常“seasoning期”（通常要等1年）。

当然这样的好处是在于指数巨头都在为SpaceX打破常规，更早纳入指数，很多被动型基金指数ETF（如SPY、VOO等）都会被动买入，带动股价走高。

中国对日加卤化丁基橡胶征收反倾销税
中国从星期六（3月14日）起对原产于日本和加拿大的进口卤化丁基橡胶，征收反倾销税，征收期限为五年。
中国商务部星期五（13日）在官网公告，调查机关最终裁定，原产于日本和加拿大的进口卤化丁基橡胶存在倾销，国内卤化丁基橡胶产业受到实质损害，而且倾销与实质损害之间存在因果关系。
反倾销税税率为介于13.8%至30.1%。
卤化丁基橡胶主要用于无内胎的气密层、耐热内胎、耐热软管和输送带、药用瓶塞、防震垫、粘合剂和密封材料等。

美国军方：一架美军加油机在伊拉克西部坠毁
美国军方说，一架美军KC-135空中加油机在伊拉克西部坠毁，另一架涉事飞机安全着陆。
负责中东地区美军指挥的美国中央司令部（CENTCOM）星期五（3月13日）发布声明说：“一架飞机在伊拉克西部坠毁，另一架安全着陆。这并非敌方火力或友军误击。”
然而，伊朗军方通过国家电视台发表声明称，伊拉克境内的一个盟友组织用导弹击落了这架飞机，机上人员全部遇难。
法新社报道，由伊朗支持的伊拉克各派系组成的松散联盟——伊拉克伊斯兰抵抗组织（ISRI）声称击落了一架KC-135飞机。他们还说，他们曾攻击另一架飞机，但飞机逃脱了。
自中东战争爆发以来，这个联盟每天都声称对美国在伊拉克和整个地区的目标发动袭击，但很少透露具体目标。
KC-135至少是这场战争中损失的第四架美军飞机，此前三架F-15战斗机在科威特上空被友军误击落。
据美国空军称，KC-135已服役60多年，通常有三名机组人员，分别为一名飞行员、一名副驾驶和一名负责操作加油吊杆的人员。然而，据空军的一份情况说明书指出，部分KC-135任务需要导航员，飞机最多可搭载37名乘客。
