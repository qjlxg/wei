据英国政府发布的公告，这项禁令适用于咖啡因含量超过150毫克每升的饮料，茶和咖啡除外，覆盖商店、自动售货机和网络零售商等所有零售形式。违规企业将面临高达2500英镑（约4300新元）的罚款。
新华社报道，英国政府援引的估算数据显示，英格兰约有10万名儿童每天饮用高咖啡因能量饮料。
研究显示，儿童饮用这类饮料与多方面健康问题以及学业成绩下降存在关联，相关健康问题包括头痛和胃痛频率增加，睡眠时间缩短、质量下降，焦虑、抑郁等情绪问题风险上升等，但现有证据尚不能证实直接因果关系。
英国政府此前已就禁售措施开展为期12周的公众咨询。英国一些大型零售商已自愿停止向16岁以下儿童销售此类饮料，但政府认为，各地执行情况并不一致，有必要通过立法建立统一标准。

受短暂暴雨侵袭 上海部分老旧小区淹水
受短暂暴雨侵袭，上海有部分老旧小区淹水，受灾居民被紧急安置。
综合澎湃新闻、红星新闻和都市快报等报道，上海市防汛办透露，受强雷暴云团影响，上海星期天（7月19日）午后起普降中到大雨，局部暴雨到特大暴雨。因小时降雨强度远超排水能力，杨浦、虹口等地出现道路积水、小区进水。
上海市防汛办紧急调度泵站强排、移动泵车支援抢排。18个重点保障地区均有移动泵车前置驻守。城投集团也调集30多支队伍驰援杨浦等区。
报道引述杨浦区四平路街道工作人员称，辖区内部分小区为建成五六十年的老旧小区，整体地势低洼，本身属于汛期易积水区域，这次极端集中强降雨导致积水影响范围扩大。
工作人员表示，险情发生后，街道第一时间启动常态化演练的防汛应急预案，联动社区居委会、志愿者及专业抢险队伍开展抢险工作，启用储备抽水机持续强排积水，同时重点排查高龄、行动不便的特殊居民，及时将受积水影响的群众转移至小区活动室、邻里中心等安全区域，并保障受灾居民基本就餐需求。
报道也指，杨浦区国权路上的一家大型地下超市被淹。有视频画面显示，这是属于盒马旗下的平价社区超市。复旦大学在校学生提供的图片显示，学校池塘的锦鲤已经游到了校内的马路上。
有网民发布的视频显示，受路面严重积水影响，暴雨过后有市民乘坐皮划艇涉水通行，还有车主下车借助皮划艇转移人员。
上海市气象台星期天中午12时至下午16时的雨量统计显示，降水多集中在西部和中北部地区。其中杨浦五角场街道降水最大，达197.8毫米，市区徐家汇站为57.2毫米。
上海当地官方媒体星期天晚报道，经过多部门连夜抢险作业，杨浦区辖区主干道积水已基本消退，道路交通恢复正常，部分积水小区居民的日常生活逐步恢复。受小区地势高低差异影响，目前少量低洼点位仍存有残余积水，排涝善后收尾工作仍在持续推进。
