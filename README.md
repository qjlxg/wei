今年春运上海机场航班量客流量创历年最高纪录
今年春运，上海浦东、虹桥两大机场共保障航班起降9.6万架次，保障进出港旅客1573.8万人次，航班量客流量均创历年春运最高纪录。
据中新网报道，上海机场集团星期六（3月14日）提供的快报数据显示，浦东、虹桥两大机场保障的航班起降量较去年春运增长2.3%。
其中，浦东机场6.4万架次、虹桥机场3.2万架次，两场日均2407架次；保障的进出港旅客量较去年春运增长5%，其中浦东机场1009.9万人次、虹桥机场563.9万人次，两场日均39.4万人次。
其中，2月20日大年初四至2月22日大年初六，上海机场单日客流量分别为42.5万人次、43.1万人次、43.8万人次，连续三天刷新历史纪录。浦东机场首次春运保障进出港旅客超1000万人次，其中出入境旅客438万人次，较去年春运增长6.5%。
据了解，今年春运，入境游、北方冰雪游、南方避寒游热度不断提升，上海机场积极服务航空公司加密了前往首尔、曼谷、新加坡、吉隆坡，以及哈尔滨、长白山、三亚、海口、西双版纳等热门目的地的国际国内航线。
