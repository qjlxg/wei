台外长出访图瓦卢 回程过境澳洲悉尼
台湾外交部长林佳龙上周访问南太平洋邦交国图瓦卢（台称吐瓦鲁），回程过境澳大利亚悉尼，并与澳洲新南威尔斯州议员麦克德莫特会面。
据台湾外交部上星期六（3月7日）发布的新闻稿，林佳龙应图瓦卢政府的邀请，以台湾总统特使身份于上星期三（3月4日）至星期四（3月5日）率团访问图瓦卢。访团此行受到图瓦卢政府及人民热切欢迎，行程结束后，已在上星期六（3月7日）清晨经澳洲顺利返抵台湾。
台湾外交部称，此次赖清德指派林佳龙担任特使访图，彰显对台图情谊的珍惜与重视。
台湾外交部也提到，台图建交至今，双方在诸多领域长期深入合作，成果有目共睹。未来台湾也将在“总合外交”的理念下，推动“荣邦计划”之进一步落实，促成公私协力，加强布局台湾友邦国家。
民视新闻网报道称，林佳龙访问图瓦卢，回程罕见过境悉尼，造访悉尼歌剧院及悉尼会展中心。他还特地在悉尼歌剧院外，为世界台湾商会联合总会活动录制开幕致词视频。
麦克德莫特（Hugh McDermott）星期一（3月9日）在脸书发文说，上星期五（3月6日）他很高兴与林佳龙会面。
麦克德莫特也说，澳洲和台湾拥有共同的民主原则，并同样关注一个开放、包容，以及稳定的印太地区。

台海周边大陆军机数量骤降引发质疑
台湾官方过去10天里有九天未在台湾海峡周边侦获任何中国大陆军机，令专家对解放军活动数量骤降感到不解。
据法新社报道，北京近年持续对台湾施加军事压力，几乎每天都出动军机和军舰绕行台湾海峡。
不过，法新社根据台湾国防部每日发布的数据统计显示，自2月28日以来，当局24小时内在台海周边仅侦获两架大陆军机，比去年同期的86架显著减少。这是法新社2024年开始统计相关数据以来，大陆军机零侦获的最长时段。
统计也显示，当局过去10天在台海日均侦获六艘大陆军舰，与去年同期持平。
根据统计，今年1至2月在台海周边活动的大陆军机数量较去年同期下降42%，大陆军舰数量则较一年前减少4.5%。
专家预测，大陆显著减少派遣军机绕台，可能与中国全国两会正在北京举行，以及解放军高层近期被免职有关。其他可能原因，则包括美国总统特朗普即将在本月底访华，以及美伊冲突等。
新加坡南洋理工大学拉惹勒南国际研究院高级研究员唐安竹（Drew Thompson）上星期五（3月6日）在线上原创媒体平台Substack写道，“我没想过为解放军暂停在台海周边活动感到担忧，但缺乏合理的解释令人不安”。
总部位于美国华盛顿的国际战略研究中心的中国实力项目研究员贺博然（Brian Hart）则对近期在台海周边活动的大陆军机数量骤降不感意外。
他上星期六（7日）在X平台写道，在每年两会期间，大陆军机飞入台湾防空识别区的数量几乎降至零或接近于零。如果这种情况持续至两会结束后，那就显得不寻常。“我不觉得现在有任何显示不寻常的证据”。
台湾媒体上星期四（5日）引述台湾军事专家苏紫云分析，这是中共内部整肃未止，且企图透过营造和平假象和氛围，影响台湾支持国防特别预算民意的“心机城府”。　　
一名台湾国安官员对法新社说，北京或许在尝试制造一个假象，即大陆降低对台湾的威胁，以蒙骗美国让他们减少对台湾安全的支持。“我们不能松懈防备”。

哈梅内伊之子任伊朗最高领袖 中国称伊朗基于宪法做出决定
伊朗宣布已故最高领袖哈梅内伊之子穆杰塔巴为新任最高领袖后，中国外交部星期一（3月9日）称，这是伊朗基于本国宪法做出的决定。
中国外交部发言人郭嘉昆星期一主持例行记者会。他在回应有关伊朗最高领袖人选的提问时说，“我们注意到有关报道。这是伊方基于本国宪法作出的决定”。
彭博社报道，在中东战争进入第九天之际，伊朗专家会议以压倒性多数票推举穆杰塔巴（Mojtaba Khamenei）为伊朗伊斯兰共和国第三任最高领袖。伊朗半官方的法尔斯通讯社星期天（3月8日）报道，穆杰塔巴在伊朗专家会议上以“决定性投票”当选，但这个消息迟至数小时之后才对外公布。
伊朗伊斯兰革命卫队在一份声明中宣誓全面效忠这位新领袖。
伊朗专家会议是负责选举、监督、罢黜伊朗最高领袖的最高权力机构。原任伊朗最高领袖哈梅内伊2月28日在美国和以色列对伊朗的袭击中身亡，终年86岁。
