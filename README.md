参与会议的六国外长——阿联酋、巴林、沙特、阿曼、卡塔尔和科威特（均曾遭伊朗袭击）审议了“伊朗卑劣袭击造成的广泛破坏”，并讨论恢复地区稳定的措施。 
会后声明指出，海湾各国“将采取一切必要措施，捍卫安全与稳定，保护领土、公民和居民，包括对侵略作出回应的选项”，并要求“立即停止这些袭击”，强调海湾地区稳定不仅是区域问题，更是全球经济稳定的基石。 
据报导，伊朗在美以对其发动一轮致命打击、导致其最高领导人死亡后，开始向海湾多处目标发射导弹与无人机。 
（来源：法新社／通讯社）

2026年3月2日，印度金奈——一名28岁斯里兰卡籍女子因涉嫌以伪造材料取得印度护照并企图出境，在金奈国际机场被警方拘捕。
警方称，外国人登记处（FRRO）报案后，印度中央刑事分局护照诈骗组立案调查。被捕者为S. Nishanthini。据调查，她的家人2009年自斯里兰卡抵印，曾居住在拉曼那塔普拉姆的曼达帕姆难民营。
Nishanthini在科印布托尔（Coimbatore）接受大学教育，并办理了Aadhaar、PAN和选民卡，随后利用这些证件取得护照。她在登机前往斯里兰卡时被拦截并带走调查。

英国首相基尔·斯塔默周日表示，英方已同意美方请求，允许美军使用英国基地对伊朗储存库或发射装置内的导弹实施“特定且有限的防御性打击”。
斯塔默在X发布的视频中称，英国未参与上周六美以对伊朗的联合空袭（该空袭据称造成伊朗最高领袖阿里·哈梅内伊遇难），并不会加入后续更多轮打击。
他说，伊朗随即在区域内发动持续报复，其导弹击中包括有英国公民居住的机场和酒店，构成对英国人的重大威胁。英国战机已参与协调防御并拦截部分袭击。
斯塔默表示，唯有在源头摧毁储存库或发射器内导弹才能终止威胁；基于“长期盟友的集体自卫”和保护英国人生命的考虑，英方依国际法接受了该请求。
（来源：路透社）
