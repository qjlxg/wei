澳大利亚总理宣布将制定全球首套人工智能国家标准
澳大利亚总理安东尼·阿尔巴尼斯宣布，将制定全球首套统一的人工智能国家标准，该标准将明确大型数据中心的运营规则，保护版权，并规范人工智能开发资源的使用。
阿尔巴尼斯在悉尼大学发表讲话时表示：“今天，我宣布，为了把握人工智能带来的机遇并引导其发展，我国政府将制定澳大利亚人工智能标准。”
据他介绍，新标准将把大型数据中心的各项要求整合到一个统一的强制性监管体系中。政府计划在与企业界和贸易伙伴磋商后，于明年初向议会提交相应法案。
澳总理还宣布，将在总理和内阁部内设立人工智能办公室，负责协调政府在该领域的政策。
阿尔巴尼斯表示，版权保护将受到特别重视。他强调，未经版权所有者许可，任何公司都不得使用艺术作品、书籍、音乐或媒体内容来训练神经网络。
“澳大利亚的作家、音乐家、艺术家和记者必须保留对其作品的所有权和控制权。任何公司都不得在未经作者监督的情况下使用澳大利亚的书籍、音乐、艺术作品或新闻来开发或训练人工智能……否则就是盗窃。”澳总理说道。
他还指出，澳大利亚将成为首个将数据中心选址、电力供应和用水量要求整合到国家统一标准体系中的国家。

法国议会通过法案 绝症患者可申请协助死亡
法国国民议会周三最终通过一项法案，允许患有不治之症的成年人享有“协助死亡”的合法权利。
法案规定，在符合严格条件的情况下，患有不治之症的成年人可获得致死药物。​​药物既可由当事人自行服用，也可在当事人因身体状况无法自行操作的情况下，由医生或护士协助给药。
申请“协助死亡”（assisted dying）的资格仅限于身为法国公民或在法合法居留的成年人；这些人须患有严重且无法治愈、危及生命并处于晚期或终末阶段的疾病，因疾病承受持续的身心痛苦，并能够作出自由且知情的选择。
在最终表决中，法国国民议会以291票赞成、241票反对通过法案。
此次投票意味着法国总统马克龙三年多前提出的这项立法正式完成议会程序。法国由此加入少数允许这一权利的国家行列，其中包括荷兰、比利时、瑞士和加拿大。美国也有几个州允许为绝症患者提供医疗协助死亡。
民调显示，法国公众广泛支持允许协助死亡。据民调机构Ifop今年2月发布的调查结果，这项法案获多达84%的法国人支持。
支持者认为，法案将在维持严格保障措施的同时，赋予那些在生命尽头承受难以忍受痛苦的人们更大的自主权，让他们能掌控自己的死亡方式。
包括部分医疗界人士和宗教团体在内的反对者则认为，将协助死亡合法化可能会给弱势群体带来压力。天主教会也是反对法案的团体之一，甚至有一名主教威胁称，将拒绝向支持法案的议员提供圣餐。

中国拟实行统一税务行政处罚裁量基准
中国拟在全国范围内实行统一的税务行政处罚裁量基准。
据中新社报道，中国国家税务总局研究制定《税务行政处罚裁量基准（2026年版）》，星期三（7月15日）起向社会公开征求意见。
中国国家税务总局有关司局负责人介绍，近年来，东北、西南、西北、中南、华北、华东等区域先后发布区域内统一的税务行政处罚裁量基准，在规范税收执法、强化税款征收等方面发挥了积极作用。但也存在体例结构不统一、科学合理性不足、裁量因素不一、指引作用发挥不足等问题。
负责人表示，制定税务行政处罚裁量基准，统一全国税务行政处罚尺度，有利于压缩税务行政执法自由裁量空间，防止同案不同罚，平等维护各类经营主体合法权益。为了更好服务全国统一大市场建设，国家税务总局研究制定《税务行政处罚裁量基准（2026年版）》，在全国范围内实行统一的税务行政处罚裁量基准。
该负责人介绍，此次制定的裁量基准有三个特点：统一体例结构，逐项列明法定依据、裁量阶次、适用条件和具体标准，一目了然，便于执行；统一基准内容，遵循立法本意，“一把尺子量到底”，实现对违法行为的科学评价；防止“一刀切”。充分运用“定额＋幅度”方式设置处罚标准，为个案处理留有合理的裁量空间，防止因简单化“一刀切”导致执法不公。

粮农组织：健康饮食成本飙升 全球三分之一人口无力负担
联合国周三发布的数据显示，过去五年间，健康饮食的成本上涨了25%，导致全球三分之一的人口无力负担。
根据联合国粮食及农业组织星期三（7月15日）发布的《2026年世界粮食安全和营养状况》报告，食品成本的上涨意味着，人均每日维持健康饮食所需的费用（按购买力平价计算）已达到4.28美元（约5.51新元）。
粮农组织首席经济师库伦在纽约联合国总部举行的新闻发布会上说：“结果是，仍有26亿9000万人，相当于全球近三分之一的人口无法负担健康饮食。”
这份报告的完整版将于7月21日在粮农组织位于罗马的总部发布。
根据报告，各项成本的构成并不均衡：例如，豆类、谷物等主食仅占健康饮食成本的13%，而动物性食品占近30%，果蔬类则占16%。
库伦说：“因此，挑战不在于生产足够的热量，而在于让营养丰富的食品变得更可负担。”
库伦认为，政府应调整补贴方向，将资金更多地投向营养丰富的食品，而非大米和小麦等谷物。他也建议，投资本国基础设施和物流将显著降低成本，因为“健康饮食成本中70%至75%的部分是在食品离开农场后产生的”。
库伦指出，全球健康饮食成本最高的地区是拉丁美洲，特别是加勒比地区。这可能是因为该地区侧重于食品出口，而未能生产足够的食品以降低本地价格。

四川巴士撞护栏坠河事故致六死11伤
中国四川省发生一起巴士撞护栏坠河事故，导致六人死亡、11人受伤。
据中国央视新闻报道，四川雅安宝兴县公安局星期四（7月16日）发布警情通报称，宝兴县陇东镇中岗村灯笼沟路段星期三（15日）下午6时许发生一起中巴车碰撞护栏、坠入河滩的事故。

经核实，该车核载17人、实载17人，事故共造成车上六人死亡、11人受伤，其中一人重伤（生命体征平稳），10人轻伤（无生命危险），伤者均已送医治疗。事故调查工作目前已经展开。

印有特朗普头像1美元硬币将投入生产
美国财长贝森特周三（7月15日）宣布，美国铸币局将开始铸造一款印有总统特朗普头像的一美元硬币，以庆祝美国建国250周年。
硬币正面印有“自由”（Liberty）、“我们信仰上帝”（In God We Trust）和“1776-2026”字样，背面则印有总统徽章上的白头海雕图案。

铸造印有特朗普肖像的硬币是特朗普政府正在推进的多项举措之一，旨在将总统的个人印记融入美国货币体系。相关计划还包括今年在新版纸币上添加特朗普的签名，以及推动发行印有特朗普肖像的250美元面值纸币。
