马尼拉讯——国家安全委员会（NSC）、国防部（DND）与菲律宾武装部队（AFP）周一发表联合声明表示，尽管中东冲突加剧，经核实并无对菲领土或与《加强防务合作协定》（EDCA）有关的军事设施的直接威胁。
NSC发言人科尔内里奥·瓦伦西亚重申，菲律宾境内没有美军基地，EDCA地点为菲律宾军方所有、控制与管理的设施。三方表示将持续与中东局势保持密切监测，但紧张态势仍局限于该地区。
国防部发言人阿尔塞尼奥·安多隆回应称，EDCA设施不会因中东冲突成为目标；他并指出，近期伊朗针对的为与其有海陆接壤的国家中的美方设施，这类情形不适用于菲律宾。安多隆强调，EDCA仅用于联合训练、人道与灾难救援（HADR）及后勤预置，且为轮换性准入，不会被用作对外进攻，亦不属于美国所有。此立场与总统费迪南德·马科斯（小马可斯）一贯说法一致。
AFP发言人弗朗塞尔·帕迪利亚上校补充，EDCA安排旨在增强防卫准备与威慑，不削弱国家安全，并指出菲律宾在南海与台湾海峡等区域的战略重要性更多由地理位置决定。
三方呼吁公众保持冷静，依赖经核实信息，警惕并抵制错误信息。
