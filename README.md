位於菲律賓以東海域的潛在風暴預料較大機會週末在本港以東登陸，但其與本港的距離須視乎中心整合位置和速度，以及季風槽和副熱帶高壓脊的互動。
潛在風暴過去一兩日發展速度較慢，尚未建立和鞏固中心。人工智能模式預測潛在風暴短期內中心整合速度較快，繼而較早進入南海，預測登陸位置亦略為偏東。部分傳統電腦模式則預測潛在風暴仍須一段時間整合中心，因此起跑時間較遲，預測登陸位置亦略為偏西。
人工智能和傳統電腦模式對風暴初期中心整合的預測有別，在去年數個案例亦有出現，但現時未有足夠數據判斷誰優誰劣。
至於潛在風暴進入南海後的路徑北份量，則須視乎副高往南伸展的幅度。早前傳統電腦模式預測季風槽有殘餘渦度，因而阻礙副高南伸。然而，傳統電腦模式過去一兩日有加大副高南伸幅度，以致潛在風暴預測路徑往東修正的趨勢。此外，新一輪西風波動會在週末通過中國東北部，亦令副高短期內較難進一步往西伸展。因此，現階段預測潛在風暴東登的機會較大。
東登風暴須較接近本港，以及登陸後採取較為偏西的路徑，才會對本港造成顯著的風雨影響。相信潛在風暴中心完成整合後，本港週末風雨預測會更為明朗。

俄罗斯国家杜马7月22日通过一项由政府提出的法案，自2027年1月1日起，将建立覆盖商品流通全链条的价格监测机制，对纳入俄罗斯联邦国家统计局消费者价格指数核算范围的商品和服务开展价格监测，覆盖从生产商或进口商到零售终端的各个环节，以分析价格上涨的动态及成因。据《俄罗斯报》报道，首批监测对象可能包括肉类、蔬菜、牛奶、谷物等社会重要商品，具体名单将由政府确定。
根据法案，俄罗斯政府将指定有权获取数据的部门，由俄罗斯联邦税务局提供企业销售量、收入、支出和利润等经营数据，信息来源包括纳税申报表、财务报表及税控收银设备记录。相关部门将在法定权限内利用这些数据分析价格走势及形成因素，除《俄罗斯联邦税法》规定的情形外，相关商业数据不得公开。
对于该法案，支持者认为，新机制有助于监管部门利用交易数据更准确分析价格上涨环节及其原因；部分学者和零售业专家则担忧，扩大政府获取商业交易数据的范围可能增加信息泄露风险，而物流、租金、人力等成本并不能完全从税务数据中反映，因此该机制对降低商品价格的实际效果仍有待观察。

中国籍数学家首次获颁菲尔兹奖
有数学界诺贝尔奖之称的菲尔兹奖（Fields Medal）首次迎来两名中国籍得奖者。
综合澎湃新闻和湖北日报等报道，2026年国际数学家大会星期四（7月23日）在美国费城召开，大会为四名有卓越成就的数学家颁发菲尔兹奖。北京大学校友邓煜（37岁）和王虹（35岁）荣获这项殊荣，为中国数学取得历史性突破。
其他两名得奖者是加拿大数学家斯默尔曼（Jacob Tsimerman）和美国数学家帕顿（John Pardon）。
上述颁奖礼举行前一周，网上已流传邓煜与王虹两人将得奖。
菲尔兹奖每四年颁发一次，每次获奖者不超过四位，以奖励当年未满40岁、在数学领域做出杰出贡献的学者。这是国际数学界最高荣誉之一，被称为“数学界的诺贝尔奖”。
王虹在1991年出生于广西桂林，2007年考入北京大学地球与空间科学学院，后转入数学科学学院。她先后获得巴黎第十一大学硕士学位和麻省理工学院博士学位，现任纽约大学柯朗数学科学研究所教授及法国高等科学研究所终身教授。
2025年2月，王虹与国际知名的数学学者、南开大学讲席教授扎尔（Joshua Zahl）合作解决了三维挂谷猜想（Kakeya conjecture）。这一猜想此前困扰数学界逾百年。
邓煜在1989年出生于广东深圳，2007年被保送至北京大学数学科学学院，2009年转入麻省理工学院，2015年获得普林斯顿大学数学博士学位，现任芝加哥大学教授。
邓煜与美国密歇根大学Donald J.Lewis助理教授马骁、密歇根大学数学系教授哈尼（Zaher Hani）合作解决了狭义希尔伯特第六问题，首次以数学严格方式推导出从微观粒子系统到宏观玻尔兹曼方程的完整逻辑链条。
自1936年创立以来，菲尔兹奖长期由欧美学者主导，始终未有中国籍得主登顶。1982年，丘成桐凭借证明卡拉比猜想、广义相对论正质量猜想等成果，成为首位华人菲尔兹奖得主。2006年，陶哲轩凭借在调和分析、数论等多领域的突破性研究获奖。但丘陶两人获奖时分别为美国籍和澳洲籍。
