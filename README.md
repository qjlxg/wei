新关税将于美东时间8月19日零时01分生效，并将在现有相关关税、税费及其他收费基础上叠加征收。
根据白宫发布的情况说明书，此次关税涵盖葡萄酒、曲棍球棒、水泥等，还将适用于其他多种产品，包括乳制品、游泳池、家具、钓鱼竿、种子、服装、假发等。

美国贸易代表格里尔在一份声明中说：“尽管特朗普政府继续致力于与贸易伙伴达成公平和互惠的贸易协定，但加拿大与其他合作伙伴及盟友不同，仍继续针对美国重塑贸易平衡和保护国家安全敏感领域产业的努力进行报复。”
加拿大总理办公室未立即回应置评请求。
