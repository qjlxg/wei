在听取专家和企业家发言后，李强说，今年以来，各地区各部门克服各种不利因素，努力促进经济持续向好。要全面客观认识当前经济形势，既肯定取得的成绩，又清醒看到存在的问题，对经济运行中出现的新变化新动向，要深入分析思考，善于研机析理，更好地趋利避害。
他也说，要注重从整体上把握经济形势，“特别是要看到我国（中国）经济正全面转入高质量发展轨道，长期向好的内在支撑更加坚实”。
李强强调，做好下半年经济工作，直接关系全年发展目标实现，关系“十五五”（2026年至2030年）开局。要加大逆周期调节力度，用好用足存量政策，预研储备增量政策，切实巩固拓展经济稳中向好态势。要系统施策释放内需潜力，抓好服务业扩能提质政策落实，扩大优质服务供给，加快培育消费新增长点。
李强提到，要坚持投资于物和投资于人协同发力，着力推进重大基础设施网络建设，加强教育、培训等人力资本投入，完善医疗、养老、托育等公共服务，在改善民生中扩大内需。要推进新旧动能平稳接续转换，加快制造业数智化转型，推动先进制造业和现代服务业深度融合，促进人工智能等新技术规模化商业化应用，培育更多新产业新赛道。
他还说，要做好稳就业稳企业工作，实施好稳岗扩容提质行动，积极拓展新职业新岗位，确保就业大局稳定。要及时回应企业关切，持续优化营商环境，注重用改革的办法激发市场活力，支持企业更好发展。
李强希望广大企业家坚定发展信心，深入挖掘需求升级和产业升级机遇，持续做强实体经济，为稳就业促发展多作贡献。希望专家学者发挥专业优势，加强战略性前瞻性研究，多提建设性意见。

中国人民大学撤销蒋方舟硕士学位
中国人民大学在月初认定中国青年作家蒋方舟没有学术不端行为后，本周再发通报，称接到新线索并调查后，认定蒋方舟构成学术不端行为，决定撤销其硕士学位。
中国人民大学星期一（7月13日）晚在微博通报称，近日，网上出现关于中国人民大学文学院2019届硕士毕业生蒋方舟学位论文涉嫌学术不端的新线索。学校高度重视，立即组建由多位校内外知名专家参与的调查组，通过文献溯源比对、依规问询、听取当事人申辩等，深入开展核查工作。
经核查，蒋方舟硕士学位论文有九处与境外某篇期刊论文存在文字重合，且相关内容未标注引用、未列明参考文献。人大依据中国《高等学校预防与处理学术不端行为办法》《中华人民共和国学位法》的相关规定，认定蒋方舟构成学术不端行为，并研究决定，撤销其硕士学位。
通报称，学校将以此为鉴，持续深化学风建设，优化论文质量管理，夯实学术道德教育，坚决维护学术规范的严肃性和权威性。

北京密云一副局长自费开发防汛应用
中国北京市密云一副局长发挥编程特长，自行开发防灾小程序“叫应”。
据北京电视台北京新闻报道，今年汛期，北京市规划和自然资源委员会密云分局副局长谢陨石发挥编程特长，自购10亿token（词元，AI处理数据的基本单位），花了近一个月时间，自主编程了密云专属防灾App“叫应”。
据报道，这套小程序收录了全区地灾隐患点、受威胁险户、包保人员，能做到实时更新山体预警、雨情变化和群众转移状态，也能跳转导航到隐患点查看。
话题“密云局长买10亿token手搓防汛APP”登上中国微博热搜。北京新闻评论称，这件事的意义，不在于“局长会写代码”这个标签，而在于它提供了一个基层数字化治理的鲜活样本。
《北京晚报》则评论，谢陨石“手搓”APP之所以引发关注，就是因为他跳出了被动完成任务的思维，主动发现防汛工作刚需、主动研判防汛风险隐患，主动探索数字化解题路径。“叫应”的出现不仅是基层治理的微创新，还体现了基层干部的数字化自觉。
评论还称，谢陨石“手搓”APP对基层干部开展工作颇有借鉴意义。这不是说让基层干部都去学编程、自掏腰包买token，更不是在制作数字化工具上盲目开“卷”，而是启示基层干部更加积极地拥抱数字化潮流，提升智能化思维。
不过，也有网民质疑谢陨石为什么有这么多“业余”时间开发软件；也有人爆料，谢陨石是使用了美国人工智能企业Anthropic的工具Claude Code开发的软件，而Claude Code已经禁止在中国使用，怀疑可能泄露关键信息。

也门国防部说，也门武装部队袭击首都萨那国际机场的跑道，以阻止一架伊朗飞机降落。

萨那目前由伊朗支持的胡塞武装组织控制；受到包括沙特阿拉伯等波斯湾国家在内国际社会承认的也门政府，在南部城市亚丁运作。

乌克兰总统泽连斯基宣布：撤换总理改组内阁
（基辅综合电）乌克兰总统泽连斯基星期天宣布将大规模改组政府，提议更换总理，并调整部分执法机构负责人。这凸显他决心避免让乌克兰在今年冬天重演去年冬季的困境。
泽连斯基没有说明谁将接替现任总理斯维里登科，不过消息人士透露，泽连斯基可能任命国营乌克兰石油天然气公司（Naftogaz）首席执行官科列茨基（Sergii Koretskyi，48岁），或前总理什梅加尔（Denys Shmyhal，50岁）接任。什梅加尔在能源领域也有丰富的经验。
泽连斯基星期天（7月12日）发声明说，为即将到来的冬季做好准备是这次领导层调整的原因之一，因为乌电力和供暖基础设施仍极易遭俄弹道导弹攻击，而且，乌拦截弹储备也不足。他说，乌克兰必须为任何可能出现的威胁做好应对准备。
一名消息人士透露，斯维里登科将出任乌克兰驻美国大使。泽连斯基也说，他会改组执法机构。
这是俄乌战争四年多前爆发以来，泽连斯基第二次撤换总理和改组内阁。去年7月，他要求担任总理六年的什梅加尔辞职。 
近期乌军队收复了部分失地，并对俄罗斯展开远程及中程无人机打击。这些袭击扰乱俄石油业，引发燃油短缺。 
泽连斯基星期一出席在巴黎举行的意愿联盟（Coalition of the Willing）会议，向西方盟友寻求为乌克兰争取更多空防承诺。
法国总统府官员对媒体说，此次会议的重点是反导领域的合作，涵盖采购更多美国爱国者导弹，以及探讨欧洲与乌国防工业如何开发替代方案等。
其中一个选项，是由欧洲国家合作开发一个能与欧洲曼巴空防系统（SAMP-T）和美国爱国者系统互补的武器系统，并让乌克兰在生产中发挥重要作用。
