俄罗斯国家杜马7月22日通过一项由政府提出的法案，自2027年1月1日起，将建立覆盖商品流通全链条的价格监测机制，对纳入俄罗斯联邦国家统计局消费者价格指数核算范围的商品和服务开展价格监测，覆盖从生产商或进口商到零售终端的各个环节，以分析价格上涨的动态及成因。据《俄罗斯报》报道，首批监测对象可能包括肉类、蔬菜、牛奶、谷物等社会重要商品，具体名单将由政府确定。
根据法案，俄罗斯政府将指定有权获取数据的部门，由俄罗斯联邦税务局提供企业销售量、收入、支出和利润等经营数据，信息来源包括纳税申报表、财务报表及税控收银设备记录。相关部门将在法定权限内利用这些数据分析价格走势及形成因素，除《俄罗斯联邦税法》规定的情形外，相关商业数据不得公开。
对于该法案，支持者认为，新机制有助于监管部门利用交易数据更准确分析价格上涨环节及其原因；部分学者和零售业专家则担忧，扩大政府获取商业交易数据的范围可能增加信息泄露风险，而物流、租金、人力等成本并不能完全从税务数据中反映，因此该机制对降低商品价格的实际效果仍有待观察。

日本连续四天出现“酷暑日” 追平历史纪录
日本中部岐阜县观测到40摄氏度高温，这是日本连续四天出现40摄氏度以上的“酷暑日”，追平历史纪录。
新华社报道，日本气象协会星期五（7月24日）发布数据显示，当天午后在岐阜县美浓市观测到40摄氏度的高温。这是日本连续第四天出现“酷暑日”，也是继2013年8月10日至13日、2025年7月30日至8月2日之后，日本第三次连续四天观测到40摄氏度以上高温。
日本气象厅今年决定将40摄氏度及以上高温天气称为“酷暑日”。21日，岐阜县郡上市测得40摄氏度高温，这是日本今年首次出现“酷暑日”。22日和23日，日本又有多地观测到气温达40摄氏度或以上。

中东局势推高油价 国泰8月起上调客货运燃油附加费
中东战火重燃带动国际油价再次攀升，香港国泰航空宣布从8月起全面上调客货运燃油附加费。
综合路透社和香港星岛网报道，国泰航空星期五（7月24日）宣布，鉴于近期航空燃油价格出现升势，将从8月1日起上调燃油附加费。其中，长途航班的客运附加费将大幅增加41%至1362港元（约224新元）。这是国泰自今年4月以来，首次调高相关收费。
往返香港与中国大陆的航班，客运附加费将从165港元上调至198港元，增幅达20%；飞往南亚航班的附加费从448港元增至633港元；其他航线则从241港元增至339港元。
货运方面，自香港始发的货物燃油附加费也将同步上调：长途航线增至每公斤10.10港元，中途航线增至每公斤5.10港元，短途航线则增至每公斤2.80港元。
国泰指出，航空燃油价格于今年4月触及高位后曾一度回落，但受近期中东局势动荡影响，油价再度回升。为反映成本变化，集团决定于8月调整收费。不过国泰强调，即便经过此次上调，最新的附加费水平仍低于4月时的高峰。
国泰引述国际航空运输协会（IATA）的数据指出，截至7月17日当周，全球平均航空燃油价格已攀升至每桶149.4美元（约192.8新元），较前一周的127.06美元及7月3日当周的119.13美元大幅飙升。
国泰表示，现行机制能有效且适时地反映油价波动。自今年5月起，燃油附加费也曾随着油价回落而多次下调。未来，国泰将继续维持每两周检讨一次燃油附加费的机制，确保收费紧贴市场实际状况。

胡塞武装：海上禁运针对沙特 曼德海峡未封锁
也门胡塞武装发言人强调，在本周宣布对敌对国家沙特阿拉伯实施海上禁运后，他们并未封锁战略要道曼德海峡（Bab al-Mandeb Strait）的交通。
这个由伊朗支持的组织在宣布封锁后，袭击了附近红海的一艘沙特船只。此前，伊朗也封锁了沙特唯一的另一条海上通道——霍尔木兹海峡（Strait of Hormuz），此次封锁威胁到这个全球最大的石油出口国向世界各地输送石油的能力。
由于伊朗与美国之间再次爆发冲突，全球能源市场已经动荡不安。胡塞武装发动袭击后，布伦特原油价格星期四一度飙升至每桶100美元（近130新元）以上。
法新社引述胡塞武装高级谈判代表穆罕默德·阿卜杜勒萨拉姆在社交媒体发表的声明说：“正如一些人所暗示的那样，曼德海峡并未封锁。”他补充道，其组织的立场“仅限于海上封锁，而且只影响沙特一方，”是对沙特“封锁”胡塞武装控制区的回应。
星期一，胡塞武装通过无线电向船只发出警告，称他们将攻击“属于敌人”的船只，除非它们遵守封锁令。
法新社星期三分析的数据显示，在胡塞武装宣布实施海上封锁后，有九艘船只掉头，但其中一些已经恢复航向。
尽管如此，曼德海峡的交通仍在继续。在霍尔木兹海峡通道变得危险之后，曼德海峡已成为沙特出口的关键通道。
根据海运追踪机构Kpler的数据，中东战争爆发后，2026年3月至7月中旬，沙特阿拉伯经曼德海峡的海上原油出口量比2025年同期增长了八倍。

罗马尼亚首次击落犯境无人机
罗马尼亚在其领空击落无人机，这是自俄罗斯全面入侵乌克兰以来的首次。
据与乌克兰接壤的北约成员国罗马尼亚国防部星期五（7月24日）称，这架无人机在距离布加勒斯特约114公里（70英里）处被击落。
法新社报道，罗马尼亚总统尼库索尔·达恩在新闻发布会上说：“我们的领空出现了一架无人机。几分钟前，大约上午11时（新加坡时间下午4时），一名驾驶F-16战斗机的罗马尼亚飞行员将其击落。”
他补充道：“该地区无人居住，因此飞行员能够从高空开火。目前，来自各个机构的小组正在调查此事，以提供有关该事件的所有细节。”
国防部称，这架无人机是在帕迪纳村（Padina）上空被击落的。
路透社引述北约空军司令部说，调查正在进行中，北约正与罗马尼亚当局保持密切联系。
罗马尼亚与乌克兰共享650公里（400英里）的陆地边界。自2022年莫斯科全面入侵乌克兰、开始隔着多瑙河攻击乌克兰港口以来，罗马尼亚已多次报告其领空遭到俄罗斯无人机侵犯。
