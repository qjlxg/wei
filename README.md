马尼拉 — 社会福利与发展部（DSWD）下属国家儿童照护署（NACC）助理秘书Rowena Macalintal周六呼吁社交媒体平台，尤其是Meta，加大打击网上“婴儿买卖”和非法领养群组的力度。
Macalintal在广播节目中表示，NACC已加强与菲律宾国家警察及妇女儿童保护中心的联络，向警方通报多处在Facebook上监控到的“收养”、“Babies for Sale”及“在线领养”群组，这些群组成为卖家与买家联络交易的渠道。她同时说明，因警方正进行诱捕行动，要求不要全面下线相关群组，以保全案件证据并抓获涉案人员，目前已有多人被逮捕并起诉。
调查显示，这类群组内婴儿成交价从8,000披索到130,000披索不等，有些群组成员多达1.3万人，凸显Meta监管不足的问题。Macalintal指出，扩展版反贩运法规定，若互联网中介明知或因重大疏忽允许其平台被用于贩运，亦须承担法律责任。
NACC呼吁政府与Meta进行正式对话，敦促平台阻止此类群组的设立。她重申，在线领养在菲律宾属非法，买卖双方均可被追究责任；正式领养程序若文件齐全，约需九个月。无法照顾子女的父母可向NACC、DSWD各办事处、持证儿童照护机构及其他授权收容机构寻求评估、咨询与干预，以保障儿童与父母的安全与合法权益。

马尼拉讯 — 菲律宾总统费迪南德·马科斯二世下令外交部（DFA）与移民劳工部（DMW）尽速查明并确保在伊朗的菲律宾公民安全，此前美以两国周六对伊朗展开大规模军事行动。
“我们收到有关空袭击中伊朗的报告，正在获取更多信息，”马科斯在社交媒体表示，并强调在此期间保障在伊朗及中东地区菲律宾公民安全至关重要，已指示两部在有关地区确认我侨民下落并采取即时安全措施。
宫廷新闻官克莱尔·卡斯特罗转述外交部长玛·特蕾莎·拉萨罗称，菲律宾对中东局势升级表示关切，呼吁各方立即回到对话与谈判，以恢复地区和平与稳定。
外交部表示，驻德黑兰、特拉维夫及海湾合作委员会国家使馆均已进入高度戒备状态；目前尚无菲律宾人受伤或受影响的报告，使馆准备协助在外国民，并继续劝诫菲籍人士留在家中或安全地点，避免前往公共场所。
美以此次对伊朗多座城市发动空袭，德黑兰出现爆炸与浓烟。美国总统唐纳德·特朗普称目的是“消除伊朗的迫在眉睫威胁”，以色列国防部长以色列·卡茨将此次行动称为“预防性打击”。

2026年2月28日，斯里兰卡外交、外劳与旅游部发布声明称，将密切关注中东地区迅速变化的安全形势。政府表示，约有100万斯里兰卡公民在中东居住工作，其安全与福祉为首要任务。
声明提醒在中东的公民遵守安全指引：及时关注所在国官方与媒体通报；非必要尽量避免出行，远离边境、易发生军事行动的地区、大型集会与示威场所；与就近的斯里兰卡使领馆保持联系，确保手机有电并随身携带护照或身份证复印件。
外交部已指示所有驻中东使领馆开通24小时应急热线；领事事务处与斯里兰卡外劳局（SLBFE）将尽快公布供国内家属查询的专线号码。外劳局已公布咨询专线：+94 71 980 2822。外交部表示，各使领馆与本部保持密切联系，目前对在中东的斯里兰卡人影响尚不大。

美国总统特朗普周六表示，美军已在伊朗境内展开“重大作战行动”，并警告可能出现美方伤亡。白宫称此次打击目标是摧毁伊朗导弹并重创其海军，五角大楼在社交平台上将行动命名为 EPIC FURY。

德黑兰方面称正准备进行报复，一名伊朗官员称回应将是毁灭性的；以色列军方表示伊朗已向以色列方向发射导弹作为回应。 
特朗普在社交平台上说，政府已尽最大努力降低在该地区美方人员风险，但伊朗政权企图杀戮，可能会有美国英雄阵亡或受伤。他同时呼吁伊朗革命卫队放下武器，承诺放下者可获豁免，否则面临“必死”下场。 
华盛顿与德黑兰近期就核问题进行多轮会谈，周四的最新会谈未达成协议。特朗普此前曾在伊朗全国抗议期间鼓励示威者，称轰炸将遍及并敦促民众在事后接管政府。

斯里兰卡民航局（CAASL）表示，局长兼首席执行官Daminda Rambukwella机长主持实施一项特别应对计划，以便妥善安排进出斯里兰卡的旅客与航班。 
此举系应中东及周边地区冲突导致多国关闭空域的局面。受影响航线中，斯里兰卡航空和FitsAir已临时暂停飞往相关国家及经由该空域的航班。 
民航局提醒拟前往上述目的地的旅客，在前往机场前务必向旅行社或航空公司确认航班信息。同时在港口与民航部部长、副部长及秘书的指导下，局方会同相关利益方已制定应急行动方案。 
马塔拉·拉贾帕克萨国际机场已做好备降准备。民航局敦促公众不必过度惊慌，并将根据形势继续发布最新信息。
