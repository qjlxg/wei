大埔火灾调查听证会结束 工人天台抽烟等照片首次曝光
香港大埔宏福苑火灾独立委员会星期五（7月17日）完成最后一天的听证会，并发布书面总结陈词文件，其中首次展示承建商宏业工人在去年11月大火发生前，于最先起火的宏昌阁天台上抽烟。
综合网媒“香港01”和香港《星岛日报》报道，宏福苑去年11月26日发生致命大火，造成168人遇难。港府去年12月宣布成立独立委员会调查这起事故。
委员会经过六轮听证会后，在星期五发布书面总结陈词文件，让大火当天的照片和闭路电视画面首次曝光，其中包括身穿承建商宏业制服的工人在首先起火的宏昌阁天台上吸烟。
这份以英文书写的陈词文件长达627页，已上载委员会网站。陈词中提到，这场大火发生在当天下午2时30分至2时40分之间。最早的火灾迹象是由F座宏昌阁后方的脚踏车棚上的闭路电视拍摄到。从2时33分开始，录影显示有淡淡的烟雾从外面的棚架飘入脚踏车棚内。
到下午2时42分，闭路电视拍到第一道火光，是宏昌阁104室及105室窗外的棚架，之后火屑滴落的频率越来越高，到2时44分48秒，脚踏车棚的地下已有可以自行燃烧的火苗。
陈词称，火势之后迅速蔓延到屋苑其他楼宇，宏昌阁旁边的E座宏泰阁天台在2时58分已有起火迹象，较远的C座宏新阁天台则在3时12分已有火苗，同样来自被烧熔的棚网。
陈词指出，火势除形成12条垂直火柱外，主要透过飞散的棚架及被烧熔的棚网火屑在各座楼宇之间扩散，燃点起其他楼宇外墙的可燃物。
至于火灾的源头，委员会排除了自燃、电力故障、烧焊等工地作业的典型火灾成因，估计有可能来自工人吸烟，并称听证会上已有足够证据证明工人持续在地盘吸烟，直至火灾当日。
陈词展出的照片显示，火灾当日下午，承建商宏业与顾问公司鸿毅的WhatsApp群组中，流传G座宏盛阁的楼梯间有大量烟头的影片。同时，闭路电视拍到当日下午2时45分，有身穿宏业制服的工人，在首先起火的宏昌阁天台上吸烟。
