【311大地震十五週年】2011年3月11日日本時間下午2時46分，東北地區宮城縣對開海域發生9級猛烈地震。這場世紀強震轉眼已過十五載，其引發的巨大海嘯席捲了日本東北及關東沿岸，並觸發福島第一核電廠核事故，整場災難導致約二萬人喪生，至今仍有二千多人下落不明。
311地震為一次罕見且發生機制複雜的「連動型地震」。其震央雖然位於宮城縣對開海域，但整體的斷層破壞面卻異常廣闊，並由岩手縣一直延伸至茨城縣，合共長達約500公里，東西闊約200公里。在短短五分鐘內接連引發了至少三處大範圍的斷層錯動，累積釋放的能量極為龐大。
災後十五年，受災地區目前在安全性、基礎設施、觀光及出口等方面持續進行復興工作。然而，部分災民的心理創傷，以及福島第一核電廠為降低潛在輻射風險而持續推進的廢爐作業（如近年展開的取出核燃料碎片樣本、處理水排海及移除部分機組乏燃料），依然是這場災難留下的長遠挑戰。
而日本三陸近海未來仍然存在強震風險。三陸北部近海至北海道東部近海的岩盤可能已累積足以引發巨大地震的能量。最壞情況下地震規模可達9級或以上，北海道至東北可能出現超過20米的巨大海嘯。

食安办将健全跨部门会商、联合检查、联合执法机制，针对群众反映强烈的校园食品、“幽灵外卖”、食品添加剂滥用等重点领域、重点品种，开展常态化专项整治。
食安办将持续强化食用农产品、食品贮存运输、网络销售、进口食品等关键环节监管，严厉打击违法违规行为，形成“露头就打、严惩不贷”的高压态势。
食安办还将针对校园食品安全，将持续深化“互联网+明厨亮灶”建设，完善校园食材集采机制，强化常态化联合检查，严防不合格食材进入校园。
另外，食安办将组建抽检“蓝军”队伍，聚焦人民群众急难愁盼问题，提升监督抽检靶向性，精准排查风险隐患，实现防患于未然。

伊朗空军指挥官遇袭身亡 伊朗称在波斯湾北部击沉船只
新华社引述伊朗法尔斯通讯社的报道说，伊朗伊斯兰革命卫队空军指挥官萨达尔·伊斯梅尔·德甘和他的妻子、两个孩子等，在伊朗中部阿拉克遇袭身亡。
法新社则引述伊朗革命卫队星期四（3月12日）发布的消息说，他们在波斯湾北部击沉一艘悬挂马绍尔群岛国旗的船只，并声称这艘船为美国所有。
革命卫队在网站Sepah News上说，这艘名为“Safesea”的船是“美国恐怖分子军队的资产之一”，并且“在无视警告和警报后，于波斯湾北部遭到袭击”。
另据路透社引述伊朗议会议长卡利巴夫星期四在社交媒体X发布的贴文说，美国任何针对伊朗海湾岛屿的侵略行为都将促使德黑兰“放弃一切克制”。
