能源成本上升 欧元区制造业复苏艰难
荷兰国际集团首席经济师贝尔特·科莱恩指出，欧元区工业产出1月已降至2024年底以来的最低水平，随着中东冲突推高能源价格并加剧供应链扰动，欧元区工业生产尤其是能源密集型行业面临新风险。
星期五（3月13日）公布的数据显示，继去年12月下降0.6%之后，今年1月欧元区工业产出环比下降1.5%，为2024年12月以来的最低水平。当月，该地区主要经济体德国、意大利和西班牙的工业产出较上月均有所下降。
新华社引述科莱恩说，市场原本期待欧元区公共投资增加能带动制造业走强，特别是国防和基础设施领域的支出将推动制造业增长。然而，如果能源价格再次持续上涨，能源密集型行业的复苏前景将受到冲击。
科莱恩指出，2021年至2022年能源价格飙升导致欧洲能源密集型产业持续承受较大压力。近来欧元区制造业信心刚有所恢复，地缘政治紧张局势再次给该地区工业前景带来明显下行风险。

德国总理：美国放松对俄石油制裁为错误之举
德国总理默茨批评美国放松对俄罗斯制裁、试图通过暂时允许俄罗斯出售石油来缓解伊朗战争引发的油价压力这一决定。
特朗普政府已经再一次发出许可证，允许买家接收已在海上的俄罗斯石油，此举扩大了上周给予印度的临时豁免范围。此举引发乌克兰盟友担忧，认为克里姆林宫可能从能源价格飙升中受益，并有更多资金为持续四年的对乌克兰战争提供支持。
默茨星期五（3月13日）在与挪威首相约纳斯·加尔·斯特勒共同举行的新闻发布会上说：“让我说得非常清楚：我们认为现在出于任何原因放松制裁都是错误的。”
彭博社引述这位德国保守派领导人说：“我们将而且必须继续加大对莫斯科的压力。我们不会允许伊朗战争让我们因此退缩或分心。”

古特雷斯访贝鲁特 吁真主党和以色列“停战”
到贝鲁特访问的联合国秘书长古特雷斯呼吁黎巴嫩真主党和以色列“停止战争”，说“现在已不再是武装团体的时代”。
古特雷斯星期五（3月13日）在黎巴嫩总统府发表讲话时说：“我强烈呼吁真主党和以色列停火，停止战争，让黎巴嫩成为一个独立的国家，让黎巴嫩当局掌握武力使用权。”此时，黎巴嫩战争已进入第11天。
法新社引述古特雷斯说：“现在已不再是武装团体的时代，现在是强大国家的时代。”
联合国将于星期五发起一项紧急人道主义筹款活动，以援助黎巴嫩境内80万多名登记在册的流离失所者。
古特雷斯星期五抵达贝鲁特，他称之为“声援”访问。据黎巴嫩卫生部统计，自3月2日以来，黎巴嫩战争已造成至少687人死亡。
他指出，战争爆发正值穆斯林庆祝斋月和基督徒守斋之际，本应是和平时期。
“不幸的是，黎巴嫩卷入了一场人民不愿面对的战争。”
路透社报道，战争持续近两周，已有2000人丧生，其中大部分在伊朗，黎巴嫩也有许多人遇难，波斯湾地区的死亡人数也在不断增加。几十年来，波斯湾地区首次成为中东冲突的前线。美军也遭受了伤亡。美国军方证实，一架在伊拉克西部坠毁的加油机上的所有六名机组人员遇难。

美国去年四季度经济增速下修至0.7% 全年增长2.1%
美国商务部公布的修正数据显示，去年四季度美国实际国内生产总值（GDP）按年率计算增长0.7%，较首次预估值1.4%显著向下修正。
美国商务部星期五（3月13日）发布的数据显示，去年最后三个月的美国经济成长率数据调整，是受到出口、消费者支出、政府支出及投资额均下修的影响，不过进口额萎缩幅度未如原先估算严重。
法新社引述最新的数据报道，在美国2月28日携手以色列对伊朗发动打击之前，这个世界最大经济体的状况就已经比预期更为疲软。此次空袭将这个原油资源丰富的地区拖入了战争泥潭。战争爆发以来，能源市场受扰，导致燃油价格飙升，加剧人们对通货膨胀的担忧。
2025年美国全年经济成长率数据，则由先前估算的2.2%略微下修至2.1%。
这一疲软的经济表现为总统特朗普重返白宫后的第一个完整年度画上了句号，与此同时，人们对就业市场降温和通胀持续高企的担忧也在加剧。

美国消费者信心降至三个月低点 战争推高油价担忧
美国消费者信心降至三个月低点，因为近期对伊朗战争会影响汽油价格的担忧加剧。
密歇根大学星期五（3月13日）公布的数据显示，3月消费者信心指数初值从2月的56.6降至55.5。调查时间为2月17日至3月9日。彭博调查的经济师的预估中值为54.8。
彭博社报道，星期五的数据显示，消费者预计未来12个月的通胀率为3.4%，与前月持平。他们预计未来五至十年通胀率为3.2%，较前月略有回落。
展望未来，在美国与以色列对伊朗采取军事行动背景下，燃料价格上涨可能进一步打击消费者情绪，尤其是在生活成本高企的情况下。2月疲弱的就业报告反映出劳动力市场仍然脆弱，也可能继续压制信心。
另有星期五公布的数据显示，即便在伊朗战争爆发前，美国联邦储备局青睐的核心通胀指标1月环比上升0.4%，涨幅较为坚挺。
对于普遍预计将在下周政策会议上维持利率不变的美联储官员而言，通胀前景的不确定性可能限制未来几个月降息的空间。
该调查负责人Joanne Hsu在声明中说：“在伊朗军事行动前完成的访谈显示，信心较前月改善，但随后九天内的回落完全抹去这些初步涨幅。”
她指出，伊朗遭到空袭后，人们对汽油价格的担忧升温，促使3月对未来一年汽油价格的平均预期升至2022年以来最高水平。
消费者对自身财务状况和整体经济的展望均出现恶化。
