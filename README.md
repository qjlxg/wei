泰国工商会主席博·阿兰瓦塔南（ดร.พจน์）表示，今日（3月2日）与总理会商，评估中东冲突对泰国经济与企业的影响，并制定全面应对方案。政府已邀请安全、经济与民间部门共同研判，以减轻对民众与企业的冲击。 
报告指出，霍尔木兹海峡为全球海运石油约每日14–20百万桶、约占海运石油的20%，亦是供应亚洲液化天然气的重要航道。一旦航运受限或保险与安全风险上升，将立即收紧能源供给，推动油价快速上涨并加重通胀压力。中国、印度、日本、韩国等亚洲国家受影响显著；对能源净进口的泰国而言，油气价格波动将传导为生产成本、电价、运输费与民生开支上升。 
工商会与物流业评估的具体影响包括：一、油价不可避免上涨，推高全球海运、空运与陆运成本，增加整体物流费用；二、对波斯湾航线的“战争风险保费”会显著提高，初步估计运费每航次上升约50%；三、对中东的出口可能短期延缓或暂停，承运减少订舱或改航线将拉长运期并增加库存与资金压力；四、即便对欧洲的航线已有改道绕行好望角，若油价继续上涨，运费与运输时间仍将进一步上升，最少延长约一月，影响双程箱运与出口竞争力。 
若油价与运费同时上升，将对全球贸易与泰国出口竞争力形成双重压力。工商会建议企业尽快审查运输合约、保险条款与能源成本，做好多情景规划并维持充足流动性；同时把握农业、食品加工及日用消费品等可能增长的出口机会。 
政府经济部门正密切跟进并准备能源、金融與貿易等配套措施，工商会表示愿与政府紧密合作，深入评估行业影响并协助中小企业稳住营运。工商会呼吁企业与民众关注官方信息、谨慎经营，共同维护国家经济稳定。

阿克罗蒂里位于塞浦路斯南端一块呈方形的半岛上，过去曾被用于伊拉克、叙利亚和也门的军事行动。

第一财经梳理历年《四川省人力资源和社会保障事业发展统计公报》发现，近年来四川农民工转移输出数量持续增长，尤其在2023年创下新高。

中新网报道，全晓锋称，四川将持续大规模开展职业技能提升培训，今年计划培训包括农民工在内的各类城乡劳动者50万人次。
根据人民网2024年报道，四川农民工总量常年保持在2600万人左右，占全国的十分之一。
另据中国国家统计局官网消息，中国国家统计局上星期六（2月28日）公布2025年国民经济和社会发展统计公报显示，截至2025年末中国就业人员为7亿2504万人，其中农民工总量首次突破3亿，达到3亿零115万人，比上年增长0.5%。
其中，外出农民工1亿8006万人，增长0.8%；本地农民工1亿2109万人，增长0.1%。
统计公报也提到，中国农民工人均月收入5075元人民币（941.91新元），比上年增长2.3%。

特朗普对伊朗升级军事行动 分析：欲转移国内焦点
在国内丑闻缠身、政策受挫之际，美国总统特朗普突然升级对伊朗的军事行动。有分析认为，这不仅是一次地缘政治决策，更是一场带有明显政治动机的“转移性战争”，试图以外部冲突重塑舆论焦点，压过国内争议。
英国《卫报》评论员奇维斯（Christopher Chivvis）撰文指出，当前局势与2003年伊拉克战争前夕存在令人警惕的相似之处。
当年，美国在尚未充分回答“是否应该开战”的前提下仓促入侵，结果付出沉重代价。如今，华盛顿同样在缺乏公开审慎辩论的情况下，将军事行动推向以政权崩溃为目标的方向，却几乎没有认真评估战争是否必要，以及能否带来稳定的政治结果。奇维斯认为，特朗普的决策逻辑并非围绕长期战略展开，而更偏向于展示力量本身。在这种思维框架下，军事行动不再是服务于战略目标的工具，而成为吸引注意力、塑造强硬形象的手段。战争的政治效果，甚至比战争的实际结果更为重要。这一判断并非脱离背景。近期，特朗普面临多重国内压力，包括公民权利争议、已故性罪犯爱泼斯坦档案风波，以及最高法院否决他关税政策的法律依据。
肯塔基州众议员马西（Thomas Massie）日前在社交平台X上发文指出，这场战争无法转移他推动司法部公开所有与爱泼斯坦相关文件的努力。
他说：“温和轰炸地球另一端的一个国家，也无法让爱泼斯坦档案消失，这就像道琼斯指数涨到五万点也不会让问题消失一样。”
奇维斯指出，特朗普在此情势下升级对外冲突有助于重置公共议程，将国内焦点转移至国家安全与对外威慑。文章进一步提醒，即便短期内军事打击削弱伊朗军力，也难以保证出现稳定的政治继任结构。冷战后多次干预的经验显示，削弱中央权威往往带来长期碎片化与权力真空，而非自动过渡至更可持续的秩序。军事力量擅长摧毁，却未必擅长重建。此外，冲突升级或冲击海湾航运与能源市场，推高油价；伊朗局势若失控，也可能引发新一轮人道危机与难民外溢效应。在美国同时应对亚洲战略竞争与欧洲安全承诺之际，另一场长期中东冲突势必分散资源，削弱整体威慑力。奇维斯总结指出，威慑不仅建立在实力之上，也依赖可预测性与联盟凝聚力。若外交政策由表演驱动，而非战略思考主导，即便初期看似强势，也可能在长远中削弱国家实力。历史已经表明，为展示而发动的战争，往往难以善终。

以色列拉马特甘地区大型医院将病房转入地下，应对伊朗危机加剧
位于特拉维夫附近的谢巴医疗中心（Sheba），在连续两天有伊朗导弹飞越区域的情况下，已把原本的地面停车场改造为地下医院综合体，容纳产科、急诊及专科病房等设施。医院表示，已将包括约2000张床位在内的所有病房转入地下。
产科负责人米哈尔·菲谢尔·巴塔尔称，医护人员在接到警报后迅速改造停车库，短时间内搭建了产房、三间引产室及剖宫产手术室等10间分娩和产房。谢巴副总裁约埃尔·哈尔·埃文指出，此举是出于对来袭导弹破坏力的担忧——这些导弹最多可携带约1000公斤炸药，任何地面建筑都难以承受。
报道称，2月28日美以对伊朗发动大规模攻击并造成伊朗最高领袖哈梅内伊死亡后，事态升级；伊朗随后发射报复性导弹，促使多家医院将病人转入地下避险。文中并回顾称，2025年6月曾有导弹击中南部索罗卡医院，造成多人受伤与严重损毁。

菲律宾政府正安排将因以色列空袭遭弹片击中身亡的菲籍看护玛丽·安·贝拉兹克斯·德维拉（Mary Anne Velazquez de Vera）遗体遣返。 
移民劳工部部长韩斯·里奥·卡克达克表示，部方与驻特拉维夫大使馆正与德维拉的丈夫（亦在以色列从事看护工作）协调遣返事宜。 
政府紧急救助基金将协助其丈夫生活开支，海外劳工福利管理局（OWWA）将提供葬礼與下葬補助。 
总统费迪南德·马科斯表示，德维拉来自潘加西南，事发时正在护送年长病人前往防空掩体，未能及时抵达便遭炸弹碎片击中身亡。 
阿布·卡比尔法医所通过生物识别记录确认其身份，丈夫已到场肯定认认遗骸。 
卡克达克称，以色列居民使用警报应用提示来袭导弹与最近避难所位置，但他们当时未能及时入避难所。

泰国工商会主席博·阿兰瓦塔南（ดร.พจน์）表示，今日（3月2日）与总理会商，评估中东冲突对泰国经济与企业的影响，并制定全面应对方案。政府已邀请安全、经济与民间部门共同研判，以减轻对民众与企业的冲击。 
报告指出，霍尔木兹海峡为全球海运石油约每日14–20百万桶、约占海运石油的20%，亦是供应亚洲液化天然气的重要航道。一旦航运受限或保险与安全风险上升，将立即收紧能源供给，推动油价快速上涨并加重通胀压力。中国、印度、日本、韩国等亚洲国家受影响显著；对能源净进口的泰国而言，油气价格波动将传导为生产成本、电价、运输费与民生开支上升。 
工商会与物流业评估的具体影响包括：一、油价不可避免上涨，推高全球海运、空运与陆运成本，增加整体物流费用；二、对波斯湾航线的“战争风险保费”会显著提高，初步估计运费每航次上升约50%；三、对中东的出口可能短期延缓或暂停，承运减少订舱或改航线将拉长运期并增加库存与资金压力；四、即便对欧洲的航线已有改道绕行好望角，若油价继续上涨，运费与运输时间仍将进一步上升，最少延长约一月，影响双程箱运与出口竞争力。 
若油价与运费同时上升，将对全球贸易与泰国出口竞争力形成双重压力。工商会建议企业尽快审查运输合约、保险条款与能源成本，做好多情景规划并维持充足流动性；同时把握农业、食品加工及日用消费品等可能增长的出口机会。 
政府经济部门正密切跟进并准备能源、金融與貿易等配套措施，工商会表示愿与政府紧密合作，深入评估行业影响并协助中小企业稳住营运。工商会呼吁企业与民众关注官方信息、谨慎经营，共同维护国家经济稳定。

特拉维夫一名菲律宾护理员回忆：2月28日早上8点，她和两名子女被以色列政府发来的短信警报惊醒——“导弹将至，请就近避难”。48岁的Irene Dela Cruz Mohtaseb说，这类短信成为他们在以美对伊朗发动先发制击时判断何时入避难所的重要信号。
她与家人住在一所设有地下避难所的学校旁，来自世界各地的外籍劳工在警报期间聚集在地下室躲避空袭。首日警报频繁，每15分钟一响，连上厕所也成了惊慌的折返：一坐下警报又响，孩子们既习惯又害怕。
Irene说丈夫是以色列人，习惯警报选择留在家中；第二天空袭较首日缓和。她表示自己作为护理员被视为必要岗位，仍坚持上班，照顾依赖她的高龄病人。尽管有报道称一名菲律宾护理员在最新空袭中被弹片击中身亡，Irene暂无回国计划，许多外籍人士也不打算撤离。
她向菲律宾的家人报平安：“我们这里还算安全，不用担心，上帝会保佑我们，我们非常想念你们，希望很快见到你们。”
