中国科技媒体“晚点LatePost”星期三（3月4日）获悉，中国连锁咖啡品牌瑞幸咖啡（Luckin Coffee）背后的大钲资本竞得蓝瓶咖啡（Blue Bottle Coffee），即将和雀巢完成交易。
一名接近瑞幸的人士称，大钲买下蓝瓶咖啡全球门店，雀巢将保留蓝瓶的咖啡机和胶囊业务。另一名接近大钲的人士称，目前交易已签字，尚未完成交割。
不过雀巢对上述媒体的应询不予置评，私募股权投资机构大钲则没有回复。
另据彭博社引述知情人士报道，大钲资本和瑞幸一直在评估包括蓝瓶、可口可乐公司旗下连锁咖啡品牌Costa Coffee，以及日本% Arabica咖啡品牌在中国境内的运营商等作为竞购目标。

G7下周召开财长会议 讨论中东局势经济影响
法国经济与财政部长莱斯屈尔说，法国计划下周召开七国集团（G7）财长会议，讨论中东危机，各国央行行长也将出席。
法新社报道，莱斯屈尔星期三（3月4日）告诉法国信息电台说：“我已经与多名同行进行沟通，特别是与美国财政部长贝森特。我们一致同意在下周初举行会议。”
他也说：“我们希望先观察一周，看看冲突如何发展，市场如何波动。届时各国财长和央行行长也将出席。”
莱斯屈尔说，预计下周的会议将是一次意见交流。“我们将认真听取来自基层、企业和世界各地经济学家的意见。我们的目的是讨论当前局势，以便评估在必要时可能需要采取的应对措施。”
他说：“目前这场冲突虽然局限于某个地区，但却具有全球性的影响，因此协调行动显然至关重要。”
美国和以色列上周六（2月28日）轰炸伊朗引发中东战争，导致股市大幅下跌，尤其是在欧洲和亚洲。
投资者也担心霍尔木兹海峡的石油和液化天然气运输中断，导致油气价格飙升。霍尔木兹海峡承担着全球约20%的海运石油和液化天然气运输。
法国目前担任七国集团轮值主席国，集团成员包括加拿大、德国、意大利、日本、英国和美国。
