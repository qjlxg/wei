王毅：东京审判开庭80周年再给日本扪心自省契机
中国外长王毅星期天（3月8日）表示，中日关系走向何方，取决于东京的选择。他指出，今年是东京审判开庭80周年，历史又给日本一次扪心自省的契机，希望广大日本人民能擦亮眼睛，不允许今天再有人自不量力，重蹈覆辙。
十四届中国全国人大四次会议于星期天（3月8日）上午举行外交主题记者会。王毅在回答关于中日关系的提问时，作出上述表示。
王毅表示，去年是中国抗日战争胜利80周年，在这个特殊年份，日本理应做的就是深刻反省过去走过的错误道路，包括侵略殖民台湾的劣迹。
王毅指出：“但是我们看到，日本的现职领导人竟然声称台湾有事就构成日本的所谓存亡危机事态，就可以据此来行使所谓的集体自卫权。众所周知，行使自卫权是以本国遭到武装攻击为前提的。”
他续问：“台湾事务是中国的内政，日本有什么资格来插手？中国的台湾地区出了事，日本有什么权利来行使自卫权？所谓集体自卫权，是否意味着要掏空规定放弃交战权的和平宪法？”
王毅表示，联想到当年日本军国主义曾以存亡危机事态为借口对外发动侵略，不能不使中国和亚洲各国人民高度警惕和忧虑日本要向何处去。
他指出，今年是又一个80周年，就是东京审判开庭80周年。80年前，11个国家的法官历经两年半的庭审，以海量的铁证昭示了日本军国主义的累累罪行。
王毅表示，东京审判审的是人类良知，判的是历史公正。
他说：“80年后的今天，历史又给了日本一次扪心自省的契机。历史为鉴，可事兴替。前事不忘，后事之师。希望广大的日本人民能够擦亮眼睛，不允许今天再有人自不量力，重蹈覆辙。已经发展壮大起来的中国和14亿中国人民，也绝不允许任何人再为殖民张目，为侵略翻案。”
