2016年3月3日，商务部长素帕吉·苏塔潘表示，受中东局势影响，国际能源与航运市场出现波动，已对泰国出口产生潜在冲击。她通报，3月2日13:30在政府大楼召开评估会，由总理阿努廷·占威鲁主持。会议认为，尽管泰国与冲突方的直接贸易比重有限，但地区性扩散对全球经济、能源与海运稳定具有重要影响，需密切关注。 
素帕吉指出，泰国与中东直接贸易并不高——2015年对该地区出口额约1,247.56亿美元，占出口总额约3.67%，尚未出现大规模取消或推迟订单的明显迹象。但海运路线受限和安全风险已导致改航、航程延长，致使运费、保险费和物流成本上升，同时部分航线出现集装箱与航班紧张，影响出口商。国际能源价格波动亦可能传导至生产与运输成本，形成间接冲击。 
为应对，商务部提出六项主动措施：
(1) 强化商品价格监管，防止囤积与不当涨价，监测能源成本向品价的传导；
(2) 协调进口商检视库存，分散采购来源并鼓励使用国内原料；
(3) 支持出口商并优化物流管理，评估运费与保费上升影响，提供成本管理与交付条款咨询；
(4) 与船公司及物流服务商紧密联动，跟踪航线、港口拥堵与供应链连续性；
(5) 发挥驻外商务参赞作用，及时报告贸易状况与进口商信心并为企业提供风险管理建议；
(6) 分析能源与运费上涨对通胀及价格稳定的影响，提出针对性政策建议。 
她表示，政府将整合相关单位与民间力量，落实价格、成本与出口支持措施，维持泰国经贸与价格稳定，应对外部地缘政治不确定性。

2016年3月3日，商务部长素帕吉·苏塔潘表示，受中东局势影响，国际能源与航运市场出现波动，已对泰国出口产生潜在冲击。她通报，3月2日13:30在政府大楼召开评估会，由总理阿努廷·占威鲁主持。会议认为，尽管泰国与冲突方的直接贸易比重有限，但地区性扩散对全球经济、能源与海运稳定具有重要影响，需密切关注。 
素帕吉指出，泰国与中东直接贸易并不高——2015年对该地区出口额约1,247.56亿美元，占出口总额约3.67%，尚未出现大规模取消或推迟订单的明显迹象。但海运路线受限和安全风险已导致改航、航程延长，致使运费、保险费和物流成本上升，同时部分航线出现集装箱与航班紧张，影响出口商。国际能源价格波动亦可能传导至生产与运输成本，形成间接冲击。 
为应对，商务部提出六项主动措施：
(1) 强化商品价格监管，防止囤积与不当涨价，监测能源成本向品价的传导；
(2) 协调进口商检视库存，分散采购来源并鼓励使用国内原料；
(3) 支持出口商并优化物流管理，评估运费与保费上升影响，提供成本管理与交付条款咨询；
(4) 与船公司及物流服务商紧密联动，跟踪航线、港口拥堵与供应链连续性；
(5) 发挥驻外商务参赞作用，及时报告贸易状况与进口商信心并为企业提供风险管理建议；
(6) 分析能源与运费上涨对通胀及价格稳定的影响，提出针对性政策建议。 
她表示，政府将整合相关单位与民间力量，落实价格、成本与出口支持措施，维持泰国经贸与价格稳定，应对外部地缘政治不确定性。
