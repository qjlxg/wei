 对外投资领域的第一部专门行政法规，以后个人和企业出去投项目监管更严了，尤其是涉及国家安全、技术转移、敏感行业的，备案和审查会更细。

 这个比较猛，重点是铸牢中华民族共同体意识，对境外破坏民族团结的行为有了明确法律追责。影响最大的地方在于民族地区和涉及民族事务的工作，以后相关宣传、教育和跨境管理会更严。
3. 《中华人民共和国社会救助法》
 以前低保边缘家庭、支出突然困难的家庭救济不太明确，现在把救助对象扩大了，还建了分层分类的体系。
4. 《超龄劳动者基本权益保障暂行规定》
 用人单位必须按时足额发工资、少安排加班、岗位强度要合适，还得给上工伤保险。影响最大的是现在很多返聘老人和灵活就业的群体
😂
5. 新修订的《中华人民共和国民用航空法》
 加强了航空安全，尤其是低空经济和无人机管理，要求适航许可、产品识别码什么的更严格。影响最大的是民航行业和新兴低空飞行领域，以后安全标准更高，产业的自由发展必须在严格的管理下自由。
6. 离境退税措施优化
 “即买即退”时间延长到28天，还搞无纸化、小额抽检，退税商店覆盖面扩大。影响最大的是来中国旅游购物的人，尤其是境外游客，消费更方便。
7. 电动汽车安全新国标
 电动车新增“一键断电”、刮底测试等安全要求。

Anthropic封堵中国用户 中国开发者质疑被“暗中监视”
美国人工智能企业Anthropic据报正采取行动，防止中国开发者绕过限制使用AI模型Claude。而中国科技圈则指控，该企业识别用户电脑时区等线索，是在对中国用户进行“暗中监视”。
过去两周，多个中国自媒体称使用的Claude账号被封，怀疑Anthropic正在加紧对中国用户的限制。
据《南方都市报》，有技术人员在Reddit发帖爆料，指控Anthropic在4月推出的AI编程工具Claude Code的新版本客户端里，暗藏了一段针对中国用户的“监视代码”，能检测用户是否身处中国境内、使用的代理是否属于中国域名，以及是否与某个中国人工智能实验室存在关联等。
本周三（7月1日），负责Claude Code的Anthropic工程师萨里克·希希帕尔（Thariq Shihipar）在个人的X账号对此事作出回应。他解释，“这项检测机制是今年3月启动的一项实验，旨在防止未经授权的转售商滥用账户以及可能存在的模型蒸馏行为。”另外，目前团队已找到更好的应对方案，在次日发布的新版本中将删除这串特殊代码。
Anthropic并不开放Claude给中国用户，但中国用户，尤其是技术开发者，一直通过代理等各种方式绕开限制使用Claude。英国《金融时报》星期五（7月3日）报道了Anthropic的行动，并引述知情人士说，包括蚂蚁集团等在内的中国公司，曾通过云服务供应商和海外子公司等变通方式，使用 Claude Code。
中国自媒体“财经故事荟”称，Anthropic的做法是对中国用户“打标“，是“AI时代的新种族歧视”。
