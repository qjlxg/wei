东京高温天气逼近东南亚 日企酷暑调整工作模式
（东京讯）日本东京夏季的气温和湿度已接近东南亚城市的水平，随着中暑病例持续上升，日本企业开始重新调整工作模式，以应对高温天气。
日本夏季正变得越来越炎热。《日经亚洲》根据美国国家海洋和大气管理局（NOAA）的气象资料进行分析后发现，东京7月和8月的最高气温和湿度，已接近曼谷和新加坡的水平，全国每年因高温损失的工作时间超过28亿小时。
在大阪堺市的一处医院建筑工地，约100名工人会在工作间隙到休息室，利用移动式空调和刨冰来消暑降温。
负责这项工程的建筑公司鸿池组（Konoike）去年因高温而中暑的员工人数较前一年增加一倍，公司因此决定在酷暑期间减少工作时间，允许员工每月可连续休假九天，并在气温较低的月份增加工时，以维持全年总工时。
公司还推出多项防暑措施，例如配备内置风扇的降温工作服。
日本已在2025年修订《劳动安全卫生条例》，要求雇主必须为高温环境工作的员工采取预防措施，并建立中暑紧急应变机制。不过，那年因工作导致的中暑和死亡人数仍超过1800起，连续第二年创下历史新高。
日本产业医科大学校长堀江正知说，如果政府数据也包括请假三天或以下的轻微中暑个案，中暑人数肯定要多出一位数。
他认为，日本必须改变工作方式，例如在高温期间延长企业之间的交货期限，以减少员工暴露于酷热环境的时间。
日生基础研究所（NLI Research Institute）首席分析师篠原拓也则强调，把防暑对策“从降温转向缩短人体暴露在高温下的时间”至关重要。
他说，企业有必要采取更多措施，例如把工作时间调整至清晨和傍晚、根据季节重新安排全年工时，以及进一步推广远程办公等。
