迹象显示，北京方面希望进一步扩大人民币的国际使用。中国人民银行上月加大对跨境融资的支持力度，一些中国知名经济学家也主张在当前历史机遇下适度放松资本管制，以增强人民币的全球吸引力。自去年年底以来，华尔街分析师也纷纷预测人民币将进一步走强。
人民币融资需求增长的一个关键原因是中国利率较低，国内持续面临通缩压力。

共和党成功保驾护航 特朗普对伊军事行动获国会支持
美国众议院星期四（3月5日）投票否决了一项旨在阻止总统特朗普对伊朗发动空袭的决议案。这项表态意味着美国国会参众两院都支持共和党政府继续当前的军事行动。 
据路透社报道，众议院是以219票对212票，否决了“战争权力决议案”（War Powers Resolution）。投票结果明显按照党派路线，由特朗普领导的共和党凭借微弱多数优势保住特朗普的军事行动。值得一提的是，有两名共和党议员倒戈支持提案，四名民主党议员则投反对票。
支持提案的民主党人批评特朗普政府在未获国会授权、未设定清晰目标及终战计划的情况下，发动了一场“选择性的战争”。
纽约州民主党籍众议员米克斯（Gregory Meeks）直言，政府至今未能向国民解释这些军事行动如何保障美国人的安全。“这是一场由本届政府未经授权发动的选择性战争，既没有明确的目标或终极目标，也没有解释他们打算如何保障美国人的安全。”
共和党人则反击指责民主党人将此事付诸表决，只因为他们反对特朗普，这让美国人面临更大的风险。众议院情报委员会主席、亚利桑那州共和党议员克劳福德（Rick Crawford）说：“我们都知道，如果总统的名字不是唐纳德·特朗普，我们今天根本不会坐在这里讨论这个话题。”
在表决前，众议院两党议员曾高票通过另一项动议，重申“伊朗仍是全球最大的恐怖主义支持国”。
尽管此次投票结果对特朗普有利，但根据美国1973年《战争权力法》，总统在未获国会正式宣战或授权的情况下，只能在面临紧急袭击时动用武力。特朗普政府辩称伊朗构成“紧迫威胁”，因此所发动的军事行动是合法的。
《战争权力法》也明确规定，未经授权的军事行动必须在60天内终止。这意味着特朗普政府最迟须在4月底前向国会寻求正式授权。
自上周六（2月28日）美伊开战以来，冲突已导致超过1000人丧生，其中包括至少六名美军士兵，整个中东局势陷入剧烈动荡。参议院已于星期三（4日）投票表决支持特朗普的军事行动。
