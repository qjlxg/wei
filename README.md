天文学家首次在星际空间发现糖
天文学家首次在星际空间发现一种单糖，显示构成生命的重要有机分子，或早在恒星和行星形成前，就已能在太空中自然产生。
美国有线电视新闻网（CNN）报道，西班牙天体生物学中心领衔的研究团队，在银河系中心附近一团名为“G+0.693−0.027”的气体和尘埃云中，探测到赤藓酮糖。
赤藓酮糖（erythrulose）是一种含有四个碳原子的单糖，少量存在于覆盆子等水果中，也常用来制作美黑产品。
糖类在生命系统中扮演重要角色，不仅能提供能量和构成生物结构，也是核糖核酸（RNA）和脱氧核糖核酸（DNA）等遗传物质的组成部分。科学家此前已在陨石和小行星样本中发现核糖、葡萄糖等糖类，但从未在星际介质中直接探测到糖。
研究人员用西班牙两座大型望远镜，捕捉这团太空气体和尘埃发出的无线电信号。不同分子会留下不同的信号，就像人的指纹一样。研究团队发现，其中一组信号与实验室里的赤藓酮糖完全吻合，因此确认这种糖存在于星际空间。
研究团队原本想寻找的是结构更简单、含有三个碳原子的糖，但并没有发现；相反，他们发现了结构更复杂、含有四个碳原子的赤藓酮糖。
这让研究人员感到意外，因为过去天体化学界普遍认为，星际分子就像“搭积木”一样，会一个碳原子一个碳原子地增加，由简单逐步变复杂。
按照这种理解，三碳糖应该先形成，或者至少比四碳糖更容易被探测到；但这次观测结果正好相反。
这说明，星际糖类的形成可能并不只是靠逐步增加碳原子，还可能存在其他化学途径。研究人员认为，赤藓酮糖可能是由一些更简单的分子，在寒冷的星际尘埃表面结合形成的。
研究人员还说，这项发现证明，较复杂的糖可以在恒星和行星诞生前形成，并可能随着尘埃、彗星或小行星进入新形成的行星系统。地球早期所需的一部分有机物，因此可能并非全部在地球上产生，而是由太空中的物质带来。
不过，发现赤藓酮糖并不等于找到外星生命，也无法解释生命如何从这些分子中诞生。
这项研究的重要性在于，它扩大科学家对太空化学复杂程度的认识，也提高未来在星际空间发现核糖等与生命起源更直接相关分子的可能。
研究成果已刊登在《自然·天文学》期刊。

台湾总统、民进党主席赖清德星期天（7月19日）在民进党全代会上称，反对中国大陆“红色恐怖”对台湾社会的威胁，强调绝不让“民主台湾”走回头路变成“中国台湾”。同一天，他率领19名县市长参选人同台造势，吹响2026年底“九合一”地方选举的选战号角。
综合台湾《自由时报》《联合报》报道，民进党星期天召开第22届全代会。本届开幕式首度邀请多国驻台使节出席，包括圣基茨和尼维斯、海地、图瓦卢及马绍尔群岛等国驻台代表均受邀。
赖清德在致辞时说，中国大陆本月实施的《民族团结进步促进法》，显示大陆对台法律战已从《反分裂国家法》的“反对独立”，走向披着民族团结外衣的“强制统一”。
他批评，这部法律不仅企图将台湾纳入民族工作范畴，迫使民众自我审查，更滥施跨境镇压，将威权触角伸向世界。赖清德承诺，政府将建立完善的预防与反制机制以保护全台民众安全。
他也呼吁党内团结一致，共同抵御“红色恐怖”渗透，齐心守护民主自由的生活方式。
在地方选举布局上，赖清德强调台湾正处于关键时刻，并提出备战年底“九合一”选举的“三大原则”，包括新世代扛起新时代、推出共同政见打造台湾好生活和爱民如亲。
赖清德说，民进党提名具备“台湾优先”理念与专业治理能力的新生代人才，结合优秀的各级民代，共组新世代领航团队。
他续称，民进党年底选举的共同政见“台湾好生活”，希望通过“加薪、减税、增福利、少负担”政策，逐步迈向“福利国家”愿景，照顾全年龄层与弱势群体。
赖清德还呼吁参选人以温暖亲切的行动赢得人民信赖，让民众确信选择民进党就是选择稳健改革。
针对近期延烧的毒油风波，台湾行政院长卓荣泰在会上预告，行政团队将于星期一（20日）向民众详细报告《食安法》修法方向。修法将明确厘清中央与地方的权限，以及未来通报机关的即时责任与义务。
卓荣泰也承诺，市面查获的所有问题油品批号，将在20日完成全面检验并公布报告。
全代会会议结束后，民进党将进行中执委、中评委及中常委改选，重组党中央最高决策核心，牵动各派系在2028年总统、立委提名布局。

俄军对乌克兰发动规模最大的单次弹道导弹袭击
俄罗斯星期天（7月19日）对乌克兰首都基辅发动空袭，造成一人死亡、13人受伤。此前一天，乌克兰发动无人机袭击了俄罗斯境内的电商仓库。
法新社报道，基辅市长克利奇科和地方当局称，索洛拇扬卡区的一栋公寓楼遭到袭击，一家超市起火，斯维亚托申区的一栋房屋也燃起了大火。
据报道，第聂伯区的一家购物娱乐中心、舍甫琴科区的一栋公寓楼和一座非住宅建筑也遭到袭击。
乌克兰中部第聂伯罗彼得罗夫斯克州的官员甘扎在Telegram上说，俄罗斯无人机袭击造成一人死亡。
乌克兰代理外长西比哈星期天说，这是自俄乌战争全面爆发以来，俄罗斯发动的规模最大的一次弹道导弹袭击。
他在社媒X平台上说：“俄罗斯向乌克兰首都发动了残酷的恐怖袭击，发射约48枚弹道导弹，这是自战争爆发以来，单次袭击中弹道导弹数量最多的一次，造成多人死伤。”
西比哈呼吁国际社会作出“恰当且强有力”的回应。“我们需要对莫斯科施加毁灭性的压力，以结束这场恐怖行动。”
此前，乌克兰星期六（18日）派出无人机摧毁了莫斯科和坦波夫地区的电商仓库，造成八人死亡并引发重大火灾。

韩驻华科创机构建议韩企多赴华参加科技展会
韩国科学技术信息通信部旗下全球创新中心（KIC中国）建议韩企多赴华参加科技展会，实地了解中国技术和产业发展的最新动向。
据韩联社报道，KIC中国代表金钟文星期六（7月18日）在接受采访时表示，中国科创能力快速提升，市场竞争日趋激烈，韩国企业应主动参加世界人工智能大会等中国主要科技展会。
第八届世界人工智能大会于17日至20日在上海举行，展览面积超过10万平方米，吸引1100余家企业参展，展出3000余个产品。KIC中国组织八家韩国中小企业参展，这是韩国AI企业首次组团亮相世界人工智能大会。
金钟文指出，韩国企业亟须对标全球及中国市场，尽快摸清自身技术所处水平。参加此类展会是能对标优劣、盘点韩方优势技术与拳头产品的重要途径。
他说，韩国企业如果脱离中国产供链体系，很难维系市场竞争力。企业需要低成本快速打样、迭代测试产品，仅凭韩国现有的研发和生产速度，恐难适应当前市场竞争节奏。他强调，中国技术实力快速提升，相关企业数量众多，韩国业界必须正视中国市场的激烈竞争环境。

纽约暴雨洪灾接踵而至 加拿大野火蔓延多地空气质量恶化
（纽约综合电）在经历数日的浓烟与酷热天气后，纽约市及周边地区星期六遭遇了雷暴、倾盆大雨和突发洪水的袭击。
据美国国家气象局数据，截至星期六（7月18日）午后，降雨量已达2至4英寸（约5到10公分），且预计降雨强度最高可达每小时1英寸。
气象局已针对纽约市全部五个行政区、威斯特彻斯特县南部以及新泽西州东北部发布了洪水预警，预警将持续至星期天（19日）凌晨2时（新加坡时间下午2时）。
气象局纽约办事处警告，部分地区可能出现严重洪涝，导致交通中断、地下室进水以及需要进行水上救援的情况。
暴雨也导致拉瓜迪亚、肯尼迪及纽瓦克机场航班延误或取消。新泽西州多个县发布山洪警报，纽瓦克有五辆汽车被困积水中，司机和乘客获救，暂未传出伤亡。
加拿大方面，政府星期六说，全国目前有955处活跃野火，其中近200处位于安大略省，过火总面积约2万8500平方公里。
美国东北部多个城市持续遭受加拿大林火引发的浓烟侵袭。多伦多、纽约和华盛顿特区星期六的空气质量降至全球最差水平之列，促使当地政府发布健康警告，呼吁民众减少户外活动。
针对美国总统特朗普指加拿大林火烟霾侵美，扬言要加征关税追责，安大略省省长福特星期六回应称，特朗普的批评是“不可接受”。
他说，省内约65万5000公顷森林正在燃烧。如果特朗普希望加拿大清理森林，就应取消美国对软木木材征收的关税，“这将解决我们目前面临的许多问题”。

一只猫咪在委内瑞拉地震废墟中获救 带来“希望”
一只虚弱的猫星期六（7月18日）在委内瑞拉大地震后的居民楼废墟中获救。一名救援人员称，这带来了“一线希望”。
法新社报道，委内瑞拉于6月24日遭遇两次地震，造成超过5100人死亡，其中沿海的拉瓜伊拉州受灾最为严重。
志愿救援人员卡瓦哈尔说，他在拉瓜伊拉一处居民楼废墟中进行搜救时发现了这只猫。
“我们看到了这只猫，它受惊后又钻回了楼里，钻得更深了。”目前尚不清楚小猫是什么时候被困住的。
“我进去后，脱下手套，把一些猫粮放在胳膊上……小猫慢慢地靠近，当然，它带着一丝绝望吃了起来。”
卡瓦哈尔在头盔上写了“el gato”（猫），他说这是他从小学起就有的绰号。他和几名委内瑞拉中央大学的学生成立了一个协会，协助开展救援行动。
获救后，这只猫被送到一个小型营地，那里有兽医随时待命。小猫接受输液并被清洗干净后，被送往收容所。
卡瓦哈尔说：“我们找到它真是太高兴了。”
“面对任何生命，我们都无法不心生怜悯。找到这只小猫无疑是一线光明、一线希望。”

DeepSeek V4满血版据报最快7月20日推出
中国人工智能（AI）初创公司深度求索（DeepSeek）据报最快将于星期一（7月20日）发布新模型V4的“满血版”（正式版）。新版本在性能上直追OpenAI的最新模型GPT-5.6，还将引入峰谷计费机制。
综合《每日经济新闻》和中国科技媒体“36氪”报道，多名开发者透露，DeepSeek V4的GA（全面开放）版本已开启小范围灰度测试，最快可能于20日正式亮相。目前该模型分为V4 Flash与V4 Pro两个版本。
测试者反馈显示，新版本在编程设计和智能体（Agent）生成能力上有显著提升。部分开发者认为整体表现已能与Anthropic的Claude 5抗衡，但也有观点指出，Pro版本的性能领先优势相比Flash版并不算悬殊。
“36氪”引述一名开发者Pankaj Kumar的评价称，V4“满血版”的整体表现接近Anthropic的高端模型Opus 4.8，编码能力直追OpenAI的GPT-5.6 Sol。虽然V4智能体能力大幅增强，但在处理同一任务时，V4所需的迭代运算轮数仍多于Anthropic的Fable 5。
此外，也有测试者认为，V4在绝对性能上或许略逊于近期刚发布的月之暗面Kimi K3，但V4的核心优势在于极高的性价比。
报道指出，此次升级真正引发关注的可能不仅是性能，而是定价策略。根据DeepSeek此前向API（应用程序接口）用户发送的邮件，V4正式版上线后将首次采用峰谷计费模式。
根据这一定价模式，DeepSeek V4 Pro每百万输出Token（词元）的平时收费为0.87美元（约1.12新元），高峰期则翻倍至1.74美元；缓存未命中时的输入收费为0.435美元。轻量级的V4 Flash每百万输出Token平时收费仅0.28美元，高峰期为0.56美元，缓存命中时的输入费用更是低至0.0028美元。
高峰时段为每天上午9时至中午12时，以及下午2时至6时。
分析指出，面对海外头部大模型动辄每百万Token几十美元的定价，DeepSeek V4即便引入了峰谷计价机制，整体价格依然维持在极低水平。
