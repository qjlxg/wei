与墙内通常能够快速对普通网民以“寻衅滋事”或“散布谣言”治罪相反，这种利用真实信息差的“听床”行为让现有法律和执法程序陷入了双重盲区。文章引用警方办案人员的话称：
“网民在微信公众号发布‘官员简历’不属于编造谣言或者泄露国家秘密行为，公安机关暂无法律依据对相关网民开展立案侦查或予以行政处罚。此外，对夸大落马官员贪腐情节、编撰绯闻轶事等行为，公安机关同样无权率先辟谣。‘警方不掌握相关落马官员的具体犯罪事实，因此无法认定网民发布的信息为谣言。’”
官方目前对此类现象持严厉打压态度。文章将此类“听床”信息定性为“国家重要的政治信息”，并称这种炒作“破坏了相关工作的严肃性”。

会议也要求各级公安机关，专项重点打击盗掘盗窃全国重点和省级文物保护单位、盗窃国有馆藏文物等犯罪，集中打击以“探宝”为名盗挖地下文物、依托网络平台非法贩卖文物、以拍卖等合法渠道为掩护倒卖文物、走私文物、在建设施工过程中故意损毁文物等犯罪。
会议也提出，要快破现案，深挖扩线，全力追缴被盗文物，彻底摧毁犯罪链条。要全面梳理盗掘古文化遗址古墓葬、盗窃馆藏文物、盗窃石窟寺石刻、倒卖文物等积案，开展破案攻坚。
中国公安部也公布，自5月15日开展“云鸮”行动以来，公安机关与文物部门共立案侦办盗掘、盗窃、倒卖、走私、损毁等各类文物犯罪案件170余起，抓获犯罪嫌疑人390余名，追缴涉案文物2700余件，其中珍贵文物230余件。
早前发生的南京博物院文物流失事件掀起风波后，中国今年5月发布《博物馆藏品管理办法》意见稿，对已沿用40年的规章首次作出重大调整，对已沿用40年的规章首次作出重大调整，新增博物馆法定代表人离任必须点交藏品、不允许征集来源不明藏品等规定。
此外，有网民发文称，收藏家马未都创办的观复博物馆（北京）展览的一尊佛像，与海口市五公祠2000年丢失的一尊宋末元初铸铜佛像相似。观复博物馆曾回应称，已上报，并将服从主管部门鉴定结果。

朝鲜外长访问莫斯科 与俄外长会谈
朝鲜外长崔善姬定星期六（7月18日）抵达莫斯科，与俄罗斯外长拉夫罗夫举行会谈。
法新社报道，俄罗斯外交部长办公室宣布，应拉夫罗夫的邀请，崔善姬将于7月18日抵达俄罗斯联邦进行正式访问。
自2022年俄罗斯入侵乌克兰以来，平壤和莫斯科的关系日益密切。两国在2024年俄罗斯总统普京访问平壤期间签署了一项防务协议。
朝鲜已向俄罗斯提供导弹、弹药和派遣数千名士兵协助它在乌克兰的军事行动，分析人士称，作为回报，莫斯科正在向这个外交上孤立的国家提供财政援助、军事技术、食品和能源。
今年4月，朝鲜领导人金正恩承诺帮助俄罗斯赢得其“神圣”战争。崔善姬支持俄罗斯对乌克兰的战争。
俄罗斯国防部长别洛乌索夫等多名高级官员于4月访问平壤，就加强军事关系举行会谈。别洛乌索夫说，莫斯科准备签署一项2027年至2031年的合作计划。
