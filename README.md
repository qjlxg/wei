巴黎/罗马讯 — 法国两名官员3月13日表示，法国正推进组建多国护航队的外交准备，拟在地区安全形势缓和后保护霍尔木兹海峡通航安全。此消息此前因有报道称巴黎正就本国船只的安全通行进行谈判而被关注。
随着美以与伊朗冲突升级、伊朗对以色列、美军基地及海湾国家发动打击，航道受影响、油价上扬，欧洲国家正在寻求维护自身利益的方案。法方过去一周与欧洲、亚洲及海湾阿拉伯国家磋商，酝酿由军舰为油轮护航的计划，但一位官员指出，目前条件尚不允许部署部队，首先要做的是外交对接，随后才能进入细节并可能走向实操。
有媒体称法意正在谈判保障两国船只通行。法国总统府未置评，意大利外交部则否认有针对性“私下交易”，强调意方倾向总体降级而非牺牲部分商船换取个别通行保障。
欧盟在该地区主要行动是2024年启动的“阿斯皮德”红海护航任务，法国已向该行动贡献军舰并计划增至两艘。法国总统马克龙表示，法国将在该地区部署八艘军舰、航母打击群及两艘直升机航母，必要时可扩展到霍尔木兹海峡以支援商船。

意大利政府据报将与耐倍力及投资者商讨中国控制权争议
路透社星期五（3月13日）引述两名知情人士透露，意大利政府据报将于下星期一（16日）与耐倍力和两大主要投资者的代表会面，商讨解决围绕中国股东中化集团对公司影响力引发的争议。
据报道，意大利轮胎制造商倍耐力（Pirelli）两大股东中国国有企业中化集团和Camfin分别持有34%和25.3%股权，是这场公司控制权长期争议的核心单位。
Camfin以及倍耐力方面均呼吁，限制中化集团的持股地位，并称如若不然将使倍耐力在美国这一高端轮胎业务关键市场的扩张计划复杂化。
意大利政府使用黄金权利保护属于国家利益的资产，正考虑解决相关争议的选项，包括将中化集团的股权转移至保密信托，冻结集团的投票权。
其中一名知情者星期五称，中化集团倾向一个临时方案，即允许集团长期持有耐倍力的股权，但治理权利受限。
上述两名知情者称，意大利政府预计将于4月中就耐倍力的治理权作出裁决。
另两名知情人士对路透社说，中化集团也考虑发行可转换债券，将暂时中断集团对耐倍力的持股。
意大利通讯社Radiocor最先报道，意大利政府下星期一与耐倍力和两大主要投资者的代表会面的消息。意大利总理梅洛尼办公室官员将参与会面。
中化集团今年1月提交一份提案，以解决耐倍力的治理问题，以及美国潜在的监管担忧。
彭博社去年9月引述知情者报道，美国商务部工业和安全局7月中旬致函意大利官员，称意大利在2023年为了限制中化影响力而对倍耐力动用的否决权，不足以保护公司免受美国限制。
据彭博社去年4月报道，倍耐力董事会在接到监管要求后，作出了将中化治理权降级的决定。

伊朗石油出口枢纽遭袭 加剧供应中断风险
承载伊朗大量石油出口的岛屿首次遭到大规模袭击，不仅威胁到伊朗的命脉，也让在持续两周的中东冲突中，早已苦不堪言的全球供应市场陷入更深担忧当中。
美国总统特朗普在社交媒体上发文称，美国轰炸了波斯湾哈尔克岛（Kharg Island）上的军事目标，但暂时放过石油基础设施。他还警告伊朗领导人，如果干扰通过霍尔木兹海峡（Strait of Hormuz）的船只，他将立即重新考虑这一决定。
伊朗严重依赖哈尔克岛，全国十分之九的原油从这里出口，大部分输往中国。如果冲突升级，影响到这些原油的流通，将进一步扰乱石油市场。
彭博社引述新美国安全中心的高级研究员瑞秋·齐姆巴（Rachel Ziemba）说：“这很可能会削弱伊朗的出口能力，特朗普试图通过升级局势来缓和局面，伊朗是否会进行报复，是石油市场和战争面临的最大风险。”
法新社报道，根据与革命卫队有关的实体向当地媒体发表的评论，伊朗威胁如果该国能源设施遭到攻击，将打击中东与美国有关的石油目标。
摩根大通在3月8日的报告中指出，如果哈尔克岛被迫关闭，将会迅速引发上游减产，伊朗多达一半的原油产量将面临风险。报告称，伊朗原本被视为将是波斯湾产油国中最晚停产的国家之一，但现在可能会早于科威特和阿联酋。
“哈尔克岛面临美国如此直接的军事攻击威胁，货轮可能会犹豫该不该装船，”华盛顿咨询机构华盛顿能源市场和政策咨询公司Rapidan能源集团的总裁鲍勃·麦克纳利（Bob McNally）说。“如果伊朗仍在干扰霍尔木兹海峡的航运，可能会更加犹豫不决。”
