越南拟将燃油进口税降至0%以应对中东战事引发的能源危机
越南政府拟以紧急措施暂免各类燃油进口税，缓解由美以与伊朗冲突对全球能源供应造成的冲击。财政部表示，正审议一项草案，将部分燃油进口税降至0%，以稳定国内市场并维护能源安全。
官员称，若冲突持续且霍尔木兹海峡（全球约五分之一石油运输经此）被封锁，全球供应将进一步紧张并推动价格上扬。战事爆发一周以来，越南国内燃油价格已大幅飙升，政府因此启动价格应急议程。
官方媒体报道，常用汽油上涨约21%，至每升27,040越南盾；柴油上涨逾50%。目前越南对无铅汽油征收10%进口税，对柴油、航空燃油和煤油征7%，草案若获批，相关税率将暂时取消至4月底。
尽管尚未出现大范围断供，数十家小型加油站已因库存减少而临时停业或缩短营业时间。民众生活成本显著上升，胡志明市一名市民表示油价飞涨迫使其考虑步行上班。
受中东局势影响，国际原油接近每桶120美元，为2022年俄乌冲突以来高位；七国集团（G7）财长将于今日召开紧急会谈，讨论共同释放战略储备以应对油价上涨。来源：AFP
