今天是三八妇女节，在传统以男性主导的基金行业，中国女性公募和私募基金群体再受关注。财联社引述Wind数据报道，截至3月6日，中国公募行业共有在任女性基金经理1110位，较去年同期增加22位，占全部基金经理比重约26.75%。
规模方面，按含共管口径计算，女性基金经理参与管理的基金规模达18.95万亿元，占全行业在管规模的51.06%，女性基金经理单独管理的基金规模达到6.83万亿。
报道称，女性基金经理参与管理的基金中，近一年实现正收益的共有3721只，占比约83.35%，其中129只收益率超过50%，20只实现翻倍。
私募基金方面，私募排排网统计显示，截至2月底，女性私募从业人员已达1787位，其中有1045位是基金经理。担任实控人的有241位，其中女性基金经理共有48位，占全部女性基金经理的4.59%。
报道称，整体来看，女性基金经理已在权益与固收两大主线中形成较为完整的人才梯队。
