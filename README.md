世界银行星期二（7月7日）在官网发布中国经济简报。报告指出，中国第一季度GDP同比增长5.0%，但房地产行业继续适应住房需求下降，消费者支出因房价下跌带来的负财富效应和劳动力市场疲软而保持谨慎。
报告预计，中国经济增长将在2026年放缓至 4.4%，原因是国内需求持续疲弱；消费预计仍将低迷，私人投资则受到房地产行业持续调整和部分行业企业盈利能力较低的制约，但公共基础设施支出和高科技行业持续强劲将部分抵消这些逆风。
报告说，中国经济前景面临的风险大体均衡。若房地产下行超出预期，可能加大家庭资产损失，并收紧房地产开发商融资条件，从而进一步加剧消费支出以及房地产和相关行业投资所面临的压力。

官宣被查八个月后 中国官方通报方红卫被双开
官宣被查整整八个月后，中国官方通报中共西安市委原书记方红卫严重违纪违法，被开除党籍和公职。
中央纪委国家监委网站星期二（7月7日）公布，经中共中央批准，中央纪委国家监委对第二十届中央候补委员，陕西省委原常委、西安市委原书记方红卫严重违纪违法问题进行了立案审查调查。
通报指方红卫丧失理想信念，忘却初心使命，对组织不忠诚不老实，履行全面从严治党主体责任不力，干扰巡视工作，对抗组织审查，搞迷信活动；违规接受宴请、旅游安排；不按规定报告个人有关事项，违规选拔任用干部。
方红卫也被指违规收受礼金，违规拥有非上市公司股份，利用职权和职务影响为亲属谋取利益，纵容、默许配偶利用本人职务影响谋取私利；家风不正，对家人失管失教；利用职务便利为他人在工程项目承揽、干部选拔任用等方面谋利，并非法收受巨额财物。
公开资料显示，方红卫今年60岁，1989年从清华大学毕业后，他在陕西汽车制造总厂、陕汽集团任职长达26年。2015年，方红卫出任宝鸡市委常委、副市长，其后多次履新，担任过渭南副市长、汉中市长、汉中市委书记。
2021年11月，方红卫任陕西省委常委、西安市委书记。中央纪委国家监委网站去年11月7日通报，方红卫被查。
香港《明报》曾引述西安消息人士报道，在官宣被查前两天，方红卫当天原定调研西安市宗教工作，并拟对行程作公开报道，以就外界指他“神隐”出事辟谣。不过，方红卫在半路就被中共中央纪委人员拦下带走。
报道也引述一名在当地经商的外省企业家称，方红卫不仅本人涉贪，还纵容、默许亲属利用他的影响力谋取私利。 “当地商界都知道，方红卫妻子、妹妹‘胃口很大’。”
去年10月举行的中共二十届四中全会，方红卫是其中一名在递补中央委员时被“跳过”的候补中委。

研究：老龄少子化 经济仍可蓬勃发展
全球生育率下降引发许多人对经济影响的担忧，人们预测经济增长将放缓，创新能力也将下降。但一项最新研究发现，事实情况可能正好相反。
彭博社报道，根据2024年诺贝尔经济学奖得主达龙·阿西莫格鲁（Daron Acemoglu）、大卫·奥特（David Autor）、基兰·伯恩（Keelan Beirne）、安德鲁·斯科特（Andrew Scott）的一项新研究，历史经验表明人口老龄化和萎缩实际上提高了经济体的劳动力人均产出，而且对国内生产总值（GDP）总量没有产生抑制作用。
研究发现，工人和企业会转向技术来弥补劳动力减少的不足，从而提高每一名工人的生产力。
四名作者在美国国家经济研究局（NBER）发布的一篇论文中写道：“我们的研究结果挑战了目前普遍存在的悲观论调，出生率下降以及由此导致的人口老龄化和萎缩，实际上提高了而非降低了人均GDP。”
他们说，这一增长足以完全抵消人口下降的负面影响，使GDP总量基本不受影响。
过去70年间，各大洲的出生率均有所下降。这篇论文指出，全球出生率已从1950年的每100人3.78人（即生育率3.78%）降至2025年的1.71人。然而，生育率每下降一个百分点，劳动力人均GDP就会增长26.8%。
论文说：“跨国数据显示，生育率下降会导致更高的全要素生产率、更大的资本存量、向高科技行业出口转变，以及更多节省劳动力的专利研发。”
在美国，这一趋势表现为劳动力向高科技产业转移，以及节省劳动力的发明专利增加。
四名作者说：“正是这种技术应对措施促成了生育低谷与后来经济繁荣之间的正向关系。”
他们还研究了生育率下降推动生产率提高的其他可能途径，例如促使更多女性就业或促进产业从农业向制造业转型，但并未发现任何证据表明这些方法会产生显著影响。
阿西莫格鲁和奥特是麻省理工学院的经济学教授，伯恩是麻省理工学院的经济学博士生，斯科特是伦敦商学院的经济学教授。

日本舆论质疑高市早苗“轻视国会”
多家日本媒体和各界人士近日对日本首相高市早苗出席国会审议及答辩的情况提出批评，质疑她“轻视国会”。
新华社引述日本《每日新闻》的报道，截至星期一（7月6日），高市早苗在本届国会出席集中审议八次，累计29小时36分钟，仅为过去五年间前首相菅义伟、岸田文雄、石破茂平均出席时间的40%左右。此外，根据日本朝野2025年达成的协议，在4月至6月例行国会会期内，应每月举行一次党首讨论，但高市早苗仅在今年5月出席过一次。
在野党方面6日在国会指出，本届国会立法进程明显放缓：2023年至2025年三届例行国会期间，政府提交法案的通过率均接近98%；而本届国会会期临近结束时，法案通过率只有73%。
日本共产党政策委员长山添拓7日在社交媒体上发文，批评高市早苗变相“拒绝国会审议”，且即使出席国会，也经常回避正面回答议员提问，一再表现出不诚实的答辩态度。
东京大学研究生院教育学教授本田由纪说，高市早苗用机械化的固定说辞应付答辩，导致国会审议陷入停滞。她批评高市早苗、执政党及其支持者缺乏应有的反思与责任意识。
《朝日新闻》报道说，本届国会期间，高市早苗政府凭借执政党在众议院的席位优势压缩预算案审议时间，不顾在野党反对，强行推进多项法案，导致朝野对立持续升级。报道引述一名自民党前阁僚的话说：“这是我第一次见到如此糟糕的国会。”

台致癌油风暴：卢秀燕公布三企业各罚300万台币
台湾食用油致癌物超标风波持续延烧，台中市长卢秀燕公布，将针对设籍台中的三家问题油品厂商各罚300万元（新台币，下同，12万新元）。
综合台湾《联合报》《自由时报》报道，卢秀燕星期二（7月7日）上午主持市政会议前说，此案事关重大，影响到全民，市政府经过反覆查证，决定对涉及致癌油的上述三家食油厂中联、福寿与福懋，根据《食安法》采取最高罚则，每家开罚300万元。
此事始于4月时的首次检验、下游业者反映异常后展开追查，历经复验确认超标、主动通报主管机关，到食药署星期日（5日）公布使用问题油品的257家业者名单，并要求全面下架回收，整个流程历时近三个月。截至星期一（6日）受影响业者已达到360家。
涉事上游食用油供应商为中联油脂，这家企业由福懋油脂、福寿实业及泰山企业三家油脂企业合资成立。
台湾总统赖清德强调将依法究责，绝不宽贷；但在野阵营批评赖政府迟迟未公布下游厂商名单，并要求相关官员下台负责。
台湾官方星期一公布含量20%以下致癌油产品免下架后，卢秀燕批评中央对问题油品先盖牌、延迟一周才公布，现又称20%以下不用下架，对此她无法认同、恕难配合，只要有致癌物就不容许存在，“不管掺多少要求全下架”。
