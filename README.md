消息：美国卡塔尔欲购买乌无人机 对付伊朗无人机
（基辅／华盛顿综合电）消息人士透露，美国和卡塔尔正同基辅商讨购买乌克兰拦截无人机，作为击落伊朗无人机的廉价替代方案。
消息说，这项初步会谈，目前只涉及政府官员之间，没有任何公司参与。讨论的技术，包括监听来袭无人机并干扰其通信。卡塔尔和美国五角大楼都没有对此置评。
乌克兰总统泽连斯基星期四（3月5日）深夜在社媒平台X发文说，美国请求基辅协助击落伊朗见证者（Shahed）无人机，他已指示提供必要资源。
西方外交官透露，乌克兰将分别派代表团到卡塔尔和阿联酋，分享乌方防御无人机经验。
泽连斯基表明愿意分享经验，但他也提出要求。“我们的要求非常简单，希望填补爱国者导弹的库存缺口，并获得一定数量的拦截弹。”
泽连斯基建议阿拉伯国家协助说服俄罗斯总统普京同意停火，以让乌方能够腾出人手向中东派遣反无人机专家。
彭博社分析说，美国总统特朗普多年来一直指责欧洲躲在美国的安全伞下，但在他对伊朗发动攻击后，却发现欧洲仍掌握美国所需的关键资源：军事基地、领空和战略地缘位置。
这反映了一个基本现实，虽然欧洲仍通过北约等机构在安全上高度依赖美国，但仍保有影响力。美国要将军力投射到中东，最有效的方式仍是借助盟友的地理优势，那就是德国的后勤枢纽、英国的空军基地、西班牙的海军设施，以及确保战斗机无障碍飞越的许可。
分析也说，尽管存在分歧，欧洲不大可能与特朗普彻底决裂。欧洲大陆在国防方面仍非常需要美国，大多数欧洲领导人也会尽可能地配合美国的要求。
