国民党军购拍板“3500亿＋Ｎ” 卓荣泰盼1.25万亿不打折
台湾在野国民党拍板确定，提出“3500亿元（新台币，下同，141亿新元）＋Ｎ”的国防特别预算条例版本；行政院长卓荣泰则强调，行政院去年提出为期八年的1.25万亿元版，是经过防卫专业缜密评估，不容许打折。
国民党内部对军购金额原有不同主张，经内部协商后，党中央星期四（3月5日）先举行记者会，强调国民党一向支持对美军购，但坚持军购必须符合三项基本原则：第一，军购内容必须符合防卫作战需求；第二，必须完成建案与程序；第三，必须取得美国正式发出的发价通知书。
国民党文传会主委吴宗宪说，国民党无法接受行政院1.25万亿元版本，“把军购与商购包在一起，胁迫在野党空白授权，逼全民吞下去”。
国民党主张，优先支持政府对政府的军售，但涉及商业采购及内部采购部分，必须严格审查，并回归正常预算程序，确保采购透明，可以监督。
国民党立法院党团随后召开党团大会，凝聚共识后，在上述三原则下，宣布该党的主张为“3500亿元＋Ｎ”。这指的是，除了美国已公布111亿美元（142亿新元）军购内容外，只要美国启动下一阶段对台军售，正式提出新的军售报价书，主管机关应立即研拟第二阶段特别条例草案，送请立法院审查，立法院会从速完成三读程序。
国民党主席郑丽文在苗栗受访时指出，这1.25万亿元并非皆给美国人，过去商购的拉法叶舰弊案曾闹出人命，民进党政府也曾发生“钽震”弊案，因此国民党必须严格把关。
国民党团书记长林沛祥则说：“希望以分阶段、具体明确的方式来推动军购，而不是一次性授权一张没有金额上限、没有内容明细的空白支票。”
台湾行政院长卓荣泰回应说，行政院去年提出为期八年的1.25万亿元国防特别预算条例，是经过防卫专业缜密评估，考虑防卫需求和财政年度规划，包括台湾之盾、用人工智能（AI）高科技强化击杀链，以及发展军工产业等三个目的，不容许打折。
根据朝野协商，过去被封杀10次的行政院版本与国民党的3500亿元+N、民众党4000亿元版本，将一起在星期五（3月6日）立法院会交付委员会审查。
民进党团干事长庄瑞雄说，当天不需表决即可付委，相信在野党会遵守协商结论。

胡润全球富豪榜 中国十亿美金企业家达1110位排名第一
胡润研究院星期四（3月5日）发布2026胡润全球富豪榜，中国以1110位十亿美金企业家位居第一。
据胡润研究院官网消息，中国超越美国，重新成为十亿美金企业家最多的国家。中国以1110位十亿美金企业家位居第一，比去年增加了287位，而美国以1000位十亿美金企业家位居第二，增加了130位。
排名第三的印度拥有308位十亿美金企业家，增加了24位。德国以171位（增加30位）攀升至第四位，超过了英国。其他增长较快的国家包括俄罗斯、澳大利亚、新加坡、巴西、加拿大和沙特阿拉伯。
在2026胡润全球富豪榜中，西方国家占41%，其中1000位来自美国，743位来自欧洲，75位来自加拿大，61位来自澳大利亚。东南亚有181位十亿美金企业家，增加七位，其中新加坡以59位居首，印度尼西亚有44位，泰国有40位。马来西亚和菲律宾各有16位，越南有六位。
