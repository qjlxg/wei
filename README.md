马赛，3月12日 — 极右党派国民阵线（RN）在马赛市长选举中借助以治安为主的竞选视频撬动选情，吸引原本犹豫的选民支持。候选人弗朗克·阿利西奥承诺将市警人数增至三倍、监控摄像头翻倍，并在每个行政区设警务岗，誓言“把幸福带回马赛”。
民调显示，阿利西奥在周日第一轮投票前与现任社会党市长贝努瓦·帕扬并列领先，这使RN在法国第二大城夺取市政权力成为可能。2024年参议院报告指出，马赛已成为全国性可卡因使用激增的中心。民调显示，治安是3月15日与3月22日日两轮市选的首要关切，这对主打严打犯罪的RN有利。RN在全国走势走强，民调甚至显示其可能角逐2027年总统职位。
Ifop民调指出，结果取决于左派是否联合：若左翼集中支持单一候选人，帕扬可领先约10个百分点；若左翼分裂，竞局将收窄。法国市长在地方治安上权力有限，市警无全国警力的权限与资源，但阿利西奥与帕扬都把打击犯罪作为竞选核心。阿利西奥称马赛是“毒品之城”，批评多年未有作为；帕扬则指责极右煽动恐惧，并称官方数据显示去年马赛总体犯罪较2024年下降4.1%，涉毒杀人自2023年高峰后有所回落。
法兰西药物与成瘾趋势监测中心社会学家克莱尔·杜波特表示，尽管与往昔相比并非整体上升，但涉毒杀戮从有针对性的报复变为更无差别的暴力，居民的不安全感有其合理性却不必然反映总体趋势。为增强治安话语权，帕扬请来了22岁反毒活动家阿明·凯萨西，他曾失去两名兄弟，表示社会党将以医疗、教育、交通和住房应对贩毒根源，而极右方案无法根治问题。
在受毒品暴力影响最深的北部社区拉·比塞林，一些居民对选战中被安全议题主导感到反感。社区志愿者法德拉·乌伊迪夫说，极右总是在设定话题，把“安全”作为万能答案，其潜台词是将阿拉伯人和黑人描绘为“问题制造者”。她担忧RN上台会削减社会服务，加剧贫困与违法，反而不利治安改善。

OpenClaw曝安全风险 中国多所高校禁校内使用
代理式人工智能OpenClaw在中国爆红后，被曝存在信息泄露等安全隐患。为防范风险，中国多所高校已发通知，限制或禁止在校内使用这一智能体。
综合澎湃新闻和《南方日报》报道，江苏师范大学信息化建设与公共资源管理处星期三（3月11日）发布提醒，建议师生若需使用OpenClaw，应优先通过云服务器、虚拟机或容器等隔离方式部署，避免将服务直接暴露在公网或校园网。
安徽师范大学网络安全与信息化办公室星期二（3月10日）发布预警，指OpenClaw存在明显安全隐患，建议“非必要不部署使用”。通知要求避免在接入校园网的设备、办公电脑或存有敏感信息的设备上使用，并严禁在教学科研数据、行政办公信息和学生信息等工作场景中使用，以防数据泄露或系统受攻击。
珠海科技学院同日发布通知，明确严禁在校内使用OpenClaw软件，要求已安装者立即卸载，并清除相关配置、缓存和日志文件。校方称将不定期开展网络安全扫描，一旦发现违规安装或使用，将依规处理。
华中师范大学也在星期一（3月9日）发布提醒，要求排查校内是否存在OpenClaw部署，重点检查公网暴露、权限配置和凭证管理情况，并明确禁止在信息化办公室分配的服务器上安装该软件。
由奥地利工程师施泰因贝格尔（Peter Steinberger）创建的OpenClaw今年1月对外开源，可部署在电脑、手机或云服务器上，整合调用不同大语言模型和应用软件。只要发布语音指令，就能自主执行文件管理和数据汇总等任务。
今年2月，中国工信部发布风险提示，称OpenClaw可能因指令诱导、配置缺陷或被恶意接管而执行越权操作，导致信息泄露或系统被控制。中国国家互联网应急中心星期二也警告，OpenClaw默认安全配置较为脆弱，一旦被发现漏洞，攻击者可能轻易获取系统控制权。
中国工信部星期三（3月11日）还就此发布六条安全建议，包括严格控制互联网暴露面、坚持最小权限原则等。

中东战火抬高肥料价格 中化化肥要求客户不参与市场炒作
霍尔木兹海峡危机推高化肥价格之际，中国国有中化化肥提示客户，不得参与化肥市场炒作、哄抬价格。
财联社报道，中化化肥星期四（3月12日）发布通知，称当前正值中国国内春耕用肥关键时期，公司为进一步贯彻落实国家化肥保供稳价精神，要求公司客户不得参与化肥市场炒作、哄抬价格、囤积居奇等，要主动维护化肥市场秩序稳定，严禁客户转卖和囤积，要求客户合理定价销售。
联合国贸易和发展会议星期二（10日）发布报告指出，霍尔木兹海峡是全球能源运输要道，承载全球约四分之一的海运石油贸易以及大量液化天然气和化肥运输。
中东局势升级下霍尔木兹海峡航运受阻，已推高能源和化肥价格。
日经中文网星期四报道，中东局势紧张的背景下，农业肥料价格正在上涨。国际行情达到三年高点。除了中东停止生产肥料之外，还面临原料采购困难，肥料价格的上涨和供应受限预计将对食品价格形成上行压力。

宝马CFO：今年在华销量有望恢复至去年水平
德国汽车制造商宝马首席财务官默特尔认为，宝马在中国的销量有望在2026年恢复至去年的水平。
据路透社报道，宝马星期四（3月12日）公布，2025年净利润降至74.5亿欧元（109.75亿新元），同比下滑3%。受关税和中国市场竞争加剧影响，公司去年在中国的销量同比下降12.5%。
宝马指出，在经历上一年的销量下滑后，公司希望在2026年稳住中国市场。默特尔（Walter Mertl）则说，过去几个月的平均销售数据显示，中国市场销量有可能回到去年的水平。
目前，宝马的全球汽车销量已降至2017年以来最低水平。公司去年10月曾下调利润预期，并警告中国市场销量不及预期。

陆委会：大陆民族团结进步促进法 或变成处理两岸问题法律依据
台湾陆委会针对中国大陆人大会议刚刚通过的《民族团结进步促进法》称，该法有可能会被外溢，变成处理两岸问题的一个法律依据。
中国大陆全国人大星期四（3月12日）以2756票赞成、三票反对、三票弃权，通过《民族团结进步促进法》。
据台湾《自由时报》报道，在《民族团结进步促进法》通过前，陆委会副主委沈有忠星期四告诉记者，这部法律重点是摆在对中国大陆境内少数民族，但也涉及了境外，并提到要推进民族团结和国家统一，该法也有可能会被外溢，变成处理两岸问题的一个法律依据。
沈有忠也说，台湾方面会认为，中国大陆的很多作为名义上是对内宣称维稳，“事实上有可能转化为长臂管辖或跨境镇压，甚至具有破坏两岸关系的可能性，所以要提早因应”。

伊朗称袭击以色列基地 库尔德斯坦一处意大利基地遇袭
伊朗军方宣布，在冲突进入第13天之际，他们袭击了以色列军事基地和以色列安全机构辛贝特。
法新社引述伊朗军方星期四（3月12日）通过国家电视台播出的声明称：“伊朗伊斯兰共和国军队的无人机袭击了犹太复国主义政权的帕尔马希姆（Palmachim）和奥夫达（Ovda）空军基地以及情报机构辛贝特（Shin Bet ）总部。”
另据意大利外交部长塔亚尼（Antonio Tajani）说，位于伊拉克库尔德斯坦城市埃尔比勒（Erbil）的一处意大利军事基地遭到“袭击”，但无人受伤。
塔亚尼星期四通过社交媒体X发布消息说：“我刚刚与意大利驻伊拉克大使通了电话。幸运的是，我们所有的士兵都安然无恙地待在掩体里。”
埃尔比勒响起防空系统击落无人机的声音，法新社记者引述一名库尔德斯坦安全部门消息人士称，星期三，驻扎在埃尔比勒机场的以美国为首的国际部队在埃尔比勒上空击落了无人机，没有人员伤亡报告。
路透社引述国防部消息人士说，无人机可能并非有意攻击意大利基地，而是在失去高度后意外击中。基地指挥官皮佐蒂（Stefano Pizzotti）告诉天空电视台TG24频道，军方人员在袭击发生前数小时已接到空中威胁警报并躲入掩体。
他说，目前尚不清楚袭击来自何处，并补充说，空袭警报已经解除，但专家仍在检查和确认该区域的安全。
意大利国防部在其网站上说，意大利在埃尔比勒驻扎了约300名士兵，加入国际部队，负责训练库尔德安全部队。
