位於菲律賓以東海域的潛在風暴預料較大機會週末在本港以東登陸，但其與本港的距離須視乎中心整合位置和速度，以及季風槽和副熱帶高壓脊的互動。
潛在風暴過去一兩日發展速度較慢，尚未建立和鞏固中心。人工智能模式預測潛在風暴短期內中心整合速度較快，繼而較早進入南海，預測登陸位置亦略為偏東。部分傳統電腦模式則預測潛在風暴仍須一段時間整合中心，因此起跑時間較遲，預測登陸位置亦略為偏西。
人工智能和傳統電腦模式對風暴初期中心整合的預測有別，在去年數個案例亦有出現，但現時未有足夠數據判斷誰優誰劣。
至於潛在風暴進入南海後的路徑北份量，則須視乎副高往南伸展的幅度。早前傳統電腦模式預測季風槽有殘餘渦度，因而阻礙副高南伸。然而，傳統電腦模式過去一兩日有加大副高南伸幅度，以致潛在風暴預測路徑往東修正的趨勢。此外，新一輪西風波動會在週末通過中國東北部，亦令副高短期內較難進一步往西伸展。因此，現階段預測潛在風暴東登的機會較大。
東登風暴須較接近本港，以及登陸後採取較為偏西的路徑，才會對本港造成顯著的風雨影響。相信潛在風暴中心完成整合後，本港週末風雨預測會更為明朗。

法国6月热浪期间超额死亡人数逾5700人
法国卫生部门星期三说，由于上个月法国遭遇破纪录的热浪，死亡人数比往年同期高出5764人，并称15岁及以上人群的超额死亡情况“前所未有”。
法新社报道，这是自2003年造成约1万5000人死亡的毁灭性高温事件以来，法国热浪期间出现的最高超额死亡率。
自5月以来，法国已遭受三次热浪侵袭。由于许多房屋、学校和医院的设计并未考虑防暑降温，也没有空调，高温天气造成了严重破坏。
法国公共卫生局说，今年6月17日至7月2日期间增加的死亡病例属于“所有死因”，目前尚无法明确将其直接归因于高温。最终数据预计将在秋季公布。
相比之下，去年6月至9月中旬，法国报告了5722例与高温相关的超额死亡病例。
上个月的超额死亡病例中，约三分之二是75岁及以上的老年人，不到五分之一为45至64岁的人群。其中一半的死亡病例发生在6月25日至27日这短短三天内。
法国上个月经历了自1947年有记录以来最热的六月。法国公共卫生署说，6月份巴黎地区的死亡人数上升了80%。
在6月的最后一周，法国各地有四名幼童因被困在滚烫的汽车内而死亡。
5月份，法国遭遇了罕见的热浪，导致至少300人死亡。
科学家说，人为造成的气候变化毫无疑问是高强度热浪于6月下旬席卷欧洲的原因。
据法国国家气象局统计，自1947年以来，截至6月下旬，法国全国共记录到53次热浪，其中一半发生在2010年之后。

他强调，和平稳定是自然资源治理的坚实保障。国际社会应秉持共同、综合、合作、可持续的安全观，坚持和平解决争端，敦促有关冲突方停止军事对抗，支持地区国家和组织加强协作，共同打击自然资源非法开采与走私，加强供应链透明度和可追溯性，从源头上切断武装冲突伸向自然资源的黑手。

他强调，要打破资源和冲突伴生的恶性循环，就必须支持当事国实现自主可持续发展。国际社会，特别是发达国家有责任履行援助承诺，支持资源国特别是发展中国家，持续完善资源治理体系，提高资源产品附加值与收益，真正造福当地民众，为和平打造更坚实的根基。
孙磊形容中国始终是自然资源治理与合作的倡导者和推动者，并说中国愿同国际社会一道，深化自然资源互利合作，更好帮助有关国家将资源禀赋转化为发展动力，走上长治久安与发展繁荣之路。

中国进入主汛期，多地出现超警雨水和洪涝灾害，防汛形势严峻复杂。为防止违法不良信息误导公众、干扰救援秩序，官方将开展集中清理整治。
中国国家网信办星期四（7月23日）在微信公众号发文称，为有效阻断汛期灾害事故违法不良信息传播，进一步规范汛期灾害事故信息发布行为，中央网信办和应急管理部近日联合印发通知，在全国范围内部署开展相关违法不良信息集中清理整治。
据介绍，整治行动重点聚焦五类问题。一是移花接木与恶意剪辑类问题，包括恶意剪辑国外或外地发生的灾害事故影像资料，选择性摘取片面信息，通过去水印、改标题、换字幕等手段，伪装成发生在本地的实时灾情，混淆视听、误导公众。
二是旧闻翻炒与历史嫁接类问题。将往年甚至几十年前发生的灾害事故现场图片或视频重新包装，删除原始时间信息，谎称即时新闻，渲染恐慌情绪。
三是夸大其词与虚构数据类问题。杜撰编造受灾情况，刻意捏造夸大伤亡人数、房屋损毁数量、经济损失规模等核心数据，或篡改灾害等级、虚构次生灾害，干扰救援秩序。
四是编造险情与冒充权威类问题。编造“水库溃坝”“堤防决口”“化工厂泄漏”等严重虚假险情，盗用官方机构、主流媒体名义或标识，伪造公文格式，发布虚假公告，损害政府公信力。
五是技术伪造与借机诈骗类问题。利用人工智能（AI）生成、拼凑剪辑、剧情摆拍等手段，制作灾情图片或视频，杜撰事故现场、救援细节等。假借灾情发布虚假募捐链接、求助二维码等，导致公众财产损失和个人信息泄露。
中国官方也要求，各地网信部门要聚焦典型问题及隐形变异问题，开展全面排查，督导网站平台压实主体责任，健全内容审核及账号管理机制。
