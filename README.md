3月3日，泰国民主党副党首กรณ์·查提卡瓦尼（Korn Chatikavanij）在脸书表示，他率党内青年到社区企业Aromatic Farm考察“香水椰子”的价格危机。田间价曾降至每颗2–3铢，而平均成本为5–6铢；与此形成反差的是，2023年泰国椰子出口额接近280亿铢。
กรณ์指出，问题并非销路不足，而是产业链中“价值分配结构”失衡；中游掌握议价权，农户各自为战，价格波动最终落在产地。解决之道不能仅靠短期托市，而应进行真正的结构性调整。
Aromatic Farm的模式以“人—地—利”（People–Planet–Profit）和可持续为核心，通过对会员公平收购、建立自有营销渠道、提升品质与专业化管理来降低脆弱性。กรณ์并提出应借鉴Fonterra等合作社经验，让农户入股掌握更多价值链权益。
他建议的关键措施包括：支持农户联合以增强议价能力；建立透明公平的竞争框架；监管收购商（ล้ง）不得压榨产地；建设农户可直接参与的农业碳信用体系。
กรณ์警告，若放任“2铢椰子”继续，农户将持续亏损、无力维护树木与土壤，最终导致产量下降。他呼吁将泰国椰子从易波动的农产品，转型为能为农民提供长期保障的“绿色产业”，并勇于进行结构改革。

菲律宾危机警戒级别：海外菲律宾人需知
迪拜讯（2026年3月3日）——记者 Tricia Gajitos
菲律宾外交部（DFA）在美以与伊朗冲突升级之际，加强与中东各地使领馆协调，保护海外菲律宾劳工（OFWs）。DFA表示，相关使领馆“仍在当地并保持高度警觉”，并会根据各国安全情势检视并更新危机警戒级别，以便提供必要协助。
危机警戒系统是启动紧急遣返与出入境限制的依据，适用于疫情、医疗危害及治安或外部攻击等情况。各级别对应不同建议与政府措施：
一级（预防期）：出现不稳迹象或威胁限于小范围，建议采取防范、避免非必要外出，仍可正常旅行但需谨慎。
二级（限制期）：暴力或不稳扩散或外部攻击趋近，建议就地避险、远离高风险区域；使领馆启动应急计划。
三级（建议自愿撤离）：动荡或外部威胁显著升高时，建议民众自愿撤离或遣返，强烈不建议前往受影响国。
四级（强制撤离）：发生大规模内乱或全面外部攻击时，实施政府主导的强制撤离与全面应变，禁止前往受影响国。
阿联酋目前未被列入任何警戒级别，官员称该国拥有世界领先的防空与反导系统，呼吁居民以官方与经核实的信息为准，保持冷静与警觉。

危机警戒系统是启动紧急遣返与出入境限制的依据，适用于疫情、医疗危害及治安或外部攻击等情况。各级别对应不同建议与政府措施：
一级（预防期）：出现不稳迹象或威胁限于小范围，建议采取防范、避免非必要外出，仍可正常旅行但需谨慎。
二级（限制期）：暴力或不稳扩散或外部攻击趋近，建议就地避险、远离高风险区域；使领馆启动应急计划。
三级（建议自愿撤离）：动荡或外部威胁显著升高时，建议民众自愿撤离或遣返，强烈不建议前往受影响国。
四级（强制撤离）：发生大规模内乱或全面外部攻击时，实施政府主导的强制撤离与全面应变，禁止前往受影响国。
