英国寻求限制伊朗危机对经济的冲击
伦敦，3月9日 —— 财政大臣瑞秋·里夫斯表示，随着伊朗局势升级推高能源价格并增加通胀风险，英国希望主要经济体同意释放应急石油储备以缓解影响。
自冲突爆发一周多来，英国借贷成本大幅上升，涨幅超过其他欧洲国家与美国。投资者担忧油气价格飙升会推高已居高不下的通胀并迫使政府增加举债。
首相凯尔·斯塔默警告称，危机若长期化将损害经济；当日短期英国国债价格一度出现自特拉斯时代市场动荡以来的最深单日跌幅。斯塔默说，事态越久，影响经济的可能性越大。
利率期货显示市场已不再预计英格兰银行今年会降息。随着油价从高点回落，债券价格当日收盘时大部分回稳。
能源价格飙升可能迫使政府出手救助家庭，然而财政空间有限且此举不受欢迎。斯塔默称政府正努力限制后果，财政大臣与英格兰银行每日沟通，关注居民能源价格问题。
七国集团表示愿意采取“必要措施”应对油价上涨，但未承诺释放库存。里夫斯在议会要求协调释放国际能源署储备，并称中东快速降级是保护居民免受能源账单上升的最佳途径；在局势发展期间，她的优先事项是帮助家庭应对生活成本并维护公共财政。
里夫斯还表示，能源价格上升可能在未来数月对通胀形成上行压力，并正与劳合社等保险机构就经霍尔木兹海峡航运问题进行磋商。英国对天然气价格尤为敏感，通胀为七国集团最高。2022年俄乌战争后，保守党政府曾为能源补贴花费逾400亿英镑。
多数住户因受调控关税保护可避免今冬取暖电费的即时冲击至7月。花旗估计若在未来两年将居民关税维持在当前水平，政府成本约为90亿英镑，约为上周估计的两倍。德意志银行则估算若不干预，能源价格上升将使通胀上升约1个百分点，并使经济产出下降近0.5%，在此前约1%增速的预测中影响显著。路透社报道。
