【通胀担忧与强劲经济数据共振 美债收益率全线攀升】金十数据3月5日讯，由于市场担忧美伊冲突带来的通胀影响，导致投资者开始消化更高的借贷成本，美国国债收益率随之上升。Sky Links Capital Group的Daniel Takieddine在报告中表示：“中东局势升级引发的地缘政治不确定性继续深刻影响着金融市场。”此外，他还指出，近期公布的经济指标——包括好于预期的2月份ISM服务业PMI数据和ADP劳动力市场数据，进一步强化了美国经济保持韧性的观点。根据Tradeweb的数据，两年期美国国债收益率上升2.5个基点至3.567%，而十年期收益率上涨3.7个基点至4.118%。

日本商业航天公司小型火箭发射第三度失败
日本商业航天公司“太空一号”再度尝试发射小型火箭“凯洛斯”三号，不过火箭在上升过程中飞行中断。这是继2024年两发两败后，这一系列火箭第三次发射失败。
继星期三（3月4日）发射临时中止后，“太空一号”公司（Space One）于当地时间星期四（5日）11时10分（新加坡时间10时10分）从位于和歌山县（Wakayama Prefecture）的纪伊太空发射场发射“凯洛斯”三号。直播画面显示，火箭升空后很快出现飞行姿态不稳，飞行轨迹异常。“太空一号”公司判断火箭难以完成送卫星入轨的任务，遂决定采取飞行中断措施。
新华社报道，“凯洛斯”三号搭载五颗超小型卫星。其发射最初定于今年2月进行，但因为天气等原因多次推迟。星期三上午，火箭发射前安全系统被触发，导致发射紧急停止。
“凯洛斯”（Kairos）火箭采用四级串联构型，全长约18米，直径约1.35米，重约23吨。其近地轨道运载能力为250千克，太阳同步轨道运载能力为150千克。“太空一号”公司原先希望把这款火箭打造成全球范围内从签约到发射所需时间最短的航天运载工具。
“凯洛斯”火箭曾于2024年3月和12月两次尝试运载卫星上天，均以失败告终。其中，“凯洛斯”一号升空数秒后在空中爆炸解体，二号在发射后不久便飞行中断。

卡尼吁加澳携手合作 加强对抗霸权
（堪培拉综合电）加拿大总理卡尼星期四提出，加拿大与澳大利亚应成为战略伙伴而非竞争对手，两国携手合作将加强与包括美国在内的超级大国谈判的实力。
卡尼星期四（3月5日）访问澳洲期间向澳洲国会发表演讲，这是自2007年以来加拿大领导人首次在澳洲国会发言。
卡尼说，他此次来访是为了在全球架构因接连不断的危机而分崩离析之际，重申渥太华与堪培拉的同盟关系。他指出，加澳虽然在地理上相隔遥远，但理应成为“战略表亲”。
他说：“在旧世界……我们倾向于把彼此视为竞争对手。在新世界，我们应该成为战略合作伙伴，在提升战略自主性的同时，加大投资、加快技术合作，增强供应链韧性，以及扩大国内加工能力。”
卡尼说，在大国竞争的世界中，中等强国面临的选择是：要么争取大国青睐，要么联合起来增强实力。他认为，中等强国的当务之急是制定规范，并参与制定新规则，以共同应对日益被霸权大国主导的世界格局。
他强调：“中等强国不能仅仅筑起更高的墙，并退到墙后。我们必须共同合作。”
澳洲总理阿尔巴尼斯星期四在国会致辞时也说，卡尼来访反映了加澳在世界舞台上发挥更大作用的共同抱负。
阿尔巴尼斯说：“澳洲和加拿大是正在变化的世界中的中等强国。我们无法逆转变革，但可以支持自己与我们的公民，同时互相支持。”
卡尼指出，两国可以合作发展战略优势的领域包括关键矿物、人工智能（AI）、贸易，以及无人机和卫星等防御设备。
他也宣布，加澳当天早些时候签订了一系列与关键矿产有关的新协议，包括澳洲加入七国集团（G7）矿产联盟。
这个联盟由加拿大主导，是全球拥有主要矿产储备的民主国家组成的最大集团，旨在促进全球关键矿产的多样化，并保障生产和供应。加拿大和澳洲共生产全球约三分之一的锂和铀，以及超过40%的铁矿石。
澳洲是卡尼亚太之行的第二站。他此前已访问印度，星期五（6日）离开澳洲后将续程前往日本。

中国春节票房遇冷 甄子丹吁回归电影人初心拍出佳品
针对中国电影春节票房遇冷情况，中国全国政协委员、香港演员甄子丹星期四（3月5日）在参加全国政协界别小组讨论后，接受媒体采访时，呼吁电影人回归初心，拍出佳品，必定能获得观众的喜爱。

意大利计划向波斯湾国家提供防空援助但不参战
意大利将向遭受伊朗空袭的波斯湾国家提供防空援助。意大利总理梅洛尼在接受当地媒体采访时说，意大利计划同英国、法国和德国一道向波斯湾国家提供防御性质的军事援助，主要涉及防空领域。她同时强调，意大利不打算加入战争。 
梅洛尼星期四（3月5日）在接受RTL 102.5电台采访时说：“我们表明要提供防空援助，这不仅是因为这些国家是友好国家，还因为波斯湾地区有数万意大利公民和约2000名意大利士兵，我们必须保护他们。”
法新社引述她说：“波斯湾地区对意大利和欧洲的能源供应至关重要。”
当记者问及意大利境内三个美军基地可能卷入战争的问题时，梅洛尼说，这些基地已经获得了进行“非轰炸行动”的授权，任何变更都必须由意大利政府以及意大利议会考虑。
新华社引述梅洛尼说，目前没有收到相关请求，并强调“意大利没有卷入战争，也不打算加入战争”。
英国、法国和德国领导人1日晚发表联合声明说，可能对伊朗采取“必要且相称的防御行动”，以摧毁其发射导弹和无人机的能力。
