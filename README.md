英国新任首相伯纳姆星期二（7月21日）启动降低民众生活费的政策计划，其中包括拟取消家庭电费账单中的增值税。
法新社报道，伯纳姆说，政府将从10月起取消电费账单中的增值税。这项政策预计将使英国政府本财年减少约8亿5000万英镑（约14亿7000万新元）的税收。

中国海关查获藏匿在旅客裤袋两濒危活鹦鹉
中国海关查获藏匿在旅客裤袋的两只濒危活体鹦鹉。
中国海关总署办公厅星期二（7月21日）在微信公众号通报，罗湖海关在旅检进境渠道，连续查获两名女性旅客在各自裤子口袋内藏匿活体鹦鹉。
根据通报，海关关员在对进境旅客进行监管时，发现两名旅客相继进入海关监管区，均未主动申报。两人裤子宽松，口袋处有明显鼓起，海关关员立即对两人进行拦截检查。

美国总统特朗普星期一（7月20日）签署行政令，规定2027年1月起，国防承包商在申请豁免令向“非盟友外国”（non-allied foreign nations）购买关键矿产时，除了过去规定的须证明这些供应商是最方便或最便宜的选择，还须证明业者已尝试寻找替代来源，并须披露原材料来源。
“非盟友外国”一般指中国、朝鲜、俄罗斯和伊朗。未能遵守新条例的承包商，可能面对采购方面的后果，包括失去竞标政府合同的机会。

五角大楼正持续推动包括洛克希德马丁和波音公司在内等主要国防承包商加速扩大武器生产。只不过，许多用于生产导弹、飞机和其他先进系统的关键矿产和生产材料，仍高度依赖中国供应商；新条例意味着这些公司今后难免会在加速生产武器供应美军和盟友，以及减少依赖中国两大政策目标之间陷入两难。

在特朗普与中国国家主席习近平今年5月举行峰会后，中国开始采购新一季美国大豆。双方在峰会上维持中国承诺2028年前每年采购2500万吨美国大豆的安排。

联合国：2025年亚洲诈骗损失或达近1500亿
联合国星期二（7月21日）发布的一份报告称，去年，东亚、东南亚、澳大利亚和新西兰的跨国网络诈骗受害者损失金额高达1141亿美元（约1472亿新元），预计较2023年增长至少两倍。
法新社报道，联合国毒品和犯罪问题办公室的最新报告显示，去年上述地区的网络诈骗给受害者造成的损失估计在883亿美元至1141亿美元之间。
报告指出，这远远超过2023年的180亿美元至370亿美元，反映出“这种犯罪经济的急剧扩张”。
报告指出，过去两年，中国大陆、韩国和台湾遭受的诈骗损失最为严重，各自损失数十亿美元。
柬埔寨政府正在加大力度打击境内的网络诈骗园区，几名头目已经从柬埔寨被引渡到中国。
报告作者指出，尽管一些头目已被逮捕，但犯罪活动仍在继续，一些从被查封的窝点获释的受害者，将他们的技能带去了其他地方。
联合国毒品和犯罪问题办公室区域代表尚茨说：“受害者与那些利用自己在犯罪团伙中的经历，学习犯罪技能并将其转化为牟利手段的人之间，界限总是很模糊。”
她指出，这些人回到自己的原籍国后，会利用并加入当地已有的犯罪网络。她还指出，这种情况不仅发生在欧洲，尤其是巴尔干地区，也出现在非洲。
联合国毒品和犯罪问题办公室表示，东南亚的犯罪生态体系还涉及非法毒品交易，以及儿童性剥削和房地产投资等活动。这些犯罪活动利用既有的贸易路线，并借助加密货币等手段隐藏资金流向。

中国官方承诺主动回应市场关切 增强市场内在稳定性
中国证券监督管理委员会承诺将主动回应市场关切，增强市场内在稳定性，更好推动资本市场长期平稳健康发展。
中国证监会在官网公布，证监会中共党委班子成员7月20日下午至21日分别召开座谈会，就促进资本市场稳定健康发展听取各方面意见建议，来自上市公司、证券基金机构和专家学者的29位代表参加会议。
证监会说，与会代表围绕当前资本市场形势进行了深入研讨，并对证监会工作提出意见建议。与会代表围绕进一步加强资本市场基础制度建设、落实中长期资金长周期考核机制、不断提升上市公司规范运作和治理水平、进一步规范量化交易行为、加大市场违法违规惩处力度、强化资本市场预期引导、进一步推动稳市机制制度化等方面提出了意见建议。
证监会负责人说，希望上市公司着力增强合规经营能力、提升核心竞争力，更好回报投资者；行业机构加大逆周期布局力度，切实提升投资者服务水平；专家学者积极建言献策、理性发声，共同营造良好市场环境。
负责人也说，证监会将认真研究吸收各方面提出的意见建议，主动回应市场关切，统筹防风险、强监管、促高质量发展，持续加强市场基础制度建设，增强市场内在稳定性，完善投资者合法权益保护长效机制，更好推动资本市场长期平稳健康发展。
中国证监会在星期一（7月20日）召开投资者座谈会。证监会主席吴清在会上说，证监会将坚持一体推进资本市场防风险、强监管，全力维护市场平稳运行，提高上市公司透明度和真实性、更好回报投资者。
