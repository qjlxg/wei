央视新闻客户端星期四（7月9日）引述中国气象部门预测，受今年第九号台风“巴威”（超强台风级） 影响，浙江南部、福建东北部等部分地区在星期四五至下星期一（7月10日至13日）的累计降雨量达150至300毫米，浙江东南部、福建东北部局地500至600毫米。
中国自然资源部地质灾害技术指导中心组织专家研判，受这次台风降雨过程影响，上述地区发生地质灾害的风险高。自然资源部于星期四晚上7时对浙江、福建启动地质灾害防御三级响应，并派工作组赴浙江、福建现场指导防御应对工作。
据央视新闻星期四消息，根据台风“巴威”防范应对保障需要，中国国家防灾减灾救灾委员会办公室、应急管理部会同国家粮食和物资储备局向浙江、福建调拨折叠床、夏凉被、家庭应急包等五万件中央救灾物资，全力支持做好群众避险转移和安置救助工作。

船只在霍尔木兹海峡遇袭 卡塔尔暂停液化天然气增产计划
卡塔尔暂停迅速恢复全球最大液化天然气设施产能的计划，因为旗下一艘液化天然气运输船在霍尔木兹海峡遇袭，加剧对这条关键航道通行风险的担忧。
彭博社引述知情人士说，卡塔尔能源公司的高层在星期二（7月7日）袭击事件发生后召开一系列会议，首席执行官萨阿德·卡比（Saad Al-Kaabi）决定暂停提高拉斯拉凡（Ras Laffan）综合体产量的计划。
部分知情人士称，出于安全考虑，该设施将维持最低限度运营，未来几天计划靠泊工厂的船舶数量也将减少。
这是本周局势升级带来的最受关注的影响之一。霍尔木兹海峡附近多艘船舶接连遭袭，美国又连续两天对伊朗发动空袭。 
美国总统特朗普星期三（8日）甚至提及再次爆发全面战争的可能性，这对于正逐步从冲突影响中恢复的中东能源生产商而言将是最坏的情形。 
卡塔尔能源公司未立即作出回应。

美国首次申请失业救济人数略减 裁员规模仍然有限
美国上周首次申请失业救济人数变化不大，表明企业裁员规模仍然有限。
彭博社报道，截至7月4日当周（涵盖美国独立日假期），美国首次申请失业救济人数减少2000人，至21万5000人。彭博对经济师的调查显示预期中值为21万7000人。
根据美国劳工部星期四（7月9日）公布的数据，此前一周持续申领失业救济人数小幅升至181万人。
首次申请失业救济人数维持在较低水平，表明雇主仍不愿大规模裁员。
不过，6月非农就业报告显示，当月招聘步伐有所放缓，令近期劳动力市场的强劲势头略有减弱。
报告还显示，许多美国人退出劳动力市场，这一变化也可能是首次申请失业救济人数维持低位的原因之一。
