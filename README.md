中国西部直辖市重庆市市委书记袁家军在当天的活动上回答提问时说，重庆作为内陆城市，将聚焦做好陆上贸易规则改革创新、对接高标准国际经贸规则等，努力为内陆地区对外开放，探索可复制可推广的有益经验。 
袁家军介绍，重庆正打造智能网联新能源汽车之都，把推动“渝车出海”作为“通道带物流、物流带经贸、经贸带产业”的抓手和着力点。
他也提到，去年是中新战略性互联互通项目10周年，重庆持续深化与新加坡在金融、航空、物流、信息四个重点领域的合作，同时把合作领域拓展至科技创新、教育医疗、新能源、临空经济以及超大城市治理。 
中国经济重镇上海市市长龚正星期五在代表团活动上指出，上海要强化内需主导作用，努力推动经济实现质的有效提升和量的合理增长，为全国发展挑大梁、作贡献。
他说，消费是经济增长的持久动力；要深入实施提振消费专项行动，加快建设国际消费中心城市，积极打造服务消费新场景，持续放大“文商旅体展”融合效应。

阿联酋两家航空公司未来几天恢复航线运营
全球最大的国际航空公司阿联酋航空宣布，公司预计将在未来几天内恢复全部航线网络运营。此前公司因战事相关的空域封闭已暂停运营一周。
阿联酋航空（Emirates）星期五（3月6日）在声明中说，恢复商业航班运营取决于空域可用性和所有运营要求的满足情况。
彭博社引述阿联酋航空称，星期四共有约3万名乘客从迪拜出发。到3月7日，阿联酋航空将每日运营106个往返航班，飞往83个目的地，占到其航线网络的近60%。
竞争对手阿提哈德航空（Etihad Airways）也在准备恢复飞往约70个目的地的航班运营。这家国有航空公司说，将自3月6日起，在其阿布扎比枢纽与欧洲、印度、美国以及包括利雅得在内的中东部分目的地之间恢复有限的商业航班运营。
阿提哈德称，航班服务仍需获得运营审批，并可能根据地区空域状况进行调整，并非所有目的地都会每日执飞。
阿拉伯联合酋长国仍面临来自伊朗的威胁。星期五上午，阿布扎比机场工作人员在乘客手机收到导弹警报后，将人员引导至避难所。

美国汽油零售价升至2024年来最高水平
由于中东冲突扰乱了该地区的能源供应，美国汽油零售价格上涨至2024年9月以来的最高水平。
彭博社引述美国汽车协会数据说，星期四（3月5日）汽油零售价格上涨至每加仑3.32美元（4.24新元）。汽油期货本周上涨了27%，势将创下自2022年3月以来最大单周涨幅。
