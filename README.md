美国和以色列上星期六（2月28日）联手空袭伊朗后，全球能源命脉霍尔木兹海峡的石油运输几乎停摆。伊朗当局未发出封锁海峡声明，但伊朗媒体称这条全球石油供应的重要通道“实际上已经关闭”，有船舶企图通过结果被击中。
霍尔木兹海峡连接波斯湾和阿曼湾，是沙特阿拉伯、伊拉克、卡塔尔、阿联酋等中东产油国的原油出口必经之路
据法新社报道，中远海控星期天（3月1日）在声明中称，已进入海湾的船舶，在安全条件允许的情况下完成相关作业后，已被指示前往安全水域停留或抛锚。
声明称，驶往海湾的船舶“已被通知优先确保航行安全，相关措施包括减速、前往安全水域，或在指定的避风锚地等待进一步指示”。
声明补充称，中远海控正在评估受影响船舶上所有货物的应急方案，其中包括潜在的替代卸货港。
包括马士基和地中海航运在内的其他航运巨头，也已宣布暂停在该地区的运营。
伊朗媒体报道称，一艘悬挂帕劳共和国国旗的“天光号”（SKYLIGHT）油轮星期天在试图“非法通过”海峡时被击中。
英国海上贸易行动办公室（UKMTO）则通报，一艘船当天在阿曼首都马斯喀特以北50海里处遭到不明投射物袭击。袭击发生在阿曼北部海岸外海域，在霍尔木兹海峡附近。
