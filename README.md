尽管泰伊双边贸易规模不大（2568年合计1.4618亿美元，泰国出口1.3682亿美元、进口0.0936亿美元），但泰国与15个中东国贸易总额达405.3563亿美元，进口高达280.6005亿美元并出现155.8447亿美元的逆差，显示我国产业对能源价格和物流成本波动高度敏感。
若霍尔木兹海峡运输受阻、冲突扩大，油价可能从当前约70美元/桶升至100–105美元/桶，进而推高能源、运输及生产成本。依输入产出表估算，柴油价格上涨将重创钢铁、染整、基础化学品、水泥、金属制品、玻璃及纺织服装等行业；宏观模型显示，若油价短期涨10–20%（≈77–85美元/桶），工业增加值将减少约27–30亿泰铢（约占工业GDP 0.06%）；若涨50%（≈100–105美元/桶），减少约101.25–120亿泰铢（约占0.15%）。
ธนกร强调，สศอ.要加强与企业沟通，深入评估影响，并拟定包括能源成本监控、针对中小企业的缓解措施、鼓励节能改造、强化运输及汇率风险管理与出口市场多元化等配套政策，以维护产业竞争力。

眾議院法案8190將路怒定義為：駕駛人或乘客因交通糾紛故意實施的侵略性行為，旨在恐嚇、威脅、騷擾、報復或傷害他人，並對生命、肢體或財產構成明顯且即時的危險。
法案列舉示例，包括以危險方式駕駛使他人受危害、用車輛恐嚇或追逐、以及在交通糾紛中威脅或攻擊他人。馬科斯表示，明確定義可協助檢方識別慣犯並追究責任，需建立專門法律來處罰常導致嚴重後果的路面暴力行為。

此法案是在一起據報發生於北呂宋高速公路、疑涉總統之子車隊的路怒爭議後提出。馬拉坎南否認事件發生，宮廷發言人克萊爾·卡斯特羅表示媒體應向陸運局查詢。菲律賓路怒事件常因行車畫面在社交媒體上流傳引發關注。

马尼拉 — 眾議院多數黨領袖、伊洛科斯北第一選區代表桑德羅·馬科斯周一提出法案，擬將「路怒」定為刑事犯罪，指出道路上的侵略、恐嚇與暴力行為危及生命。
眾議院法案8190將路怒定義為：駕駛人或乘客因交通糾紛故意實施的侵略性行為，旨在恐嚇、威脅、騷擾、報復或傷害他人，並對生命、肢體或財產構成明顯且即時的危險。
法案列舉示例，包括以危險方式駕駛使他人受危害、用車輛恐嚇或追逐、以及在交通糾紛中威脅或攻擊他人。馬科斯表示，明確定義可協助檢方識別慣犯並追究責任，需建立專門法律來處罰常導致嚴重後果的路面暴力行為。
若路怒造成財產損失或人身傷害，將處2年零1天至4年徒刑，或罰款10萬至20萬比索，或並科；若未造成損害或傷亡，處6個月零1天至1年徒刑，或罰款2萬至10萬比索，或並科。調查期間暫停駕照，定罪後永久喪失駕照資格。
此法案是在一起據報發生於北呂宋高速公路、疑涉總統之子車隊的路怒爭議後提出。馬拉坎南否認事件發生，宮廷發言人克萊爾·卡斯特羅表示媒體應向陸運局查詢。菲律賓路怒事件常因行車畫面在社交媒體上流傳引發關注。
