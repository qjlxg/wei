【非官方預測：MET WARN最新評估】
MET WARN 最新評估，天文台就現時位於南海的熱帶低氣壓發出三號強風信號的機會為「中等」。
【預測：離岸海域及高地風勢較大】
在西南氣流及副熱帶高壓脊的偏東氣流匯合下，預料風暴環流會較為廣闊；加上在與副熱帶高壓脊的其同影響下，華南沿岸氣壓梯度頗大，風暴東側的強風區會頗為接近珠江口一帶。
預料週五起，本港天氣會漸趨不穩。隨著風暴移至海南島一帶，週五及週六驟雨將逐漸增多，風勢增強。預料本港屆時主要受東至東南風影響，離岸海域及高地可能吹強風，高地或間中達烈風程度。
【變數：熱帶低氣壓與本港距離、環流大小】
電腦模式預報普遍支持系統移向海南島至北部灣一帶，並會與本港保持逾500公里的距離。然而，由於風暴中心初期整合的實際位置，以及副熱帶高壓脊的強度仍然存在不確定性。因此，風暴與本港的距離及環流大小存在變數，現階段仍未能斷言本港會否普遍吹強風。
現階段亦未能排除本港天氣在脫離熱帶低氣壓的主導後才普遍吹起強風，因此直接由一號信號改發強烈季候風信號的機會存在。
註：「中等」代表基於現有資料，MET WARN 評估天文台發出相關信號與否機會相若，可以用「五五波」形容。

《检察日报》星期五（7月3日）发表题为《千年泰山插入135公里刀片刺绳，项目是否科学要经得起公众追问》的评论文章，指出泰山作为公共景观资源，对其管理决策的科学性论证不能只停留在文件里，而应该让公众看得见、信得过，经得起公众拷问、经得起时间检验。

对于景区回应，文章指出，如果当初的立项经过了充分的可行性研究、环境影响评估和安全风险研判，那么面对公众的追问，景区大可以用翔实的数据、专业的论证来讲明道理，让公众清楚地看到决策的科学性和合理性，眼见为实、心服口服。但目前的回应恰恰缺少这些内容。
文章进一步指出，公众对一项公共决策是否科学的追问，恰恰是与法治政府建设同向的。假如景区管理部门拿不出科学决策的充分依据，甚至被证明存在决策不科学、不合规的问题，就要依法依规承担相应的责任。
“这也表明，公共决策不是关起门来的事，花的是公共财政，就必须恪守法律法规要求，经得起公众的审视。”文章最后强调，用科学的论证说服人，用坦诚的态度打动人，这是管理者的本分，也是该有的担当。

美国与欧盟就中国新生效的《民族团结进步促进法》表达关切，尤其担忧新法的域外适用条款或将为“长臂管辖”提供法律依据。
中国自7月1日起正式施行《民族团结进步促进法》，其中规定境外组织和个人若实施破坏民族团结进步、制造民族分裂的行为，将被依法追究法律责任。
据路透社报道，欧盟发言人星期四（7月2日）发表声明指出，该法案恐将进一步限制少数民族的文化、语言及宗教权利，而这些权利理应依照国际人权标准及中国在联合国框架内的承诺得到维护。
该发言人强调：“我们对该法的域外适用深表担忧，欧盟反对任何违反国际法的第三国立法进行域外延伸。我们敦促任何第三国，克制在欧盟境内及其他地区实施跨国镇压的企图。”
美国国务院发言人同一天也发表声明，直指该项法律“存在问题”。声明称，该法迫使中国境外人士“积极推动中共的‘民族团结’议程，否则将面临中国当局的报复”。
美国国务院发言人重申：“美国将维护我们的主权，保护境内个人免受外国政府与政权过度执法的侵害，防止他们遭到噤声、恐吓、骚扰、伤害或胁迫。”
这项新法在台湾也引发警惕，舆论担忧这项法律可能为北京提供另一个法律依据，借以惩治被界定为“台独”分裂分子的台湾民众。
台湾行政院长卓荣泰星期四称，中国大陆持续制定具有境外管辖与制裁效果的法律工具，形成日益绵密的长臂管辖网络，并不断扩大适用对象，意图迫使台湾人民服从大陆设定的政治框架，强制对台统一政策的法制化推进。
针对外界质疑上述规定涉及“长臂管辖”，中国司法部副部长胡卫列6月24日回应说，该规定立足国情、契合法理、符合国际通例，责任追究将依法依规、审慎规范，不影响正常中外人文交流、学术研讨和经贸合作。
