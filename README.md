強雷雨帶正靠近本港西南部，並繼續往北移動。短期內廣泛地區要提防大驟雨、強陣風和雷暴。
受活躍偏南氣流影響，本港附近及廣東沿岸持續有強雷雨區不時發展，讀者今早要繼續留意最新天氣情況。

---

超強颱風巴威逐步逼近台灣、浙江、福建和八重山群島一帶。計劃前往或正逗留當地的讀者應留意當地天氣警示及航班安排。
巴威環流廣闊，且會維持較高強度，對台灣構成頗大威脅。按照現時預測，巴威將於週六近距離掠過台灣以北，屬對台灣北部影響最大的「西北颱」路徑，會帶來猛烈風勢、豪雨和風暴潮。
若巴威未登陸台灣，對浙江、福建威脅亦會相當大。然而，目前未能排除風暴近岸受地形影響，先登陸台灣的可能。
本港方面，受巴威外圍下沉氣流影響，週五起將迎接高溫天氣。天文台預料，市區週六氣溫高達35度，達極端酷熱水平。同時，高溫或觸發大驟雨及雷暴，但熱對流發展位置可以相當隨機。
未有外遊計劃的讀者，應做好防曬及防中暑措施，並多關心身邊慢性疾病人士及長者健康。計劃進行戶外活動時，應留意高溫及驟雨可能造成的影響。

---

一道低壓槽會在星期一稍後開始在華南沿岸徘徊，屆時本港天氣漸轉不穩，大驟雨及狂風雷暴的風險會在本週中期持續。
預料低壓槽內可能有低空渦旋發展。除非該渦旋發展位置顯著偏南，疊加副熱帶高壓脊相關的東風充分配合，否則其發展為熱帶氣旋的機會暫時偏低。
視乎低空渦旋掠過的位置，水氣有機會被加強抽入至沿岸抬升，並誘發本週中期強雷雨區發展。本港雨勢強弱、雨區發展時間和影響範圍仍有變數。
雖然低壓槽及低空渦旋會在本週後期消退，但緊接而來的是活躍西南氣流，因此本週後期天氣仍會不穩定。讀者本週外出前應留意最新天氣消息，並留意已安排的戶外活動可能受到影響。

---

天文台表示，南海北部廣闊低壓槽上有一個低壓區會逐漸發展，並靠近廣東沿岸。若該低壓區發展為熱帶低氣壓，會評估是否在今日傍晚至明早發出一號戒備信號。
天文台亦表示，低壓系統相關的對流雲團現時集中於南側，預料尚需一些時間整合，在較為接近廣東沿岸時，才能確定會否發展成熱帶低氣壓。
按照現時預測，該低壓系統明早最接近，會在本港100公里範圍內掠過。天文台預料明日本港會有大驟雨、雷暴及狂風，可能需要發出暴雨警告信號或局部地區大雨提示；離岸吹強風，部分低窪地區有機會出現水浸。

---

受南海北部低壓系統影響，預料本港明早（星期二）天氣轉差，有大驟雨和狂風。星期三、四天氣繼續不穩。
目前低壓系統對流集中在南側，而且尚未發展出明確和穩定的中心。然而，受近岸地形輻合影響，未能排除中心加快整合，導致低壓系統發展為熱帶氣旋的可能。
按照現時預測，低壓系統明早會近距離掠過本港，但實際路徑要視乎中心整合情況。由於低壓系統大風區集中在東側和南側，若在本港以西掠過，風力會較大；至於東登的話，風力會在低壓系統掠過本港以北時有所上升。
低壓系統靠近時，近岸地形輻合或誘發對流發展；此外，低壓系統登陸後會引進西南氣流，亦有利強雷雨區發展。因此，預料未來兩三日本港不時受大驟雨影響，強雷雨區掠過期間或有短暫風力上升，有機會伴隨雷暴。然而，強雷雨區發展位置和影響時間存在一定變數。
無論掛波與否，本港未來兩三日天氣亦會轉差，讀者要密切留意天氣情況，MET WARN 亦會跟進最新消息。

1.若天文台發出黃色或紅色暴雨警告信號，所有放榜程序將會如常進行。
2.若天文台於早上6時或以前發出黑色暴雨警告信號，考評局則會在除下「黑雨」後2小時內公布最新安排，並開放指定辦事處讓學校領取成績單以作派發，並在3小時後向全港考生（包括自修生）發放SMS成績通知。如果「黑雨」維持至下午1時或更後，所有放榜及各院校的收生程序會相應順延一日。

澳大利亚总理宣布将制定全球首套人工智能国家标准
澳大利亚总理安东尼·阿尔巴尼斯宣布，将制定全球首套统一的人工智能国家标准，该标准将明确大型数据中心的运营规则，保护版权，并规范人工智能开发资源的使用。
阿尔巴尼斯在悉尼大学发表讲话时表示：“今天，我宣布，为了把握人工智能带来的机遇并引导其发展，我国政府将制定澳大利亚人工智能标准。”
据他介绍，新标准将把大型数据中心的各项要求整合到一个统一的强制性监管体系中。政府计划在与企业界和贸易伙伴磋商后，于明年初向议会提交相应法案。
澳总理还宣布，将在总理和内阁部内设立人工智能办公室，负责协调政府在该领域的政策。
阿尔巴尼斯表示，版权保护将受到特别重视。他强调，未经版权所有者许可，任何公司都不得使用艺术作品、书籍、音乐或媒体内容来训练神经网络。
“澳大利亚的作家、音乐家、艺术家和记者必须保留对其作品的所有权和控制权。任何公司都不得在未经作者监督的情况下使用澳大利亚的书籍、音乐、艺术作品或新闻来开发或训练人工智能……否则就是盗窃。”澳总理说道。
他还指出，澳大利亚将成为首个将数据中心选址、电力供应和用水量要求整合到国家统一标准体系中的国家。

中东局势升温 中国靠电动德士减轻油价冲击
随着中东紧张局势持续推高国际油价，中国正凭借快速普及的电动德士（中国称出租车）和网约车，降低交通运输对石油的依赖，并为潜在能源供应冲击提供缓冲。
据路透社报道，今年5月，中国德士和网约车出行量达30.5亿人次。官方数据显示，自2月底伊朗战争爆发以来，相关出行量较去年3月至5月增长6%。
这一增长反映出中国交通市场的一个特殊现象：尽管汽油价格上涨，德士和网约车车费却持续下降。
分析人士指出，在经济增长放缓背景下，大量求职者涌入网约车行业，加上价格低廉的电动车广泛投入运营，压低出行成本，吸引更多乘客放弃自驾、转而搭乘德士或网约车。
一名36岁的李姓北京兼职网约车司机说，自六个月前开始接单以来，车费已经下降10%至15%，“竞争非常激烈”。
自3月油价开始上涨以来，中国社交媒体上出现大量帖文，认为搭乘德士或网约车比自己开车更划算。
一名45岁的燃油车车主杨女士说：“尤其在油价高的时候，距离远到不能骑车的地方，我宁愿搭德士。这样既不用找停车位，也不用承担油费。”
随着德士车队加速电动化、网约车市场持续扩张，中国交通体系正逐步摆脱对石油的依赖，从而增强应对霍尔木兹海峡关闭等能源供应风险的能力。
中国交通运输部数据显示，全国约130万辆德士中，约一半为电动车；在主要城市，这一比例接近100%。
中国主要网约车平台滴滴说，去年平台新增注册200万辆混合动力车或电动车，使非化石燃料车辆总数增至800万辆，其中电动车贡献了平台总行驶里程的75%。
受此影响，尽管中国公路货运量同比增长2%，五一假期公路出行量创下历史新高，今年5月汽油和柴油消费量仍分别同比下降10%和14%。
环保组织绿色和平预计，到2035年，中国德士和网约车总行驶里程中，约90%将由电动车完成。
交通与发展政策研究所东亚区首席代表刘岱宗说：“随着燃油价格上涨，人们减少驾驶自己的燃油车。但整体出行需求仍在增加，因此更多出行正转向德士、地铁等公共交通方式。”

据CDC统计，全美34个州已记录了近7000起环孢子虫病（cyclospora）确诊或疑似病例。这是一种由微小寄生虫引发的感染。

美国医学会警告，这种疾病可能出现复发情况，即症状消失后又再次出现。

据卫生部门统计，截至7月13日，全国范围内的感染已导致141人住院治疗，目前尚未报告死亡病例。相关部门正展开调查以确定感染源。
密歇根州卫生部门怀疑，感染源与沙拉和生菜有关。据报，美国快餐连锁店塔可钟（Taco Bell）正在接受卫生部的调查。密州的一些塔可钟快餐店已采取预防措施，停止供应生菜。
