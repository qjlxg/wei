衛星雲圖可見，巴威風眼清晰，圍繞風眼的對流旺盛，短期內強度有繼續上升的可能。

香港以南海域持續有強雷雨帶發展並由南向北移動，先前未受大驟雨及狂風雷雨影響的其他地區亦將會受到影響。
與美沙克外圍雨帶相關的強雷雨帶呈南北走向，並正由南而北影響本港。因此，大驟雨、強陣風及雷暴的風險可能於下午維持一段時間，讀者應保持高度警覺。

[暴雨速報] 天文台在下午3時15分取消黃色暴雨警告信號。
雖然暴雨警告已經取消，但市民仍須警覺有關河道氾濫可能帶來的危險。

---

美莎克是今年首個影響本港的熱帶氣旋，期間間中為本港帶來大驟雨及猛烈陣風。
美莎克最終只在本港西南面590公里左右掠過，但在副熱帶高壓脊共同影響下，廣東沿岸的氣壓梯度較為緊密，即使在沒有雨帶影響期間，本港高地、西南部及新界東北離岸亦間中錄得強風。
在強雨帶影響期間，本港風力曾短暫上升，並錄得猛烈陣風。八個風球參考測風站中有三個錄得強風，其中長洲更一度錄得烈風，而西貢僅差一公里便達強風程度（該站其後於強烈季候風信號生效期間錄得七級強風）。除長洲外，位於高地的昂坪及「前哨站」橫瀾島亦錄得烈風，而昂坪的陣風更一度達颶風程度。
雖然技術上是次風暴可視為「八中三」，惟由於風勢並不持續，最終只維持一號戒備信號，情況類似2019年熱帶氣旋白鹿。

---

熱帶氣旋美莎克為本港帶來狂風驟雨後，先後登陸海南島和越南。風暴登陸越南前，一度迅速發展出完整眼壁，其風眼結構甚至在陸地上維持一段時間。
熱帶氣旋進入北部灣後，不時出現快速增強，主因是狹窄的海灣地形可化身為風暴加速器，有助加強地形輻合；此外，北部灣夏季海溫較高，亦為風暴提供更多能量。
美莎克登陸地沒有密集的氣象站觀測網絡，但鄰近廣西近岸有測風站錄得颶風。由於該測風站有一定海拔，數據可否參考，以及美莎克實際增強幅度仍有待氣象機構判斷。

---

強雷雨帶正影響大嶼山，本港以南海域亦有強雷雨區發展並往北移動。
未來數小時本港部分地區有機會受大驟雨、雷暴和強陣風影響。由於強雷雨帶的發展位置有機會往東移動，因此未能排除相關天氣擴展至其他地區的可能。

然而，由于美以与伊朗的战争，霍尔木兹海峡被封锁，导致包括沙特阿拉伯、科威特和伊拉克在内的OPEC成员国的油轮无法通行，产量受到限制，因此这次增产在很大程度上仍停留在纸面上。
市场分析师西卡莫尔说：“这个数字基本符合预期。鉴于阿联酋已退出石油输出国组织，而且由于冲突后产量仍在逐步恢复，配额可能仍未得到满足，我认为目前这些增产措施意义不大。”
阿联酋已于5月1日退出石油输出国组织。波斯湾已开始恢复因伊朗战争而中断的石油供应，并正在增加出口。
