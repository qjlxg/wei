报告作者认为，美国感到担忧的原因不仅在于中国对拉丁美洲投资的规模，还在于其性质。作者解释称：“对中国在该地区许多国家经济政策产生的战略影响、具有双重用途的基础设施（如深水港可用于部署军舰）以及贸易流向的转变，这些都对华盛顿不利，因此特朗普政府认为必须予以遏制。”

报告中写道：“美国当前的政策依据是‘门罗主义’——这份1823年的文件认为，除美国以外，任何国家均无权在整个西半球实现其地缘政治野心。白宫再次正式将拉丁美洲视为自己的‘后院’，以此‘捍卫美国免受外国扩张主义大国的威胁’。只不过，如果说19世纪时针对的是欧洲殖民帝国，而两百年后，矛头则指向了中国。”
在该地区的一些国家，华盛顿已取得了一定成效。例如，巴拿马成为拉美地区首个拒绝参与中国“一带一路”倡议的国家。
报告中强调：“美国政府也对秘鲁当前动荡的地缘政治局势感兴趣。中国提出的穿越秘鲁和巴西领土、连接太平洋与大西洋的跨大陆通道项目及其他倡议，对‘门罗主义’构成了重大威胁。其中还包括巴拿马运河可能失去作为绕过南美洲的唯一航道地位的风险。”
然而，并非该地区所有国家都愿意轻易放弃中国的投资，以换取与美国的合作。报告作者总结道：“通常情况下，无论是关税还是直接威胁，都无法说服其他国家改变立场。”

法新社报道，美国公告说，美国商务部调查发现，外国行为“持续损害”美国商用飞机制造业，而且美国过度依赖外国供应链。
但公告也说，美国商务部长卢特尼克建议“暂不要立即加征关税”，以应对因进口商用飞机、喷气发动机及相关零部件而带来的潜在国家安全风险。
尽管如此，这份9日发布的公告仍为美国日后采取进一步行动保留空间。
美国总统特朗普自去年重返白宫以来，已向美国的贸易伙伴加征广泛关税，并对钢铁、铝、汽车等行业祭出了高额关税。
尽管特朗普的多项全球性关税最终被最高法院推翻，但他针对特定行业的关税依然有效。
针对商用飞机的调查是援引1962年《贸易拓展法》第232条进行的，特朗普此前也曾以第232条款对其他行业征收关税。
不过，特朗普政府目前正寻求与贸易伙伴进一步磋商，以“调整”进口规模，确保进口量“不至于对美国国家安全构成威胁”。

中国网信网说，自今年5月部署推进短视频内容标注工作以来，中共中央网信办加大巡查力度，从严处置一批发布虚构演绎内容、未按规范进行内容标注的短视频账号。
中国网信网也列出部分典型案例，其中包括针对部分账号未进行标注，演绎“残疾妻子再次怀孕”“重病患儿募捐求助”等剧情，利用残障人士形象卖惨，消费公众同情心问题，对特定网络账号予以关闭处置。
另一起典型案例为针对部分账号未进行标注，演绎婆媳争吵冲突桥段，虚构“相恋五年双方因彩礼僵持不下”“三角恋争夺情人”等情节，刻意放大家庭或情感矛盾，激化社会焦虑问题，对特定网络账号予以禁言或关闭处置。

中国网信网在5月宣布，中央网信办总结12家平台试行经验，指导各地各网站平台全面推进落实几项工作。
