中国规范国际科技组织成立登记
中国三部门发文规范国际科技组织成立登记，明确其成立条件、申请程序等。
据新华社报道，记者星期六（7月4日）从中国民政部获悉，为支持国际科技组织在华设立和发挥积极作用，进一步规范其成立登记，民政部、科技部、中国科协制定《国际科技组织成立登记指引》。
据介绍，登记指引适用于在民政部申请成立登记的国际科技组织，即由中外高等学校、科研院所、科技组织、企业等单位或者个人共同发起，在中国境内成立登记，旨在促进自然科学和工程技术领域国际合作、国际交流和国际治理的国际性（或者区域性）、非营利性、非政府的社会团体法人。在经济、教育、文化、卫生、体育、环保等领域发起设立国际性社会组织，可参照适用。
登记指引要求，成立国际科技组织须有适当的中外发起人，发起人为初始会员；有规范的名称，应按照国家有关规定经过批准，冠以“国际”“世界”“全球”“亚洲”等体现组织国际性或者区域性的字样；有合法的资产和经费来源，成立时的活动资金为捐赠资金，不低于200万元（人民币，约38万新元）等。
在申请程序上，主要包括五个环节，分别是明确业务主管单位，业务主管单位审核，登记管理机关审查，准予登记和发放证书，以及刻制印章、办理税务登记和开立银行账户。

中国国内航线燃油附加费下调  暑期机票预订升温
中国国内航线燃油附加费再次下调，旅客暑期出行成本降低，机票预订升温。
据央视新闻报道，记者从国航、东航、南航等多家航司了解到，自星期天（7月5日）起，中国国内航线客票燃油附加费将下调。
调整后的收费标准为：800公里（含）以下航线燃油附加每航段为人民币50元（约9.5新元），800公里以上航线燃油附加每航段为人民币100元。相较于调整前，分别降低了30元和50元。
去哪儿大数据研究院研究员石珂分析，燃油附加费下调有利于降低旅客暑期出游成本，带动暑期国内国际航线机票预订升温。目前来看，受航司暑运增班等多重因素共同影响，今年暑期机票价格是近年来的“洼地”。

俄罗斯推出无人机识别系统 紧急情况下可一键迫降
俄罗斯企业开发出一款名为“红色按钮”的无人机识别系统，可在紧急情况下强制降落民用无人机。
新华社引述俄罗斯媒体报道称，俄罗斯系统由“格洛纳斯”公司和“元素”公司联合开发，可嵌入任何民用无人机中，利用俄罗斯“格洛纳斯”全球定位系统追踪无人机，在无蜂窝网络信号的情况下依然能够完成识别。当出现紧急情况或违规飞行时，监管部门可向无人机发送指令，强制其着陆。
俄罗斯媒体引述“红色按钮”开发企业管理人员的话报道，这一系统在指令传输环节采用加密技术，防止未经授权人员擅自篡改数据。目前系统已进行了首次测试，计划推广到无人驾驶车辆，尤其是无人驾驶卡车中。
近年来，俄罗斯民用无人机飞行次数迅速增长，主要用于物流、农业、监测、航拍等领域。俄罗斯无人机专家帕维尔·卡姆涅夫介绍，根据俄政府相关法令，在俄飞行的无人机系统必须有可识别或可被防空部门强制拦截的设计。
