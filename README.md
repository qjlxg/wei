（北京讯）中国正迅速扩张长时储能系统项目版图，并悄然领跑这个全球能源转型新赛道。与通常只能储存四小时或更短时间电能的传统锂离子电池不同，长时储能可储存和释放电能长达数小时甚至数天。 
研究机构彭博新能源财经星期一（3月2日）发布的报告显示，去年全球新增长时储能（ long-duration energy storage，简称LDES）装机容量达到创纪录的9.6吉瓦时，较2024年增长约30%。
报告指出，几乎所有在2025年建成的长时储能项目都位于中国。尽管西方经济体出现多家资金雄厚的初创企业，包括美国储能公司Form Energy以及加拿大先进长时储能技术开发商Hydrostor，但项目落地情况远不及中国。
随着间歇性可再生能源在全球电力供应中占比越来越大，长时储能对平衡能源供需至关重要，而中国这方面正遥遥领先于世界其他国家。
彭博新能源财经分析师认为，中国在长时储能领域的领先地位，很大程度上源于政策推动。北京在2023年将长时储能明确列为实现净零排放路线图中的关键组成部分。 
中国能源监管部门2024年并启动了56个新型储能示范项目，部分项目使用长时储能技术，由此引爆本轮快速扩张。
从在建和规划项目来看，中国的领先优势更为明显。目前全球规划中的长时储能项目总规模约为422吉瓦时，其中95%计划在中国落地。美国仅占全部规划项目的约2%，位居第二位却远远落后。

（旧金山综合电）《华尔街日报》引述知情人士说，OpenAI正在考虑与北约达成一项合同，将它的人工智能（AI）技术部署在北约的“非机密”网络。
OpenAI是美国AI巨头，它刚于上周宣布与美国国防部达成合作协议，将技术部署在五角大楼的机密网络中。此前，在美国总统特朗普的指示下，联邦政府机构纷纷停用AI公司Anthropic的产品，转向竞争对手如OpenAI。
Anthropic被封杀，是因为它反对美国军方和情报机构将AI技术用于大规模国内监视以及部署在全自主武器系统。 
OpenAI首席执行官奥尔特曼周二（3月3日）在全体员工大会上说，公司在跟五角大楼合作时，无权干预后者如何使用AI软件。他暗示，Anthropic就是因为试图对此有话语权，而与五角大楼关系闹僵。 
他说，五角大楼已表明会听取OpenAI在技术应用方面的专业意见，但并不希望OpenAI对某些军事行动的是非对错发表看法。
奥尔特曼也提到，跟五角大楼合作是正确的决定，但对于公司的品牌会带来极大挑战，在短期内对公司的形象会有极负面的冲击。
他此前坦言，OpenAI与国防部仓促达成的协议显得“投机且草率”，公司正在与国防部商讨，“在协议中加入一些条文，以非常清晰地阐明我们的原则”，其中包括确保AI不会用于国内监控，以及确保美国国家安全局等情报机构不可使用OpenAI的服务。
奥尔特曼也说，他希望缓解五角大楼与Anthropic之间的冲突，目前正在劝五角大楼放弃将Anthropic列入“供应链风险”的限制清单。通常只有敌对国家的公司才会被列入此清单。

破除职场内卷 中国政协委员建议推广弹性工作制
对于职场内卷现象，中国全国政协委员陆铭建议试点推广弹性工作制。
据《新民晚报》报道，上海交通大学安泰经济管理学院特聘教授陆铭说，破解内卷式加班陷阱是一项系统工程，需要多方联动、多措并举。
他建议，推广科学管理与柔性工作模式，引导企业引入先进管理方法，以综合绩效而非工时长短或简单的量化指标作为核心考核依据。
陆铭建议，支持在条件允许的行业与企业，试点并推广弹性工作制等灵活形式。
去年的中国政府工作报告中，首次提出要综合整治“内卷式”竞争，其中涉及因素既有企业之间的恶性竞争、无序价格战，地方政府招商引资时的过度消耗，也被解读为加大对员工劳动权益的保护，规范企业内部的运作。
此后，中国多家知名企业开启“反内卷”行动，提出强制员工下班、禁止无效加班、办公楼准时熄灯等措施，以规范企业内部管理。

伊朗无意与美国进行谈判 “将继续打下去”
已故伊朗最高领导人哈梅内伊的高级助手穆赫贝尔说，伊朗无意与美国进行谈判，并可根据需要继续这场中东战争。 
法新社报道，穆赫贝尔接受国家电视台采访时说，伊朗“不信任美国人，我们与他们没有任何谈判的基础”，并称“只要我们想打仗，我们就可以继续打下去”。
另据伊朗司法部门旗下的在线新闻网站（Mizan）报道，伊朗司法部长埃杰伊说，在由美以联合空袭伊朗引发的中东战争期间，任何协助伊朗敌人的人员“都将受到果断严厉的惩处”。
他说：“鉴于我们正处于战争状态，任何以言语或行动迎合侵略敌方非法欲望和利益的个人，都将根据现行法律法规受到果断严厉的惩处。”
