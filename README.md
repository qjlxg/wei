中美高官今在巴黎会谈 中国官媒：是机遇也是考验
中美高级官员星期天（３月15日）在法国巴黎举行两天经贸磋商，预计为三周后两国元首北京峰会铺路。中国官媒星期天发评论，指新一轮经贸磋商是机遇也是考验，呼吁双方应算大账，多看合作带来的长远利益，拉长合作清单、压缩问题清单，争取更多积极进展，为中美关系打开新的合作空间。
美国财政部长贝森特、贸易代表格里尔和中国副总理何立峰，将于星期天至星期一（16日）在巴黎举行第六轮经贸磋商。
新华社星期天发表题为“维护中美经贸关系健康、稳定、可持续发展”的评论文章，指新一轮中美经贸磋商在即，在全球经济复苏乏力、国际局势变乱交织的当下，国际社会普遍期待中美相向而行，为两国经贸合作与世界经济注入更多确定性和稳定性。
评论指出，去年以来，中美元首身体力行，在最高层次上保持了良好交往，为两国关系的改善发展提供了重要战略保障，也推动中美关系历经跌宕起伏实现了总体稳定。
中美双方已先后开展五轮经贸磋商，取得一系列积极成果，充分证明确保中美这两艘巨轮共同前行，不偏航、不失速，就必须坚持两国元首的战略引领不动摇，坚持落实两国元首重要共识不打折扣，坚持相互尊重、和平共处、合作共赢的原则不放松。
评论认为，中美应共同落实好、维护好两国元首共识，通过平等协商，妥善管控分歧，拓展务实合作，维护双方经贸关系健康、稳定、可持续发展。
此前五轮经贸磋商取得的成果还证明，只要双方秉持平等、尊重、互惠的精神，通过平等对话磋商，就能够找到解决经贸分歧的办法。
在最近一次、去年10月的吉隆坡磋商中，双方围绕美对华海事物流和造船业301措施、延长对等关税暂停期、芬太尼关税和执法合作、农产品贸易、出口管制等达成了一系列成果共识。
此后，中美经贸团队通过经贸磋商机制在各层级保持密切沟通，就落实两国元首釜山会晤共识和吉隆坡经贸磋商成果、解决彼此经贸领域关切及时交换意见。
新华社在评论中指出，解决中美经贸关系中长期积累的结构性矛盾和深层次分歧不可能一蹴而就。今年以来，又出现一些新的动向。美国宣布根据122条款对来自所有国家和地区的商品加征进口附加费。
此后，美国又相继以产能过剩和未禁止进口强迫劳动产品为由，分别对包括中国在内的16个经济体和60个经济体发起301调查，令全球贸易不确定性再度上升。中国正密切关注并将全面评估美国相关举措，将保留采取一切必要措施的权利，坚决捍卫自身合法权益。
评论认为，即将开启的新一轮经贸磋商能否取得积极进展，关键在于美国能否秉持对华理性务实的认知，按照相互尊重、和平共处、合作共赢的原则，在行动上做到与中国相向而行。
中美“合则两利、斗则俱伤”是经过实践反复验证的常识，中美“相互成就、共同繁荣”是看得见、摸得着的实景。新一轮经贸磋商是机遇也是考验。
评论最后呼吁中美应该算大账，多看合作带来的长远利益，秉持平等、尊重、互惠态度，拉长合作清单、压缩问题清单，争取更多积极进展，为双边关系打开新的合作空间，更好造福两国人民和世界人民。
