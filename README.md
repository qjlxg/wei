但自从美以袭击伊朗，中东爆发冲突以来，飞往中东枢纽的取消航班已超过2万7000架次。尽管部分航空公司已开始逐步恢复有限运营，但仍然有数以千计的旅客滞留在海湾地区，许多人不得不选择绕行且费用更高的路线，经由沙特阿拉伯和阿曼的机场继续行程。
Flight Centre指出，由于冲突导致航线受阻，商务旅客并未普遍取消澳大利亚与欧洲之间的商务出行，而是倾向寻找替代路线。
