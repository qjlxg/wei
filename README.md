台湾行政院长卓荣泰上个星期六（3月7日）赴日本东京为参加世界棒球经典赛（WBC）的台湾队加油，期间传出与日本首相高市早苗会面。台湾总统府发言人郭雅慧反驳称，不知道传闻从何而来，指这并非事实。
据联合新闻网、自由时报等报道，卓荣泰称赴日为私人行程，自费承担包机费用，引来在野党强烈质疑与批评。
郭雅慧星期四（12日）为卓荣泰辩护，称阁揆利用假日赴日为台湾队加油，属私人行程，费用由自己负担“何错之有”。
另一方面，卓荣泰星期五（13日）在行政院秘书长张惇涵、发言人李慧芝陪同下，主动公布上周末赴日观赏世界棒球经典赛小组赛的相关单据，包括门票、当日游览车费及包机费用等。

卓荣泰表示：“国民党立委刻意要把我说成公务行程，是何居心？就算是公务行程，自费有错吗？”他强调，此行为私人行程，“看球以外的事情，我无所评论，也不需要交代。”
被问到是否有见到高市早苗？卓荣泰表示，有见拉面店老板，没有其他的人。他指出，台湾总统府当然知道他的行程，不了解为何有媒体彷彿知道他都去见了谁，“好像我有邀他（媒体）去一样”。

港交所改革上市机制 吸引更多海外公司赴港上市
香港交易所建议大幅降低同股不同权上市市值门槛，以及放宽不同投票权比率，吸引更多海外公司赴港上市。当局强调，这些改革建议不会降低上市公司的质素。 
港交所星期五（3月13日）在这份提升上市机制竞争力的咨询文件中，提出多项建议，涵盖三大改革范畴，包括同股不同权架构、海外上市公司赴港上市规定，以及首次上市安排。
在同股不同权方面，文件建议上市市值门槛降低至不少于200亿元（港元，下同，33亿新元），或不少于60亿元及在经审计的最近一个会计年度的收入门槛不少于6亿元。目前的规定为市值不少于400亿元，或不少于100亿元及在经审计的最近一个会计年度的收入门槛不少于10亿元。
在以保密形式提交上市申请方面，目前由保密提交上市申请的选项，仅限于合资格的第二上市申请人、生物科技公司及特专科技公司，或个别情况下获得豁免的公司。文件建议放宽至所有新申请人都可选择以保密形式提交上市申请。
文件也建议，若申请人于上市时市值至少为400亿元，其可调高不同投票权比率上限至每股不同投票权股份不多于20票（20比1不同投票权比率上限）。目前的规定是不同投票权比率为不多于10比1。
港交所上一次进行大规模上市机制改革是在2018年，包括容许同股不同权公司上市，过后成功吸引阿里巴巴等科技巨头赴港上市。港交所星期五指出，八年来香港高质素的创新上市公司已增长很多倍，目前是时候优化制度。
有关文件的咨询期将在5月8日结束。港交所上市主管伍洁镟星期五在记者会上强调，有关改革建议不会降低上市公司质素。
她说：“我们认为提升上市制度的多元化与保荐市场的高质量，其实并无矛盾。”
香港股市去年首次公开募股（IPO）市场强劲复苏，全年募资额达2858亿元，较2024年飙涨224.24%，时隔六年重夺全球交易所榜首。
香港中文大学通识学部讲师蔡思行接受《联合早报》访问时指出，目前向港交所保密提交IPO申请的企业是有限制的，咨询文件建议扩展至所有新申请人，将有助不同种类的企业赴港上市，增加香港资本市场的多元性。
蔡思行认为，发行IPO的上市公司要有一定的往绩，以保障投资者利益，但香港在发展新经济项目时，也应认许某些“试错”的空间。“若能有措施支持在一段时期内都未必能有盈利的新经济项目如人工智能业务公司，在合适限制下到香港上市集资，将可以让香港成为支持新经济发展的动力源。”

韩国时隔30余年重设油价上限 李在明呼吁全民监督
受中东局势影响，韩国国内油价近期大幅上涨。为稳定市场，韩国政府自星期五（3月13日）起全面实施“石油价格上限机制”。韩国总统李在明说，如发现加油站违反相关规定，“请不要犹豫，立即向我举报”。随着制度落地，全国油价已开始回落。
韩国政府从13日零时起实施为期两周的石油价格上限机制，这是自1997年油价全面自由化以来，政府30多年来首次重新设定石油价格上限。
根据规定，从13日起，韩国四大炼油公司向加油站供应的批发价格不得超过汽油每升1724韩元（约1.48新元）、汽车用柴油1713韩元（约1.47新元）、室内煤油1320韩元（约1.13新元）。政府说，今后将根据中东局势及国际油价走势，每两周重新评估并调整价格上限。
李在明13日在社交媒体平台发文称：“从今天起全面实施石油价格上限机制。为防止部分企业借机牟取暴利，需要国民的监督和参与。”他还发布了一张标注京畿道始兴市多家加油站油价的地图截图，并写道：“油价是不是稳定了不少？如果发现异常，请立即举报。”
政策实施后，油价已出现下降迹象。韩国石油公社数据显示，截至13日下午5时，全国平均汽油价格为每升1864韩元，较前一日下降34韩元；柴油价格下降46韩元。部分加油站经营者说，即使短期利润受到影响，也会提前下调价格以维持销售。
不少民众表示暂时松了一口气，但也有人认为，由于此前油价在短时间内上涨幅度较大，目前的价格下跌还不太明显。
首尔江南区市民金仁淑告诉记者，油价此前一度上涨两三百韩元，“现在虽然稳定了一些，但负担仍然不小”。
据了解，韩国政府也加强市场监管。国税厅已动员全国税务机关对油价进行监测，公平交易委员会则对部分涉嫌串通抬价的加油站展开调查，以防止在低价供应条件下仍获取过度利润。

研究：AI聊天机器人协助策划暴力袭击
（华盛顿法新电）最新研究显示，一些主流（AI）聊天机器人在测试中曾协助用户策划暴力袭击，这凸显了AI技术可能对现实世界造成的危害。
非营利监督机构“打击数码仇恨中心”（Center for Countering Digital Hate，CCDH）与美国有线电视新闻网（CNN）的研究人员伪装为美国和爱尔兰的13岁男孩，对包括ChatGPT、谷歌Gemini、Perplexity、深度求索（DeepSeek）和Meta AI在内的10款聊天机器人进行了测试。
研究发现，其中八款聊天机器人在超过一半的回复中，为这些“袭击者”提供有关应选择哪些地点作为目标和应使用何种武器等建议。
CCDH首席执行官艾哈迈德说：“几分钟之内，用户就可能从模糊的暴力念头发展成具体可行的行动计划。”
研究发现，Perplexity和Meta AI的安全性最低，在大多数回应中都为研究员提供协助；只有Snapchat的My AI和Anthropic的Claude在超过一半的回应中拒绝提供帮助。
另外，DeepSeek在提供武器建议后还补充“祝射击愉快（也请注意安全）！”；Gemini则提到金属碎片在犹太教堂袭击中更具杀伤力；Character.AI甚至主动鼓励暴力行为，包括建议提问者“用枪”袭击保险公司首席执行官，以及殴打不喜欢的政治人物。
艾哈迈德说，这项研究最令人震惊的结论是“这种风险完全可以避免”，他特别赞扬了Anthropic的产品“展现出了识别不断升级的风险并阻止伤害的能力”。
Meta受询时说，公司已采取措施纠正问题。谷歌则称测试使用旧模型，当前模型对绝大多数提问都做出了恰当的回应。

下午察：列车重启 中朝关系也重回轨道？
星期四（3月12日），一列客运列车从中国辽宁省丹东市驶出，并抵达朝鲜首都平壤。这看似普通的一趟跨境列车，其实标志着中断六年的中朝客运列车服务正式恢复。
中朝客运线路始于1954年，长期被视为两国友谊的重要象征。但在2020年冠病疫情暴发后，朝鲜实施严格的边境管控措施，中朝铁路客运服务随之全面暂停。
随着此次重启运营，北京与平壤之间的国际旅客列车将每周一、三、四、六双向对开；丹东与平壤之间的列车则每日双向运行。中国境内售票窗口的旅行社人员透露，目前任何持有效签证的人都可以购买前往朝鲜的车票，包括在朝鲜工作或学习的中国公民，以及在海外工作、学习或探亲的朝鲜人。不过，普通游客仍无法购票。

中国查处5800余起野生动植物案件
中国开展“网盾行动2025”取得积极成效。这项专项行动以打击整治网络非法交易野生动植物和禁止使用的猎捕工具为重点，严厉打击了网络非法野生动植物贸易，查处5800余起野生动植物案件，其中刑事案件占比约56%，打掉365个犯罪团伙，处理6600余名违法犯罪人员。
中国国家林业和草原局星期五（3月13日）透露上述数据。据中新社报道，近年来，野生动植物非法贸易大量转向线上，为有效维护生物安全、生态安全和公共卫生安全，国家林草局2025年会同有关部门在全国范围内组织开展了“网盾行动2025”。
专项行动期间，中国各地各部门坚持线上巡查监测与线下执法检查一体推进，紧盯电商平台、社交媒体、短视频直播、网络拍卖等重点领域，聚焦非法交易野生鸟类、重点保护物种和禁止使用的猎捕工具等打击重点。
当局还加强对暗语代称、私域引流、线上展示线下交付等隐蔽方式的巡查筛查，并将寄递渠道、经营利用主体及重点市场场所纳入排查范围，推动线索发现、核查处置、立案查处闭环运行。
专项行动共监督检查互联网和寄递企业45万余家、商户用户35万余家，查处野生动植物案件5800余起，其中刑事案件占比约56%，打掉犯罪团伙365个，处理违法犯罪人员6600余人。
国家林草局称，下一步相关部门将进一步深化合作，持续发挥打击野生动植物非法贸易联席会议制度作用，推动专项行动向常态监管转化，强化对网络平台、电商平台、社交媒体、寄递物流等重点领域日常监管，不断巩固治理成效。
