芬兰计划取消长期以来禁止在本国领土部署核武器的规定。政府表明，此举将使芬兰在核威慑政策上与北约更加一致。
英国广播公司（BBC）报道，芬兰国防部长哈卡宁说，自2022年俄罗斯全面入侵乌克兰以来，芬兰以及欧洲的安全环境已经“发生根本且重大变化”。

根据芬兰1987年的《核能法》，在芬兰境内进口、制造、持有或引爆核爆炸装置均被禁止，即便在战争时期也是如此。
政府提出的修订将改变这一规定。哈卡宁说，若与芬兰的军事防御有关，将可以“把核武器带入芬兰，或在芬兰境内运输、交付或持有核武器”。

北约的核心原则是集体防御，即对任何成员国的攻击都被视为对所有成员国的攻击，这也是北约核威慑战略的基础。
由于北约内部拥有多个核武国家，这意味着若成员国遭受直接攻击，可能引发核回应。根据军控与防扩散中心的资料，美国核武器目前部署在多个欧洲国家。
芬兰政府的提案需要修改《核能法》和刑法。
芬兰执政的右翼联合政府在议会拥有多数席位，并表示提案将先进行公众咨询至4月2日，之后才会正式提交议会审议。
芬兰与俄罗斯接壤，边界长达1340公里，是欧盟和北约成员国中与俄罗斯边界最长的国家。芬兰领导人多次警告，自俄罗斯入侵乌克兰以来，芬兰的安全环境明显恶化。
芬兰于2023年4月成为北约第31个成员国，此举被广泛视为对俄罗斯总统普京的战略挫折，因为他长期批评北约向东扩张。芬兰的邻国瑞典则在2024年加入北约。
北约在北极和波罗的海地区，以及扩展后的东翼地区加强了军事存在。
