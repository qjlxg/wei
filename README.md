英国广播公司（BBC）报道，尽管遭到一些左翼人士的批评，法国参议院（上议院）和国民议会（下议院）星期二（7月21日）仍一致通过禁止15岁以下者使用社媒的法案。

中国广东省珠海市原副市长张宜生被查，成为近四个月以来第三名落马的原珠海市级领导层官员。
中共中央纪委国家监委网站星期三（7月22日）通报，珠海市政协原党组成员、专职常委张宜生涉嫌严重违纪违法，目前正接受广东省纪委监委纪律审查和监察调查。
这是自今年3月以来，第三名曾在珠海担任市级领导职务的官员被查。此前落马的广东省政协原副主席郭永航、广东省人大常委会财经委原副主任委员何宁卡，曾分别任珠海市委书记和珠海市市长。
据财新报道，张宜生在6月底至7月初被有关部门带走。知情人士称，张宜生长期参与珠海城市开发和重大项目协调，他主政高新区及担任副市长期间的工作经历，可能是调查关注的方向。
公开资料显示，今年61岁的张宜生曾任珠海市香洲区委副书记、香洲区委书记、珠海（国家）高新技术产业开发区党委书记等职。2017年1月，他出任珠海市副市长，并兼任珠海高新区党委书记，同年3月卸任高新区职务。
担任副市长期间，张宜生分管领域多次调整，先后负责国土规划、城乡建设、城市更新、西部城区开发，以及工业和交通运输等工作。2021年1月，他曾在新闻发布会上介绍，珠海当年计划推进79项交通基础设施重点项目，涵盖高铁、城际轨道、高速公路、城市快速路及机场改扩建等。
2021年7月15日，珠海兴业快线（南段）一标段石景山隧道施工段发生坍塌透水事故，造成14名施工人员遇难，直接经济损失逾3678万元人民币（701万新元）。广东省政府同年10月批复事故调查报告，认定事故为重大生产安全责任事故。
事故发生后，纪检监察机关对属地党委政府、监管部门及国企等27名责任人员进行追责问责。时任珠海市副市长的张宜生因对事故负有责任，被给予政务警告处分。
2022年珠海市领导班子换届后，张宜生不再担任副市长，转任珠海市政协党组成员，直至此番被查。

中国最后一名长征女红军去世
中国最后一名参加过长征的女红军王全英去世，享年105岁。
据央视新闻报道，记者从王全英的大女儿刘桂华处了解到，王全英于星期三（7月22日）凌晨0时30分去世。
王全英1921年生于四川阿坝州金川县一个藏族家庭。1935年6月，14岁的她加入中国工农红军，随红四方面军参加长征，负责护理伤员、筹粮运粮等工作。
1936年春，红军在丹巴草地作战时被江水冲散，王全英与部队失去联系，未能走完长征全程，此后隐姓埋名。
1984年中国有关部门找到王全英，为她颁发“红军流落人员证书”并落实相关政策。后来，王全英又先后获得中国工农红军长征胜利80周年、庆祝中华人民共和国成立70周年纪念章。

伊朗多地遭美军袭击 首都德黑兰启动防空系统
伊朗媒体星期三（7月22日）报道，当天凌晨，包括首都德黑兰在内的伊朗多地遭到美军袭击。目前尚无人员伤亡报告。
22日凌晨3时（新加坡时间22日早上7时30分）许，新华社记者看到，德黑兰防空系统启动，在夜空中密集拦截不明飞行物，远处可听到爆炸声。德黑兰防空系统多次启动应对敌方袭击。
伊朗媒体引述当地官员的话报道，伊朗布什尔市郊区有两地遭到美军袭击，尚无人员伤亡报告；大不里士市附近一个军事设施遭到空袭，市内未遭袭击；阿卜达南附近一处地点遭两枚导弹袭击，目前尚未收到人员伤亡报告；坎加瓦尔遭到美军导弹袭击；恰巴哈尔港传出四声爆炸声，随后听到敌方战斗机的飞行声；锡里克地区居民听到来自海上方向的爆炸声。此外，霍马、卡布达拉汉吉、马赫夏赫尔港等地也遭美军袭击。
美国中央司令部称，美军于美国东部时间21日晚上8时15分（新加坡时间22日早上8时15分）结束对伊朗境内军事目标新一轮打击，这是美军连续第11晚采取此类行动。 
打击目标包括伊朗的军事行动中心、海上作战设施、飞机机库、无人机储存设施以及军事后勤基础设施，“以进一步削弱伊朗在霍尔木兹海峡威胁商业航运的能力”。

欧盟重罚阿里速卖通 中国商务部表示强烈不满
针对欧盟委员会对阿里巴巴旗下全球速卖通处以5.5亿欧元（8.1亿新元）罚款，中国商务部表示强烈不满和严正关切。
欧盟委员会本周一（7月20日）以销售非法商品，包括仿冒服装、不安全玩具、危险化妆品等为由，对阿里巴巴旗下的零售电商巨头全球速卖通处以5.5亿欧元的罚款。
中国商务部星期三（22日）以答记者问形式回应，对此表示强烈不满和严正关切。
中国商务部新闻发言人表示，中方坚决反对欧方以平台监管为由设置数字壁垒，并采取歧视性措施，限制打压中国电商企业在欧正常经营。
“中方敦促欧方停止利用法律条文的模糊性滥用自由裁量权，公平、公正对待中国企业。中方将坚定支持中国企业运用法律武器维权，并将采取有力措施坚定维护企业利益。”
阿里巴巴此前已经表示对欧盟罚单不认同，认为该处罚“不成比例”，忽视了平台已投入的重大合规改进，并声明将对该决定提起上诉。

福利审批不能全交给AI 澳洲收紧自动决策规定
澳大利亚政府将收紧联邦机构使用人工智能（AI）的规定，确保涉及福利和公共服务的决定公平、准确，同时加强个人资料和消费者权益保护。
澳洲广播公司（ABC）报道，澳当局星期天（7月19日）公布一系列AI消费者保护重点。包括福利机构Centrelink和澳洲服务局在内的联邦机构，日后使用AI协助决策时，须遵守新框架，确保决定“公平、准确”，并接受适当监督。
政府近年来面对越来越大压力，要求在社会福利等涉及民众切身权益的领域保留人工介入，同时利用AI提高行政效率。
去年，澳洲联邦监察专员发现，近1000名求职者的收入补助因为自动化取消机制被违法停发。政府用算法处理居家养老服务评估，以及全国残障保险计划参与者的预算，也引发争议。今年早些时候，监察专员已开始调查养老评估工具。
就业部长里什沃思今年5月宣布改革失业援助制度时承诺，求职者补助的审批过程将继续由人类参与。
政府也准备重新审视消费者保护和隐私保护规定。其中一个重点是“监控式定价”，也就是零售商收集消费者的行为数据，再根据不同个人制定不同价格。此前，政府虽然加强了部分隐私保护，但未能推动限制定向广告的措施。原本提出的改革包括：让民众可以选择退出定向广告，并限制企业使用敏感个人信息。新政策还将研究如何通过消费者法加强监管，并把AI企业纳入“数字谨慎责任”立法，要求平台主动识别和处理可能造成的伤害。
澳洲总理阿尔巴尼斯此前说，AI带来的改变已经发生，劳动者必须参与塑造转型，否则将失去发言权。他也承诺，不会允许AI企业在没有补偿的情况下，使用澳洲书籍、音乐和艺术作品训练模型。
