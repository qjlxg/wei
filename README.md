据日前报道，韩国前总统尹锡悦4月在二审中被判处有期徒刑七年，刑期较一审明显加重。裁决不仅是围绕“戒严事件”的首次二审判断，也是首尔高等法院内乱专责法庭成立以来的首个判决，多项一审无罪认定被推翻，具有标志性意义。
今年1月16日，尹锡悦一审被判有期徒刑五年。
首尔高等法院内乱特别法庭4月29日宣判，认定尹锡悦构成妨碍特殊公务执行、滥用职权妨碍权利行使、制作虚假公文及教唆违反《总统警卫法》等多项罪名。
在维持一审有罪部分的同时，法院将大部分无罪指控改判有罪，但量刑仍低于独立检察组此前提出的10年有期徒刑。
判决指出，尹锡悦去年1月动员总统警卫处人员阻挠公调处执行逮捕令，构成妨碍特殊公务执行及滥用职权；同时，在内乱调查过程中指示删除涉军方指挥官通信记录，被认定为教唆违法。

自民党联合会议：中国发射导弹对日本和亚洲安全构成巨大威胁
中国星期一（7月6日）向太平洋海域试射一枚带有训练模拟弹头的潜射战略导弹后，日本自民党外交部会等星期三（7月8日）举行联合会议，与会者纷纷表达严重关切，并称中国此举对日本及亚洲地区的安全保障构成巨大威胁。
据日本放送协会（NHK）报道，日本自民党星期三举行联合会议。自民党外交部会部会长高木启在会上指出：“此事极具象征性地显示出了究竟是哪个国家在进行不透明的军事扩张。这对我国乃至亚洲和大洋洲的安全保障构成了极其巨大的威胁。”
国防部会部会长本田太郎说：“此次发射的导弹有可能转化为核战斗力，这给国际社会带来了巨大影响。研究出台妥善的应对措施至关重要，我国推进加强防卫力量的方向不会改变。”
据报道，在日本政府方面对各国的反应等进行说明后，有与会者发表意见称，应当让国际社会更广泛地关切中国日益频繁的军事活动。
另据路透社报道，美国国务院一名官员说：“中国在发射前几个小时才通知美国，而且提供的细节不充分，远未达到其他所有五常（P5）核武器国家所遵循的标准。”
五常指的是联合国安理会五个常任理事国。根据1968年《不扩散核武器条约》，这些国家也是唯一被公认为核武器国家。
这名官员也说：“此次试射正值中国迅速且不透明地扩充核武器之际，引起了该地区的极大关注。”
中国海军一艘战略核潜艇星期一向太平洋海域试射一枚带有训练模拟弹头的潜射战略导弹。这是中国时隔近两年后，再度公开在太平洋试射洲际弹道导弹。
据法新社报道，监测人员称，这枚导弹似乎落入位于所罗门群岛、瑙鲁和图瓦卢之间的太平洋海域。
中国外交部发言人毛宁说，这是中方年度军事训练的例行安排，并事先向有关国家通报，符合国际法和国际惯例，希望有关国家不要过度解读。

中俄联合演习进入海上演练阶段 设置防空反导对海打击课目
中俄“海上联合-2026”联合演习星期四（7月9日）进入海上演练阶段。根据计划，海上演练阶段设置防空反导、对海打击等课目。
据新华社报道，中俄“海上联合-2026”联合演习港岸筹划阶段完成，星期四正式进入海上演练阶段。
港岸筹划阶段，演习联合导演部、联合指挥部组织多轮指挥推演和战术协同。双方商定了作战行动要点、演练规则，细化了海上协同流程和应急处置预案。
根据计划，海上演练阶段在青岛附近海空域进行，设置联合侦察、防空反导、对海打击、联合援潜救生等课目。
中俄联合演习星期一（7月6日）开幕，参演舰艇将赴青岛附近海空域组织联合侦察等多个课目演练，并开展实际使用武器训练。
此前，有“航母杀手”之称的瓦良格号导弹巡洋舰，星期天上午率俄海军舰艇编队抵达青岛一座军港靠泊，中国军方在码头举行欢迎仪式。
此次俄军参演兵力由瓦良格号、护卫舰凛冽号、大型潜艇乌法号，以及救生船伊戈尔·别洛乌索夫号等组成。
解放军今年则派出更多更新的战舰参演，以北部战区海军为主，包括“万吨大驱”055型导弹驱逐舰鞍山舰、054A型导弹护卫舰芜湖舰、052DL型导弹驱逐舰开封舰。055型是中国海军现役第四代驱逐舰，也是航母编队的核心主力。
