上周末，多瑙河在罗马尼亚境内的水位降至1996年以来的最低水平，导致农民面临灌溉限制，沿岸的航运和旅游业也受到部分干扰。
路透社气候监测数据显示，今年欧洲各地的水道均受到持续热浪和干旱的影响。周一（7月20日），西欧地区的平均最高气温预计将达到26.2摄氏度，比7月20日的正常最高气温高出2.7摄氏度。
罗马尼亚国家水务管理机构说，星期天（19日），这条欧洲第二长河在罗马尼亚入境处的水流量为每秒1700立方米，而7月份的平均流量通常在每秒4700立方米左右。

水务管理部门说，受多个县普降大雨影响，预计多瑙河水位将从当地时间周一开始逐渐回升。

伊朗战事推高能源成本 全球清洁能源需求升温
（北京彭博电）洁净能源过去主要被视为应对气候变化的工具，但俄乌战争和伊朗战事接连扰乱全球油气市场，也暴露出各国依赖进口燃料的风险。相比石油和天然气，太阳能和风能可以在本地开发利用，成本也比较稳定，因此能源安全正成为各国加快能源转型的新动力。
彭博新能源财经（BloombergNEF）数据显示，今年3月至5月，也就是伊朗战事爆发后的前三个月，中国家用锂离子电池出口量同比增加45％，其中对澳大利亚和印度的出口分别增长103％和77％。
中国太阳能电池出口也在3月和4月同比分别增加80％和60％。不过，部分增长是因为企业赶在中国4月取消太阳能产品出口退税前提前出货；到了5月，出口量同比下降11％。
菲律宾是需求快速增长的一个例子。当地住宅电价在东南亚名列前茅，今年前五个月从中国进口的太阳能板同比增加超过一倍，目前已超过巴基斯坦，成为仅次于荷兰的中国太阳能板第二大买家。
汽油价格上涨也让电动车更有吸引力。战事爆发后的前三个月，中国电动车出口量同比增加57％，其中对菲律宾、泰国和越南的出口分别增长95％、70％和64％。今年3月，超过30个国家的电动车月销量创下新高。
面对能源冲击，亚洲和欧洲多国也推出措施，加快能源转型。
中国计划到2030年让风能和太阳能占总装机容量超过一半，目前占比已达47％；抽水蓄能容量则计划从66吉瓦增至160吉瓦。菲律宾为住宅洁净能源设备提供低息贷款，并计划加快大型可再生能源项目；法国提高电动车和热泵补贴，英国则要求新建住宅安装热泵和太阳能板。
不过，并非所有国家都选择加快转型。一些政府暂时削减燃油税，以减轻民众负担；印度尼西亚则提高煤炭产量配额，增加国内供应。
中国可能成为这股趋势的最大受益者。目前，中国生产全球约92％的太阳能电池、83％的电池芯和66％的电动车。如果化石燃料价格持续高企，各国继续优先考虑能源安全，全球对中国洁净能源产品的需求可能进一步上升。
不过，近期出口激增是否会成为长期趋势仍不确定。部分增长可能来自退税政策变化引发的提前采购，但若化石燃料价格长期居高不下，各国继续把能源安全列为优先事项，洁净能源需求上升的动力可能持续。
