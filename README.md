上述调查指哈佛大学涉嫌向中国留学生提供经济援助，以换取来自中国的捐赠。
美国司法部助理部长迪伦（Harmeet Dhillon）在声明中说：“学校不能一方面接受联邦资金，过后再接受外国资助来发放故意排除美国公民的助学金。”
这是特朗普政府向这所常春藤盟校挥出的又一记重拳。自去年以来，哈佛一直与白宫就数十亿美元的联邦资金冻结问题进行着法律诉讼。
哈佛发言人纽顿在声明中说，哈佛“遵守必须呈报捐赠的法律，也一直遵循校方的法律义务，从未根据种族群或国籍发放助学金”，并补充“我们正审阅司法部发给哈佛的最新信函，将就相关议题与政府接洽”。
美国政府上述举动显示，官方可能加大对美国高校中国影响力的打击力度，尤其针对哈佛大学。去年，美国国务卿鲁比奥提出，调查哈佛是否违反制裁，与中国机构进行学术合作。
美国总统特朗普去年签署行政令，要求私人大学要提供详尽的外国捐赠资料，美国司法部就此开展审查。

迪伦在给哈佛的信中写道，校方必须在7月31日前向调查员提供记录与中国相关的捐赠来源资料的文件。如果不遵从，将面对包括联邦资助被暂停的措施。
