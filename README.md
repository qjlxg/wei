位於菲律賓以東海域的潛在風暴預料較大機會週末在本港以東登陸，但其與本港的距離須視乎中心整合位置和速度，以及季風槽和副熱帶高壓脊的互動。
潛在風暴過去一兩日發展速度較慢，尚未建立和鞏固中心。人工智能模式預測潛在風暴短期內中心整合速度較快，繼而較早進入南海，預測登陸位置亦略為偏東。部分傳統電腦模式則預測潛在風暴仍須一段時間整合中心，因此起跑時間較遲，預測登陸位置亦略為偏西。
人工智能和傳統電腦模式對風暴初期中心整合的預測有別，在去年數個案例亦有出現，但現時未有足夠數據判斷誰優誰劣。
至於潛在風暴進入南海後的路徑北份量，則須視乎副高往南伸展的幅度。早前傳統電腦模式預測季風槽有殘餘渦度，因而阻礙副高南伸。然而，傳統電腦模式過去一兩日有加大副高南伸幅度，以致潛在風暴預測路徑往東修正的趨勢。此外，新一輪西風波動會在週末通過中國東北部，亦令副高短期內較難進一步往西伸展。因此，現階段預測潛在風暴東登的機會較大。
東登風暴須較接近本港，以及登陸後採取較為偏西的路徑，才會對本港造成顯著的風雨影響。相信潛在風暴中心完成整合後，本港週末風雨預測會更為明朗。

香港住宅租金料创10年来最大暑期涨幅
随着香港房地产市场复苏势头增强，当地住宅租金预计将创下10年来最大暑期涨幅。
据彭博社报道，中原地产数据显示，今年上半年，香港住宅租金较去年下半年上涨近4%。房地产中介预计，随着中国大陆专业人士和学生持续赴港，加上外籍人士的租房需求增加，香港住宅租金将出现2016年夏季以来的最大涨幅。
网上租赁平台Spacious.hk首席运营官费舍尔（James Fisher）说：“学生和大陆人士仍是租赁市场炽热的主要推动力。与过去几年夏季租金的季节性上涨相比，今年最大的不同无疑是外籍人士市场需求强劲。”
报道称，租金飙升反映香港楼市正从持续多年的低迷中稳步复苏。截至今年5月，香港房价已连续12个月上涨，创2018年以来最长连涨纪录。此前，大批外籍人士在冠病疫情后离开香港，楼市一度承压。
楼市回暖有利于在上环和山顶等热门地区持有住宅单位的个人业主。尽管香港房地产发展商持有的出租物业不多，但租赁市场蓬勃可提高投资者需求，进而带动住宅销售。
不过，租金上涨也加重学生和新抵港人士的负担。按每平方英尺计算，香港部分住宅的租金甚至高于纽约和伦敦。
过去几年，香港推出多项新工作签证计划，吸引中国大陆专业人士赴港，为当地租赁市场带来支持。港府推动大学招收更多非本地学生，也进一步推高租房需求和租金价格。
在邻近商业区、以中产阶级居民为主的上环，一套面积约500平方英尺的典型两居室公寓，平均月租约3万4000港元（5591新元），与纽约曼哈顿同等面积的公寓相若。
香港住宅一向以面积较小、价格较高著称。尽管纽约租金近年来持续上涨，缩小了两地差距，但香港住宅通常仍比其他金融中心更狭小、昂贵。
Spacious.hk数据显示，山顶、清水湾、苏豪区和浅水湾等外籍人士聚居区的租金涨幅高于平均水平。与此同时，来自海外、浏览该平台出租房源的流量同比增加9.4%，其中来自美国、新加坡、英国、澳大利亚和日本的搜索量均录得双位数增长。
分析师预计，香港住宅租金将继续上升。美联物业研究分析师沈锐培（Benny Sham）认为，目前推动租金上涨的两大动力，即人才签证计划和赴港学生人数，短期内将继续增长。

英首相宣布酒吧等场所明年起商业税减20%
英国首相伯纳姆宣布，英国全国的酒吧、俱乐部和现场音乐场所的商业税明年4月起将下调20%。
英国政府星期四（7月23日）发布声明说，这项减税措施预计惠及近3万2000家企业，一般酒吧明年可节省约1100英镑（约1726新元）。
这是伯纳姆出任首相以来，三天内宣布的第三项政策。
唐宁街说，这项减税措施每年耗资约1亿英镑，政府将通过检讨目前给予部分“对当地社区没有积极贡献”企业的税务优惠来筹资，例如电子烟店。此外，当局也将加强打击通过网络市场销售、却不履行纳税义务的企业。
不过，最大的现场音乐场所将不享有这项20%的税务减免，具体详情将在秋季财政预算案中公布。
伯纳姆说，这项政策表明政府将“支持人们希望在社区中看到的商业场所”，而不是袖手旁观，任由“备受珍视的场所从当地商业街上消失”。

官媒：重庆山体崩塌失联人员存活率很低
致多人被埋的重庆彭水山体崩塌事故已过去近一周，重庆官媒引述救援队现场指挥长称，由于现场崩塌体垂直压覆，“失联人员存活率很低了”。
重庆市彭水县上周五（7月17日）上午9时许突发山体垮塌，多栋居民楼和一辆载有多人的客运巴士被埋。根据官方事发当天公布的初步统计，这起事故至少造成八人死亡和34人失联。当局此后没有再进一步更新伤亡和失联数字，也未对外说明是否有人员获救。
据中共重庆市委机关报《重庆日报》报道，根据“深度救援”需要，彭水山体崩塌现场在星期三（7月22日）进行了新一轮巨石爆破。
报道引述中国自然资源部汛期防御重庆驻守专家组组长李长明称，彭水是个千年老县城，长时间看地质是比较稳定的，自然界的地质演变导致此次山体崩塌，崩塌下来的石头特别巨大，最大的巨石达到了5300立方米。
李长明称，爆破后，堆积的土石方达到7.8万立方米，这个土石量也非常巨大。崩落的巨石块成分以硬质灰岩为主，质地较为致密完整，从30米高的山体近乎垂直掉落，产生的势能巨大、冲击力极强，危害性极大，造成了巨大的破坏。同时，这些巨石坠落后，活动的距离不远，因此没有完全解体。“这种情况下，生命的存活率很低，科学救援尤其重要。”
李长明说：“现在到了深度救援阶段，要实行爆破，更要确保作业的安全，救援人员要对崖体进行监测并开展数据分析，分析评估后再进行下一轮的作业，这都需要时间。目前，救援工作一直在进行中。”
据报道，经监测数据和专家分析评估，现场指挥部慎重研究，针对巨石，采取定向爆破、逐层推进的方式进行清理，严防爆破有害效应发生。目前已进行了三次爆破，核心区外露的最大块石障碍已基本排除。
中国安能重庆救援队现场指挥长冯宇说，现场已采取多种救援措施，开展多轮搜救，但现在到了深度救援阶段，由于现场崩塌体垂直压覆，失联人员存活率很低了，但按照“救人为先”的原则，在每轮爆破之前，救援人员还是进行多轮生命探测。
冯宇介绍，救援队下一步将对现场石渣进行清运，针对剩余的部分裸露及埋在石渣下的大型块石，将灵活采取定向爆破、机械破碎等方法进行处置。爆破作业完成后，救援人员将对巨石掩埋区域展开拉网式、地毯式搜寻，尽最大努力持续开展搜救工作。

英国政府上周宣布将中国民营企业控股的英国钢铁公司收归国有后，中国外长王毅呼吁英国为在当地经营的中企提供公平、公正、非歧视的营商环境。
据新华社报道，王毅星期四（7月23日）在菲律宾马尼拉与英国外长米利班德会面时作出以上表述，并呼吁英国恪守一个中国原则，维护中英关系的政治基础。
英国政府上星期四（16日）宣布，将把中国民企敬业集团旗下的英钢公司全面国有化。即将离任的英国首相斯塔默在声明中强调，这项决定将“保障英国未来的钢铁生产，保护技能型就业，并维护一项至关重要的国家能力”。
敬业集团2020年以7000万英镑（1亿2000万新元）收购英钢。英钢位于英格兰北部斯肯索普（Scunthorpe）的工厂，有英国仅存的两座使用原生矿石炼钢的高炉。
英钢去年3月称，母公司敬业集团已投入超过12亿英镑维持公司运营，但每天亏损约70万英镑。英钢当时评估，两座高炉财务上不可持续，次月传出英钢取消原材料订单，引发关于英钢将停炉关厂的担忧。
为保留其原生钢生产能力，英国国会去年4月通过紧急法案接管英钢，政府今年5月宣布将提出立法，促成英钢国有化。
对此，中国商务部新闻发言人上星期五（17日）指责英国“罔顾敬业对英经济社会做出的重要贡献，以国家安全之名，强行接管、进而对英钢实施国有化，严重损害了敬业的正当权益，严重挫伤了中国企业对英投资信心”。
发言人也说，将支持中企运用法律手段维权，并将采取有力措施坚定维护中企利益。

鲁比奥与拉夫罗夫会谈 讨论美俄关系及俄乌战事
美国国务卿鲁比奥星期四在菲律宾马尼拉与俄罗斯外长拉夫罗夫会面，双方讨论美俄关系及结束俄乌战争的必要性。
法新社报道，两人在亚细安峰会期间举行会谈。美国国务院说，双方也可能讨论中东冲突。俄罗斯近期曾谴责美国袭击伊朗。
鲁比奥星期三（22日）说，俄乌战争在许多方面阻碍美国和俄罗斯就其他议题寻求共识，但这不意味着双方不会继续寻找潜在合作领域。他也说，希望看到俄乌战争结束。
俄乌停火谈判目前陷入停滞，美国的注意力转向中东，而俄罗斯仍坚持要求乌克兰割让领土。
乌克兰总统泽连斯基星期四说，他已与美国特使威特科夫和特朗普女婿库什纳就“如何重新推动外交努力”进行“良好且重要的对话”。
美国总统特朗普支持乌克兰对俄罗斯能源基础设施发动远程无人机袭击，认为这可能有助于结束战争。乌克兰近期加强袭击俄罗斯炼油厂、油库和海上货船，导致俄国内出现燃料短缺。
俄军则持续对乌克兰发动导弹和无人机袭击。6月底，俄军对基辅发动大规模空袭，造成至少30人死亡。

莫迪誓言严惩考试舞弊涉案人员
印度总理莫迪周四誓言，将对引发学生愤怒的考试舞弊事件涉案人员给予“严厉惩处”。这是莫迪首次就针对政府大规模抗议活动直接发表评论。
莫迪周四（7月23日）在社媒发文说，“没有什么比我们年轻人的福祉和未来更重要”，并承诺设立快速审理法庭，以惩处涉及试题泄露事件的人员。
他补充说：“那些试图损害我们年轻人未来的人，绝不会被放过。”
印度数以万计的“蟑螂运动”支持者周三继续在首都新德里聚集，与警察爆发新一轮冲突。示威者主要是学生，他们坚持要求教育部长普拉丹辞职，为试题泄露、技术故障等考试违规问题负起责任。 
这些抗议活动已成为莫迪自2024年连任第三届总理以来面临的最大挑战之一。
警方说，连日的暴力冲突至少造成178人受伤，其中包括安保人员。

印度阿萨姆邦洪灾一周夺走36人命
印度东北部阿萨姆邦持续暴雨引发洪灾，过去24小时至少10人死亡，使过去一周的死亡人数增至36人。当地官员形容灾情造成“前所未有的破坏”。
路透社报道，阿萨姆邦灾害管理部门星期三（7月22日）发布通报说，暴雨已影响全邦11个地区超过65万人。
阿萨姆邦首席部长萨尔马在社媒发文说，这次洪灾造成的破坏规模“前所未有”，但未透露更多详情。
邦政府报告指出，这次洪灾主要由邻邦那加兰邦（Nagaland）7月18日至20日的特大暴雨引发，导致河流水位异常上涨。
阿萨姆邦是世界主要河流之一布拉马普特拉河的流经地区。每逢季风季节，河流水量就会增加，几乎每年都给当地造成洪灾和破坏。

中国驻菲大使回应官媒猴子视频：政府管不了所有媒体内容
中国官媒视频将菲律宾人描绘成猴子引发马尼拉外交抗议。中国驻菲律宾大使井泉称，媒体行为与中国政府无关，北京并不能控制中国所有媒体机构制作的内容。
菲律宾总统小马可斯星期二（7月21日）就阿云津礁（中国称仁爱礁）冲突在总统府召见井泉。在与小马可斯进行数小时的闭门会谈后，井泉接受了多家菲律宾和中国媒体的群访。针对中国官媒《中国日报》在一段人工智能（AI）视频中将菲律宾人描绘成猴子引发的种族歧视争议，有记者提问北京对相关内容的立场。
综合菲律宾ABS-CBN新闻网和《马尼拉公报》报道，井泉用英语回应说：“我认为我们的发言人已经非常明确地表示，这只是媒体的行为，与政府没有任何关系。”
中国外交部发言人林剑上周曾在例行记者会上说，对《中国日报》的视频不予置评，相关内容不代表中国官方立场。
井泉还补充说，中国的媒体数量众多，“我们不可能要求每家媒体、每个视频、每篇文章、每份报纸都去管控”。
井泉说：“实际上你知道，当我们说我们不予置评时，那就是我们的立场。”
对此，菲律宾左翼政党阿克巴扬党（Akbayan Party）党代表森达尼亚星期三（7月22日）发声明说，《中国日报》是中共宣传机构控制的媒体，井泉的言论是在公然误导菲律宾人。
2016年的南中国海仲裁案裁决出炉10周年之际，中共中央宣传部主办的《中国日报》英文版（China Daily）7月10日在社媒平台X和脸书上传了一段约一分钟的视频。
视频中，一只穿着菲律宾传统服饰“巴隆”（barong）、戴着斗笠且神态惊恐的猴子，被两只分别印有美日国旗的巨手推上摇摇欲坠的卡拉OK舞台要求唱歌。在中国互联网上，一些中国网民长期以来使用猴子来贬损菲律宾人形象。
这只猴子唱道：“我们主张绕过中国推进所谓海域划界谈判。”紧接着，画外音厉声呵斥：“唱反了！蠢猴子，我们给你的不是这份歌词！”
随后，猴子掏出一张印有“南中国海仲裁案裁决”英文字样的纸张，随即被巨手抛入海中，并遭水炮轰击。视频结尾，被揉烂的裁决纸张漂浮在水面上。
菲律宾外交部7月17日发声明说，菲国副外长埃雷拉林已当面向井泉提出外交抗议，要求立即删除视频，称这类内容只会加剧菲中两国间的不信任。声明称，视频诉诸于“对菲律宾人贬低性、非人化和种族歧视的描绘”，已完全超出合法的政治评论范畴。
