总统尼科斯·克里斯托杜利德斯表示，此次为自1986年利比亚武装分子火箭袭击以来首次针对该英军设施的攻击，局势显著升级。他称凌晨0:03一架沙阿德（Shahed）型无人机撞入军事设施，仅造成轻微破坏，并强调“我们的国家不会以任何方式参与，也不打算参与任何军事行动”。

英国首相基尔·斯塔默称，英国已同意美国请求，允许使用其基地对伊朗导弹储存库或发射装置实施防御性打击。
