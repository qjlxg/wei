据萍乡学院官网上星期六（3月7日）消息，也是萍乡学院副院长的吴代赦发布了题为《前排的幸福眩晕——近距离感受习近平总书记的风范》的文章。文章首先写道，上星期五（3月6日）下午，习近平看望参加全国政协十四届四次会议的农工党、九三学社、医药卫生界、社会福利和社会保障界委员，并参加联组会，听取意见和建议。吴代赦作为九三学社界别委员现场聆听了习近平的讲话。
根据文章，吴代赦回忆，他走进全国政协界别联组会议的会场友谊宫聚英厅时，一度在座位分布图上找不到自己的名字，工作人员随后指引说，他的座位可能在最前排。“我循着指引望去——嘿！还真在正面对习近平总书记的前排，我的名字静静地待在那里。那一瞬间，一股幸福的眩晕感涌上心头。”
文章写道：“这不只是一个位置的巧合，更是一次近距离感受领袖风范的珍贵机缘。接下来的差不多两个小时，我就这样坐在总书记的前方，见证着他的每一次提问、每一句点评、每一个眼神。”
吴代赦还以五个“范”形容习近平的形象，分别是科学家的“专业范”、实干家的“务实范”、战略家的“格局范”、长者的“人情范”，以及倾听者的“亲和范”。
另据《明报》报道，吴代赦接受江西电视台采访时也回忆说：“我想这怎么可能呢？结果一看，哇，我的名牌就摆在总书记的正对面，一下子幸福瞬间笼罩了我，真的有点眩晕的感觉。”

中国海军055型导弹驱逐舰东莞舰和安庆舰亮相
中国央视的《新闻联播》公开了海军最新两艘055型导弹驱逐舰，舷号分别是东莞舰（109）和安庆舰（110）。
《新闻联播》星期天（3月8日）发布了东莞舰和安庆舰的画面，《环球时报》报道称，这意味着这两艘舰艇已经服役。自此，公开亮相的“万吨大驱”增加到10艘。
《环球时报》引述军事问题专家宋忠平分析称，虽然从表面上看不出最新的两艘055型导弹驱逐舰，与此前八艘有什么明显区别，“但其‘五脏六腑’发生了变化”。
军事问题专家张军社说，首先，在舰体上层建筑上采用了新型材料，应用了新的设计理念，大幅度减少了雷达反射截面积（RCS），降低了电磁信号特征，进一步提升了舰艇隐身性能。其次，舰艇换装了新型雷达，对隐身战机的探测距离大幅增加，侦察预警和信息感知能力得到进一步增强。
第三，动力系统有所改进。改进后的动力系统推力更大、功率更强。第四，可能装载更先进的导弹，尤其是高超音速导弹，使得导弹打击距离更远、威力更大。
张军社也说，当前中国已有三艘航母，未来航母与两栖攻击舰的数量还将进一步增加。同时，远海交通线、能源通道的守护以及海外利益的维护等需求，均需要大量大型水面作战舰艇作为支撑。“现有055型万吨驱逐舰的规模仍显不足。”
据央视新闻去年6月报道，055型导弹驱逐舰突破了大型舰艇总体设计、信息集成、总装建造等系列关键技术，具有较强的信息感知、防空反导和对海打击能力，是当前中国海军装备技术水平的综合体现。
