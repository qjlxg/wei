针对中国男子篮球职业联赛（CBA）上海队外籍球员被检出兴奋剂事件，中国篮球协会发布声明，表示对任何形式的兴奋剂违规行为持“零容忍”态度，将对违规方进行严肃处理。
中国反兴奋剂中心星期五（7月10日）在网站公开，CBA上海队外籍球员哈桑·怀特塞德（Hassan Whiteside）在5月6日进行比赛中被检测出禁用物质氧雄龙代谢物和氯米芬，球员放弃B瓶检测。
中国篮球协会星期六（11日）在官方微博发布声明表示，协会对此高度重视，正按照相关程序，积极配合中国反兴奋剂中心开展调查。待中国反兴奋剂中心做出最终认定后，中国篮协将严格依据反兴奋剂规则及协会相关规定，对违规方进行严肃处理。
中国篮协在声明中强调，对任何形式的兴奋剂违规行为持“零容忍”态度。协会将以此为戒，持续强化各参赛单位及从业人员的红线意识与反兴奋剂教育管理工作，坚决维护公平公正、健康积极的竞赛环境。
据悉，怀特塞德被查出兴奋剂违规的比赛是2025-2026赛季CBA季后赛上海队主场对阵山东队的首场较量，而他此后还参加了上海队与山东队8进4系列赛的第二场比赛，以及与北京队的所有半决赛系列赛。总决赛期间，怀特塞德缺席上海队与广厦队的系列赛，最终上海队以总比分4:1战胜广厦队，夺得本赛季CBA联赛总冠军。
《北京日报》指出，一位兴奋剂检测违规的球员仍然能出场比赛，这种情况绝对是不公正、不合理的，对北京队、山东队都是不公平的。上海队进入总决赛后，怀特塞德没有再登场。对于处罚结果为何迟迟公布，以及怀特塞德被查出使用兴奋剂后为何还能参赛，其所参加的五场比赛是否有效，中国反兴奋剂中心、CBA公司目前都没有表态。

英国前国务部长家中遇害 警列谋杀案调查
英国前保守党议员维德科姆被发现倒毙在英格兰西南部家中，身上有严重伤势。警方已按谋杀案展开调查。
局英国广播公司（BBC）和彭博社星期六（7月11日）报道，警方星期四（9日）早上接到救护车服务部门报警后，前往维德科姆（Ann Widdecombe）位于德文郡海托尔的住家调查，在屋内发现维德科姆已经死亡。
一名26岁男子当天在德文郡被捕，后来获释。警方说，他不是此案的调查对象。
维德科姆在遇害前一天曾接受媒体采访。
警方也说，现阶段没有迹象显示案件与恐怖主义或政治动机有关。内政部长马哈茂德敦促公众不要妄加猜测。
政界领袖对维德科姆遇害深感震惊。首相斯塔默称维德科姆为一位“杰出的政治家”。他也说，政界人士的安全至关重要，并表示近年来政府已经加强这方面的保护。
英国改革党党魁法拉奇赞扬维德科姆对脱欧运动的贡献，并表示“我们所有人都会怀念她”。
前议员布兰德雷斯以风趣、活泼好斗、令人敬畏形容维德科姆。
维德科姆曾是保守党党员，在1987年至2010年间当选国会下议院议员，曾任内政部负责监狱事务的国务部长，退党后加入英国脱欧党并当选欧洲议会议员。英国脱欧后，脱欧党更名为改革党，维德科姆担任移民与司法事务发言人。

台致癌油风波 中联油脂总经理获准2000万新台币交保
台湾食用油致癌风波持续延烧，刚被检方声押禁见的中联油脂总经理余凌冲获准以2000万元（新台币，下同，80万5620新元）交保，检方随即提出抗告。
综合台湾三立新闻网、《中国时报》《联合报》等报道，台湾食用油龙头中联油脂公司生产的大豆沙拉油检出致癌物“苯并芘”超标四倍，冲击全台食安。
台湾食药署星期二（7月7日）向中联油脂开罚1亿6520万元后，台中地检署星期四（9日）搜索中联等四家油品公司，认定中联总经理余凌冲有串灭证之虞，星期五（10日）依违反《食品安全卫生管理法》，将余凌冲声押禁见。
台中地方法院合议庭在评估涉案情节与相关证据保全状况后，于星期五晚上近11时决定准予余凌冲以2000万元重金保释，并要求他限制住居、限制出境、出海，向警方定期报到。
余凌冲的家属与律师团队在短短数小时内便筹足现金，在星期六（11日）凌晨1时许在台中地检署办妥交保手续。面对媒体询问，余凌冲不发一语、持随身物遮挡镜头，快步上车离去。
台中地检署认为，余凌冲有羁押的原因与必要，已当庭依法提起抗告。
台中地检署侦办上述案件，星期五依涉嫌违反《食品安全卫生管理法》将中联油脂、福寿实业和福懋油脂等四大油品公司董事长和公司高层共11人传唤到案。
复讯后，检方谕令中联董事长蔡清松以2000万元交保；福寿实业董事长洪尧昆、福懋油脂董事长吴星澄和泰山企业董事长刘伟龙各以1000万元交保，吴星澄因一时筹不出交保金，改以200万交保；其余被告以600万元至50万元交保。

国际能源署：美伊冲突再升温 重建石油库存进程或受扰
国际能源署（IEA）指出，美国与伊朗敌对行动再度升温，可能扰乱今年晚些时候全球重建石油库存的进程。这也再次凸显如果冲突持续升级，全球经济所面临的风险。
据彭博社报道，国际能源署星期五（7月10日）发布月度报告说，美国与伊朗于上个月达成临时和平协议后，全球石油供需关系一度开始恢复正常。IEA称，在阿联酋创纪录产量的带动下，波斯湾地区原油运输量已恢复至正常水平的约三分之二，全球原油库存也出现四个月来的首次增长。
随着运输逐步恢复，全球石油市场原本有望在第四季重新进入供应过剩状态，并可能在2027年出现更大规模的过剩。然而，本周爆发的新一轮冲突导致多艘油轮遇袭、霍尔木兹海峡通行放缓，使这一前景再次面临威胁。
IEA警告：“冲突升级给前景蒙上阴影，并可能颠覆市场明年转向供应过剩的预测。”
