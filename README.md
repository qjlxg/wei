中国限制关键矿产出口 对日本企业带来冲击
（东京路透电）中国限制关键矿产出口引发的供应短缺开始波及日本整体经济，让日本政府更迫切地寻求替代方案。
中国海关上月公布的数据显示，自去年11月至今年5月，中国未向日本出口铽（terbium）或氧化镝（dysprosium oxide）；自去年12月以来，氧化钇（yttrium oxide）的出口量也微乎其微，切断制造高性能磁铁所需的原料供应。
在5月和6月近200份提及稀土的日本企业文件中，超过三分之二称中国的出口管制已对业务造成负面影响，或未来可能带来冲击。
瑞穗研究技术首席经济师东深泽武史指出，在2010年中国实施贸易限制期间，日本经济曾损失约相当于国内生产总值（GDP）的0.9%，而稀土如今已成为供应链中不可或缺的一环，这次中国禁令造成的冲击可能更大。
乐天证券大宗商品分析师吉田悟认为，各行业受影响程度不一，主要是各公司的稀土储备量不同。“供应受到限制，但每个人都开始使用稀土，这只会让稀土更加稀缺。”
日本政府对此采取一些相应措施，包括去年10月与美国签署一项框架协议，旨在协调关键矿产和稀土资源，同时讨论联合开发深海矿藏。然而，商业规模的生产仍需数年时间。 
日本也已启动一些稀土回收项目，同时寄望于七国集团6月达成的一项加强储备协调计划。
不过，Yuri Group咨询公司首席执行官亨伯说，这些项目何时投入运营和最终产量如何仍不明朗。“我认为长达一年的出口限制会造成很大的问题，而我们已经历四到五个月。”
他也认为，日本政府显然希望保持低调，以免引发恐慌，也避免让中国过早取得优势。
去年11月高市早苗将“台湾有事”列为日本的“存亡危机事态”后，中国今年1月起加强管制对日本出口两用物项。

法国债务压力加剧 恐在大选前持续恶化
法国借钱的成本越来越高，投资者和经济师警告，法国本来已经背负3.5万亿欧元（约5万亿新元）公共债务，若明年总统选举前政治角力加剧、财政改革继续受阻，债务可能会越滚越大，形成所谓“滚雪球效应”。
路透社报道，所谓“滚雪球效应”，是指政府债券平均利率高于经济增长率时，如果政府无法长期维持基本预算盈余，债务占经济规模的比重就会继续上升。
经济合作与发展组织（OECD）秘书长科尔曼早些时候说：“如果什么都不做，到2050年，法国公共债务可能达到国内生产总值（GDP）的203%。因此，严格的预算纪律对于稳定公共债务至关重要。”
官方数据显示，今年第一季，法国公共债务突破3.5万亿欧元，占GDP的117.5%，接近冠病危机期间的水平。法国审计法院指出，法国是欧元区内唯一尚未把债务负担从疫情后高位降下来的国家。
法国去年公共债务利息支出达660亿欧元，正迅速成为国家最大支出项目之一，并可能超过教育和国防预算。审计法院警告，到2029年，这笔支出可能逼近1000亿欧元。
穆迪预计，英国、法国、德国、意大利和西班牙这欧洲五大借款国的债务比率都将进一步恶化，其中法国利息支出相对于公共债务的增幅最大。
自2024年提前议会选举造成无多数议会以来，已有数届少数派政府在推动预算案时倒台，法国国债持续承压。随着政府准备在今年秋季推动2027年预算案过关，投资者要求持有法国国债而非德国国债的风险溢价，已回到去年10月养老金改革暂停后出现的高位，并超过意大利与德国国债之间的利差。
摩根士丹利上周以财政担忧为由，建议客户降低法国债务敞口。经济师预计，在明年总统选举前，法国债券市场波动仍将维持在高位。

晚婚与女性收入提高推动 美国年轻人更愿意签婚前协议
在美国，婚前协议正逐渐成为年轻夫妻和准新人之间更常见的财务安排，也不再只是富人的选择。
彭博社报道，最新调查显示，截至今年5月，45岁以下已订婚或已婚的美国人中，53%已签署婚前协议，高于2022年调查中Z世代的41%和千禧一代的34%。家庭法律师说，提出签署婚前协议的人群已明显改变。
曼哈顿律师安德里亚·瓦卡说，过去通常是男性，尤其是较富有的一方提出签约；如今女性、收入较低者，以及不希望重蹈父母离婚覆辙的情侣，也会主动谈婚前协议。美国人口普查局数据显示，2025年美国女性首次结婚平均年龄为28.4岁，男性为30.8岁，均比2005年延后超过三年。由于结婚年龄推迟，许多人婚前已积累资产、债务和财务经验，也更希望提前厘清继承财产、学生贷款、房产，甚至冷冻卵子或胚胎等问题。
女性也是推动这股趋势的重要力量。
在线平台Hello Prenup数据显示，平台约52%的婚前协议由女性发起，使用平台的女性中近四分之一年收入达到15万美元（约19万新元）以上，客户多来自医疗、科技和金融行业。
这股趋势也催生律师、应用程序和播客等相关产业。通过Hello Prenup或First等平台，情侣花约600美元（近780新元）即可通过在线问卷生成协议；若加入律师审核，费用可升至2000美元或以上，但仍可能低于直接聘请律师处理复杂财务安排的成本。
过去，婚前协议常被视为婚姻不稳的信号，如今则越来越被看作务实的财务规划。27岁的软件工程师尼尔德主动提出签署婚前协议，并形容这是“一种爱的表现”。
2025年结婚的纽约财经播客从业者拉沃伊也说，她签署协议是为了避免离婚时承担对方债务。她认为，随着社会观念转变，婚前协议不再是难以启齿的禁忌话题，而是一种务实的自我保障。她签署协议并不是不信任丈夫，而是希望为自己多一层保障。
