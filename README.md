以色列外长萨尔说，以色列政府不打算在未来几天与黎巴嫩进行直接谈判，也没有向美国通报以方的导弹拦截器库存告急。
以色列《国土报》14日援引消息人士的话报道，以色列和黎巴嫩代表预计在未来几天内举行直接会谈。美国新闻网站Semafor的报道则称，以色列已通知华盛顿，指以色列的导弹拦截器库存已告急。
路透社报道说，当被问起上述两则报道时，萨尔表明“针对这两个问题，答案都是否定的”。
另一方面，黎巴嫩媒体和政府星期天（3月15日）称，黎巴嫩南部连夜遭遇空袭，造成至少四人死亡。
以军星期天发表声明称，军方继续打击真主党在黎巴嫩各地使用的基础设施，并轰炸了南部卡特拉尼村的“多个真主党发射点”，称真主党正准备从该地发射导弹。
声明还称，已摧毁了真主党拉德万部队位于贝鲁特的“指挥中心”。

香港保安局长首谈2019年立法会“空城计”争议
（香港综合讯）香港发生反修例抗争至今近七年，时任警务处副处长、现任保安局局长邓炳强，星期天首度谈及当年围绕立法会“空城计”的争议。他称，当时警方从立法会撤退，属“最不坏”的做法。
2019年7月1日香港反修例抗争期间，大批抗争者在游行后冲击并一度占领立法会大楼。当时有分析认为，警方在与抗争者对峙时突然撤离大楼，让抗争者长驱直入攻占大楼，是为了抢占道德高地，扭转舆论风向，令一般民众对抗争者的破坏行为产生反感。
时任警务处处长卢伟聪曾公开否认警方设下“空城计”的说法。
综合星岛头条、网媒“香港01”、香港电台报道，邓炳强星期天（3月15日）在香港电台节目《旧日的足迹》说，当时立法会大楼外有数以万计抗争者，许多人受到煽动。警方难以在不造成更大伤害的情况下进入大楼。
邓炳强解释，当时有议员责骂警察进入大楼，甚至带领抗争者在大楼内逐层巡逻；考虑到抗争者曾切断电线，大楼内一度漆黑一片等极高风险因素，抗争者和执勤警员都可能面临严重危险。
对于外界指警方就此使出“空城计”，邓炳强回应称，当时警方面对的所有选择都不好，“但都要选一个‘最冇咁唔好’（最不坏）的做法”。他进一步说，警方在衡量情况及困难后，选择不采取比较激烈的行动。

港官员谈中东局势：香港有风险也有机遇
中东局势持续紧张，香港财经事务及库务局局长许正宇称，这对香港有风险也有机遇，因为最近有海外资金因应目前情况正考虑更多元化的配置。
综合网媒“香港01”、有线新闻网、《信报》等报道，许正宇星期天（3月15日）接受商台节目《政好星期天》访问时说，中东局势对香港既有风险也有机遇。他说，相比中东其他地区，香港的安全、稳定与可预见性日益突显，吸引部分原本计划投放于中东的资金配置转向更分散。
他说：“你也看到有报纸指，最近不论律师、银行也收到查询，指一些家族办公室或财富管理业务对来香港有兴趣，所以我们也正筹谋很多与作为资产管理中心相关的工作。”
许正宇称，香港特区政府正就资产管理中心相关工作积极筹谋，并向立法会提交了税务条例修订，扩大对家族办公室及基金的税务豁免范围，包括贵金属交易与数码资产等，以向国际市场发出香港作为资产管理中心的强烈信号。
香港运输及物流局局长陈美宝星期六（3月14日）受访时说，持续的中东动荡已严重扰乱全球贸易与航空网络，也凸显香港需要扩大能源供应来源，降低对传统石油的依赖。
目前，战争的外溢影响已严重冲击航空业，不少航线受到干扰，许多乘客推迟长途出行，尤其是需要经由中东转机的航班。陈美宝指出，在此背景下，香港应把握机遇，加强直飞欧洲和美国的航班，并提升中转服务能力。
