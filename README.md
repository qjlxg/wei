法国地方选举要点
法国将于3月15日和22日举行两轮市镇选举，此次投票被视为明年总统选举前的重要试金石。近3.5万个市镇的市长来自基层，被选民视为最贴近日常生活的信任民选官员，地方结果也可能影响全国政治势头。
极右的国民联盟（RN）将此次选举视为为2027年总统选举积累势能的关键一步，计划提名约650份名单，力图巩固已有据点并进军更大城市。目前只有约十来位市长属于或获RN支持，RN仅执掌超过10万人的城市佩皮尼昂一城。能否在两轮间与其他政党结盟，打破长期排斥极右的传统，是观察重点；119名RN国会议员中有33人参选，显示其重视程度。
重点看点城市包括马赛、巴黎、土伦，以及此前左派在2020年胜出的南特、蒙彼利埃，以及绿党执政的里昂和斯特拉斯堡。极左的“法国不屈”亦欲在塞纳-圣但尼等地取得进展。保守的共和党（LR）在全国性选举中失利，但在市政层面长期具优势；马克龙盟友掌市数量有限，限制了反政府投票的空间。里维埃拉小镇芒通也受关注，前总统萨科齐之子路易是该地候选人。
选情难以预测：约3.5万场独立选举多以地方议题为主，很多未被民调覆盖。民调显示，治安、住房和地方税收是选民关心的主要议题。选举制度亦增加变数：若首轮获得绝对多数即获胜；若无，得票达到10%的名单进入次轮，得票5%以上者可并入他名单，常导致三四方对决，从而使第二轮高度不确定。
此外，二月一名极右活动家在与疑似极左分子冲突中被打死一事，可能影响“法国不屈”的民意支持。
投票时间：3月15日当地8:00开始（格林尼治时间07:00），各地闭票时间在格林尼治时间17:00至19:00不等。多数中大型城市将于3月22日举行第二轮投票，计票结果将在当晚陆续公布。

中国科技部部长：摆脱“先有成果后转化”的惯性思维
中国科技部部长阴和俊说，要摆脱过去“先有成果后转化”的惯性思维，加快推动形成科技创新和产业创新一体的新格局。
据中新社报道，十四届全国人大四次会议首场“部长通道”集中采访活动星期四（3月5日）在人民大会堂举行。
阴和俊说，“十五五”时期，将坚持科技创新瞄准产业需求、产业发展紧紧依靠科技创新，摆脱过去“先有成果后转化”的惯性思维，加快推动形成科技创新和产业创新一体谋划、一体部署、一体推动的新格局，从而打通从科技强到产业强、经济强、国家强的通道。
他提出，强化高质量科技供给。抓紧部署实施一批国家重大科技项目，特别是加强集成电路、人工智能、生物制造、量子科技、脑机接口、核聚变能等领域的科技攻关，为产业发展提供更强有力的科技支撑。
阴和俊说，要强化企业科技创新主体地位。企业是创新链和产业链的结合点。要加快培育壮大科技领军企业，把更多的创新资源向企业集聚，支持企业主导产学研融通创新，支持企业牵头组建创新联合体，承担更多攻关任务。进一步提高研发费用加计扣除比例，调动企业加大科技投入。
阴和俊表示，要进一步促进科技成果高效转化应用。关键是要深化科技成果转化改革，统筹建设概念验证、中试验证平台，加大应用场景培育和开放力度，让新技术、新产品有更多应用场景。同时，大力发展科技金融，引导资本“投早、投小、投长期、投硬科技”，为科技创新和产业创新融合发展提供更多金融产品。
他还说，支持地方先行先试。引导各地立足资源禀赋，锻造创新长板和特色产业，鼓励跨区域合作、城市群协同，支持推动各地因地制宜发展新质生产力。
中国全国人大会议星期四（3月5日）上午在北京开幕。加紧培育新动能被列为今年中国政府工作的第二项重点任务，中国总理李强介绍，将打造生物医药、低空经济等新兴支柱产业，并培育发展具身智能、脑机接口等未来产业。

中国推广中小学春秋假  旅行预订激增
中国政府提出支持有条件的地方推广中小学春秋假，成都、南京等地出发的旅行预订激增。
据21世纪经济报道消息，随着春季学期开学，中国多地陆续公布2026年春假安排，成都、南京、苏州、常州、扬州、乐山、眉山、绍兴等地已官宣春假时间，明确春假时间为4月1日至3日，与清明节相连。春假与清明连休，凑出的六天长假也刺激了更多出行。
去哪儿旅行数据显示，截至3月4日，预订4月1日至6日从成都出发机票量同比去年激增1.5倍，环比上周增长28%。从目的地增长情况来看，增长最快的城市中，北海和上饶居于榜首，增长数十倍；丽江、揭阳、南昌、武汉、无锡增长均超五倍；封关后的海南三亚和海口也将迎来更多成都亲子游客，同比去年增长五倍和三倍。
此外，南京出发，2026年春假期间机票同比增85%，热门目的地前十位包括广州、三亚、揭阳、昆明、海口、贵阳、西安、厦门、南宁、深圳。
十四届中国全国人大四次会议3月5日上午开幕，政府工作报告提到，支持有条件的地方推广中小学春秋假，落实职工带薪错峰休假制度。

日本车企本田将在日销售中国产电动汽车
激烈竞争导致在华销量下滑之际，本田计划成为首个将其在中国生产的电动汽车进口回日本销售的日本车企。
日本共同社星期三（3月4i日）获悉，本田计划将其在中国生产的电动汽车进口至日本国内，最早今年春季开始销售。由于在中国市场与本土车企竞争加剧、销量持续下滑，本田希望通过扩大销路来提高中国工厂的开工率。
据报道，进口回日本的车型将以自2022年起在中国销售的“e:N”系列为基础。本田目前在日本国内销售的电动汽车仅有“N-ONE e:”和“N-VAN e:”两款微型车，公司计划通过扩充额外的产品线来应对新需求。
本田去年在中国卖出64万6971辆汽车，同比下降24.2%，已是连续五年减少。工厂开工率也被认为在下降，重振业务已成当务之急。去年12月下旬，本田在华工厂一度因半导体短缺而停产，并在今年1月19日起陆续恢复生产。
