根据斯坦福经济学家 Jacob Light 的研究，2025-2026 学年美国计算机科学专业入学人数近二十年来首次下降，但这种下滑是否是一种长期趋势还有待观察。数据显示，过去十年美国计算机和信息科学专业的学士学位授予数量增长了一倍多，从 2014 年的约 5.6 万增至 2024 年的约 12.2 万。2025 年秋季入学数据显示，四年制大学计算机与信息科学及支持服务专业的本科生入学人数同比下降 8.1%，从 2024 年的约 659,700人 降至 2025 年的 606,100 人，但仍然高于 2022 年的 574,333人，表明人数下降是在经历了多年快速增长之后出现的。Light 认为人数下滑的原因可能有多种：基于大模型的辅助编程工具的流行、IT 就业市场疲软等等。

香港交易所取消午间休市
（香港彭博电）香港交易所提出一项初步方案，建议取消午间休市并延长交易时间，此举引来批评声音。当地券商尤其质疑交易量的增长，是否足以抵消工作时间延长所带来的负担。
若这些建议最终获采纳，将是香港交易及结算所有限公司（港交所）十多年来首次延长交易时间，也将打乱香港股市长年形成的交易习惯；许多人依赖午休时间完成上午交易的收尾工作，并洽谈新业务。延长交易时间也可能对大型券商有利，因为他们拥有更多员工，能够灵活安排轮班。
相关讨论目前尚处于早期阶段，不过正在受考虑的方案，还包括将开市交易时间提前30分钟（即上午9时），以及增设晚间交易时段，以便与美国股市的早盘交易时段重叠。此举也将使香港市场与全球市场的运作更加接轨。
香港证券及期货专业总会永远名誉会长王国安说：“为了这项改变所需投入的人力与资源实在太多，根本无法合理化最终结果。”
与纽约、伦敦等西方金融中心全天连续交易的模式不同，亚洲各地（包括东京、新加坡、上海和深圳）的主要市场普遍保留了午休时段。在香港，交易时段的安排历来都是敏感议题。2011年，港交所曾计划将午休时间从两小时缩短至一小时，约1000名股票经纪人和餐饮业从业者在王国安组织之下，走上街头抗议，但港交所最终成功落实了这项调整。
许多全球性银行和大型券商通常会采取错峰午休的安排，以确保有员工值守，从而兼顾期货交易（期货交易不设午休）以及其他区域市场的交易需求。
从事香港及中国大陆股票业务的小型券商面临较大冲击。在中环、金钟和上环等金融机构密集的区域，初级职员通常利用午间休市时段核对上午的交易并处理行政事务，随后在附近的餐馆匆匆吃顿午饭，为下午的交易时段做准备。
资深交易员和券商老板也会利用午休时间与客户共进午餐，这也是维系客户关系、交换市场看法及闲谈叙旧的良机。在新的交易时段安排下，此类活动恐怕将难以继续。
港交所此举反映了全球各地交易所为争取更多全天候交易量而采取的普遍行动。韩国交易所已提议将每日交易时间延长近一倍，达到12小时。印尼证券交易所将考虑在现有交易时段的前后，各延长一小时；台湾方面则计划在年底前将开市时间提前至上午9点，也在酝酿将全天交易时长延长最多90分钟。

港大研究：两航空枢纽加强旅客监测 提早约三天侦测变异病毒
（香港综合讯）香港大学医学院团队研究发现，策略性监测两个国际航空枢纽的旅客病毒基因，可将新兴病毒变异株的侦测时间提前约2.6至3.3天，提升全球防疫预警能力。
港大星期一（7月20日）公布研究成果，团队利用一个整合了950万条病毒基因组、流行病学数据及全球航空旅运模式的精密模型，重建冠病变种病毒奥密克戎（Omicron）的传播途径，并模拟各类监测策略如何应对未来大流行威胁。 
结果显示，只须在两个高度连通的航空枢纽，如香港和阿拉伯联合酋长国，对过境旅客加强病毒检测，可推前新兴病毒变异株的全球侦测时间。
团队指出，长期维持疫情大流行级别的监测水平，须投入庞大人力、资金和物流资源，对许多地区构成挑战。研究团队提出的策略易于实施，适合长远推行，即使诊断或基因排序资源减少逾50%，仍优于现行方法，具成本效益和应用潜力。
研究团队也进一步评估不同疫情情境，证实无论变异株传播力或群体免疫水平如何，这套策略都能维持稳定成效。
相关研究结果已于今年3月在科学期刊《自然—通讯》发表。
港大医学院公共卫生学院教授潘烈文说，提早数天发现新变异株，对实验室工作和防疫准备流程可带来关键作用。在变种病毒大规模传播前，能及早审视现有的诊断测试是否更新、开发评估免疫逃脱的工具、开展疫苗原型设计，以及为防疫部署争取先机。
他说，加强各地本土监测仍是全球卫生安全的基石，这套策略将发挥国际旅运网络的汇聚效应，无须从其他地区重新调配资源，就可建立一套让全球受惠的早期预警系统，加快病毒侦测速度。
潘烈文在电台节目《千禧年代》解释，研究最初针对全球最繁忙的10个机场，其后选择人流较多的香港和阿联酋，两地也较接近第三世界国家，有助于找到变异病毒。

中国商务部星期三（7月22日）以答记者问形式回应，法国近日出台的“反超快时尚”法对中资跨境电商平台采取了歧视性限制，以立法方式严重扭曲公平竞争，中方对此表示严正关切。
中国商务部新闻发言人表示，中方认为，该法以制定所谓“环保”和“可持续”标准为名，行排他性措施为实，涉嫌违反世贸组织非歧视原则，已构成了对华贸易壁垒，严重背离法国对外标榜的公平竞争和自由贸易准则，不仅将损害中方企业的正当合法权益，更将伤及法国消费者的切身利益。
发言人表示，敦促法方恪守世贸组织规则，立即纠正上述歧视性做法，为中资企业提供公平、透明、非歧视的营商环境。“中方将密切关注相关立法实施细则制定与落地执行情况，一旦中国企业合法权益受到侵害，中方将采取必要措施予以回击”。
这是一天内，中国商务部第二次对欧洲对中国电商平台的相关监管发出声明。星期三下午，中国商务部也对欧盟委员会重罚阿里巴巴旗下电商平台全球速卖通表示抗议。
中国商务部新闻发言人表示，中方坚决反对欧方以平台监管为由设置数字壁垒，并采取歧视性措施，限制打压中国电商企业在欧正常经营。
“中方敦促欧方停止利用法律条文的模糊性滥用自由裁量权，公平、公正对待中国企业。中方将坚定支持中国企业运用法律武器维权，并将采取有力措施坚定维护企业利益。”

（华盛顿综合讯）美国人工智能（AI）公司OpenAI在进行网络安全测试时，旗下先进AI模型自主突破隔离限制，接入互联网，并入侵供开发者存放和分享AI模型及数据集的Hugging Face平台。
OpenAI星期二（7月21日）说，参与测试的包括先进AI模型GPT-5.6 Sol，以及另一款尚未公开、能力更强的模型。为了测试这些模型发现和利用网络漏洞的能力，公司暂时放宽了安全限制，并把测试放在一个不能直接上网的封闭系统内。
不过，AI利用测试系统中一款第三方软件的漏洞，绕过限制并接入互联网，随后侵入Hugging Face，并试图窃取其数据库里的秘密信息，以完成测试任务。
Hugging Face说，这次入侵从寻找漏洞到闯入系统，全程由AI自主完成。联合创始人德朗格透露，公司当时因攻击手法复杂，怀疑幕后黑手是一家前沿AI实验室。OpenAI承认涉事模型来自公司后，德朗格感叹：“想到这一切都是自主完成的，实在令人震惊！”

研究：男青少年成瘾性使用社媒 或加重多动症症状
美国一项长期研究发现，成瘾性使用社交媒体程度上升的男青少年，一年后更可能出现注意力缺陷多动障碍（ADHD，简称多动症）症状加重。
研究人员提醒，家长不应只关注孩子使用屏幕的总时长，也应留意他们是否出现无法自控等问题使用模式。
《华盛顿邮报》报道，根据发表于医学期刊《美国医学会杂志·网络开放》的研究，加利福尼亚大学旧金山分校团队追踪超过1万1000名美国青少年五年。参与者从平均12岁起，每年接受一次调查，直至平均16岁。
研究发现，青少年的成瘾性社交媒体使用程度上升后，下一年的ADHD症状往往也随之增加。这种关联在14岁和15岁的男孩中尤其明显，但在女孩中没有发现明确关联。
研究发现，一般是青少年越来越沉迷社交媒体之后，ADHD症状才会变得更明显，而不是先出现更严重的ADHD症状，再进一步沉迷于社交媒体。
不过，研究人员也提醒，这只能说明两者之间有关系，并不能证明“沉迷社交媒体”一定会导致ADHD症状加重。
所谓成瘾性使用社交媒体，是指青少年明知社交媒体已影响睡眠、学业或人际关系，却仍控制不了使用时间，想减少也做不到；一旦不能使用，还可能感到焦躁或不安。
参与者平均12岁时，约23％经常惦记着社交媒体，18％会靠刷社交媒体暂时忘掉烦恼，15％虽然想少用，却控制不住。
永田指出，社媒平台不断提供即时满足和新鲜刺激，也鼓励用户频繁切换任务，可能妨碍青少年维持注意力，并影响工作记忆和执行控制。部分研究显示，青春期男孩可能对即时奖赏和追求刺激更为敏感。
他建议父母观察孩子能否自行停止使用社交媒体，以及社交媒体是否影响学业、睡眠和人际关系。家庭也可设立无屏幕时段，不在孩子的卧室放置屏幕设备，并规定全家用餐时不使用手机。
父母本身也应以身作则。永田说，过去研究显示，父母使用屏幕的习惯是预测孩子屏幕使用情况的重要因素，因此家长制定规则后也应共同遵守。
