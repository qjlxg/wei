萧美琴批大陆民族团结新法：恐成打压海外异议人士工具
针对中国大陆新生效的《民族团结进步促进法》，台湾副总统萧美琴说，这项法律可能成为北京将打压异议人士和批评者的手段延伸至境外的新工具，令人担忧。
综合台湾中央广播电台和《自由时报》等报道，萧美琴星期二（7月7日）出席CITW 2026开幕式时，以六个面向阐述她对中国大陆全球影响力扩张的观察。
她提到，第一，中国近年来持续扩大在亚洲、拉丁美洲、中亚和中东等“全球南方”地区的影响力，通过安全合作、伙伴关系和经济联系深化布局，对民主制度和公民社会空间构成挑战。
第二是资讯环境。萧美琴称，北京运用数码平台和人工智能技术塑造境外舆论，并针对台湾进行资讯操作，相关影响已从区域问题演变为全球性挑战。
第三是科技治理。她指出，中国大陆大规模发展人工智能驱动的监控技术，包括人脸识别、大规模资料搜集和行为预测等，并逐步向海外输出相关技术。
第四是经济影响力。萧美琴认为，北京正将关键矿产、供应链和基础设施融资等经济资源转化为政治胁迫工具，“这对于北京具备重要的战略意义”。
第五是中国大陆安全体系的跨境延伸。萧美琴称，北京持续向海外侨民社群、异议人士和学术机构施压，并通过非传统方式扩大境外影响力。她提到，新实施的《民族团结进步促进法》可能成为北京“既有工具箱中的另一项工具”，将打压异议人士的逻辑延伸至境外，适用于海外批评者与相关社群。
第六是台湾安全。她说，台湾持续面临来自中国大陆的军事压力，包括军机、军舰活动以及以“海上执法”名义展开的行动。她强调，这绝非可以轻忽的问题，任何对台湾安全的破坏将影响印太地区稳定，对全球经济带来巨大冲击。
萧美琴最后指出，台湾需要与理念相近国家及世界各地公民社会，深化以共同价值为基础的协调与合作，“台湾希望成为一名建设性伙伴，在有需要的地方发挥倍增力量”。
