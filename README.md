香港警方搜查两家独立书店拘捕五人，保安局长邓炳强就事件回应，目前五人获准保释外出，警方会继续调查；并称书店老板不可能不知道书本的内容就售卖，卖书的人有责任确保书籍不违法。
据香港电台网站星期六（7月18日）报道，邓炳强在电台节目指出，法律清楚规定，通过一些物品包括刊物、煽动对特区政府或司法机构的不信任，或挑拨憎恨即属违法，以往的“羊村案”便是相关案例。
邓炳强还说，卖书的人有责任确保出售的书不违法，他举例称，好比卖食品的人要确保食物不会令人中毒、腹泻，虽然未必每样食物都要咬一口，但总有方法知道食物来源或者成分。
他认为，书店老板不可能不知道书本的内容就售卖，并指“没打开看过”并非是借口。
对于港人外游买书回港、入境时才被发现有问题，邓炳强表示，凡事要讲证据，有足够证据才会拘捕和检控。至于最严重后果是否会没收书籍，他则称每宗案件情况不同。
邓炳强还强调，国家安全工作只有进行时、没有完成时，因为国安风险仍存在，包括外部势力、本土恐怖主义、“软对抗”和潜逃人士，需不停检视法律制度和执行机制，才能做到最好。
港警国家安全处星期三（15日）对“留下书舍”及“田园书屋”两家独立书店展开搜查行动，拘捕五人，涉嫌违反《维护国家安全条例》的“出于煽动意图作出具煽动意图的作为”罪。
被拘捕的五人被带到警署扣留调查后，星期五（17日）已获准保释候查。

台湾8月起停止对柬埔寨公民免签
台湾官方公告，将从8月起停止对柬埔寨公民实行的免签措施。
台湾外交部星期五（7月17日）在官网发布的新闻稿中指出，台湾自2018年8月起开放柬埔寨国民适用“东南亚国家人民赴台先行上网查核”（TAC）有条件式免签，以及“观宏专案”等签证便利措施入境台湾，盼借由增加两地民间往来，促进柬埔寨政府与人民对台湾了解。
然而，柬国政府长期并未以平等互惠方式回应台湾善意，更屡次公开配合中国大陆发表贬抑台湾主权甚至隐含支持大陆武统的声明，损害台湾尊严及人民情感。另外，考量近期诸多跨国重大诈骗案件也与柬埔寨有关，相关负面发展引起台湾各界及民众高度重视与严重关切。外交部因此决定自8月1日起停止对柬埔寨国民适用的上述签证便利措施。
外交部强调，未来拟赴台的柬国民众均须向台驻外馆处申办适当签证。至于何时恢复相关签证便利措施，将视柬国政府对台政策再予研议调整。　
同日，台外交部也公告，自今年8月1日起，延长试办泰国、文莱及菲律宾国民赴台免签证措施一年至明年7月31日；另对印度、印尼、越南、缅甸、老挝等五国国民续行“观宏专案”一年至明年12月31日，并依现行作法持续办理TAC有条件式免签证入境措施。

乌无人机袭俄物流中心七死24伤 另300多架飞向莫斯科
俄罗斯西部一处物流中心星期六（7月18日）遭乌克兰无人机袭击，造成七人死亡，24人受伤。与此同时，莫斯科也再次遭到无人机袭击。 
法新社报道，俄罗斯坦波夫州（Tambov）州长佩尔维绍夫说：“Wildberries物流中心遭敌方无人机袭击，造成七名夜班员工死亡。”
他说，根据初步信息，这起发生在科托夫斯克市（Kotovsk）的袭击还造成24人受伤。物流中心仓库明火已被扑灭，消防救援工作仍在进行中。
同日，莫斯科市长索比亚宁说，夜间有370多架无人机飞向莫斯科地区。
索比亚宁在俄罗斯政府支持的通讯与社交平台MAX发文说：“大多数无人机在逼近前已被防空部队拦截。64架敌方无人机在接近莫斯科时被摧毁。”
近几个月来，由于华盛顿将注意力转向伊朗，由美国主导、旨在结束俄乌战争的外交努力陷入停滞。
