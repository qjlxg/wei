美国第四季度劳动生产率增幅超过市场预期，进一步表明企业正努力提升效率以控制成本。美国劳工统计局周四公布的数据显示，第四季度生产率——非农雇员每小时产出——环比折年率增长2.8%，此前的第三季度增幅被上修至5.2%。经济学家的调查中值预测为增长1.9%。近期效率提升的趋势，有助于抑制工资压力，印证了美联储官员的观点，即劳动力市场已不再是通胀来源。对许多企业而言，劳动力成本是最大支出，因此公司会通过引入新技术和设备来提高员工效率。对人工智能（AI）等技术的投入，使部分企业能够以更精简的人手维持运营，这也在一定程度上导致去年招聘偏弱。单位劳动力成本——即企业为生产一单位产出而向员工支付的成本——在第四季度同样上升2.8%。尽管去年年底经济增长放缓，但这在很大程度上反映了美国史上持续时间最长的政府停摆所带来的影响：联邦政府支出出现自1972年以来最大降幅。不过，企业投资仍以稳健速度增长。劳工统计局的报告还显示，2025年全年生产率增长2.2%，劳动力成本上升1.9%。

欧盟周三公布一揽子工业与基础设施保护政策，目标是重振停滞的制造业并降低对外依赖。多项提案均指向对在欧中资企业收紧监管，应对价格低、质量提升的中国商品和投资带来的安全与竞争压力。
核心为“工业加速法案”，拟将制造业在欧盟经济中的比重从2024年的14.3%提升至2035年的20%。欧盟官员称，自2019年以来产量约下降近20%，大量转向中国。欧盟工业主管斯特凡·塞久内表示，此举标志着从自由市场倾向向产业保护政策的教条性转变。
委员会估计若不采取紧急措施，未来十年欧洲汽车业或流失60万就业岗位。欧盟约一半的电池与逆变器、94%光伏组件由中国生产，依赖程度被视为不可持续。
法案要求：对欲在欧盟电池、电动汽车、太阳能或关键原材料供应链投资超1亿欧元并寻求政府支持或合同的第三国企业，须转让技术与知识产权；对不允许欧企进入其政府采购市场的国家企业，将被排除在钢铁、水泥、汽车、太阳能、风电及核能等招标之外；并在重工业、国防和清洁高科领域优先推广“欧制”产品。
同时发布的港口与海事产业策略将强化对外资、尤其是对港口与双用途设施的审查，并考虑对渡轮、海上风电安装船等行业的不公平竞争采取专项贸易工具，推动出口信贷与公共采购优先欧制船舶和技术。
草案将提交欧洲议会和成员国理事会审议，预计经年余谈判才可能成法。最初草案曾七次延后并被大幅削弱，人工智能、芯片、量子与机器人等高科领域已被剔除出“优先欧制”清单，焦点转向钢铁、水泥、铝、化工、石油精炼与塑料等传统重工业及电动汽车、光伏、风电、电池、热泵、电解槽与核裂变等净零技术。
对于电动汽车，若要享受公共采购或国家支持，本地化率需达70%（不含电池价值）。法案要求在六个月内实现包括电池电芯在内的三项关键电池部件在欧生产，三年内增至五项。此外，若某国在特定产品市场占比超40%，其企业要入围政府合同须满足六项条件中的四项，包括技术转移、项目员工逾半在欧、外方股比不超49%、至少一名欧盟合作方、在欧研发投入达全球营收1%、以及至少30%零部件在欧采购。
观察者称这些条款可能冲击中资车企和电池厂在欧投资，有评论将之比作“反向邓小平”式产业政策。欧盟官员回应称法案并非“反中”，合规的中国龙头企业仍可投资，政策与其他地区多年的市场准入条件相仿。

习近平强调经济大省支撑全国大盘 江苏要练好内功做强自身
中共总书记习近平星期四（3月5日）指出，经济大省发展底盘稳，抵御外部冲击能力强，才能支撑全国经济大盘稳定。他要求江苏在增强经济韧性上持续用力，练好内功，做强自身。
作为中国第二经济大省，江苏省去年地区生产总值（GDP）超过14万亿元（人民币，2.6万亿新元），占全国GDP的十分之一，与排在首位的广东省差距进一步缩小。
中国经济增长放缓之际，习近平星期四下午参加他所在的全国人大会议江苏代表团审议时，对这一经济大省作出上述指示。
根据央视新闻联播报道，习近平指出，江苏要全面融入全国统一大市场，助力畅通国内大循环；同时扩大高水平对外开放，广泛开拓全球市场，更好联通国际循环，并以底线思维防范各种风险。
习近平说，完成“十五五”（2026至2030年）规划目标，需要应对更加复杂的环境，解决更多深层次矛盾。“江苏等经济大省处在改革开放前沿，要在研究新情况、解决新问题上下功夫、出经验。”
他还提到，发展新质生产力，对于推动高质量发展，增强经济竞争力至关重要。江苏这方面基础较好，要努力走在前列，力争在加强原始创新和关键核心技术攻关、抢占科技制高点上实现新突破。
中国总理李强星期四上午在全国人大会议开幕会上，公布了“十五五”的主要目标和任务。当天发布的2026年政府工作报告中，也多次强调“原始创新”和“关键核心技术攻关” 。 
习近平对江苏代表团说，要在优化提升传统产业，培育壮大新兴产业，超前布局未来产业上开创新局面；在进一步深化改革、破除制约新质生产力发展的体制机制障碍上取得新成果。
习近平还说，江苏要准确把握新形势下人民群众对美好生活新期待和民生工作新特点，探索推进共同富裕的有效途径。
他最后强调，要认真组织开展树立和践行正确政绩观学习教育，以全面从严治党的实际成效取信于民，凝聚推进事业发展的磅礴力量。

法国允许美军支援飞机使用基地 但不得参与对伊朗行动
法国军方宣布，美军支援类军机已获准进入法国南部伊斯特尔空军基地，但前提是相关军机不得参与美国在伊朗境内的军事行动。
法国商业调频电视台星期四（3月5日）引述法国武装部队总参谋部的声明说，获准使用伊斯特尔（Istres）空军基地的是支援类军机（非作战飞机）。此次授权不涉及法国在中东地区的军事基地。
新华社引述法国军方说，上述安排是在北大西洋公约组织框架内的例行程序。鉴于当前局势，法方已明确要求，相关军机不得以任何形式参与美国对伊朗的军事行动，仅能严格用于支持法国在该地区伙伴的防务需求。军方声明说:“法国已获得这方面的充分保证。”
法新社报道，此前，法国总统马克龙星期二（3日）指出，美国与以色列在伊朗的军事行动“违反了国际法”。

世卫组织：核实伊朗境内13起医疗机构遇袭事件
世界卫生组织总干事谭德塞说，世卫组织已核实伊朗境内13起针对医疗基础设施的袭击事件，这些袭击事件发生在美国与以色列失事军事行动期间。世卫组织正在核实有关四名医务人员遇难、25人受伤的报道。
谭德塞星期四（3月5日）在新闻发布会上说：“世卫组织已核实伊朗境内13起针对医疗机构的袭击事件，黎巴嫩境内一起。”但他并未指明袭击者或提供细节。
路透社报道，世卫组织东地中海区域主任哈南·巴尔基（Hanan Balkhy）在同一新闻发布会上补充说，伊朗境内四辆救护车也受到影响。她引述伊朗当局的消息称，附近医院和其他医疗机构因袭击而遭受轻微损坏。世卫组织此前说，袭击导致首都德黑兰的一家医院人员疏散。
巴尔基本周早些时候致函谭德塞，声称有10处设施遭到军事袭击。
巴尔基也说，由于该地区的交通管制，世卫组织位于迪拜的物流中心暂时停止运作，该中心向数十个国家提供医疗物资。
伊朗官方的烈士和退伍军人事务基金会星期四说，美国和以色列对伊朗的空袭已造成1230人死亡。
法新社目前无法核实这一数字。
据伊朗官方通讯社伊通社报道，该基金会称：“截至3月5日，在罪恶的美国和篡夺政权的以色列对伊斯兰家园发动的侵略性军事袭击中，殉难人数已达1230人。”
