马云与阿里和蚂蚁核心管理层罕见一起亮相谈AI
中国科技巨头阿里巴巴创始人马云，与阿里巴巴和蚂蚁集团核心管理层，罕见齐聚由他创办的杭州云谷学校，探讨人工智能（AI）带来的挑战和机会。
据“云谷教育”微信公众号星期二（3月3日）消息，阿里和蚂蚁最核心的管理团队当天齐聚云谷学校，探讨AI带来的挑战和机会。出席者除了马云外，还包括阿里巴巴集团主席蔡崇信、CEO吴泳铭、风险委员会主席邵晓锋、电商事业群CEO蒋凡，蚂蚁集团的董事长井贤栋以及CEO韩歆毅。
他们一行人与校长和老师们交流了一个多小时。根据“云谷教育”，马云说，AI时代已经快速到来，对社会的冲击超出想象，谁都没有做好足够的准备。但是对10几岁的孩子来说，他们最有改变的希望和机会。
他称，此次到云谷学校，目的就是把阿里巴巴最近对AI越来越清晰的洞察和老师们分享。
马云说，AI带来了一个让教育回归教育本身的机会，并建议死记硬背的时间，刷题的时间可以释放出来，用来培养创造力和想象力；孩子们可以有更多时间来玩，来学习音乐、绘画、运动，从中学会分享、学会感受和体验，学会倾听，以及学会理解。
马云也提到，看一所学校是不是属于AI时代的学校，并不是去看一所学校有多少AI服务器，有多强的AI技能。他解释，因为AI拥有的是晶片，而人类拥有的是心，AI时代对教育最大的改变，是老师们可以完完全全去做“灵魂工程师”，而不是做知识的灌输者。
马云还说，未来不是让孩子去和AI比拼计算和记忆，而是让孩子保持好奇，学会共情和担当，拥有体验感。因为，好奇心、想象力、创造力、判断力还有审美能力，才是AI时代教育需要赋予孩子的真正的能力。
“云谷教育”称，他们一行人在讨论中提到，AI的迭代以周计算，能力还在不断增长，此次的技术革命对生产效率和社会方方面面带来的变革是历史性的，以后可能一天不用工作八小时，但是很多工作种类都会消失。
此前，马云公益基金会官方微信公众号1月27日发文称，马云在线参加马云公益基金会“腊八之约”活动，期间分享了一些他对AI的思考。
马云说：“AI时代，未来我们的教育不是让孩子背得更多，记得更多，而是让他们想得更有趣，想得更有创意，想得更加独特。AI时代，我们的教育不是让一千个学生给出同样一个正确的答案，而是让一千个学生学会提出一万个不同的好问题。”

中国全国人大代表建议家暴情形不适用离婚冷静期
中国全国人大代表方燕认为，在家庭暴力情形下，离婚冷静期制度的合理性与适用性需要重新审视。
中国全国两会（人大、政协年会）星期三（3月4日）起在北京召开。据《中国新闻周刊》，全国人大代表方燕近日受访时建议，家庭暴力情形不应适用于离婚冷静期，以避免受害者遭受二次伤害。
方燕表示，《民法典》规定的30天离婚冷静期，适用于所有协议离婚情形。但是当家暴受害者与施暴者已达成离婚协议，也仍需等待冷静期届满，才能办理离婚登记时，期间可能面临施暴者反悔、威胁报复等二次伤害风险。
为绕开冷静期，家暴受害者往往只能选择诉讼离婚，而诉讼离婚存在立案、举证、审理等多个环节，耗时久、流程繁琐。因此方燕认为，需重新审视在家庭暴力情形下，离婚冷静期制度的合理性与适用性。
方燕建议，应从完善立法、证据认定、程序优化、配套保障等方面推进制度完善。这包括适时修改《民法典》，探索开展试点工作，明确可采信证据清单与核验标准，打通相关单位的数据共享通道等。
