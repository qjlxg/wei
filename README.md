中国将在集成电路等领域投资建设千亿至万亿级重大项目
中国国家发展和改革委员会主任郑栅洁表示，将在集成电路、卫星互联网、国产大飞机等领域，建设千亿（人民币，1000亿人民币约为185亿新元）至万亿级重大项目。
中国十四届全国人大四次会议星期五（3月6日）下午3时举行经济主题记者会。郑栅洁会上表示，针对高成长性行业，中国政府将突出长远布局，结合中国式现代化建设，在集成电路、卫星互联网、国产大飞机，还有全国一体化算力网等领域建设一批长链条、大体量的重大项目。
他表示，这些项目投资规模都在千亿级甚至万亿级，铸就一批打基础利长远的国之重器。

中共党政机关落实紧日子要求 会议培训经费压减10%
中国财政部长蓝佛安说，中央本级“三公”经费压减7%以上，会议、培训等经费压减10%，“用党政机关的紧日子，换老百姓的好日子”。
中国十四届全国人大四次会议星期五（3月6日）下午3时举行经济主题记者会，蓝佛安表示，全面落实党政机关过紧日子的要求，用党政机关的紧日子，换老百姓的好日子。
蓝佛安透露，今年，中央本级“三公”经费压减7%以上，会议、培训等经费压减10%；各地也从严从紧安排预算，其中20个省份节省资金超120亿元（人民币，约22.3亿新元），腾挪出更多的钱保障重点领域和关键环节。

特拉维夫上空传爆炸声 疑似伊朗导弹来袭
以色列中部的特拉维夫上空传来爆炸声，此前以色列军方探测到伊朗向以色列发射导弹。
据法新社记者报道，星期五（3月6日）以色列商业中心特拉维夫上空传来数声爆炸声之后，军方称：“不久前，以色列国防军发现有伊朗向以色列领土发射的导弹。防御系统正在运行，拦截这一威胁。”
新华社引述伊朗伊斯兰共和国通讯社的报道说，革命卫队发布公告称，已发起“真实承诺－4”第21波攻势，向以色列特拉维夫市中心发射导弹和无人机。
