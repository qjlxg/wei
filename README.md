韩国将派遣科考破冰船赴北极
韩国极地研究所（KOPRI）称，韩国将派遣本国唯一的破冰船执行北极科考任务，为未来运营北方海路进行前期筹备。
俄罗斯交通部长安德烈·尼基京3月份在接受俄罗斯卫星通讯社采访时称，韩国计划于今年下半年安排集装箱船从釜山出发，经北方海航道驶抵荷兰鹿特丹进行试航。俄罗斯国家原子能公司总裁阿列克谢·利哈乔夫也称，韩国对参与北方海路的合作表现出浓厚兴趣。
KOPRI周五在其官网发布的公告中称：“韩国唯一的破冰科考船‘阿劳恩号’（ARAON）将于7月11日从光阳港启程，执行为期83天的北极考察任务。”
