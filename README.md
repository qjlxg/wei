【非官方預測：MET WARN最新評估】
MET WARN 最新評估，天文台就現時位於南海的熱帶低氣壓發出三號強風信號的機會為「中等」。
【預測：離岸海域及高地風勢較大】
在西南氣流及副熱帶高壓脊的偏東氣流匯合下，預料風暴環流會較為廣闊；加上在與副熱帶高壓脊的其同影響下，華南沿岸氣壓梯度頗大，風暴東側的強風區會頗為接近珠江口一帶。
預料週五起，本港天氣會漸趨不穩。隨著風暴移至海南島一帶，週五及週六驟雨將逐漸增多，風勢增強。預料本港屆時主要受東至東南風影響，離岸海域及高地可能吹強風，高地或間中達烈風程度。
【變數：熱帶低氣壓與本港距離、環流大小】
電腦模式預報普遍支持系統移向海南島至北部灣一帶，並會與本港保持逾500公里的距離。然而，由於風暴中心初期整合的實際位置，以及副熱帶高壓脊的強度仍然存在不確定性。因此，風暴與本港的距離及環流大小存在變數，現階段仍未能斷言本港會否普遍吹強風。
現階段亦未能排除本港天氣在脫離熱帶低氣壓的主導後才普遍吹起強風，因此直接由一號信號改發強烈季候風信號的機會存在。
註：「中等」代表基於現有資料，MET WARN 評估天文台發出相關信號與否機會相若，可以用「五五波」形容。

美联储主席Kevin Warsh在欧洲央行论坛回答记者问题的核心内容总结如下：
记者不断试图让Kevin Warsh给出利率方向、通胀判断和政策承诺，但Warsh的回答非常一致：
第一，Fed不再轻易给forward guidance。 
第二，2%的通胀目标不会放松。 
第三，Fed会坚持独立性，不会被政治压力左右。 
第四，他希望重塑Fed的沟通、数据、通胀框架和资产负债表政策。 
第五，他对AI长期提高生产率持乐观态度，但不认为Fed现在就可以因此忽视通胀风险。
全部18个问题和回答汇总如下：
1. Fed是否还会继续给市场明确的forward guidance？
不会。他表示，在危机时期，央行可以给市场更多指引，帮助降低不确定性。但现在不是那种环境，Fed不应该继续用forward guidance来“喂养”市场，也不应该为了降低市场波动率而提前承诺利率路径。市场和实体经济应该根据真实数据自己做判断。
2. 7月28日至29日的FOMC会议会不会加息？
他拒绝直接回答。他说，利率决定应该留到FOMC闭门会议中讨论，而不是在公开场合提前透露。他强调，会议召开时委员们会充分辩论，现在不会给出任何提前指引。
3. 市场是不是误读了你之前记者会上的鹰派表态？
他没有直接说市场是否误读，而是继续强调Fed不会通过forward guidance来引导市场预期。他的意思是，市场不应该期待Fed在每次讲话中都释放交易信号。
4. Fed是否会容忍高于2%的通胀目标？
不会。他明确表示，如果有人认为这届Fed会接受高于2%的通胀目标，那会感到失望。Fed的职责是实现价格稳定，2%的通胀目标不是可以随意放松的软目标。
5. 最近美国通胀风险是否已经下降？
他承认，过去几周通胀预期和通胀风险有所下降。但他同时强调，价格水平仍然太高，Fed仍然必须专注于价格稳定。
6. 这次通胀冲击是不是temporary？
他没有接受“temporary”这个说法，还反问这是不是想把“transitory”这个词重新带回来。他明显是在避免重复此前“通胀是暂时的”那种沟通错误。换句话说，他不愿意给市场一种“通胀可以忽略”的印象。
7. 如果坚持2%的通胀目标，是否意味着会让Trump失望？
他强调Fed是独立央行，过去如此，现在也会继续如此。Fed不会因为政治压力改变自己的政策目标。
8. 最高法院关于Fed独立性的裁决，会不会影响Fed的日常运作？
他说自己看过相关裁决，但认为这不会改变Fed的日常运作。Fed之前是独立的，之后也会继续保持独立。
9. 你推动的Fed改革task forces进展如何？
他表示，下周可能会公布部分外部专家名单。改革方向包括五个方面：沟通、资产负债表、经济数据使用、生产率与就业、通胀框架。
10. 为什么Fed改革要邀请美国以外的专家参与？
他认为，外部视角有助于看清美国央行自身的问题。这些任务小组不会预设结论，外国专家的参与也可能让改革成果对其他央行有参考价值。
11. Fed资产负债表未来会怎么处理？
他认为Fed资产负债表规模应该缩小。他说，资产负债表政策在某种程度上已经接近财政政策，因此不能长期成为常规工具。不过，他也强调，任何资产负债表政策变化都会提前、充分地向市场沟通，不会突然制造冲击。
12. 利率工具和资产负债表工具，哪个更应该成为Fed的核心工具？
他更偏好利率政策。他认为利率工具更透明、更公平，而资产负债表政策更多通过资产价格和市场信号发挥作用，副作用更大。因此，Fed应该重新把利率政策放回核心位置。
13. Fed是否会改变经济数据体系？
会。他希望在未来9到12个月内，Fed能够使用新技术获得更实时的经济数据，减少对滞后、误测、低响应率政府调查数据的依赖。
14. 你怎么看美国经济增长和潜在增速？
他没有给出具体预测，因为他认为这又会变成forward guidance。但他的表述偏向于认为，美国过去几个季度的表现如果有参考意义，那么美国的潜在增长率可能已经上行。
15. AI对通胀是推高作用，还是压低作用？
他没有简单下结论。他说，AI首先带来的是资本开支大爆发，这可能在短期形成需求侧压力；但企业之所以投资AI，是因为预期未来供给能力会提升。如果AI带来的生产率提升兑现，长期可能会对通胀形成下行压力。
16. AI会不会造成大量失业？
他不是AI悲观派。他认为不能简单假设工作总量是固定的，AI长期可能创造更多就业和繁荣。但他也承认，转型过程中的时间路径很重要，Fed必须同时关注就业和价格稳定。
17. 美国是否会成为AI时代的主要赢家？
他认为美国很可能是AI的主要受益者之一。但他也强调，这不是零和游戏，并不是美国赢就意味着其他国家必须输。他希望AI带来的增长能够更加广泛。
18. AI对Fed的政策决策意味着什么？
他说，AI变化是指数级的，央行不能只依赖传统、滞后的经济数据。Fed必须更快识别趋势，这也是他强调升级实时数据体系的重要原因。

特朗普政府拟修改核辐射监管条例 加快核电发展
（华盛顿综合电）美国核能管理委员会星期三（7月1日）发布计划书，提议取消实施近50年的辐射防护原则，以更灵活、成本更低的标准取而代之。这是特朗普政府推动放松监管、加快核电发展的最新举措之一。
核能管委会建议删除一项称为“最低可合理达致水平”（As Low as Reasonably Achievable，简称ALARA）的条款，根据风险和运作情况，制定一个核辐射分级管理制度，给予业者更大灵活度，以现代化的方式评估员工和公众面对的核辐射量。
管委会指出，修改条例并非降低安全标准，而是让条规更加明确。
管委会提出的另一项提议则是修改反应堆许可证机制，包括简化建造新反应堆的审批流程。上月，它提出一系列规则修订，包括修改核电站安保标准。
在上述新规确定之前，核能管委会将在45天内收集公众意见。
核能支持者赞同放弃ALARA原则，认为这能让反应堆开发商摆脱那些既繁琐又增加成本的监管要求。批评者则认为，这么做将危及核能厂员工和公众的健康。
特朗普是在2025年签署行政令，要加快反应堆的审批流程并重组核能管理委员会，同时指示能源部与五角大楼合作建设核电站。
特朗普计划到2050年，将美国的核能容量提高至原来的四倍，以满足数据中心、电动汽车和加密货币不断增加的电力需求。

秘鲁法院裁定中资钱凯港应接受监管
（利马彭博电）秘鲁法院裁定，政府应对利马附近一座由中资运营的港口实施监管。美国近年极力遏制中国在拉美地区影响力的上升，上述裁决被视为有关努力取得的进展。
该裁定也推翻了今年1月的一项判决。此前裁决认为，位于太平洋沿岸的钱凯港可免于接受秘鲁基础设施监管机构Ositran的部分监管。
钱凯港是一座私营港口，由中国国企中远海运港口有限公司运营，其建设目标是缩短拉丁美洲与亚洲之间的航运时间。但随着美国重新加强对拉美地区的关注，这座港口逐渐成为华盛顿与北京地缘竞争的焦点。 
此前的裁决曾引发美国特朗普政府的担忧，美方警告称，这可能削弱秘鲁的主权。Ositran也批评相关裁决，认为这将使港口使用者失去监管保护。
美国国务院西半球事务局2月曾在社交平台X发文称：“希望这成为整个地区乃至全世界的警示：廉价的中国资金是以主权为代价的。”
美国驻秘鲁大使纳瓦罗在社交媒体上对最新裁定表示欢迎。上个月，他首次访问钱凯港，并向秘鲁海关捐赠两台扫描设备，以提升海关查验能力。
这座总投资13亿美元（16.8亿新元）的港口于2024年正式启用，中国国家主席习近平曾出席相关开港活动。中国目前是秘鲁最大的贸易伙伴，美国位居第二。 
法院最新裁决采纳了这样的主张：虽然钱凯港为私人所有，但它属于公共用途港口。Ositran在声明中表示，该港口应接受它依据秘鲁法律对运营方实施监管、监督、检查以及处罚等管理职能。
这一裁定仍可提出上诉。中远海运港口有限公司未回应彭博社的置评请求。
美国驻秘鲁大使纳瓦罗以及国务卿鲁比奥本周均祝贺了保守派政治人物藤森庆子赢得秘鲁总统选举，并表示特朗普政府期待与秘鲁在地区安全及贸易领域进一步深化合作。
