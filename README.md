巴黎讯 — 美国国家海洋与大气管理局（NOAA）称，今年下半年形成厄尔尼诺（El Niño）的概率为50%至60%，最早可能在7至9月出现并延续，或将推动全球气温再创新高。世界气象组织（WMO）将于周二发布最新评估。
厄尔尼诺与其相对的拉尼娜（La Niña）同属热带太平洋的ENSO振荡。19世纪秘鲁与厄瓜多尔渔民以“圣婴”命名厄尔尼诺。厄尔尼诺通过削弱东向信风，使东中太平洋海面增温，改变赤道太平洋降雨与全球风场，并将海表额外热量释放到大气中，通常可使全球平均气温短期上升约0.1–0.2℃。该现象每2至7年出现一次，常致东南亚、澳大利亚、南非南部和北巴西偏干，而非洲之角、美国南部、秘鲁和厄瓜多尔偏湿。
上次厄尔尼诺发生在2023–2024年，推动2024年成为有记录以来最热的一年。欧盟科普尼库斯气候服务主任曾表示，若今年出现厄尔尼诺，2026年可能再次破纪录；爱尔兰气象局气候学家指出，若在2026年下半年形成，其对2027年的影响会更显著，但即便无厄尔尼诺，持续的全球变暖也可能使2026年成为史上最热之一。
最新拉尼娜始于2024年12月，强度较弱并将在今年2–4月转为中性，其存在并未阻止2025年成为有记录以来第三热年。
NOAA在今年2月将旧的海洋尼诺指数（ONI）替换为相对海洋尼诺指数（RONI），以比较东中太平洋与热带其他海域的温差，NOAA称此法能更清晰、可靠地实时追踪厄尔尼诺与拉尼娜。
