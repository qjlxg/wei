报道援引《气象法》称，公众气象预报和灾害性天气警报必须由官方气象台站统一发布，其他组织或个人擅自向社会发布，可被警告、责令改正，并处最高5万元罚款。受访专家还建议平台限制此类内容。
相比之下，美国和欧洲多数国家通常区分“民间天气预测”与“政府官方预警”。
在欧美，民间可以根据公开数据自行预测，政府主要垄断的是“官方预警”的身份；在中国，普通人即使使用公开模型，只要把判断公开说成天气预报，也可能涉嫌违法。

曼谷酒吧大火逃生门遭杂物堵塞酿惨剧
泰国官员星期一（7月13日）说，曼谷乍都乍区酒吧发生大火时，酒吧的两个逃生门都被物品堵住，许多客人因此无法逃离火场。
一段发布在社交媒体平台并经路透社核实的现场视频显示，大约在12日午夜时分（新加坡时间13日凌晨1时），起初看到酒吧冒出滚滚浓烟，突然一团猛烈火焰从门口喷涌而出，尖叫声四起，许多人惊慌逃出酒吧。
这家酒吧餐厅位于曼谷北部繁华地段，交通便利，毗邻两家购物中心，步行可到电影院、大型公园以及深受外国游客欢迎的乍都乍周末市场。
泰国防灾减灾部门说，初步评估认为，起火原因是安装在天花板上方的空调发生电线短路。
当局表示，一些侥幸逃生者指火灾从酒吧舞台区域烧起并快速蔓延，许多人被迫逃到酒吧后方的厨房和厕所。
曼谷市长差猜说，这家酒吧有两个逃生门，其中一个靠近厨房，但出口被啤酒箱堵住；另一个逃生门也被一张桌子挡住，阻碍人员疏散。
他说：“刑事技术人员将对现场进行更仔细的勘查。”
目前搜救人员已发现27具遇难者遗体，另有63名伤者被送往医院，其中22人伤势严重。

中国将在下半年深化开展通航与低空经济业务
中国航空运输协会在会议上列出下半年八项任务，包括深化开展通航与低空经济业务。
据中国航协微信公众号消息，协会上星期五（7月10日）在北京召开2026年度年中工作会议，总结协会上半年工作成效，分析研判当前形势，部署安排下半年重点工作任务。
中国航协党委书记、理事长王昌顺主持会议并讲话。他说，协会接下来突出目标导向，确保任务落实，持续扩大第四届中航协（CATA）航空大会影响力，切实加强无人机操控员培训机构管理，抓好教育培训转型，不断加强研究咨询业务。
会议强调，航协下半年要聚焦世界一流协会建设和高质量发展核心任务，包括强化政策研究与行业治理；深化通航与低空经济业务开展；坚持标准引领，促进行业绿色发展；推进教育培训与人才建设；强化国内国际交流合作；推动重点项目落地，以及持续加强自身建设等。
中国科技部部长阴和俊上星期一（6日）在接受人民日报采访时明确表示，要“加快新能源、新材料、航空航天、低空经济等战略性新兴产业集群发展”。
另据上观新闻报道，以“驭低空新势，启经济新篇”为主题的2026国际低空经济博览会，将于7月22日至25日在上海国家会展中心举办。
展会现场汇集央国企国家队、外资合资企业、上市产业企业、头部初创企业四大核心市场主体，完整呈现行业多元共生、创新迭代的产业格局。
航空工业、中国航天、中国邮政等央企带来国家级低空技术方案；西门子、达索、爱普生、赢创等外资企业带来国际前沿航电、航空材料技术；亿航智能、广联航空等上市企业展示商业化落地成果，沃兰特、峰飞航空等科创企业多款eVTOL真机现场首发首秀。

中国裁判员结束世界杯执裁任务
中国籍裁判员结束了美加墨世界杯的执裁任务。
据新华社多方证实，国际足联星期天（7月12日）确定了世界杯剩余比赛裁判员名单，中国裁判员马宁、周飞不在其中。此前，中国裁判员傅明已在上星期三（8日）离开世界杯赛区。
至此，参与本届世界杯执裁工作的三名中国裁判已结束全部工作任务。
本届世界杯，马宁以主裁判身份入选，周飞以助理裁判身份入选，傅明以视频助理裁判（VAR）身份入选。
新华社报道称，中国裁判在本届世界杯上创造多项纪录。马宁成为时隔24年再度主哨世界杯的中国裁判，也是唯一一位连续两届入选世界杯裁判名单的中国裁判。傅明成为首位在世界杯上担任视频助理裁判的中国裁判，周飞成为首位在场上执法的中国助理裁判。
相比中国裁判在世界杯赛场上取得突破，中国男足则已多年无缘这项赛事。
中国男足2002年首次打进世界杯正赛，也是迄今为止唯一一次。

中国经济学家王连洲因病逝世 被誉为基金业之父
中国知名经济学家王连洲因病逝世，他曾参与起草中国信托法、证券法和基金法，被业内誉为“基金业之父”。
澎湃新闻、《上海证券报》等中国媒体报道，王连洲在星期天（7月12日）因病离世，享年87岁。他的家人在星期一（13日）发布讣告，将于星期四（16日）举行遗体告别仪式。
公开信息显示，王连洲1964年毕业于山东财经学院财政金融专业，进入中国人民银行总行印制局，从事货币印制计划管理以及中国历代货币收集研究工作，负责参与出版首部《中国历代货币》图录及说明。
1983年，王连洲参加六届全国人大一次会议快报组，同年10月调任全国人大财经委员会，历任办公室财金组副组长、组长，办公室副主任，经济法室副主任，研究室负责人、正局级巡视员。他曾担任中国信托法、证券法和基金法三部重要的行业金融法律法规的起草工作小组组长。
据报道，在中国资本市场建立初期，王连洲积极推动资本市场的立法建制，他在推动资本市场的健康发展过程中曾起到重要作用，也因此被业内誉为“基金业之父”
